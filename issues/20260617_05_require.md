# [orchestrator][phase] Define the phase state machine

## Summary
The multi-agent system needs a phase-based execution model.

## Problem
The system must control these phases:
- plan
- implement
- validate
- integrate
- finalize

## Scope
- orchestration phase logic
- phase transition rules

## Task
- define all phase states
- define allowed transitions
- define start and end conditions for each phase

## Expected Result
The orchestrator can move a run through clear phases.

## Acceptance Criteria
- [ ] all Phase 1 states are defined
- [ ] transitions are defined
- [ ] completion conditions are defined
- [ ] invalid transitions are defined
