## Design

### BUG-1: `create_schema.py` 遅延初期化

現状 (L34):
```python
logger = Logger(__name__, _get_schema_log_path())
```

変更後:
```python
_logger: Logger | None = None


def _get_logger() -> Logger:
    global _logger
    if _logger is None:
        _logger = Logger(__name__, _get_schema_log_path())
    return _logger
```

`logger.` を直接参照している全箇所を `_get_logger().` に置き換える。
`rg "logger\." scripts/db/create_schema.py` で全件確認してから修正する。
