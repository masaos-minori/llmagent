## Goal

## Design

**エンドポイント仕様:**

```
POST /events/{event_id}/ack?consumer_id=X   (consumer_id はオプション)
  → acked_at = NOW (冪等: 既に ack 済みなら上書きしない)
  → consumer_id が指定されていれば write_offset(offsets_dir, consumer_id, seq) を実行
  → 200 {"event_id": "...", "acked": true}
  → 404 if event not found

POST /events/{event_id}/nack
  → delivery_failure_count += 1, 200 {"event_id": "...", "failure_count": N}
  → 404 if event not found
```

**DB facade 追加 (db.py):**
```python
def ack_event(conn, event_id: str, now: str) -> bool:
    """Set acked_at. Idempotent. Returns True if found."""
    cur = conn.execute(
        "UPDATE events SET acked_at = ? WHERE event_id = ? AND acked_at IS NULL",
        (now, event_id),
    )
    conn.commit()
    return cur.rowcount > 0

def nack_event(conn, event_id: str) -> int:
    """Increment delivery_failure_count. Returns new count, or -1 if not found."""
    conn.execute(
        "UPDATE events SET delivery_failure_count = delivery_failure_count + 1 WHERE event_id = ?",
        (event_id,),
    )
    conn.commit()
    row = conn.execute(
        "SELECT delivery_failure_count FROM events WHERE event_id = ?", (event_id,)
    ).fetchone()
    return int(row["delivery_failure_count"]) if row else -1
```
