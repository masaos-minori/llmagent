## Assumptions

1. `_filter_env` のロジック:
   ```python
   if self._env_allowlist:          # non-empty allowlist → allowlist フィルタ
       return {k: v for k, v in req_env.items() if k in self._env_allowlist}
   if self._env_denylist:           # non-empty denylist → denylist フィルタ
       return {k: v for k, v in req_env.items() if not any(fnmatch.fnmatch(k, p) for p in self._env_denylist)}
   return req_env                   # 両方空 → pass-through
   ```
2. `_make_service(tmp_path, env_allowlist=[], env_denylist=[])` では
   `_env_allowlist=()`, `_env_denylist=()` となり、いずれも falsy → `return req_env`。
3. 既存テスト `test_no_filters_returns_env_unchanged` は引数なし (デフォルト None) の場合のみ。
   `allowlist=[], denylist=[]` を明示した場合の挙動は未テスト。
4. `allowlist=["NONEXISTENT"]` → allowlist が non-empty だが環境変数に一致するキーなし → `{}` を返す。
5. `denylist=["*"]` → すべてのキーにマッチ → `{}` を返す (grep パターン `*` は fnmatch でワイルドカード)。

## Design

### 追加テスト一覧

**Test 1: `test_both_empty_returns_env_unchanged`**
- `allowlist=[], denylist=[]` — 両方明示的に空
- `_env_allowlist=(), _env_denylist=()` → pass-through
- env に HOME と TERM が含まれていても unchanged

```python
def test_both_empty_returns_env_unchanged(self, tmp_path: Path) -> None:
    svc = _make_service(tmp_path, env_allowlist=[], env_denylist=[])
    env = {"HOME": "/root", "TERM": "xterm"}
    assert svc._filter_env(env) == env
```

**Test 2: `test_allowlist_with_no_matches_returns_empty_dict`**
- `allowlist=["NONEXISTENT"]` — 環境変数に存在しないキー
- allowlist が non-empty → フィルタリング → 結果は `{}`

```python
def test_allowlist_with_no_matches_returns_empty_dict(self, tmp_path: Path) -> None:
    svc = _make_service(tmp_path, env_allowlist=["NONEXISTENT_VAR"])
    env = {"HOME": "/root", "TERM": "xterm", "PATH": "/usr/bin"}
    assert svc._filter_env(env) == {}
```

**Test 3: `test_denylist_glob_matching_all_removes_all`**
- `denylist=["*"]` — すべてのキーにマッチするワイルドカード
- すべての環境変数が削除され `{}` が返る

```python
def test_denylist_glob_matching_all_removes_all(self, tmp_path: Path) -> None:
    svc = _make_service(tmp_path, env_denylist=["*"])
    env = {"HOME": "/root", "TERM": "xterm", "PATH": "/usr/bin"}
    assert svc._filter_env(env) == {}
```
