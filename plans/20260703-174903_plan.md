## Goal

Move `stat_tool_errors` and `stat_transport_errors` ownership from `ToolExecutor` to `ToolTransportInvoker` (the component that generates those events). Expose read-only properties returning defensive copies on both classes for backward compatibility.

**Sequencing dependency:** This plan requires require-25 (`ToolTransportInvoker`) to be implemented first.

## Scope

**In-Scope:**
- In `scripts/shared/tool_transport_invoker.py`: rename `stat_tool_errors` / `stat_transport_errors` to private `_stat_tool_errors` / `_stat_transport_errors`; expose them as read-only properties returning `dict(...)` copies
- In `scripts/shared/tool_executor.py`: replace mutable `stat_tool_errors` / `stat_transport_errors` fields with read-only properties delegating to `self._transport_invoker`
- Update `get_error_counters()` in `ToolExecutor` to use the property (no behavior change — it already reads these dicts)
- Add tests for stats ownership and defensive copy behavior

**Out-of-Scope:**
- Introducing a new shared `ToolExecutionStats` object
- Callback or recorder-based stats updates
- Moving `ctx.stats.stat_tool_errors` (integer counter in `AgentStats`) — this is a different stat in `scripts/agent/context.py`

## Design

### `ToolTransportInvoker` changes

```python
class ToolTransportInvoker:
    def __init__(self, ...) -> None:
        ...
        self._stat_tool_errors: dict[str, int] = {}
        self._stat_transport_errors: dict[str, int] = {}

    @property
    def stat_tool_errors(self) -> dict[str, int]:
        return dict(self._stat_tool_errors)

    @property
    def stat_transport_errors(self) -> dict[str, int]:
        return dict(self._stat_transport_errors)
```

Internal mutation (inside `_record_success` and `_record_transport_error`) uses `self._stat_tool_errors` and `self._stat_transport_errors` directly.

`get_error_counters()` reads `self._stat_tool_errors` and `self._stat_transport_errors` (no copy needed — it constructs a new dict anyway).

### `ToolExecutor` changes

Replace:
```python
self.stat_tool_errors: dict[str, int] = {}
self.stat_transport_errors: dict[str, int] = {}
```

With:
```python
@property
def stat_tool_errors(self) -> dict[str, int]:
    return self._transport_invoker.stat_tool_errors

@property
def stat_transport_errors(self) -> dict[str, int]:
    return self._transport_invoker.stat_transport_errors
```
