## Goal

Align search_docs schema with actual implementation by implementing or removing unsupported fields, and returning enough metadata to support follow-up get_chunk calls.

## Scope

**In-Scope**:
- Implement or remove `mode` field (bm25/grep/hybrid)
- Implement or remove `tag_filter` field
- Implement `heading_prefix` filtering
- Preserve `path_prefix` filtering
- Return metadata for follow-up get_chunk: chunk_id, source_path, heading_path, score, start_line, end_line, token_count, snippet

**Out-of-Scope**:
- Adding new tools or features
- Changes to other MCP servers' search behavior

## Assumptions

1. mode=bm25 should use FTS5 ranking (already implemented)
2. mode=hybrid should combine FTS5 with semantic embeddings (future work — remove for now)
3. tag_filter requires chunks.tags_json column (already exists in new schema)
4. heading_prefix filtering should filter by heading_path prefix match

## Affected Areas & Tool Evidence

- **Affected Files**:
  - `scripts/mcp/mdq/models.py` — update SearchDocsRequest (remove mode=hybrid, add chunk_id/heading_path/score fields to SearchResultItem)
  - `scripts/mcp/mdq/search.py` — implement heading_prefix filtering, tag_filter filtering
  - `scripts/mcp/mdq/tools.py` — update search_docs tool definition

- **Blast Radius**: Medium — changes search_docs behavior and response format. Existing consumers expecting current fields will need updates.

## Implementation Steps

1. **Phase 1: Remove unsupported mode values**
   - Remove mode=hybrid from SearchDocsRequest and tools.py schema
   - Keep mode=bm25 (FTS5 ranking) and mode=grep (keyword search)

2. **Phase 2: Implement heading_prefix filtering**
   - Add LIKE query for heading_path prefix match in search.py

3. **Phase 3: Implement tag_filter**
   - Add JSON contains check for tags_json in chunks_fts JOIN

4. **Phase 4: Update SearchResultItem fields**
   - Add chunk_id, source_path, heading_path, score, start_line, end_line, token_count, snippet fields
   - Update search.py query to return these fields from chunks/chunks_fts

5. **Phase 5: Update tools.py schema**
   - Update search_docs tool definition to match new SearchDocsRequest
