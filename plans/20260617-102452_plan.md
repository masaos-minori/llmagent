# Plan: Unify config loading and make reload results explicit (Req 29)

## Goal

Make `/reload` output clearly show what changed, what was applied, and what requires restart; document config ownership and loading paths.

## Scope

**In scope:**
- `scripts/agent/commands/cmd_config.py` ‚Äî enhance `_cmd_reload()` output to show detailed results
- `scripts/agent/services/config_reload.py` ‚Äî enhance `ConfigReloadOutcome` to categorize changes and include source file info
- `docs/05_agent_07_cli-and-commands.md` ‚Äî document /reload behavior and config ownership
- `docs/05_agent_08_configuration.md` ‚Äî document config loading path and key-to-field mappings

**Out of scope:**
- Changing the actual reload logic (which files are loaded) ‚Äî that's a separate concern
- Adding new config keys or changing config structure
- Modifying `ConfigLoader` itself

## Assumptions

1. `/reload` currently loads only `common.toml` and `agent.toml` ‚Äî this is intentional because those are the user-editable files. The other files in `_BASE_CONFIG_FILES` are infrastructure configs (tools_definitions, mcp_servers, etc.) that shouldn't be hot-reloaded.
2. The `ConfigReloadOutcome` already has `applied`, `needs_restart`, and `skipped` lists ‚Äî we need to enhance the display, not the data model.
3. "Config ownership" means documenting which config file controls which settings.

## Implementation Steps

### Step 1: Enhance `ConfigReloadOutcome` with source file tracking

Add a `source_files` field to track which files were loaded during reload:

```python
@dataclass
class ConfigReloadOutcome:
    """Structured report of what changed after a /reload."""

    applied: list[str] = field(default_factory=list)
    needs_restart: list[str] = field(default_factory=list)
    skipped: list[str] = field(default_factory=list)
    source_files: list[str] = field(default_factory=list)
```

### Step 2: Track source files in `_cmd_reload()`

In `cmd_config.py`, capture the loaded files and pass them to the outcome:

```python
def _cmd_reload(self) -> None:
    try:
        from shared.config_loader import ConfigLoader
        
        source_files = ["common.toml", "agent.toml"]
        new_cfg = ConfigLoader().load(*source_files)
        result = ConfigReloadService(self._ctx).apply_config_dict(new_cfg)
        result.source_files = source_files
        # Enhanced output (see Step 3)
    except ...
```

### Step 3: Enhance reload output format

Replace the current minimal output with a structured report:

