## Design

### 追加テスト一覧 (TestAssertAllowedRepo.test_double_slash_is_denied の後に追加)

**Test 1: `test_global_wildcard_does_not_match_any_repo`**
- `repo_allowlist=["*"]` (グローバルワイルドカード)
- `"owner/repo"` は `"*"` と exact match しない → `CicdAuthorizationError`
- ワイルドカードが任意リポジトリへの許可として機能しないことを検証

```python
def test_global_wildcard_does_not_match_any_repo(self) -> None:
    svc = _make_service(repo_allowlist=["*"])
    with pytest.raises(CicdAuthorizationError):
        svc._assert_allowed_repo("owner/repo")
```

**Test 2: `test_prefix_wildcard_does_not_match_partial_name`**
- `repo_allowlist=["owner/rep*"]` (前置一致ワイルドカード)
- `"owner/repo"` は `"owner/rep*"` と exact match しない → 拒否

```python
def test_prefix_wildcard_does_not_match_partial_name(self) -> None:
    svc = _make_service(repo_allowlist=["owner/rep*"])
    with pytest.raises(CicdAuthorizationError):
        svc._assert_allowed_repo("owner/repo")
```

**Test 3: `test_mixed_list_with_wildcard_allows_exact_match_only`**
- `repo_allowlist=["owner/*", "org/repo"]`
- `"owner/repo"` は `"owner/*"` にマッチしない (ワイルドカード無効)
- `"org/repo"` は exact match → 許可
- `"owner/repo"` は拒否される

```python
def test_mixed_list_with_wildcard_allows_exact_match_only(self) -> None:
    svc = _make_service(repo_allowlist=["owner/*", "org/repo"])
    # Exact match on "org/repo" → allowed
    svc._assert_allowed_repo("org/repo")  # must not raise
    # "owner/repo" does not match literal "owner/*" → denied
    with pytest.raises(CicdAuthorizationError):
        svc._assert_allowed_repo("owner/repo")
```
