## Design

### `test_reload_includes_security_and_tools`

```python
def test_reload_includes_security_and_tools(tmp_path: Path) -> None:
    (tmp_path / "security.toml").write_text('[security]\nallow_shell = false\n')
    (tmp_path / "tools.toml").write_text('[tools]\ntimeout = 30\n')
    result = ConfigLoader(config_dir=tmp_path).load_all()
    assert result.get("security", {}).get("allow_shell") is False
    assert result.get("tools", {}).get("timeout") == 30
```

### `test_configuration_shadowing_precedence`

```python
def test_configuration_shadowing_precedence(tmp_path: Path) -> None:
    (tmp_path / "common.toml").write_text('timeout = 5\n')
    (tmp_path / "llm.toml").write_text('timeout = 10\n')
    (tmp_path / "http.toml").write_text('timeout = 30\n')
    result = ConfigLoader(config_dir=tmp_path).load_all()
    assert result["timeout"] == 30  # http.toml wins
```

Both tests use `tmp_path` (pytest built-in fixture) for isolation; no real config files are read.
