## Design

```python
# Metadata constants
CRAWL_SCHEMA_VERSION = "1"
CHUNK_SCHEMA_VERSION = "1"

# Crawler payload additions:
payload: dict[str, Any] = {
    "schema_version": CRAWL_SCHEMA_VERSION,
    "artifact_type": "crawl",
    "created_by": "crawler",
    "url": url,
    ...
}

# Ingester validation on read:
def _validate_artifact(payload: dict, expected_type: str) -> None:
    if payload.get("artifact_type") != expected_type:
        raise ValueError(f"Expected artifact_type={expected_type!r}, got {payload.get('artifact_type')!r}")
```
