## Proposed Documentation Text

### For `store_protocols.py` module docstring (add after line 9):

```
MemoryDeleteStore — atomic cross-table deletion for memories tables.
    This protocol exists to preserve future backend replacement options.
    The current system is SQLite-only; the protocol is an extensibility boundary,
    not a sign that multiple backends already exist.
```

### For `docs/06_shared_05_db_api_and_operations.md` (add after line 172):

