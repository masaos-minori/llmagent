# Plan: M-4 Rename /context budget category from tool_results to tool_messages

Generated: 2026-07-08T10:18:48

## Goal

`/context` のバジェット表示カテゴリ名 `tool_results` を `tool_messages` に統一リネームする。
これは削除済みDBテーブル `tool_results` との混同を防ぐための名前の明確化であり、
計算ロジックは一切変更しない。

## Scope

Path A (3 ファイル以内、スキーマ/インターフェース変更を含む)。
`ContextBudget` のフィールド名変更はあるが呼び出し元は同プロジェクト内のみで外部公開なし。

変更対象ソースファイル:
- `scripts/agent/services/models.py` — `ContextBudget` フィールド名変更
- `scripts/agent/services/context_view.py` — 内部変数・docstring 変更
- `scripts/agent/commands/cmd_context.py` — 表示ラベル文字列変更

テスト更新対象:
- `tests/test_agent_cmd_context.py` — `result.tool_results` → `result.tool_messages`

## Assumptions

- 新名称は `tool_messages` とする
  (要件では `tool_messages` / `tool_context` の選択肢; LLM メッセージ構造の用語と整合する前者を採用)
- `ContextBudget` は `frozen=True` の dataclass で外部シリアライズ(JSON保存等)はしていない
- `tool_results_turn_max_chars` (config_dataclasses.py) は別概念であり変更しない
- DB テーブル名 `tool_results` はスコープ外

## Design

純粋なリネームのみ。計算ロジック・数値は変わらない。

変更の連鎖:
1. `ContextBudget` フィールド名変更 (models.py)
2. `budget_breakdown()` の返却 kwarg 更新 (context_view.py)
3. `_build_budget()` の参照更新 (context_view.py)
4. `cmd_context.py` の attribute 参照と表示ラベル更新
5. テストの attribute 参照更新

新名称の根拠: `tool_messages` は OpenAI/Anthropic の tool use API で使われる
role=tool メッセージと assistant の tool_calls JSON を指す用語と整合する。
`tool_context` より具体性が高い。
