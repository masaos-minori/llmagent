## Implementation Steps

- [ ] 1. extraction テスト: RULE/DECISION/FAILURE/CONVERSATION の各分類が正しく動作することを確認
- [ ] 2. local-only テスト: `memory_local_only=True` + 非ローカル URL → ValueError; ローカル URL は OK
- [ ] 3. cross-session recall テスト: セッション停止後に保存した DECISION memory が次セッションの `on_session_start` で返ること
- [ ] 4. branch boundary テスト: `branch="feat/x"` の memory が `branch="feat/y"` セッションで返らないこと
- [ ] 5. dedup per source type テスト: CONVERSATION は 0.85 threshold で dedup; RULE は 0.98 threshold
- [ ] 6. local-only negative test: `import rag.llm` 相当の "この module は消えているべき" スタイルのテスト（90 番で追加するもの）
