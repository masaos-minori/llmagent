## Assumptions

1. `RobustSSEParser.feed(b"data: {\"k\":1}")`:
   - `\n` がないため `while "\n" in self._buf:` が False
   - 何もパースせずに `([], False)` を返す
   - バッファには `'data: {"k":1}'` が残る
2. `_anext_or_done` は複数回呼べる:
   - ジェネレータが複数 yield する場合、各呼び出しで `(item, False)` を返す
   - 最後の呼び出しでは `(b"", True)` を返す
3. `LLMClient.stream()` が部分的な SSE フレームで終端した場合:
   - `_stream_once` の while ループが `exhausted=True` で break する
   - バッファ内の不完全データは破棄される
   - `finish_reason` が None または `"stop"` のいずれかになる
   - `stream()` は正常完了する (例外なし)

## Design

### 追加テスト一覧

**Test 1: `TestRobustSSEParserFeed.test_incomplete_line_without_newline_returns_empty`**
- `feed(b"data: {\"key\": 1}")` — `\n` なし → パース不可
- Returns `([], False)` — 空リスト、is_done=False
- バッファに残ったデータを `feed(b"\n")` で後続チャンクを送ると正常パースされる

```python
def test_incomplete_line_without_newline_returns_empty(self) -> None:
    parser = RobustSSEParser(malformed_retry=2, heartbeat_timeout=0.0)
    payloads, is_done = parser.feed(b'data: {"key": 1}')
    assert payloads == []
    assert is_done is False
    # Subsequent chunk completing the line parses normally
    payloads2, is_done2 = parser.feed(b"\n")
    assert len(payloads2) == 1
    assert is_done2 is False
```

**Test 2: `TestAnextOrDone.test_multiple_yields_then_exhausted`**
- ジェネレータが `b"a"`, `b"b"` を yield して終了
- 最初の 2 回は `(item, False)`, 3 回目は `(b"", True)` を返す

```python
@pytest.mark.asyncio
async def test_multiple_yields_then_exhausted(self) -> None:
    async def _gen() -> AsyncIterator[bytes]:
        yield b"a"
        yield b"b"

    aiter = _gen().__aiter__()
    item1, done1 = await _anext_or_done(aiter)
    assert item1 == b"a"
    assert done1 is False

    item2, done2 = await _anext_or_done(aiter)
    assert item2 == b"b"
    assert done2 is False

    item3, done3 = await _anext_or_done(aiter)
    assert item3 == b""
    assert done3 is True
```

**Test 3: `TestLLMClientStream.test_stream_ends_mid_sse_frame`**
- ストリームが SSE フレーム中断 (改行なし) で終端する
- 有効な SSE チャンクを 1 つ含んだ後、不完全なフレームで終端
- 例外が発生しない; 有効なチャンクのコンテンツが返される

```python
@pytest.mark.asyncio
async def test_stream_ends_mid_sse_frame(self) -> None:
    """Stream ends with partial SSE frame (no trailing newline) — treated as
    stream exhaustion; partial frame is discarded, no exception raised."""
    partial_stream = (
        b'data: {"choices":[{"delta":{"content":"ok"},"finish_reason":"stop"}]}\n\n'
        b"data: {incomplete"  # partial frame, no newline
    )
    client = _make_client(sse_reconnect_max=0)

    with respx.mock:
        respx.post("http://llm/v1/chat").mock(
            return_value=httpx.Response(200, content=partial_stream)
        )
        result = await client.stream(
            "http://llm/v1/chat", [{"role": "user", "content": "hi"}], []
        )

    assert "ok" in result.message.get("content", "")
```
