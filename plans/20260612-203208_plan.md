# Refactoring Plan: scripts/agent/shared — DTO整備・孤立コード除去・型安全化

## Goal

`scripts/agent/shared/` 内の孤立した定義を削除し、audit logs / approval decisions / execution results 専用の frozen dataclass DTO を追加することで、agent 層の外部境界データを型安全に扱えるようにする。

---

## Scope

**In scope:**
- `scripts/agent/shared/enums.py` — 孤立シンボルの削除（ExtractionDecision, RetrievalMode, ToolSafetyTier, WorkflowStage）
- `scripts/agent/shared/exceptions.py` — 孤立シンボルの削除（AgentSharedError, ValidationError, ConfigurationSchemaError, WorkflowStageError, UnknownTierError, UnknownRoleError）
- `scripts/agent/shared/models.py` — AuditLogEvent / ApprovalDecisionRecord / ExecutionResultRecord DTO の追加
- `scripts/agent/tool_approval.py` — deprecated `ApprovalDecision` TypedDict の削除
- `scripts/agent/tool_audit.py` — dict-based audit 構築を DTO-based に変更
- `scripts/agent/tool_models.py` — ToolExecutionResult / ApprovalOutcome をそのまま維持（既にDTO化済み）

**Out of scope:**
- `scripts/agent/memory/enums.py` — 変更しない（ExtractionDecision/RetrievalMode の正規定義として既に存在）
- `scripts/agent/commands/exceptions.py` — 変更しない（UnknownTierError の正規定義として既に存在）
- `scripts/agent/tool_enums.py` — 変更しない（RiskLevel/ApprovalDecisionType の正規定義として既に存在）
- DB スキーマ・MCP サーバー・RAG パイプライン — 影響なし

## Design

### 追加する DTO（`agent/shared/models.py`）

```python
@dataclass(frozen=True)
class ToolApprovalEvent:
    """audit_approval イベントの構造化表現。"""
    event: Literal["tool_approval"]
    task_id: str
    tool: str
    operation_type: str
    resource_scope: dict[str, str]  # path/branch キーのみ（外部境界ではない）
    risk: str
    decision: str
    args_preview: dict[str, object]
    ts: float

@dataclass(frozen=True)
class ApprovalDecisionEvent:
    """approval_decision イベントの構造化表現。"""
    event: Literal["approval_decision"]
    task_id: str
    tool: str
    risk_level: str
    decision: str
    escalation_reason: str
    ts: float

@dataclass(frozen=True)
class ToolExecEvent:
    """tool_exec イベントの構造化表現。"""
    event: Literal["tool_exec"]
    task_id: str
    tool: str
    operation_type: str
    resource_scope: dict[str, str]
    mcp_request_id: str
    is_error: bool
    args_preview: dict[str, object]
    ts: float
```

注意：`resource_scope` は外部境界ではなく内部でマスク済みの辞書なので `dict[str, str]` は許容。
`args_preview` は外部由来ではなくマスク済みの内部データ。

### 削除する孤立コード

- `agent/shared/enums.py`: ExtractionDecision / RetrievalMode（memory/enums.py が正）、ToolSafetyTier（tool_enums.py の RiskLevel が対応）、WorkflowStage（参照ゼロ）
- `agent/shared/exceptions.py`: 全クラス（参照ゼロ、各サブモジュールに独自例外あり）
- `agent/shared/models.py`: CommandResult / ValidationErrorDetail（参照ゼロ）
- `agent/tool_approval.py`: deprecated `ApprovalDecision` TypedDict

## Implementation Steps

1. **`agent/shared/enums.py` から孤立シンボルを削除する**
   - ExtractionDecision, RetrievalMode, ToolSafetyTier, WorkflowStage を削除
   - ファイル内が空になるなら `__init__.py` のコメントを更新して enums モジュールの存在意義を整理
   - 完了条件: `grep -r "from agent.shared.enums\|agent\.shared\.enums" scripts/` がゼロ

2. **`agent/shared/exceptions.py` から孤立シンボルを削除する**
   - 全クラスを削除
   - ファイル内が空になるなら削除または `pass` のみに
   - 完了条件: `grep -r "from agent.shared.exceptions\|agent\.shared\.exceptions" scripts/` がゼロ

3. **`agent/shared/models.py` の不要 DTO を削除し、3種の audit DTO を追加する**
   - CommandResult, ValidationErrorDetail を削除
   - ToolApprovalEvent, ApprovalDecisionEvent, ToolExecEvent を frozen dataclass で追加
   - 完了条件: mypy でエラーなし

4. **`agent/tool_approval.py` の deprecated `ApprovalDecision` TypedDict を削除する**
   - クラス定義と docstring を削除
   - 完了条件: `grep "ApprovalDecision" scripts/agent/tool_approval.py` がゼロ

5. **`agent/tool_audit.py` の dict-based 実装を DTO-based に変換する**
   - `audit_approval`: `ToolApprovalEvent` を生成してから `orjson.dumps(dataclasses.asdict(event))` で serialize
   - `log_approval_decision`: `ApprovalDecisionEvent` を生成して serialize
   - `audit_tool_exec`: `ToolExecEvent` を生成して serialize
   - 完了条件: dict リテラルが関数本体から消え、DTO 経由で書き込んでいる

6. **バリデーション実行（Step 7 参照）**
