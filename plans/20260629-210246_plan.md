**In-Scope**:
- `scripts/eventbus/app.py:L440` の canonical ack エンドポイント実装の確認
- `scripts/eventbus/offsets.py:write_offset()` が単調増加を強制しているか確認（現状は強制なし）
- `docs/06_eventbus_02_http_api_and_runtime.md:L99-130` の ack ドキュメント完全性確認
- `docs/06_eventbus_06_reference_api.md:L83` の endpoint リスト確認

- offset の単調増加挙動の docs への明示（強制 or 非強制）

- negative ack (nack) の設計変更
- exactly-once デリバリー
