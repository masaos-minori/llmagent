## Goal

Extract MCP transport invocation — including `HttpTransport` init, health checks, lifecycle readiness, semaphore application, transport call, error recording, and error statistics — out of `ToolExecutor` into a new `ToolTransportInvoker` class. `ToolExecutor._raw_execute()` becomes a thin resolver→delegate call.

## Scope

**In-Scope:**
- New file `scripts/shared/tool_transport_invoker.py` containing `ToolTransportInvoker`
- Update `scripts/shared/tool_executor.py`:
  - Move methods `_error_result`, `_success_result`, `_record_success`, `_record_transport_error`, `_check_health`, `_ensure_semaphores`, `_maybe_semaphore`, `_execute_with_semaphore` to `ToolTransportInvoker`
  - Replace `_raw_execute()` body with `server_key = self._resolver.resolve(tool_name); return await self._transport_invoker.invoke(server_key, tool_name, args)`
  - Delegate `set_lifecycle()`, `set_health_registry()`, `set_session_id()`, `get_error_counters()` to `self._transport_invoker`
  - Remove `_transports`, `stat_tool_errors`, `stat_transport_errors` fields from `ToolExecutor`
- New test file `tests/shared/test_tool_transport_invoker.py`

**Out-of-Scope:**
- Supporting stdio transport
- Changing `HttpTransport` API
- Changing `ToolCallResult` schema
- `deploy/deploy.sh`: no change — `scripts/` is rsync'd wholesale

## Assumptions

1. `ToolTransportInvoker.__init__` receives `http`, `server_configs`, `concurrency_limits`, `lifecycle` — same args that currently flow from `ToolExecutor.__init__`.
2. `ToolTransportInvoker` is instantiated inside `ToolExecutor.__init__` after resolver construction.
3. `stat_tool_errors` and `stat_transport_errors` move to `ToolTransportInvoker`; `ToolExecutor.get_error_counters()` delegates.
4. `ToolExecutor.stat_cache_hits` stays in `ToolExecutor` (cache concern, not transport concern).
5. `LifecycleProtocol` is defined in `tool_executor.py`; `tool_transport_invoker.py` imports it from there — or the Protocol is moved to a shared location. Simplest path: import `LifecycleProtocol` from `shared.tool_executor`.
6. `tests/shared/__init__.py` will exist (created by prior plans).

## Design

### `shared/tool_lifecycle.py`

```python
from typing import Protocol, runtime_checkable

@runtime_checkable
class LifecycleProtocol(Protocol):
    async def ensure_ready(self, server_key: str) -> None: ...
```

### `ToolTransportInvoker` interface

```python
class ToolTransportInvoker:
    def __init__(
        self,
        http: httpx.AsyncClient,
        server_configs: dict[str, McpServerConfig],
        concurrency_limits: dict[str, int] | None = None,
        lifecycle: LifecycleProtocol | None = None,
    ) -> None:

    async def invoke(
        self,
        server_key: str,
        tool_name: str,
        args: dict[str, Any],
    ) -> ToolCallResult:
        # 1. _check_health(server_key) → return error result if unavailable
        # 2. await lifecycle.ensure_ready(server_key) if lifecycle
        # 3. transport = self._transports.get(server_key); error if missing
        # 4. _ensure_semaphores(); sem = self._semaphores.get(server_key)
        # 5. try: result = await transport.call under semaphore
        #    _record_success(server_key, result); return result
        #    except TransportError: return _record_transport_error(server_key, e)

    def set_lifecycle(self, lifecycle: LifecycleProtocol | None) -> None: ...
    def set_health_registry(self, registry: McpServerHealthRegistry | None) -> None: ...
    def set_session_id(self, session_id: str) -> None: ...
    def get_error_counters(self) -> dict[str, dict[str, int]]: ...
```

### Updated `ToolExecutor.__init__` additions

```python
from shared.tool_transport_invoker import ToolTransportInvoker

# after self._resolver = ...
self._transport_invoker = ToolTransportInvoker(
    http=http,
    server_configs=server_configs,
    concurrency_limits=concurrency_limits,
    lifecycle=lifecycle,
)
```

### Updated `ToolExecutor._raw_execute()`

```python
async def _raw_execute(self, tool_name: str, args: dict[str, Any]) -> ToolCallResult:
    server_key = self._resolver.resolve(tool_name)
    return await self._transport_invoker.invoke(server_key, tool_name, args)
```

### Updated delegating methods in `ToolExecutor`

```python
def set_lifecycle(self, lifecycle: LifecycleProtocol | None) -> None:
    self._transport_invoker.set_lifecycle(lifecycle)

def set_health_registry(self, registry: McpServerHealthRegistry | None) -> None:
    self._transport_invoker.set_health_registry(registry)

def set_session_id(self, session_id: str) -> None:
    self._transport_invoker.set_session_id(session_id)

def get_error_counters(self) -> dict[str, dict[str, int]]:
    return self._transport_invoker.get_error_counters()
```
