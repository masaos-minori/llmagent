## Goal

`MessageRow` と `SQLiteSessionStore` の `message_save/message_list` に `tool_call_id` を追加し、スキーマ・DTO・ストア実装を一致させる。

## Scope

**In**:
- `scripts/db/models.py` — `MessageRow` に `tool_call_id: str | None = None` 追加
- `scripts/db/store_protocols.py` — `SessionStoreProtocol.message_save()` に `tool_call_id` パラメータ追加
- `scripts/db/store_impl.py` — `SQLiteSessionStore.message_save()` / `message_list()` に `tool_call_id` 対応追加
- `tests/test_db_store_impl.py` — `message_save` / `message_list` テストの更新

**Out**:
- `session_message_repo.py` の変更（既に `tool_call_id` を正しく処理している）
- `schema_sql.py` の変更（既に `tool_call_id TEXT` カラムあり）
- OpenAI ツールコールセマンティクスの変更
- 全メッセージ DTO の上位層リデザイン

## Assumptions

1. `session_message_repo.py` が本番の書き込み/読み取りパスであり、`store_impl.py` は `store_protocols.py` 経由の抽象 API として整合させる。
2. `MessageRow.tool_call_id` はデフォルト `None` にすることで、既存の `MessageRow()` 生成コードを壊さない。
3. `store_protocols.py` の `message_save` に `tool_call_id: str | None = None` をデフォルト引数で追加することで、既存呼び出し元は破壊的変更なしに動作し続ける。

## Design

### models.py

```python
@dataclass(frozen=True)
class MessageRow:
    role: str
    content: str
    tool_calls: str | None
    tool_call_id: str | None = None  # 追加
```

### store_protocols.py

```python
def message_save(
    self,
    session_id: int,
    role: str,
    content: str,
    tool_calls: str | None,
    tool_call_id: str | None = None,  # 追加
) -> None: ...
```

### store_impl.py

```python
def message_save(
    self,
    session_id: int,
    role: str,
    content: str,
    tool_calls: str | None,
    tool_call_id: str | None = None,  # 追加
) -> None:
    self._db.execute(
        "INSERT INTO messages (session_id, role, content, tool_calls, tool_call_id)"
        " VALUES (?, ?, ?, ?, ?)",
        (session_id, role, content, tool_calls, tool_call_id),  # 追加
    )

def message_list(self, session_id: int) -> list[MessageRow]:
    rows = self._db.fetchall(
        "SELECT role, content, tool_calls, tool_call_id FROM messages"  # 追加
        " WHERE session_id = ? ORDER BY message_id",
        (session_id,),
    )
    return [
        MessageRow(
            role=str(r[0]),
            content=str(r[1]) if r[1] is not None else "",
            tool_calls=r[2],
            tool_call_id=r[3] if r[3] is not None else None,  # 追加
        )
        for r in rows
    ]
```
