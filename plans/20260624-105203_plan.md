## Design

### Command Maintenance Procedure

```
When adding a new command:
1. Add to agent/commands/registry.py (or equivalent)
2. Add unit test in tests/test_cmd_{name}.py
3. Update command summary table in 05_agent_07 §Command Reference
4. Update short command list in 05_agent_01 §Quick Start
5. If command is operator-facing: update 05_agent_10 §Operator Commands
```
