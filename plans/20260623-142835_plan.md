# Plan: Expose Memory Embedding Fallback and Circuit-Breaker State

**Requirement:** `requires/20260622_79_require.md`

---

## Goal

Make memory retrieval degradation (fts_only fallback) visible in normal runtime paths without requiring operators to invoke `/memory status`. Three gaps exist: (1) no terminal-visible signal when retrieval falls back mid-session, (2) `/stats` doesn't show memory embedding state, (3) no cumulative fallback count — only `last_retrieval_mode` (most recent call).

---

## Scope

**In:**
- `scripts/agent/memory/retriever.py` — add `fts_fallback_count: int` counter to `HybridRetriever`; increment on each fts_only fallback in `search()`
- `scripts/agent/commands/cmd_memory.py` — show `fts_fallback_count` in `_memory_status()`; highlight `last_retrieval_mode == "fts_only"` as DEGRADED; add circuit-open warning when `circuit_open=True`
- `scripts/agent/commands/cmd_config_stats.py` — add memory embedding state line to `/stats` when `use_memory_layer=True`
- `scripts/agent/commands/models.py` — add `memory_circuit_open: bool = False` and `memory_fts_fallback_count: int = 0` to `StatsViewModel`

**Out:**
- Changing fallback semantics or circuit-breaker thresholds
- Adding warnings mid-turn to the user's message stream (disruptive; `/memory status` and `/stats` are the right surfaces)
- Adding memory state to every turn's output
- Changing `EmbeddingClient` behavior

---

## Assumptions

1. `HybridRetriever` already has `last_retrieval_mode: str = "unknown"` (verified at line 261). Adding `fts_fallback_count: int = 0` is analogous.
2. `_memory_status()` already shows: enabled, circuit, fail_count, last_retrieval_mode. The gaps are: (a) no cumulative fallback count, (b) no visual DEGRADED indicator, (c) no circuit-open warning.
3. `/stats` currently has no memory-specific row — adding one when `use_memory_layer=True` and embedding is enabled is the minimum viable addition.
4. `mem.retriever` is accessible in `_memory_status()` already (line `last_retrieval_mode = mem.retriever.last_retrieval_mode`).
5. `embed_client.get_status()` returns `EmbeddingClientStatus(enabled, circuit_open, fail_count, resets_in_sec)`.

## Design

### Change 1: `retriever.py` — `fts_fallback_count` counter

In `HybridRetriever.__init__()`, add:
```python
self.fts_fallback_count: int = 0
```

In `search()`, increment when falling back:
```python
if embedding is None:
    logger.info("retrieval: fts_only (reason=embedding_disabled_or_none)")
    self.last_retrieval_mode = "fts_only"
    self.fts_fallback_count += 1
    return fts_hits

vec_hits = self._vec.knn_search(...)
if not vec_hits:
    logger.info("retrieval: fts_only (reason=vec_returned_empty)")
    self.last_retrieval_mode = "fts_only"
    self.fts_fallback_count += 1
    return fts_hits
```

### Change 2: `cmd_memory.py` — enhanced `/memory status`

**Before:**
```python
rows = [
    ["Memory layer", "enabled"],
    ["Embedding enabled", "Yes" if status.enabled else "No"],
    ["Circuit", ("OPEN" + circuit_detail) if status.circuit_open else "closed"],
    ["Consecutive failures", str(status.fail_count)],
    ["Last retrieval mode", mem.retriever.last_retrieval_mode],
]
```

**After:**
```python
retrieval_mode = mem.retriever.last_retrieval_mode
mode_display = retrieval_mode
if retrieval_mode == "fts_only":
    mode_display = "fts_only  [DEGRADED — embedding unavailable]"

circuit_status = "closed"
if status.circuit_open:
    circuit_status = f"OPEN  [circuit breaker active{circuit_detail}]"

rows = [
    ["Memory layer", "enabled"],
    ["Embedding enabled", "Yes" if status.enabled else "No"],
    ["Circuit", circuit_status],
    ["Consecutive failures", str(status.fail_count)],
    ["FTS fallback count", str(mem.retriever.fts_fallback_count)],
    ["Last retrieval mode", mode_display],
]
```

### Change 3: `models.py` — `StatsViewModel` fields

Add fields:
```python
memory_circuit_open: bool = False
memory_fts_fallback_count: int = 0
```

### Change 4: `cmd_config_stats.py` — `/stats` memory line

In `_collect_stats()`, add:
```python
mem_services = ctx.services.memory if ctx.services is not None else None
if mem_services is not None and mem_services.retriever is not None:
    embed_client = mem_services.retriever.embed_client
    if embed_client is not None:
        status = embed_client.get_status()
        memory_circuit_open=status.circuit_open,
        memory_fts_fallback_count=mem_services.retriever.fts_fallback_count,
```

In `_cmd_stats()`, add when memory is active:
```python
if stats.memory_circuit_open:
    self._out.write("  Memory embed: CIRCUIT OPEN [DEGRADED]")
elif stats.memory_fts_fallback_count > 0:
    self._out.write(
        f"  Memory embed: fts_only x{stats.memory_fts_fallback_count} [degraded]"
    )
```

Note: Only display when degraded — no line shown when embedding is working normally (reduces noise).
