## Design

### Session Title Failure Behavior

```
Success: LLM returns non-empty title → stored in session.title
Failure cases:
  1. LLM timeout → fallback: "Session {session_id[:8]}" (non-blocking)
  2. Empty/invalid response → same fallback
  3. Generation disabled → title = None (displayed as "Untitled")
  
All failure cases: non-blocking, logged at WARNING level
```
