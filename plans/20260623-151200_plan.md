# Plan: Enhance /reload Scope and Fix Configuration Shadowing

**Requirement:** `requires/20260623_04_require.md`

---

## Goal

1. Make `/reload` reflect `security.toml` changes at runtime (currently `security_profile` and `security_lockdown_enabled` are loaded from TOML but not applied to `ctx.cfg`)
2. Document HTTP config resolution precedence in code

---

## Current State

**`/reload` scope ‚Äî already correct:**
`_cmd_reload()` (`scripts/agent/commands/cmd_config.py:50`) already calls `ConfigLoader().load_all()`, which merges all 11 TOML files in `_BASE_CONFIG_FILES` order. The requirement premise ("only loads `common.toml` and `agent.toml`") is outdated.

**Gap 1 ‚Äî `security_profile` / `security_lockdown_enabled` not applied:**
`apply_config_dict()` (`scripts/agent/services/config_reload.py:167`) dispatches to `_reload_approval_config()` for approval settings and `_reload_approval_settings()` for tool settings ‚Äî but neither handles `security_profile` or `security_lockdown_enabled` from `security.toml` (lines 487-523). These keys are in the merged dict from `load_all()` but are silently ignored.

Affected fields in `AgentConfig`:
- `ctx.cfg.tool.security_profile` (`SecurityProfile` enum, from `shared/mcp_config.py`)
- `ctx.cfg.tool.security_lockdown_enabled` (`bool`)

**Gap 2 ‚Äî HTTP precedence implicit:**
`_BASE_CONFIG_FILES` order (`config_loader.py:18-31`) defines last-writer-wins for duplicate keys. HTTP-related keys appear in `common.toml`, `llm.toml`, `http.toml` (in that load order). The precedence is `http.toml > llm.toml > common.toml` ‚Äî but there's no comment documenting this in the source.

**Gap 3 ‚Äî stale `_cmd_reload()` docstring:**
Line 35: `"""Reload config/agent.toml and apply runtime-configurable parameters."""`
`agent.toml` is not in `_BASE_CONFIG_FILES` and is not loaded.

## Design

### Change 1: `config_reload.py` ‚Äî add security fields to `_reload_approval_settings()`

**`_reload_approval_settings()` (lines 513-523):**

```python
def _reload_approval_settings(
    self,
    ctx: AgentContext,
    new_cfg: dict[str, Any],
) -> None:
    """Update approval, tool, and memory config fields when present in new_cfg."""
    self._reload_approval_config(ctx, new_cfg)
    if (lst := _get_list(new_cfg, "allowed_tools")) is not None:
        ctx.cfg.tool.allowed_tools = list(lst)
    if (v := _get_int(new_cfg, "memory_retention_days")) is not None:
        ctx.cfg.memory.memory_retention_days = v
    # security.toml fields ‚Äî hot-reloadable
    if (v := _get_str(new_cfg, "security_profile")) is not None:
        try:
            from shared.mcp_config import SecurityProfile  # noqa: PLC0415
            ctx.cfg.tool.security_profile = SecurityProfile(v)
        except ValueError:
            pass  # invalid enum value ‚Äî leave current
    if (vb := _get_bool(new_cfg, "security_lockdown_enabled")) is not None:
        ctx.cfg.tool.security_lockdown_enabled = vb
```
