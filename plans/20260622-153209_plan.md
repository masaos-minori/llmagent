## Assumptions

1. `resolve_safe()` 実装 (`scripts/mcp/file/common.py:21-35`):
   ```python
   resolved = Path(raw_path).resolve()  # 全 symlink を再帰的に解決
   for allowed in allowed_dirs:
       resolved.relative_to(allowed.resolve())  # 許可 dir 外なら FileAuthorizationError
   ```
   `Path.resolve()` は多段シンボリックリンクを再帰的に展開するため、
   連鎖シンボリックリンクも最終ターゲットで判定される。

2. 既存テストカバレッジ:
   - `test_resolve_safe_rejects_symlink_outside_allowed`: 1 段シンボリックリンク → 外部 → 拒否 (テスト済み)
   - `test_resolve_safe_allows_symlink_inside_allowed`: allowed dir 内のファイル → 許可 (テスト済み)

3. 追加すべきケース:
   - 2 段シンボリックリンク (`link_a → link_b → 外部ファイル`) → `FileAuthorizationError`
   - シンボリックリンクがディレクトリ外を指し、その下のファイルにアクセス → `FileAuthorizationError`
   - 連鎖シンボリックリンクが両方 allowed dir 内に留まる → 許可

4. シンボリックリンクの作成: `Path.symlink_to()` は WSL2/Linux で正常に動作する。

## Design

### 追加テスト一覧 (TestSecurityWrappers.test_resolve_safe_allows_symlink_inside_allowed の後に追加)

**Test 1: `test_resolve_safe_rejects_chained_symlink_to_outside`**
- `link_a → link_b → outside_file` (2 段連鎖)
- `link_a`, `link_b` は両方とも root 内
- 最終ターゲットが root 外 → `FileAuthorizationError`

```python
def test_resolve_safe_rejects_chained_symlink_to_outside(self, tmp_path: Path):
    root = tmp_path / "workspace"
    root.mkdir()
    outside = tmp_path / "secret.txt"
    outside.write_text("secret", encoding="utf-8")

    link_b = root / "link_b"
    link_b.symlink_to(outside)          # step 2: link_b → outside

    link_a = root / "link_a"
    link_a.symlink_to(link_b)           # step 1: link_a → link_b (chain)

    svc = ReadFileService(
        allowed_dirs=[root],
        max_read_bytes=1024,
        max_tree_depth=3,
        max_search_results=50,
    )
    with pytest.raises(FileAuthorizationError):
        svc._resolve_safe(str(link_a))
```

**Test 2: `test_resolve_safe_rejects_symlink_dir_pointing_outside`**
- `link_dir → outside_dir` (ディレクトリへのシンボリックリンク)
- `str(link_dir / "file.txt")` でアクセス → 解決後は outside_dir/file.txt → 拒否

```python
def test_resolve_safe_rejects_symlink_dir_pointing_outside(self, tmp_path: Path):
    root = tmp_path / "workspace"
    root.mkdir()
    outside_dir = tmp_path / "outside_dir"
    outside_dir.mkdir()
    (outside_dir / "secret.txt").write_text("secret", encoding="utf-8")

    link_dir = root / "link_to_outside_dir"
    link_dir.symlink_to(outside_dir)    # symlink to dir outside allowed

    svc = ReadFileService(
        allowed_dirs=[root],
        max_read_bytes=1024,
        max_tree_depth=3,
        max_search_results=50,
    )
    with pytest.raises(FileAuthorizationError):
        svc._resolve_safe(str(link_dir / "secret.txt"))
```

**Test 3: `test_resolve_safe_allows_chained_symlink_inside_allowed`**
- `link_a → link_b → inside_file` (2 段連鎖、すべて root 内)
- 最終ターゲットが root 内 → 許可

```python
def test_resolve_safe_allows_chained_symlink_inside_allowed(self, tmp_path: Path):
    root = tmp_path / "workspace"
    root.mkdir()
    real_file = root / "real.txt"
    real_file.write_text("data", encoding="utf-8")

    link_b = root / "link_b"
    link_b.symlink_to(real_file)        # step 2: link_b → real.txt

    link_a = root / "link_a"
    link_a.symlink_to(link_b)           # step 1: link_a → link_b (chain)

    svc = ReadFileService(
        allowed_dirs=[root],
        max_read_bytes=1024,
        max_tree_depth=3,
        max_search_results=50,
    )
    result = svc._resolve_safe(str(link_a))
    assert result == real_file.resolve()
```
