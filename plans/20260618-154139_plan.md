Derived from `01_plan.md` using requirement `requires/20260618_49_require.md`.

## Goal

The active `use_rrf` mode and `rrf_k` value are visible at runtime ŌĆö in debug output, logs, and (optionally) a status command.

---

## Scope

**In scope:**
- `scripts/rag/pipeline.py` ŌĆö plumb `rrf_k` from `cfg.fusion.rrf_k` to `FusionStage`

**Out of scope:**
