## Migration Policy

**Decision: No backward-compatibility glob.** Old `.txt` files must be manually renamed or re-crawled before running the updated ingester. Document this explicitly.

If operator needs migration: `for f in rag-src/*.txt rag-src/chunk/*.txt; do mv "$f" "${f%.txt}.json"; done`
