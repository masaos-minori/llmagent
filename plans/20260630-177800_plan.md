# Plan: Add strict mode support to `ConfigLoader.load_all()`

## Goal
`ConfigLoader.load_all(strict=True)` raises `ConfigMissingError` for missing required config files while preserving current lenient behavior when `strict=False`.

**Out-of-Scope**:

- Deployment configuration changes

## Assumptions
