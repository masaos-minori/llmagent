## Fallback chain (current behavior)

| Path | `_augment_refiner()` return | `augment()` behavior |
|---|---|---|
| Refiner returns text | `str` | Returns refined text |
| Refiner returns empty (`""`) | `None` (line 47-49: `if refined: return refined; logger.warning; return None`) | Falls through to raw chunks |
| Refiner raises HTTP/ValueError | `None` (line 50-51: `except ... return None`) | Falls through to raw chunks |
| Refiner disabled (`use_refiner=False`) | not called | Directly uses raw chunks |

The fallback is always raw chunks via `_format_chunks()` at line 364.
