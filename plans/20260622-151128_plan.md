## Goal

`tests/test_tool_approval_repos.py` に `_make_ctx()` のデフォルト値を検証するテストを追加する。

## Scope

**In-Scope:**
- `tests/test_tool_approval_risk.py` — `TestMakeCfgDefaults` クラスの新設（3 テスト）
- `tests/test_tool_approval_repos.py` — `TestMakeCtxDefaults` クラスの新設（3 テスト）

**Out-of-Scope:**

## Assumptions

1. `_make_cfg()` は `approval_risk_rules` を設定しないため、`build_agent_config()` が
   `_DEFAULT_APPROVAL_RISK_RULES` を使用する。これが `test_write_file_returns_medium` などの
   テストが "medium" / "high" を返す理由。
   - `_DEFAULT_APPROVAL_RISK_RULES["write_file"] == "medium"`
   - `_DEFAULT_APPROVAL_RISK_RULES["delete_file"] == "high"`
2. `_make_cfg()` は `tool_safety_tiers` を明示的に設定するため、これは `cfg.approval.tool_safety_tiers` に
   反映される。
   - `cfg.approval.tool_safety_tiers["list_directory"] == "READ_ONLY"`
   - `cfg.approval.tool_safety_tiers["shell_run"] == "ADMIN"`
3. `_make_ctx()` のデフォルト値は mock 属性として直接設定される:
   - `ctx.turn.current_turn_id == "test-turn-id"`
   - `ctx.workflow.workflow_id is None`
   - `ctx.services.audit_logger is None`
4. `cfg.approval` は `AgentConfig.approval: ApprovalConfig` 属性として参照可能。

## Design

### test_tool_approval_risk.py: TestMakeCfgDefaults クラス

```python
class TestMakeCfgDefaults:
    def test_default_risk_rules_write_file_is_medium(self) -> None:
        """_make_cfg() does not set approval_risk_rules; _DEFAULT_APPROVAL_RISK_RULES
        maps write_file -> medium. Tests pinning this default."""
        cfg = _make_cfg()
        assert cfg.approval.approval_risk_rules.get("write_file") == "medium"

    def test_default_risk_rules_delete_file_is_high(self) -> None:
        """_DEFAULT_APPROVAL_RISK_RULES maps delete_file -> high."""
        cfg = _make_cfg()
        assert cfg.approval.approval_risk_rules.get("delete_file") == "high"

    def test_tool_safety_tiers_shell_run_is_admin(self) -> None:
        """_make_cfg() explicitly sets shell_run -> ADMIN tier."""
        cfg = _make_cfg()
        assert cfg.approval.tool_safety_tiers.get("shell_run") == "ADMIN"
```

### test_tool_approval_repos.py: TestMakeCtxDefaults クラス

```python
class TestMakeCtxDefaults:
    def test_ctx_turn_id_default(self) -> None:
        """_make_ctx() sets current_turn_id to a known test value."""
        ctx = _make_ctx()
        assert ctx.turn.current_turn_id == "test-turn-id"

    def test_ctx_workflow_id_default_none(self) -> None:
        """_make_ctx() sets workflow_id=None by default."""
        ctx = _make_ctx()
        assert ctx.workflow.workflow_id is None

    def test_ctx_audit_logger_default_none(self) -> None:
        """_make_ctx() sets audit_logger=None; tests that rely on audit logging
        must explicitly set ctx.services.audit_logger = MagicMock()."""
        ctx = _make_ctx()
        assert ctx.services.audit_logger is None
```
