## Goal

Extract cache key generation, TTL validation, LRU ordering, store, and max-size eviction out of `ToolExecutor` into a new `ToolResultCache` class in `shared/tool_cache.py`. `ToolExecutor._execute_with_cache()` becomes a thin call that delegates hit/store to `ToolResultCache`.

## Scope

**In-Scope:**
- Add `ToolResultCache` to `scripts/shared/tool_cache.py` (alongside `CacheEntry`)
- Update `scripts/shared/tool_executor.py`:
  - Replace `self._cache: OrderedDict[str, CacheEntry]` with `self._cache: ToolResultCache`
  - Remove `_store_and_evict()` method
  - Simplify `_execute_with_cache()` to use `ToolResultCache` methods
  - Update `clear_cache()` to call `self._cache.clear()`
  - Update `apply_config()` to call `self._cache.apply_config()`
- New test file `tests/shared/test_tool_result_cache.py`

**Out-of-Scope:**
- Moving stampede protection into the cache class
- Changing cache key format
- Adding persistent cache
- Moving `stat_cache_hits` out of `ToolExecutor` (stays in Phase 1)
- `deploy/deploy.sh`: no change — `scripts/` is rsync'd wholesale

## Assumptions

1. `ToolResultCache.__init__` takes `ttl: float` and `max_size: int = 0`.
2. `ToolResultCache.get_result()` returns a fully-formed `ToolCallResult` (including `request_id=""`, `server_key=""`, correct `error_type`) — not a raw `CacheEntry`. This keeps the caller's `_execute_with_cache()` clean.
3. `ToolResultCache.get_result()` internally handles TTL check, expired-entry deletion, and LRU update.
4. `stat_cache_hits` increment stays in `ToolExecutor._execute_with_cache()` — a non-None return from `get_result()` is a cache hit.
5. `CacheEntry` is kept in `tool_cache.py` as an implementation detail of `ToolResultCache`; it is still importable for any existing usage.
6. `tests/shared/__init__.py` must exist (created by require-22 plan if processed first; create here if not already present).

## Design

### New class: `ToolResultCache`

```python
# scripts/shared/tool_cache.py — appended after CacheEntry

import logging
import time
from collections import OrderedDict
from typing import Any

from shared.json_utils import dumps as _json_dumps
from shared.transport_dto import ToolCallResult

logger = logging.getLogger(__name__)


class ToolResultCache:
    """LRU + TTL cache for successful tool call results."""

    def __init__(self, ttl: float, max_size: int = 0) -> None:
        self._ttl = ttl
        self._max_size = max_size
        self._store: OrderedDict[str, CacheEntry] = OrderedDict()

    def make_key(self, tool_name: str, args: dict[str, Any]) -> str:
        return f"{tool_name}:{_json_dumps(args)}"

    def get_result(self, key: str) -> ToolCallResult | None:
        entry = self._store.get(key)
        if entry is None:
            return None
        age = time.time() - entry.cached_at
        if age >= self._ttl:
            del self._store[key]
            return None
        self._store.move_to_end(key)
        return ToolCallResult(
            output=entry.output,
            is_error=entry.is_error,
            request_id="",
            server_key="",
            error_type="tool" if entry.is_error else "",
        )

    def store_if_success(self, key: str, result: ToolCallResult) -> None:
        if result.is_error:
            return
        self._store[key] = CacheEntry(
            output=result.output,
            is_error=result.is_error,
            cached_at=time.time(),
        )
        if self._max_size > 0 and len(self._store) > self._max_size:
            evicted_key, _ = self._store.popitem(last=False)
            logger.debug("Tool cache LRU evict: %r", evicted_key)

    def clear(self) -> None:
        self._store.clear()

    def apply_config(self, *, ttl: float | None = None, max_size: int | None = None) -> None:
        if ttl is not None:
            self._ttl = ttl
        if max_size is not None:
            self._max_size = max_size
```

### Changes to `ToolExecutor`

**In `__init__`:** replace:
```python
self._cache: OrderedDict[str, CacheEntry] = OrderedDict()
```
with:
```python
self._cache = ToolResultCache(ttl=cache_ttl, max_size=cache_max_size)
```
Remove `OrderedDict` import if no longer used elsewhere.

**Update `apply_config()`:**
```python
def apply_config(self, *, cache_ttl: float | None = None) -> None:
    if cache_ttl is not None:
        self._cache_ttl = cache_ttl
        self._cache.apply_config(ttl=cache_ttl)
```

**New `_execute_with_cache()`:**
```python
async def _execute_with_cache(self, tool_name: str, args: dict[str, Any]) -> ToolCallResult:
    cache_key = self._cache.make_key(tool_name, args)
    if (cached := self._cache.get_result(cache_key)) is not None:
        self.stat_cache_hits += 1
        logger.info("Tool cache hit: %s", tool_name)
        return cached
    result = await self._execute_with_stampede_protection(cache_key, tool_name, args)
    self._cache.store_if_success(cache_key, result)
    return result
```

**Remove `_store_and_evict()`** entirely.

**Update `clear_cache()`:**
```python
def clear_cache(self) -> None:
    self._cache.clear()
```
