# Plan: Remove legacy PipelineStageResult type and standardize on StageResult

Source: `requires/20260625_81_require.md`

---

## Goal

`scripts/rag/types.py` の `PipelineStageResult` legacy 型を削除し、`StageResult`（`scripts/rag/stage.py`）のみを pipeline stage 結果型として使用する。

---

## Scope

**In-Scope**
- `scripts/rag/types.py`: `PipelineStageResult` クラス（line 80）と `__all__` エントリ（line 23）の削除
- `PipelineStageResult` を参照する全コード・テスト・docs の更新

**Out-of-Scope**
- `StageResult` の変更
- `PipelineRunResult` の変更（別の型）

---

## Assumptions

1. `PipelineStageResult` は `types.py` に定義されているが、コード中での直接使用は確認されていない（`pipeline.py` は `PipelineRunResult` を使用）。
2. `grep -rn "PipelineStageResult"` の結果が `types.py` と `__all__` のみであれば、コード側の変更は minimal。
3. docs と tests に `PipelineStageResult` の参照がある可能性がある。
