# Plan: Clean Up Layering Violations in db/maintenance.py

**Requirement:** `requires/20260623_07_require.md`

---

## Goal

Remove `rotate_rag_db()` from `db/maintenance.py` (a session-DB module) and relocate it to `RagMaintenanceService`. Clarify that `vacuum_db()` is generic and does not violate layering.

---

## Current State

**`db/maintenance.py` RAG-specific functions:**

| Function | RAG-specific? | Used by |
|---|---|---|
| `rotate_rag_db()` (line 259) | Yes — directly reads `db_cfg.rag_db_path` | `rotate_db()` internally; `db/__init__.py` export; `tests/test_db_maintenance.py` |
| `vacuum_db(db: SQLiteHelper)` (line 112) | No — generic; takes any `SQLiteHelper` | `db_maintenance_service.py:67` with session DB |
| `_vacuum_db(target: str = "rag")` (line 312) | Dual-purpose — handles rag + session via `target` param; used by `recover_corruption()` | `recover_corruption()` — dual-purpose, stays |

**Verdict on `vacuum_db()`:** Not a layering violation — it's fully generic. The requirement's callout is incorrect. No change needed.

**Actual violation:** `rotate_rag_db()` — hardcodes `rag_db_path`, lives in the session-DB module. Moving it to `RagMaintenanceService` is the right fix.

**Circular dependency risk:** `db/maintenance.py` is lower-layer than `agent/services/`. Moving `rotate_rag_db` to `RagMaintenanceService` and calling `RagMaintenanceService` from `rotate_db()` would create `db → agent` dependency (circular). Solution: keep `_archive_db_file()` in `db/maintenance.py` (it's generic) and have both `rotate_session_db()` and `rotate_rag_db()` in `RagMaintenanceService` call it directly.

## Design

### Change 1: `db/maintenance.py` — remove `rotate_rag_db()`, inline in `rotate_db()`

**Before:**
```python
def rotate_rag_db(archive_dir: str | Path | None = None) -> Path:
    """Archive rag.sqlite to archive_dir with a timestamp suffix; returns the archive path."""
    db_cfg = build_db_config()
    return _archive_db_file(Path(db_cfg.rag_db_path), archive_dir)


def rotate_db(archive_dir: str | Path | None = None) -> tuple[Path, Path]:
    """Archive both rag.sqlite and session.sqlite; returns (rag_archive_path, session_archive_path)."""
    rag_dest = rotate_rag_db(archive_dir)
    ses_dest = rotate_session_db(archive_dir)
    return rag_dest, ses_dest
```

**After:**
```python
def rotate_db(archive_dir: str | Path | None = None) -> tuple[Path, Path]:
    """Archive both rag.sqlite and session.sqlite; returns (rag_archive_path, session_archive_path)."""
    db_cfg = build_db_config()
    rag_dest = _archive_db_file(Path(db_cfg.rag_db_path), archive_dir)
    ses_dest = rotate_session_db(archive_dir)
    return rag_dest, ses_dest
```
