## Assumptions

1. `check_allowed_root()` 実装 (`scripts/agent/tool_policy.py:143-163`):
   ```python
   root = Path(cfg.approval.allowed_root).resolve()
   resolved = Path(val).resolve()
   if not resolved.is_relative_to(root):
       return False
   ```
   相対パスは `Path.resolve()` により CWD を基準に絶対パスへ変換される。

2. 既存テスト `test_relative_path_within_root_is_allowed`:
   - `allowed_root="."`, `path="."` (両方 relative → 両方 CWD に解決) → 許可
   - 絶対 `allowed_root` + 相対 `path` の組み合わせは未テスト

3. 相対 `path` + 絶対 `allowed_root` の挙動:
   - `Path("subdir/file.txt").resolve()` → `CWD/subdir/file.txt`
   - `CWD/subdir/file.txt.is_relative_to(tmp_path)` → tmp_path が CWD でなければ False → 拒否

4. `_make_cfg()` のデフォルト `path_keys = ["path", "file_path", "source", "destination"]`

## Design

### 追加テスト一覧 (TestCheckAllowedRoot.test_source_and_destination_both_checked の後に追加)

**Test 1: `test_relative_path_denied_when_root_is_absolute`**
- `allowed_root=str(tmp_path)` (絶対パス)
- `path="relative/file.txt"` (相対パス → CWD 基準で解決)
- CWD が `tmp_path` でなければ → 拒否

```python
def test_relative_path_denied_when_root_is_absolute(self, tmp_path: Path) -> None:
    cfg = _make_cfg(allowed_root=str(tmp_path))
    # Relative path resolves to CWD/relative/file.txt; CWD != tmp_path → denied
    assert not _check_allowed_root(cfg, "write_file", {"path": "relative/file.txt"})
```

**Test 2: `test_relative_path_allowed_when_cwd_is_root`**
- `allowed_root=str(Path.cwd())` (絶対パス = CWD)
- `path="subdir/file.txt"` (相対パス → CWD/subdir/file.txt)
- CWD/subdir/file.txt は CWD 以下 → 許可

```python
def test_relative_path_allowed_when_cwd_is_root(self) -> None:
    from pathlib import Path

    cwd = Path.cwd()
    cfg = _make_cfg(allowed_root=str(cwd))
    # Relative path resolves to CWD/subdir/file.txt, which is under CWD → allowed
    assert _check_allowed_root(cfg, "write_file", {"path": "subdir/file.txt"})
```

**Test 3: `test_absolute_path_denied_when_root_is_relative_and_path_is_outside`**
- `allowed_root="."` (相対パス → CWD)
- `path="/etc/passwd"` (絶対パス、CWD 外)
- 拒否

```python
def test_absolute_path_denied_when_root_is_relative_and_path_is_outside(self) -> None:
    cfg = _make_cfg(allowed_root=".")
    # /etc/passwd resolves to itself; . resolves to CWD; /etc/passwd not under CWD → denied
    assert not _check_allowed_root(cfg, "write_file", {"path": "/etc/passwd"})
```
