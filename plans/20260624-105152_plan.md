## Design

### Canonical Partial-Completion Model

```
PartialCompletion State:
  - trigger: "sigterm" | "max_turns" | "llm_refusal" | "tool_error"
  - interrupted_at: turn number + tool call index
  - recoverable: bool (SIGTERM = True, max_turns = True, llm_refusal = depends)
  - diagnostics_key: emitted log key for operator correlation

Cross-doc references: all other docs point to 05_agent_03 §Partial-Completion Model
```
