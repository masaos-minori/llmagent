## Design
### Terminology decisions
- **File names:** Always use `crawler.py`, `ingester.py`, `chunk_splitter.py` (actual file names on disk). No legacy aliases in execution docs.

- **Legacy notes:** Only in `03_rag_90_inconsistencies_and_known_issues.md`, DOC-1/DOC-2 entries will note that the old names were resolved by this task.

- DOC-2 (`rag_ingester.py` vs `ingester.py`): Mark as **resolved** — all docs now use `ingester.py`
- SPEC-2 (3 steps vs 4 phases): Mark as **resolved** — standardized on "3 steps"
