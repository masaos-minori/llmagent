# High-Priority Refactoring: backward-compat removal — Work Plan

Source instruction: `instructions/20260604_01_instruction.md`

## Goal

`agent/` の backward-compat レイヤー除去・不要ファイル削除・session.py 分割を実施する。

## Scope

**In:** `config.py`/`context.py` compat 除去、`repl_debug.py`/`context_detection.py`/`rag_debug.py` 削除、`factory.py` グローバル除去、`session.py` 分割
**Out:** `orchestrator.py`/`llm_turn_runner.py` の LLM エラー統一（別タスク）

## Assumptions

1. `repl_debug.py`/`context_detection.py`/`rag_debug.py` はいずれも参照なし（確認済み）。
2. `config.py` の `__getattr__`/`__setattr__` が残存している。
3. `context.py` の backward-compat flat access が残存している。
4. `factory.py` の `_audit_logger_instance` グローバルが残存している。
