## Design

### Turn-State Mutation Table

| Field | When Mutated | Durable? | Notes |
|---|---|---|---|
| `messages` | After each LLM/tool call | Yes (SQLite) | Append-only |
| `current_turn` | At turn start | No (in-memory) | Reset on restart |
| `is_processing` | At turn start / end | No (in-memory) | SIGTERM flag |
| `tool_results` | After each tool call | Session only | Cleared after turn |
| `partial_completion` | On interrupt | Logged only | Not persisted |
