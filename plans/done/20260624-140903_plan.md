## Goal

`DbConfig` に `frozen=True` を追加し、docstring の "Immutable" 宣言をコードが実際に保証するようにする。

## Scope

**In**:
- `scripts/db/config.py` — `@dataclass` → `@dataclass(frozen=True)`

**Out**:
- 全 config DTO の再設計
- 他モジュールへの mutability ガード追加

## Assumptions

1. `DbConfig` フィールドを書き換えるコードは存在しない (grep で確認済み: `helper.py` はすべて読み取りのみ)。
2. `frozen=True` により `__hash__` が自動生成される。`DbConfig` が dict キーや set 要素として使われることはないが、副作用なし。
3. `__post_init__` は `frozen=True` でも動作する (Python の仕様)。
