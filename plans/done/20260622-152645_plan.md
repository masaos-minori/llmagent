## Assumptions

1. `read_text_file()` の実行フロー (`scripts/mcp/file/read_business.py:137-151`):
   ```python
   target = self._resolve_safe(req.path)
   self._require_file(target, req.path)    # FileNotFoundError ここから伝播
   size = self._check_size_limit(target)
   try:
       raw_content = target.read_text(encoding="utf-8")
   except PermissionError as e:
       raise FileAuthorizationError(str(e))  # PermissionError → FileAuthorizationError
   ```

2. `FileNotFoundError` の発生経路:
   - `_require_file()` → `require_file(target, raw_path)` in `common.py:38-43`
   - `if not target.exists(): raise FileNotFoundError(f"File does not exist: {raw_path}")`
   - `FileNotFoundError` は `read_text_file()` 内で catch されないため、そのまま伝播する

3. `PermissionError` の発生経路:
   - `target.read_text(encoding="utf-8")` が `PermissionError` → `FileAuthorizationError` に変換

4. `FileNotFoundError` テスト:
   - パス: `str(tmp_workspace / "ghost.txt")` (allowed dir 内、ファイル未作成)
   - `resolve_safe()` は `strict=False` で解決するため存在しないパスでも `FileAuthorizationError` を送出しない
   - `_require_file()` が `target.exists()` → False → `FileNotFoundError` を送出

5. `PermissionError` テスト:
   - `tmp_workspace / "no_perm.txt"` を作成し `chmod(0o000)` を適用
   - root ユーザーでは `chmod(0o000)` 後も読み取り可能なため、root では pytest.skip

## Design

### 追加テスト一覧 (TestReadTextFile.test_read_text_response_fields の後に追加)

**Test 1: `test_read_text_permission_error_raises_file_authorization_error`**
- `tmp_workspace / "no_perm.txt"` を作成、`chmod(0o000)` で全権限を剥奪
- `read_text_file()` → `target.read_text()` → `PermissionError` → `FileAuthorizationError`
- `try/finally` で `chmod(0o644)` に戻す (pytest の `tmp_path` クリーンアップを正常動作させるため)
- root では `os.getuid() == 0` をチェックしてスキップ

```python
def test_read_text_permission_error_raises_file_authorization_error(self, service):
    import os

    from mcp.file.read_models import ReadTextFileRequest

    svc, tmp_workspace = service
    no_perm = tmp_workspace / "no_perm.txt"
    no_perm.write_text("secret", encoding="utf-8")

    if os.getuid() == 0:
        pytest.skip("root can read files regardless of permissions")

    no_perm.chmod(0o000)
    try:
        with pytest.raises(FileAuthorizationError):
            svc.read_text_file(ReadTextFileRequest(path=str(no_perm)))
    finally:
        no_perm.chmod(0o644)
```

**Test 2: `test_read_text_file_not_found_in_allowed_dir_raises_file_not_found_error`**
- `tmp_workspace / "ghost.txt"` — allowed dir 内だが未作成
- `_resolve_safe()` は succeeded (allowed dir 内)
- `_require_file()` → `target.exists()` → False → `FileNotFoundError`

```python
def test_read_text_file_not_found_in_allowed_dir_raises_file_not_found_error(self, service):
    from mcp.file.read_models import ReadTextFileRequest

    svc, tmp_workspace = service
    with pytest.raises(FileNotFoundError):
        svc.read_text_file(
            ReadTextFileRequest(path=str(tmp_workspace / "ghost.txt"))
        )
```

| Risk | Likelihood | Mitigation |
|---|---|---|
