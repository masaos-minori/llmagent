## Design

### Local vs Production Config Comparison

```toml
# LOCAL-ONLY (development)
sandbox_backend = "none"

# PRODUCTION-REQUIRED
sandbox_backend = "firejail"
firejail_profile = "/opt/llm/config/shell_mcp.firejail"
```

### Startup Failure Condition

`sandbox_backend = "none"` + `production_mode = true` → `RuntimeError: "shell-mcp: sandbox_backend=none is not allowed in production mode"`
