## Design

### New Tool/Server Registration Checklist

| Artifact | Required? | Notes |
|---|---|---|
| `shared/tool_constants.py` — add to frozenset | **Required** | Registry source of truth |
| `config/agent.toml` `tool_names` | Optional | Validation only; routing works without it |
| `config/tools_definitions.toml` | Optional | Strict-mode validation only |
| Server config file (`config/<server>.toml`) | **Required** | Server must be defined |
| `deploy/deploy.sh` — add cp/install step | Required if new server | Deployment must include new server |
| `routing.md` | **Required** | Document guide must reference new server |
| Startup verification: `uv run pytest tests/test_mcp_routing.py` | **Required** | Verify routing is correct |
