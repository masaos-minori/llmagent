# Plan: _assert_allowed_repo 境界値テスト追加 — test_github_mcp_service.py

## Goal

`tests/test_github_mcp_service.py` の `TestAssertAllowedRepo` に、
空文字・スラッシュのみ・複数スラッシュを含む境界条件テストを追加し、
不正な owner/repo 入力が誤って許可されないことを検証する。

**Out-of-Scope:**

## Assumptions

1. `_assert_allowed_repo(owner, repo)` の実装:
   ```python
   slug = f"{owner}/{repo}"
   if slug not in allowed:
       raise GitHubAuthorizationError(...)
   ```
2. 境界入力のスラグ生成:
   - `("", "repo")` → slug = "/repo" (先頭スラッシュ)
   - `("owner", "")` → slug = "owner/" (末尾スラッシュ)
   - `("", "")` → slug = "/" (スラッシュのみ)
   - `("org/sub", "repo")` → slug = "org/sub/repo" (複数スラッシュ)
3. いずれも `allowed_repos=["myorg/myrepo"]` には含まれないため `GitHubAuthorizationError` が送出される。
4. `fail_open` モードでも `allowed_repos` が non-empty の場合、slug がリストにあれば通過、なければ拒否。

## Design

### 追加テスト一覧 (TestAssertAllowedRepo 末尾に追加)

**Test 1: `test_empty_owner_is_denied`**
- `owner=""`, `repo="myrepo"` → slug = "/myrepo"
- `allowed_repos=["myorg/myrepo"]` に存在しない → `GitHubAuthorizationError`

```python
def test_empty_owner_is_denied(self) -> None:
    svc = _make_service({"allowed_repos": ["myorg/myrepo"], "allowed_repos_mode": "fail_open"})
    with pytest.raises(GitHubAuthorizationError):
        svc._assert_allowed_repo("", "myrepo")
```

**Test 2: `test_empty_repo_is_denied`**
- `owner="myorg"`, `repo=""` → slug = "myorg/"
- 拒否される

```python
def test_empty_repo_is_denied(self) -> None:
    svc = _make_service({"allowed_repos": ["myorg/myrepo"], "allowed_repos_mode": "fail_open"})
    with pytest.raises(GitHubAuthorizationError):
        svc._assert_allowed_repo("myorg", "")
```

**Test 3: `test_slash_only_slug_is_denied`**
- `owner=""`, `repo=""` → slug = "/"
- 拒否される

```python
def test_slash_only_slug_is_denied(self) -> None:
    svc = _make_service({"allowed_repos": ["myorg/myrepo"], "allowed_repos_mode": "fail_open"})
    with pytest.raises(GitHubAuthorizationError):
        svc._assert_allowed_repo("", "")
```

**Test 4: `test_owner_with_slash_multiple_parts_is_denied`**
- `owner="org/sub"`, `repo="repo"` → slug = "org/sub/repo" (複数スラッシュ)
- `allowed_repos=["myorg/myrepo"]` に含まれない → 拒否

```python
def test_owner_with_slash_multiple_parts_is_denied(self) -> None:
    svc = _make_service({"allowed_repos": ["myorg/myrepo"], "allowed_repos_mode": "fail_open"})
    with pytest.raises(GitHubAuthorizationError):
        svc._assert_allowed_repo("org/sub", "repo")
```
