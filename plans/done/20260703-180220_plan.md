## Goal

Split `GIT_TOOLS` and `GITHUB_TOOLS` in `shared/tool_constants.py` into read/write/dangerous sub-classifications. Keep aggregate sets as derived unions. Update registry seeding, side-effect detection, and approval policy to use the finer-grained sets.

## Scope

**In-Scope:**
- Add `GIT_READ_TOOLS`, `GIT_WRITE_TOOLS` to `shared/tool_constants.py`; redefine `GIT_TOOLS` as derived union
- Add `GITHUB_READ_TOOLS`, `GITHUB_WRITE_TOOLS`, `GITHUB_DANGEROUS_TOOLS` to `shared/tool_constants.py`; redefine `GITHUB_TOOLS` as derived union
- Update `scripts/shared/tool_registry.py`: seed `"git"` server with `GIT_TOOLS`, `"github"` server with `GITHUB_TOOLS` (no change to behavior — derived unions cover same tool names)
- Update `scripts/shared/tool_executor_helpers.py`: add `GIT_WRITE_TOOLS | GITHUB_WRITE_TOOLS | GITHUB_DANGEROUS_TOOLS` to `_SIDE_EFFECT_TOOLS`
- Update `scripts/agent/tool_policy.py` `classify_risk()` constants-based fallback: add `GITHUB_DANGEROUS_TOOLS` → HIGH, `GIT_WRITE_TOOLS | GITHUB_WRITE_TOOLS` → MEDIUM, `GIT_READ_TOOLS | GITHUB_READ_TOOLS` → NONE
- Add tests: aggregate set equality, registry population, approval classification for representative tools

**Out-of-Scope:**
- Adding new Git/GitHub tool names
- Changing tool server ownership

## Pre-Analysis: Proposed Split

### `classify_risk()` constants-based fallback (priority 3) extension

Building on require-35 plan, add after delete/shell checks:
```python
elif tool_name in GITHUB_DANGEROUS_TOOLS:
    base = RiskLevel.HIGH

    base = RiskLevel.MEDIUM
```

Note: This depends on require-35 being implemented first (which adds the constants-fallback block). If applied in order, these lines are appended to that block.

## Assumptions

5. `_special_case_risk()` detections for gitops push patterns still take priority (they fire before base is returned).

## Design

### `tool_constants.py` additions

```python
# Git sub-classifications
GIT_READ_TOOLS: frozenset[str] = frozenset({
    "git_status",
    "git_log",
    "git_diff",
    "git_branch",
    "git_show",
})

GIT_WRITE_TOOLS: frozenset[str] = frozenset({
    "git_add",
    "git_commit",
    "git_checkout",
    "git_pull",
    "git_push",
})

GIT_TOOLS: frozenset[str] = GIT_READ_TOOLS | GIT_WRITE_TOOLS  # derived union

# GitHub sub-classifications
GITHUB_READ_TOOLS: frozenset[str] = frozenset({
    "github_search_repositories", "github_list_branches",
    "github_list_commits", "github_get_commit",
    "github_search_code", "github_get_file_contents",
    "github_list_issues", "github_get_issue", "github_search_issues",
    "github_list_pull_requests", "github_get_pull_request", "github_search_pull_requests",
})

GITHUB_WRITE_TOOLS: frozenset[str] = frozenset({
    "github_create_branch",
    "github_create_or_update_file", "github_push_files",
    "github_create_issue", "github_add_issue_comment",
    "github_create_pull_request", "github_update_pull_request",
})

GITHUB_DANGEROUS_TOOLS: frozenset[str] = frozenset({
    "github_delete_file",
    "github_merge_pull_request",
})

GITHUB_TOOLS: frozenset[str] = GITHUB_READ_TOOLS | GITHUB_WRITE_TOOLS | GITHUB_DANGEROUS_TOOLS  # derived union
```
