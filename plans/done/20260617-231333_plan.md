# Plan: Make logger file-write fallback visible (req 73)

## Goal

Replace the silent `sys.stderr.write()` fallback in `Logger._configure_logger()` with a proper logging.WARNING-level message so operators notice when file logging fails.

## Scope

**In scope:**
- `shared/logger.py` — change line 117-120 from `sys.stderr.write(...)` to emit a `logging.WARNING` via the logger instance
- `tests/test_logger.py` — add test for file-write failure producing a visible warning
- `docs/06_shared_03_runtime_and_execution.md` — document fallback behavior in Logger API reference

**Out of scope:**
- No changes to `_ContextFilter`, `_JsonFormatter`, or other Logger methods
- No changes to structured_log mode behavior

## Assumptions

1. The existing `sys.stderr.write()` at line 117-120 is the only place where file logging failure is handled.
2. At the point of failure (line 113-115), `self._logger` exists but may not have any handlers yet (FileHandler open failed).
3. The stream handler (stderr) is added AFTER the FileHandler block, so we can't rely on it being present during the error log.
4. Using `logging.warning()` directly (not via `self._logger`) ensures the message reaches stderr regardless of handler state.

## Design

### Current state

```python
# logger.py line 112-123
try:
    fh = logging.FileHandler(log_file)
    fh.setFormatter(formatter)
    self._logger.addHandler(fh)
except OSError as exc:
    sys.stderr.write(
        f"[Logger] Cannot open log file {log_file}: {exc}"
        " — falling back to stderr only.\n",
    )
sh = logging.StreamHandler(sys.stderr)
sh.setFormatter(formatter)
self._logger.addHandler(sh)
```

### Proposed design

Add a module-level fallback logger that always has a StreamHandler:

```python
# After imports (around line 20), add:
_fallback_logger = logging.getLogger("shared.logger.fallback")
if not _fallback_logger.handlers:
    _fallback_logger.addHandler(logging.StreamHandler(sys.stderr))
    _fallback_logger.setLevel(logging.WARNING)
```

Replace `sys.stderr.write()` in `_configure_logger()`:

```python
except OSError as exc:
    _fallback_logger.warning(
        "Cannot open log file %s: %s — falling back to stream handler only",
        log_file,
        exc,
    )
```
