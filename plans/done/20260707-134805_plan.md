# Plan: H-3 — Add shutdown guard to ServerLifecycleRouter

## Goal

Add a `_shutting_down` boolean flag to `_ServerLifecycleRouter` that is set to `True` at the start of `shutdown_all()` and causes `ensure_ready()`, `start_http_subprocess()`, `restart()`, and `shutdown_idle()` to return immediately without starting or restarting any subprocess.

## Scope

**In**: `scripts/agent/factory.py` — `_ServerLifecycleRouter.__init__()`, `shutdown_all()`, `ensure_ready()`, `start_http_subprocess()`, `restart()`, `shutdown_idle()`.

**Out**: No changes to `http_lifecycle.py`, `context.py`, `LifecycleManagerProtocol`, config, or DB schema.

## Assumptions

1. All start/restart entry points to HTTP subprocesses go through `_ServerLifecycleRouter`; `HttpServerLifecycleManager` is never called directly from other modules.
2. The guard must be idempotent: repeated `shutdown_all()` calls do not cause errors.
3. A debug/warning log on ignored calls is sufficient for observability; no exception is raised.

## Implementation Steps

1. **Add `_shutting_down` to `__init__()`**
   - Add `self._shutting_down: bool = False` to `_ServerLifecycleRouter.__init__()`.
   - Criterion: field exists; mypy passes.

2. **Set guard in `shutdown_all()`**
   - Set `self._shutting_down = True` as the first statement in `shutdown_all()`.
   - Criterion: subsequent calls are idempotent (setting True again is safe).

3. **Guard `ensure_ready()`, `start_http_subprocess()`, `restart()`, `shutdown_idle()`**
   - At the top of each method: `if self._shutting_down: _logger.debug("Lifecycle: %s ignored — shutting down", "<method_name>"); return`.
   - `restart()` uses `_logger.warning` (higher severity for ignored restart).
   - Criterion: none of these methods calls `_http_mgr` methods after `_shutting_down` is True.

4. **Add tests**
   - Test: after `shutdown_all()`, `restart(server_key)` does not call `_http_mgr.restart()`.
   - Test: after `shutdown_all()`, `ensure_ready(server_key)` does not call `_http_mgr.start()`.
   - Test: repeated `shutdown_all()` calls do not raise.
   - Test: ignored `restart()` emits a warning log.
   - Criterion: `uv run pytest tests/test_agent_factory.py -v` passes.
