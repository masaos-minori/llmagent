## Assumptions

1. `circuit_reset_sec` 秒が経過すると `_is_circuit_open()` が False を返し、
   `_fail_count=0`・`_circuit_opened_at=None` に auto-reset される (half-open を経ず直接 closed へ)。
2. テストで時間経過を模擬するには `client._circuit_opened_at` を
   `time.monotonic() - (circuit_reset_sec + 1)` に直接設定するか、`time.monotonic` をパッチする。
   後者は副作用が広いため、前者 (直接代入) を採用する。
3. `circuit_open_after=2` で2回失敗後に circuit が open する (既存テストと同設定)。
4. `_fail_count` は成功時に 0 にリセットされる (`if result.success: self._fail_count = 0`)。

## Design

### EmbeddingClient circuit breaker 実装詳細

```
_is_circuit_open():
  _circuit_opened_at is None → False (closed)
  elapsed >= circuit_reset_sec → _fail_count=0, _circuit_opened_at=None, return False  (auto-reset)
  else → True (open)

_record_failure():
  _fail_count += 1
  if _fail_count >= circuit_open_after → _circuit_opened_at = time.monotonic()

fetch():
  if _is_circuit_open() → return EmbeddingResult(error_kind=CIRCUIT_OPEN)
  ...attempt HTTP...
  if result.success → _fail_count = 0; return
  _record_failure()
  if _is_circuit_open() → return EmbeddingResult(error_kind=CIRCUIT_OPEN)
```

### 追加するテストケース

**1. `test_circuit_auto_resets_after_reset_sec`**
- circuit_open_after=2 で 2 回失敗 → circuit open
- `client._circuit_opened_at -= (circuit_reset_sec + 1)` で時間経過を模擬
- `client._is_circuit_open()` → False
- `client._circuit_opened_at is None` かつ `client._fail_count == 0`

**2. `test_half_open_fetch_succeeds_closes_circuit`**
- circuit_open_after=2 で 2 回失敗 → circuit open
- `_circuit_opened_at` を過去に設定
- `fetch()` を成功モックで呼ぶ → `result.success is True`
- `client._circuit_opened_at is None` かつ `client._fail_count == 0` (closed 状態)

**3. `test_half_open_fetch_fails_reopens_circuit`**
- circuit_open_after=1 で 1 回失敗 → circuit open
- `_circuit_opened_at` を過去に設定 (auto-reset)
- `fetch()` を失敗モックで呼ぶ → `fail_count=1 >= circuit_open_after=1` → 再 open
- `result.error_kind == "circuit_open"` (失敗後に即 open)

**4. `test_success_resets_fail_count`**
- circuit_open_after=5 で 3 回失敗 (circuit はまだ closed)
- `fetch()` を成功モックで呼ぶ → `_fail_count = 0`
- 再び 4 回失敗しても circuit が open しないことを確認
