# Plan: Remove `/ingest` command registration from CommandRegistry

## Goal
`/ingest` が `AgentREPL.SLASH_COMMANDS`（tab補完リスト）から除去され、REPLからこのコマンドが完全に参照されない状態になる。

## Scope
**In-Scope**:
- `scripts/agent/repl.py` の `SLASH_COMMANDS` リストから `/ingest` エントリを削除

**Out-of-Scope**:
- `_IngestMixin` の削除（`/export`, `/compact`, `/rag` を保持するため）
- `cmd_rag_export.py` への変更
- RAG ingestion pipeline スクリプト (`scripts/rag/ingestion/`) への変更
- ドキュメントの変更

## Assumptions
- `registry.py` の `_COMMANDS` から `/ingest` は既に除去済みであり、本タスクはその作業を完了させるもの
- `registry.py` の module docstring に `_IngestMixin` への言及はなく、変更不要
- `AgentREPL.SLASH_COMMANDS` は `CLIView` の tab補完にのみ使用され、dispatch には関与しない
- `test_cmd_registry_ingest_removal.py` は既に存在し全テストがパスしており、追加のテスト実装は不要
