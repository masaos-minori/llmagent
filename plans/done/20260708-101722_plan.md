# Plan: M-3 — Update /context partial completion display to reference session_diagnostics

## Goal

`/context` コマンドの partial completion 表示から `tool_result(tool_name='llm_partial_completion')` への参照を除去し、`session_diagnostics` への参照に置き換える。

## Scope

- **変更対象**: `scripts/agent/commands/cmd_context.py` のみ (`_cmd_context()` メソッド内 1 箇所)
- **変更なし**: `context_view.py`, `services/models.py`, テスト以外のファイル

## Assumptions

- `stat_partial_completions` カウンタ自体は `LLMClient` に引き続き存在し、`context_view.py` 経由で `state.partial_completions` として取得できる。
- partial completion の永続化先が `session_diagnostics` テーブルに移行済みであることを前提とする（`ToolResultStore` 廃止後の状態）。
- `/stats` (`cmd_config_stats.py`) にも同様の `tool_result` 参照 (行 119) があるが、本 require のスコープ外。
