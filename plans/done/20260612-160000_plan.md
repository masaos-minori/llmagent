## Goal

## Assumptions

1. `server.py` line 188: `except Exception` は stdio トランスポートの最終安全網。上位で `orjson.JSONDecodeError`, `(ValueError, OSError)`, `RuntimeError` が catch 済み。残余は `TypeError`, `AttributeError`, `KeyError` 等。`(TypeError, AttributeError, KeyError)` に絞る。

2. `server.py` line 162: `name = str(req.get("name", ""))` — `req` は `orjson.loads(line)` の戻り値 (外部 JSON 境界)。`name` は `str` であるべきだが、クライアントが不正な型を送った場合の対処が必要。`ValueError` raise → 上位の `(ValueError, OSError)` catch に委ねる。

3. `from_dict()` の `str(d.get("key", ""))` パターン — TOML は文字列フィールドを文字列として返すが、`isinstance` チェックなしのキャストは制約違反。TOML 境界でも `isinstance` check + `ValueError` raise に変換する。

## Design

### `from_dict()` パターンの統一変換

```python
# Before (unconditional str() coercion)
auth_token=str(d.get("auth_token", "")),

# After (validated extraction with helper)
def _require_str(d: dict[str, Any], key: str, default: str = "") -> str:
    v = d.get(key, default)
    if not isinstance(v, str):
        raise ValueError(f"Config key {key!r} must be str, got {type(v).__name__}")
    return v
```

この helper を各 `models.py` に追加し、`str(d.get(..., ""))` を置き換える。または helper を `mcp/models.py` に定義して共有する。
### `server.py` の str(req.get("name")) 修正

```python
# Before
name = str(req.get("name", ""))

# After
name_raw = req.get("name", "")
if not isinstance(name_raw, str):
    raise ValueError(f"Request 'name' must be str, got {type(name_raw).__name__}")
name = name_raw
```
