# Plan: H-6 — Remove tool_results undone marking from undo service

## Goal

`undo_last_turn()` から `ToolResultStore.mark_turn_undone()` の呼び出しを除去する。
`/undo` はconversation historyとpersisted messagesのロールバックのみを行う。

## Scope

Path A (<=3 files, no interface/schema change):
- `scripts/agent/services/undo_service.py` — `mark_turn_undone` 呼び出しと `turn_to_mark` を除去
- `scripts/agent/services/models.py` — `UndoResult.n_artifacts_marked` をdeprecated always-zero fieldとして残すか削除する

テスト更新:
- `tests/test_undo_artifact_consistency.py` — 既存テストはすべて削除またはリライト

## Assumptions

- `UndoResult.n_artifacts_marked` は `cmd_context.py` で参照されていない(確認済み: line 127で`result.n_removed`のみ使用)
- `turn_to_mark` は `mark_turn_undone` 呼び出しのためだけに存在するローカル変数であり、除去後は不要になる
- `tool_results.undone` カラム自体の削除は本issueのスコープ外(別のH-x issueで対応)
- `ToolResultStore` クラス自体は他の用途(store/list_recent/get)で引き続き使用されるため削除しない

## Unknowns (resolved)

1. **`n_artifacts_marked` の呼び出し元**: `cmd_context.py` の `_cmd_undo` では `result.n_removed` のみを表示文字列に使用。`n_artifacts_marked` は参照なし。=> フィールドを削除しても外部破壊なし。ただし要件は「always-zero fieldとして残すか削除するか決める」としているため、削除方針を採用する。
2. **`turn_to_mark` の使用箇所**: `undo_service.py` 内の`if turn_to_mark > 0` ブロックのみ。除去後は変数不要。
3. **テストの扱い**: `test_undo_artifact_consistency.py` は `n_artifacts_marked` の動作を検証しているため、全テストが無効化される。テストファイル全体を削除する(behavior自体がなくなるため)。

## Affected Areas

| `scripts/agent/services/models.py` | `UndoResult.n_artifacts_marked` フィールドを削除 |

## Design

### undo_service.py 変更後のロジック

```
def undo_last_turn(ctx: AgentContext) -> UndoResult:
    # 1. historyから最後のuserメッセージindexを探す
    # 2. メモリinjectionマーカーをさかのぼってcut_idxを決める
    # 3. ctx.conv.history をトリム
    # 4. ctx.stats.stat_turns をデクリメント
    # 5. ctx.session.undo_last_turn() を呼ぶ
    # 6. UndoResult(n_removed=removed) を返す
    # ToolResultStore呼び出しなし
```

- `turn_to_mark = ctx.stats.stat_turns` の行を削除
- `n_marked = 0` の行を削除
- `if turn_to_mark > 0 and ctx.tool_result_store is not None:` ブロック全体を削除
- `return UndoResult(n_removed=removed, n_artifacts_marked=n_marked)` を `return UndoResult(n_removed=removed)` に変更

### models.py 変更後のUndoResult

```python
@dataclass(frozen=True)
class UndoResult:
    n_removed: int
```

`n_artifacts_marked` フィールドを削除。
