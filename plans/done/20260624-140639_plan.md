## Goal

`SQLiteMemoryDeleteStore.delete_memories_before()` の多テーブル削除を `begin_immediate()` で明示的なトランザクション境界に包み、原子性をコードから自明にする。

## Scope

**In**:
- `scripts/db/store_impl.py` — `delete_memories_before()` の DELETE ブロックを `with self._db.begin_immediate():` で囲み、`self._db.commit()` を削除

**Out**:
- メモリ保持ポリシーの変更
- `SQLiteMemoryDeleteStore` の置換
- `helper.py` の変更 (`begin_immediate` は既に実装済み)

## Assumptions

1. `begin_immediate()` は `agent/memory/store.py` と `agent/workflow/state_store.py` で実績があるパターン。
2. `SQLiteMemoryDeleteStore.__init__` に渡される `db` は `maintenance.py` で `open(write_mode=True)` 済みであるため、`begin_immediate()` が呼ばれる時点で `conn` は非 None。
3. `begin_immediate()` が `COMMIT` を発行するため、内部で `self._db.commit()` を呼ぶ必要はない。
4. `fetchall(SELECT ...)` は DML でないため Python の sqlite3 自動トランザクションを開始しない。`BEGIN IMMEDIATE` はその後なので競合なし。

## Design

### 変更前

```python
def delete_memories_before(self, older_than_days: int) -> MemoryDeleteResult:
    rows = self._db.fetchall(...)
    if not rows:
        return MemoryDeleteResult(deleted=0)
    mids = [row[0] for row in rows]
    placeholders = ",".join("?" * len(mids))
    cur = self._db.execute(f"DELETE FROM memories WHERE memory_id IN ({placeholders})", tuple(mids))
    deleted = cur.rowcount
    for mid in mids:
        self._db.execute("DELETE FROM memories_fts WHERE memory_id=?", (mid,))
    for mid in mids:
        self._db.execute("DELETE FROM memories_vec WHERE memory_id=?", (mid,))
    self._db.commit()
    return MemoryDeleteResult(deleted=deleted)
```

### 変更後

```python
def delete_memories_before(self, older_than_days: int) -> MemoryDeleteResult:
    rows = self._db.fetchall(...)
    if not rows:
        return MemoryDeleteResult(deleted=0)
    mids = [row[0] for row in rows]
    placeholders = ",".join("?" * len(mids))
    with self._db.begin_immediate():
        cur = self._db.execute(f"DELETE FROM memories WHERE memory_id IN ({placeholders})", tuple(mids))
        deleted = cur.rowcount
        for mid in mids:
            self._db.execute("DELETE FROM memories_fts WHERE memory_id=?", (mid,))
        for mid in mids:
            self._db.execute("DELETE FROM memories_vec WHERE memory_id=?", (mid,))
    return MemoryDeleteResult(deleted=deleted)
```
