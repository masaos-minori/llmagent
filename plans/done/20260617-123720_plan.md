## Goal

## Scope

### In scope
- `scripts/rag/ingestion/ingester.py`: Replace `_read_chunk_json()` implementation; fix `_embed_and_store()` to use real values

## Design

### New `_read_chunk_json()` implementation
```python
def _read_chunk_json(self, path: Path) -> "dict[str, Any] | None":
    """Read and parse a chunk JSON file as raw dict; returns None on failure."""
    import orjson
    try:
        raw = path.read_bytes()
    except OSError as e:
        logger.warning("skip chunk %s: cannot read: %s", path.name, e)
        return None
    try:
        data = orjson.loads(raw)
    except orjson.JSONDecodeError as e:
        logger.warning("skip chunk %s: JSON parse error: %s", path.name, e)
        return None
    if not isinstance(data, dict):
        logger.warning("skip chunk %s: expected JSON object, got %s", path.name, type(data).__name__)
        return None
    return data
```

### Fixed `_embed_and_store()`
