# Plan: Update `/mcp` command definition and help text

## Goal
`registry.py` の `/mcp` CommandDef description が `install` を含まないことをテストで保証し、`enums.py` の未使用 `McpAction.INSTALL` を除去することで、`/mcp install` に関する残骸コードをすべて排除する。

## Scope
**In-Scope**:
- `enums.py` から未使用の `McpAction.INSTALL` を削除
- `registry.py` の `/mcp` description に `install` が含まれないことを検証するテストを追加
- `registry.py` の `/mcp` が prefix async command として登録されていることを検証するテストを追加

**Out-of-Scope**:
- `cmd_mcp.py` の変更 (install dispatch は既に削除済み)
- `registry.py` の description 変更 (既に要件通りの文言)
- `/help` の出力形式の変更

## Assumptions
- `registry.py` の `/mcp` description は既に `"MCP server status, tool list, connectivity check"` であり変更不要
- `McpAction.INSTALL` はどこからも import されておらず、削除してもランタイムに影響しない
- `test_cmd_mcp.py` の `test_invalid_subcommand_shows_usage` は `install` が `UnknownSubcommandError` になることを既に検証しているが、description レベルの検証は行っていない
