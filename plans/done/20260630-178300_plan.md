# Plan: Add validation for `k` in `SQLiteVectorStore.vec_search()`

## Goal
`SQLiteVectorStore.vec_search()` が `k < 1` の場合に DB クエリを実行せず即座に `[]` を返すこと、およびその振る舞いがテストで保証されていること。

- embedding バリデーション (`validate_embedding_blob`) の変更
- `VectorStore` プロトコル定義 (`db/store_protocols.py`) の変更
- `vec_insert`, `vec_delete`, `vec_count` の変更
- DB スキーマ変更
