# Plan: Remove or isolate `agent/services/mcp_install.py`

## Goal
`agent/services/mcp_install.py` に由来する残留シンボル (`McpInstallRequest`, `McpInstallRenderModel`, `InstallIOPort`, `StatusRenderPort`, `McpAction.INSTALL`) をすべて削除し、Agent ランタイムに install-wizard 関連コードが一切存在しない状態にする。

## Scope
**In-Scope**:
- `scripts/agent/commands/models.py` から `McpInstallRequest`, `McpInstallRenderModel` を削除
- `scripts/agent/services/io_ports.py` から `InstallIOPort`, `StatusRenderPort` を削除
- `scripts/agent/commands/enums.py` から `McpAction.INSTALL` を削除
- 上記削除後の import / re-export の整理

**Out-of-Scope**:
- `scripts/agent/services/mcp_install.py` 本体の削除 (すでに存在しない)
- `/mcp install` dispatch の削除 (cmd_mcp.py では既に install を UnknownSubcommandError にしている)
- ドキュメント変更 (`docs/04_mcp_00_document-guide.md` は削除済みの記録を正しく持つ)
- テスト追加 (参照なしシンボルのため、既存テストへの影響なし)
