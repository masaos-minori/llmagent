## Assumptions

1. `MEMORY_TYPES` は `MemoryType.value` の frozenset で、`memory_type not in MEMORY_TYPES`
   は `MemoryType(memory_type)` が `ValueError` を raise するか確認するのと等価。
   `jsonl_store.py` での置換: `MemoryType(memory_type)` を try/except ValueError で
   `JsonlFormatError` に変換。
2. `_opt_str` の `str(v)` は DB から返る TEXT 列に使うが、None 以外で str 以外が
   来ることは実装上ないはず。それでも `isinstance` チェックで制約を明示する。
3. `str(_require(d, "memory_id"))` 等 — DB から TEXT NOT NULL で返るが、
   `isinstance` で検証後 as-is で使う方が安全。
4. `tests/test_memory_types.py` の `MEMORY_TYPES` テストは `MemoryType` enum テストで
   置き換え可能。

## Design

### `types.py` — MEMORY_TYPES 削除

```python
# Before
# Derived from MemoryType enum for backward compatibility
MEMORY_TYPES: frozenset[str] = frozenset(m.value for m in MemoryType)

# After — 削除
```

### `jsonl_store.py` — MemoryType enum 直接使用

```python
# Before
from agent.memory.types import MEMORY_TYPES, MemoryEntry
...
if memory_type not in MEMORY_TYPES:
    raise JsonlFormatError(f"Invalid memory_type={memory_type!r}")

# After
from agent.memory.enums import MemoryType
from agent.memory.types import MemoryEntry
...
try:
    MemoryType(memory_type)
except ValueError:
    raise JsonlFormatError(f"Invalid memory_type={memory_type!r}") from None
```

### `mapper.py` — _opt_str の str() 除去

```python
# Before
def _opt_str(d: dict[str, Any], key: str) -> str:
    v = d.get(key)
    return str(v) if v is not None else ""

# After
def _opt_str(d: dict[str, Any], key: str) -> str:
    v = d.get(key)
    if v is None:
        return ""
    if not isinstance(v, str):
        raise MemorySchemaError(
            f"Field {key!r} must be str or None, got {type(v).__name__}"
        )
    return v
```

### `mapper.py` — `str(_require(...))` → isinstance 検証

```python
# Before
memory_id=str(_require(d, "memory_id")),
content=str(_require(d, "content")),
# (memory_type と source_type は StrEnum constructor への引数なので str() は維持)

# After
_mid = _require(d, "memory_id")
if not isinstance(_mid, str):
    raise MemorySchemaError(f"memory_id must be str, got {type(_mid).__name__}")
memory_id=_mid,

_content = _require(d, "content")
if not isinstance(_content, str):
    raise MemorySchemaError(f"content must be str, got {type(_content).__name__}")
content=_content,
```

Note: `str(_require(d, "memory_type"))` and `str(_require(d, "source_type"))` are
passed to `MemoryType()` and `SourceType()` constructors which accept str — keep
`str()` cast here as the enum constructor handles validation.
