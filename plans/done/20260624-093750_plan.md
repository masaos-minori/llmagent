## Design Decision

**Hooks remain separate from `PipelineStage`** (they accept `list[RagHit]` not `PipelineContext`), but are formalized as a typed `Protocol`:

```python
from typing import Protocol, runtime_checkable

@runtime_checkable
class PipelineHook(Protocol):
    """Post-rerank pipeline hook contract."""
    async def __call__(self, hits: list[RagHit], query: str) -> list[RagHit]: ...
```

- `strict=True`: exception propagates, pipeline fails
- `strict=False` (default): exception logged as warning, original hits returned unchanged
- Input: `list[RagHit]` + `str` (query)
- Output: `list[RagHit]` (may be modified or filtered)
