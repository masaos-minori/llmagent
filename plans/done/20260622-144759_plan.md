# Plan: 環境依存ハードコードの改善 — test_lifecycle.py / test_db_maintenance.py

## Goal

`tests/test_lifecycle.py` のハードコード URL を定数またはヘルパー関数に集約し、
`tests/test_db_maintenance.py` のハードコード URL `http://127.0.0.1:8081/embedding` を
`_make_db_cfg()` ヘルパーの引数化またはテスト定数に移動する。

## Scope

**In-Scope:**
- `tests/test_lifecycle.py` — `http://127.0.0.1:9999` 等の URL を `_http_subprocess_cfg` デフォルト値に集約
- `tests/test_db_maintenance.py` — `http://127.0.0.1:8081/embedding` を定数化

**Out-of-Scope:**
- 実装コードの変更
- テストロジックの変更
- 実際の HTTP サービスへの接続を要求するテスト (mock ベースのため不要)

## Assumptions

1. `test_lifecycle.py` の `http://127.0.0.1:9999`/`9998`/`9997`/`9996` は
   mock が HTTP 呼び出しを横断するため実際には接続しない。
   ポート番号は任意値だが、モジュールレベルの定数または `_http_subprocess_cfg` の
   デフォルト URL 変更で一元管理できる。
2. `test_db_maintenance.py` の `http://127.0.0.1:8081/embedding` は `_make_db_cfg()`
   内でハードコードされているが、テストは実際には embed_url を HTTP アクセスしない。
3. `tmp_path` は既に test_db_maintenance.py の全テストで使用されており、`/tmp` の
   ハードコードは現在存在しない (要件の誤認の可能性があるが、embed_url は改善余地あり)。

## Affected Areas

| File | 現状のハードコード | 改善方法 |
|---|---|---|
| `tests/test_lifecycle.py` | `http://127.0.0.1:9999` 等、各テストに散在 (10 箇所) | `_TEST_HTTP_URL = "http://127.0.0.1:9999"` 定数を定義 |
| `tests/test_db_maintenance.py:53` | `http://127.0.0.1:8081/embedding` in `_make_db_cfg()` | `_TEST_EMBED_URL = "http://127.0.0.1:8081/embedding"` 定数を定義 |

## Design

### test_lifecycle.py の改善

```python
# module-level constants
_TEST_HTTP_URL = "http://127.0.0.1:9999"
_TEST_HTTP_AUTH_URL = "http://127.0.0.1:8000"
```

各テストで `url="http://127.0.0.1:9999"` の箇所を `url=_TEST_HTTP_URL` に変更。
衝突回避のため異なるポートを使うテスト (9998, 9997, 9996) はそのまま個別定数化。

### test_db_maintenance.py の改善

```python
_TEST_EMBED_URL = "http://127.0.0.1:8081/embedding"

def _make_db_cfg(tmp_path, ...):
    ...
    cfg.embed_url = _TEST_EMBED_URL
    ...
```

## Implementation Steps

### Phase 1: test_lifecycle.py の定数化

- [ ] `_TEST_HTTP_URL = "http://127.0.0.1:9999"` をモジュールレベルに定義
- [ ] `TestStartHttpSubprocess` 内の `url="http://127.0.0.1:9999"` をすべて `_TEST_HTTP_URL` に変更
- [ ] `_http_subprocess_cfg` のデフォルト URL も `_TEST_HTTP_URL` を参照するよう更新

### Phase 2: test_db_maintenance.py の定数化

- [ ] `_TEST_EMBED_URL = "http://127.0.0.1:8081/embedding"` を定義
- [ ] `_make_db_cfg()` 内の `cfg.embed_url = "http://127.0.0.1:8081/embedding"` を定数参照に変更
