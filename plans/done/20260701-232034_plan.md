# Work Plan: Add eventbus_db_path to DbConfig, common.toml, and SQLiteHelper

## Goal

Add `eventbus_db_path` as a first-class DB path managed by `DbConfig` and support `SQLiteHelper("eventbus")`.

## Scope

- DB config (DbConfig dataclass)
- SQLite helper target support (DbTarget enum, SQLiteHelper.__init__)
- Default config (common.toml)
- Tests
- Documentation

Do not implement Event Bus runtime integration in this issue.

## Assumptions

1. Event Bus does NOT need sqlite-vec extension loaded (only RAG needs it)
2. The default path `/opt/llm/db/eventbus.sqlite` matches the existing convention
3. `sqlite_timeout`, `sqlite_busy_timeout_ms`, and other shared settings apply to all DBs
4. No special connection pragmas needed for eventbus beyond the defaults

## Design

### DbConfig changes

```python
@dataclass(frozen=True)
class DbConfig:
    """Immutable configuration for the SQLite database."""

    rag_db_path: str
    session_db_path: str
    workflow_db_path: str = "/opt/llm/db/workflow.sqlite"
    eventbus_db_path: str = "/opt/llm/db/eventbus.sqlite"  # NEW
    sqlite_vec_so: str = ""
    sqlite_timeout: int = 30
    sqlite_busy_timeout_ms: int = 30000
    embedding_dims: int = 384
```

### DbConfig.__post_init__ changes

Add eventbus_db_path to the parent directory validation loop:

```python
for label, path_str in (
    ("rag_db_path", self.rag_db_path),
    ("session_db_path", self.session_db_path),
    ("workflow_db_path", self.workflow_db_path),
    ("eventbus_db_path", self.eventbus_db_path),  # NEW
):
    parent = Path(path_str).parent
    if not parent.exists():
        raise ValueError(f"{label} parent directory does not exist: {parent}")
```

### build_db_config() changes

Add eventbus_db_path to the builder:

```python
def build_db_config() -> DbConfig:
    cfg = ConfigLoader().load("common.toml")
    return DbConfig(
        rag_db_path=cfg.get("rag_db_path", ""),
        session_db_path=cfg.get("session_db_path", ""),
        workflow_db_path=cfg.get("workflow_db_path", "/opt/llm/db/workflow.sqlite"),
        eventbus_db_path=cfg.get("eventbus_db_path", "/opt/llm/db/eventbus.sqlite"),  # NEW
        sqlite_vec_so=cfg.get("sqlite_vec_so", ""),
        sqlite_timeout=int(cfg.get("sqlite_timeout", 30)),
        sqlite_busy_timeout_ms=int(cfg.get("sqlite_busy_timeout_ms", 30000)),
        embedding_dims=int(cfg.get("embedding_dims", 384)),
    )
```

### DbTarget changes


### SQLiteHelper.__init__ changes

Add eventbus path resolution:

```python
if resolved == "rag":
    self._db_path = db_cfg.rag_db_path
elif resolved == "session":
    self._db_path = db_cfg.session_db_path
elif resolved == "workflow":
    self._db_path = db_cfg.workflow_db_path
elif resolved == "eventbus":  # NEW
    self._db_path = db_cfg.eventbus_db_path
else:
    raise ValueError(f"unknown target: {resolved!r}")
```

Add eventbus to the default_load_vec logic (eventbus should NOT load vec):

```python
# Change from:

self._default_load_vec = resolved in ("rag",)  # Only rag loads vec by default
```
