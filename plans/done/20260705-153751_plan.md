## Goal

Add value validation to `McpServerConfig` so that invalid timeout values, empty/duplicate tool names, non-string env, and invalid HTTP URLs fail at config construction rather than at runtime.

## Scope

**In**: Extend `McpServerConfig.__post_init__()` to validate 7 new constraints: timeout ranges, tool name validity, duplicate tool names, auth_token type, env dict types, HTTP URL scheme.

**Out**: Adding non-HTTP transports. Changing config file layout. Documentation-only updates.

## Assumptions

1. `McpServerConfig.__post_init__()` at `mcp_config.py:68` calls `_validate_enum_types()` and `_validate_cross_fields()`.
2. Current `_validate_cross_fields()` only checks: `http + no url ã error` and `subprocess + no cmd ã error`.
3. `call_timeout_sec: float = 60.0` ã `0` means no timeout (valid); negative is invalid.
4. `startup_timeout_sec: int = 30` ã zero or negative is invalid.
5. `tool_names: list[str]` ã each entry must be a non-empty string; no duplicates.
6. `auth_token: str = ""` ã empty is valid (auth disabled); non-string is invalid.
7. `env: dict[str, str]` ã keys and values must all be strings.
8. For HTTP URL validation, check: has `http://` or `https://` scheme and non-empty netloc.
9. Error messages must include the server key (passed from `_build_single_server()`).

## Design

**Key insight**: `McpServerConfig` has no `key` field. The server key is in `_build_single_server(key, v)`. Two options:
- Option A: Add optional `key: str = ""` field to dataclass; set in `_build_single_server()`.
- Option B: Validate in `_build_single_server()` after construction with key context.

**Recommended: Option A** ã add `key: str = field(default="", compare=False)` to include key in error messages.

**New validation additions to `_validate_cross_fields()`:**
```python
def _validate_cross_fields(self) -> None:
    key_prefix = f"McpServerConfig[{self.key!r}]" if self.key else "McpServerConfig"
    ...
    if self.call_timeout_sec < 0:
        raise ValueError(f"{key_prefix}: call_timeout_sec must be >= 0, got {self.call_timeout_sec}")
    if self.startup_timeout_sec <= 0:
        raise ValueError(f"{key_prefix}: startup_timeout_sec must be > 0, got {self.startup_timeout_sec}")
    # tool_names validation
    for i, name in enumerate(self.tool_names):
        if not isinstance(name, str) or not name:
            raise ValueError(f"{key_prefix}: tool_names[{i}] must be a non-empty string, got {name!r}")
    if len(self.tool_names) != len(set(self.tool_names)):
        dupes = [n for n in self.tool_names if self.tool_names.count(n) > 1]
        raise ValueError(f"{key_prefix}: duplicate tool_names: {sorted(set(dupes))}")
    # auth_token
    if not isinstance(self.auth_token, str):
        raise ValueError(f"{key_prefix}: auth_token must be str, got {type(self.auth_token).__name__}")
    # env
    for k, v in self.env.items():
        if not isinstance(k, str) or not isinstance(v, str):
            raise ValueError(f"{key_prefix}: env must be dict[str, str]; got key={k!r} value={v!r}")
    # URL validation for HTTP transport
    if self.transport == TransportType.HTTP and self.url:
        parsed = urlparse(self.url)
        if parsed.scheme not in ("http", "https") or not parsed.netloc:
            raise ValueError(f"{key_prefix}: url must be a valid HTTP/HTTPS URL, got {self.url!r}")
```
