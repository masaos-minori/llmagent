# Work Plan: `scripts/agent/commands/` リファクタリング — 無条件str()変換除去

## Goal

`scripts/agent/commands/` 配下のファイルから無条件 `str()` 変換（7箇所）を排除し、`isinstance` ガードによる境界バリデーションに統一する。

## Scope

**In scope:**
- `scripts/agent/commands/cmd_session.py` — `str(r.get("created_at", ""))` → isinstance check; `int(r["session_id"])` / `bool(r.get(...))` も型チェック追加
- `scripts/agent/commands/cmd_db.py` — `str(r["url"])` / `str(r["lang"] or "?")` / `str(r["chunk_count"])` / `str(r["fetched_at"])` → isinstance check

**Out of scope:**
- `scripts/agent/commands/models.py` — `dict[str, Any]` は "presentation-only; dict intentional" コメント付き正当用途
- `scripts/agent/commands/cmd_tooling.py` — `_decode_args() -> dict[str, Any]` は JSON 境界での正当な返り値
- `scripts/agent/commands/cmd_context.py` — `msg.get("content")` は TypedDict への正当な .get() アクセス
- `scripts/agent/commands/cmd_config.py` — `stats.latency.get(step)` は typed DTO への .get() アクセス
- `scripts/agent/commands/cmd_db.py:88` — `str(limit_raw).isdigit()` は str() への変換ではなくメソッド呼び出し
- その他 cmd_*.py の `parsed.flags.get(...)` — ParsedArgs の正当なフラグアクセス
- `except Exception`, `assert`, backward-compat code — 0件（既に対応済み）

## Assumptions

1. `AgentSession.list_sessions()` は `list[dict]` を返す（確認済み）。`r["session_id"]`, `r.get("created_at")`, `r.get("is_current")`, `r.get("title")` はすべて dict アクセス。
2. `AgentSession.list_documents()` も `list[dict]` を返す（確認済み）。`r["url"]`, `r["lang"]`, `r["chunk_count"]`, `r["fetched_at"]` はすべて dict アクセス。
3. `AgentSession.list_sessions()` の各フィールド: `session_id` は int (DB INTEGER), `created_at` は str | None (DB TEXT), `title` は str | None, `is_current` は bool。
4. `AgentSession.list_documents()` の各フィールド: `url` は str (NOT NULL), `lang` は str | None, `chunk_count` は int, `fetched_at` は str | None。
5. None や空文字列は「不在」として扱い、空文字列にフォールバックする（`or "?"` は None を `"?"` に変換する正当な処理として维持可）。

## Design

### `cmd_session.py` の修正パターン

```python
# Before
session_rows = [
    SessionRow(
        session_id=int(r["session_id"]),
        title=r.get("title"),
        created_at=str(r.get("created_at", "")),
        is_current=bool(r.get("is_current", False)),
    )
    for r in raw_rows
]

# After — ローカルヘルパー + explicit checks
def _row_to_session_row(r: dict, current_id: int | None) -> SessionRow:
    sid = r["session_id"]
    if not isinstance(sid, int):
        raise TypeError(f"session_id must be int, got {type(sid).__name__}")
    ca = r.get("created_at")
    created_at = ca if isinstance(ca, str) else ""
    return SessionRow(
        session_id=sid,
        title=r.get("title"),
        created_at=created_at,
        is_current=sid == current_id,
    )
```
