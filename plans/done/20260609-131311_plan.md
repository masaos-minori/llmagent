## Assumptions

1. `agent/services/` パターンは既存コード (`mcp_status.py`, `ingest_workflow.py`, `mcp_install.py`, `config_reload.py`) が確立しており、新サービスも同パターンで実装できる。
2. `cmd_ingest.py` はすでに `IngestWorkflowService` に委譲済み。残作業は `TYPE_CHECKING: pass` 削除と `compress_turns * 2` 内部ロジック除去のみ。
3. `cmd_mcp.py` はすでに `McpStatusService` / `McpInstallService` に委譲済み。残作業は default 動作変更のみ。
4. `SQLiteHelper._RAG_PATH` / `_SESSION_PATH` は `_ensure_config()` 呼出後にアクセス可能なクラス属性。`_print_rag_config()` の修正はこれらの属性を直接利用するか、`config_loader` で公開された値を使う。
5. `_reset_session_stats()` は `_ContextMixin` と `_SessionMixin` の 2 箇所から呼ばれている。単純な関数に切り出して両方から import できる。
6. `HistoryManager.force_compress()` は既に存在し `ctx.conv.history` を受け取る。`compress_turns * 2` の内部計算を `force_compress()` の引数なしオーバーロードまたは別メソッドで隠蔽できる。
7. `plan_mode` は `AgentContext.conv` の単純 bool フィールド。service 化の実態は「toggle ロジック + blocked tools 表示」をメソッドにまとめる程度。

## Design

### D-1: undo_service.py

```python
def undo_last_turn(ctx: AgentContext) -> tuple[bool, str]:
    """Roll back the last user+assistant turn.

    Returns (success, message).
    Strips memory injection markers, trims history, decrements stat_turns, calls session.undo_last_turn().
    """
```

### D-2: context_view.py

```python
def collect_context_state(ctx: AgentContext) -> dict:
    """Aggregate runtime context data for /context display."""
    # Moved from _ContextMixin._collect_context_state()
```

`_budget_breakdown()` も同ファイルに移す。`cmd_context.py` は `collect_context_state` を import して表示のみ担う。

### D-3: session_title.py

```python
class SessionTitleService:
    async def generate(self, ctx: AgentContext, first_input: str) -> None: ...
```

### D-4: session_restore.py

```python
def restore_session(ctx: AgentContext, session_id: int) -> tuple[bool, str]:
    """Restore session history, switch session ID, reset stats."""
```

`_reset_session_stats()` の呼出を内包する。

### D-5: db_maintenance_service.py

```python
class DbMaintenanceService:
    def stats(self) -> dict: ...
    def list_urls(self, lang, limit) -> list[dict]: ...
    def clean(self, url: str) -> bool: ...
    def rebuild_fts(self) -> None: ...
    def health(self) -> dict: ...
    def checkpoint(self, mode) -> dict: ...
    def vacuum(self) -> None: ...
    def purge(self, max_sessions, max_age_days) -> dict: ...
    def recover(self, backup_path) -> object: ...
```

### D-6: export_formatter.py

```python
def render_history_md(history) -> str: ...
def render_export(history, fmt) -> str: ...

class ExportIOService:
    def write(self, content, outfile, n_messages) -> None: ...
```

`utils.py` の `render_history_md`, `render_export`, `write_export` を `export_formatter.py` に移す。
`utils.py` はこれらを re-export して後方互換を保つか、呼出元を直接変更する。

### D-7: mixin_base.py の `_reset_session_stats` 切り出し

```python
# agent/commands/mixin_base.py
def reset_session_stats(ctx: AgentContext) -> None:
    """Reset all per-session counters."""
    ...
```

`MixinBase._reset_session_stats()` は `reset_session_stats(ctx)` モジュール関数に変更し、
`_ContextMixin` / `_SessionMixin` / `session_restore.py` からこの関数を呼ぶ。
