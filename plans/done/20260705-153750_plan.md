## Goal

Represent reachable degraded MCP servers in `McpServerHealthRegistry` by adding a `record_degraded()` method, calling it from the watchdog, and surfacing the degraded state in `/mcp status`.

## Scope

**In**: Add `record_degraded(server_key, reason=None)` to `McpServerHealthRegistry`. Update `_watchdog_check_http()` to call it. Expose degraded reason in `/mcp status` output.

**Out**: Changing watchdog restart policy. Restarting operator-action-required servers. Documentation-only updates.

## Assumptions

1. `McpServerHealthRegistry` at `shared/mcp_health.py:20` already has `McpServerHealthState.DEGRADED` but no public method to set it explicitly — `record_failure()` only sets DEGRADED on partial failure count (before threshold).
2. `_watchdog_check_http()` at `repl_health.py:464`: when `probe.reachable and not probe.restart_recommended`, sets `restart_counts[key] = 0` but does NOT call `record_degraded()` or `record_failure()` on the registry.
3. The requirement is: when `reachable=True`, `status_code != 200`, and `restart_recommended=False` → call `record_degraded(key, reason)`.
4. Degraded reason can come from `probe.body.get("reason")` or `probe.body.get("message")`.
5. `/mcp status` is in `agent/commands/cmd_mcp.py` — needs to read degraded state.
6. Per-call transport successes (`record_success()`) can still restore health from degraded.

## Design

**New `record_degraded()` in `McpServerHealthRegistry`:**
```python
def record_degraded(self, server_key: str, reason: str | None = None) -> None:
    """Record a reachable-but-degraded server. Does not trigger UNAVAILABLE."""
    self._states[server_key] = McpServerHealthState.DEGRADED
    if reason is not None:
        self._degraded_reasons[server_key] = reason
    logger.warning("Health: %r is DEGRADED (reason=%s)", server_key, reason or "unknown")

def get_degraded_reason(self, server_key: str) -> str | None:
    return self._degraded_reasons.get(server_key)
```

**Updated `_watchdog_check_http()` (line 464):**
```python
if probe.reachable and not probe.restart_recommended:
    restart_counts[key] = 0
    if probe.operator_action_required:
        logger.warning(...)
    if ctx.services_required.health_registry:
        reason = probe.body.get("reason") or probe.body.get("message")  # type: ignore[arg-type]
        ctx.services_required.health_registry.record_degraded(key, reason=str(reason) if reason else None)
    return
```

**`record_success()` clears degraded reason:**
```python
def record_success(self, server_key: str) -> None:
    ...
    self._degraded_reasons.pop(server_key, None)
```

**`/mcp status` output addition:**
Show `DEGRADED: <reason>` for each degraded server alongside existing status output.
