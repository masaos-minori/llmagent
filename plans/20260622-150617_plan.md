# Plan: compress_turns/char_limit 境界条件テスト追加 — test_history_manager.py

## Goal

`tests/test_history_manager.py` に `compress_turns=1`（最小値）と `char_limit=0`（無効化）の
境界条件テストを追加し、極端な値での圧縮動作を検証する。

## Scope

**In-Scope:**
- `tests/test_history_manager.py` — `TestCompressNoOp` または新クラスへの 3 テスト追加

**Out-of-Scope:**
- 実装コードの変更
- `protect_turns` の境界条件（別要件）
- トークン制限 (`token_limit`) の境界条件

## Assumptions

1. `char_limit=0` → `_is_over_char_limit()` が常に `False` を返す
   (`self._char_limit > 0` が偽のため)。どれだけ大きな history でも圧縮は起動しない。
2. `compress_turns=1` → `n_compress=2`。`protect_turns=0` のとき、
   `len(turn_msgs) > 2` を満たせば圧縮対象が存在する（3 メッセージ以上）。
   `len(turn_msgs) <= 2`（1 ターン対 = 2 メッセージ以下）なら `select_turns_to_compress()` が `None` を返す。
3. `_make_manager()` のデフォルトは `protect_turns=0` のため、ほとんどの境界テストでは
   `protect_turns` を明示しなくてよい。
4. `stat_compress_count` は `HistoryManager` に属し、成功時にインクリメントされる。

## Design

### 追加テスト一覧

**Test 1: `test_char_limit_zero_never_compresses`**
- `char_limit=0` (無効化)
- 大きな history (5 ターン対 = 10 メッセージ) を渡す
- Assert: `result == history`（unchanged）
- Assert: LLM の `post` が呼ばれない（`mock_http.post.assert_not_called()`）
- Assert: `mgr.stat_compress_count == 0`

```python
@pytest.mark.asyncio
async def test_char_limit_zero_never_compresses(self) -> None:
    mock_http = AsyncMock(spec=httpx.AsyncClient)
    mgr = _make_manager(char_limit=0, compress_turns=2, http=mock_http)
    h = _history(
        ("user", "q1"), ("assistant", "a1"),
        ("user", "q2"), ("assistant", "a2"),
        ("user", "q3"), ("assistant", "a3"),
        ("user", "q4"), ("assistant", "a4"),
        ("user", "q5"), ("assistant", "a5"),
    )
    result, info = await mgr.compress(h)
    assert result == h
    mock_http.post.assert_not_called()
    assert mgr.stat_compress_count == 0
```

**Test 2: `test_compress_turns_one_compresses_single_pair`**
- `compress_turns=1` (最小値), `char_limit=1` (強制圧縮), `protect_turns=0`
- 3 ターン対 (6 メッセージ) — `n_compress=2, n_protect=0`, `6 > 2` → 圧縮対象あり
- Assert: LLM が呼ばれる (`mock_http.post.assert_called_once()`)
- Assert: `mgr.stat_compress_count == 1`

```python
@pytest.mark.asyncio
async def test_compress_turns_one_compresses_single_pair(self) -> None:
    mock_http = AsyncMock(spec=httpx.AsyncClient)
    mock_resp = MagicMock()
    mock_resp.json.return_value = {
        "choices": [{"message": {"content": "Summary."}}]
    }
    mock_resp.raise_for_status = MagicMock()
    mock_http.post.return_value = mock_resp

    mgr = _make_manager(char_limit=1, compress_turns=1, http=mock_http)
    h = _history(
        ("user", "q1"), ("assistant", "a1"),
        ("user", "q2"), ("assistant", "a2"),
        ("user", "q3"), ("assistant", "a3"),
    )
    _, info = await mgr.compress(h)
    mock_http.post.assert_called_once()
    assert mgr.stat_compress_count == 1
```

**Test 3: `test_compress_turns_one_with_single_turn_skips`**
- `compress_turns=1`, `char_limit=1` (強制圧縮), `protect_turns=0`
- 1 ターン対 (2 メッセージ) — `n_compress=2, n_protect=0`, `2 <= 2` → `select_turns_to_compress()` が `None`
- Assert: `result == history`（unchanged）
- Assert: `stat_compress_count == 0`

```python
@pytest.mark.asyncio
async def test_compress_turns_one_with_single_turn_skips(self) -> None:
    mock_http = AsyncMock(spec=httpx.AsyncClient)
    mgr = _make_manager(char_limit=1, compress_turns=1, http=mock_http)
    h = _history(("user", "only question"), ("assistant", "only answer"))
    result, info = await mgr.compress(h)
    assert result == h
    assert mgr.stat_compress_count == 0
```
