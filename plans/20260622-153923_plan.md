## Assumptions

1. `fetch()` のリトライループ (`scripts/agent/memory/embedding_client.py:166`):
   ```python
   for attempt in range(self._config.max_retries + 1):
       result = await asyncio.wait_for(_fetch_embedding(...), ...)
       if result.success:
           return result
       self._record_failure()
   return last_result
   ```
   - max_retries=0 → range(1) → 1 試行
   - max_retries=1 → range(2) → 最大 2 試行
   - max_retries=2 → range(3) → 最大 3 試行

2. 既存テストカバレッジ (`TestRetry`):
   - max_retries=2: fail 1 → succeed 2nd (call_count=2) ✓
   - max_retries=2: all fail 3 times (call_count=3) ✓
   - max_retries=1: all fail, no call_count assert
   - max_retries=1: fail 1 → succeed 2nd (call_count=2) ✓
   - max_retries=0: fail → call_count=1 ✓
   - max_retries=0: succeed → call_count=1 ✓

3. **未テスト**:
   - max_retries=2, fail 2 回後に 3 回目で成功 → call_count=3, success=True
   - max_retries=1, all fail → call_count=2 (call_count の明示的検証)
   - max_retries=2, 1 回目で成功 → call_count=1 (不要なリトライなし)

## Design

### 追加テスト一覧 (TestRetry.test_max_retries_one_success_on_second_attempt の後に追加)

**Test 1: `test_max_retries_two_fail_twice_then_succeed_on_third`**
- max_retries=2, fail 1 回目・2 回目 → 3 回目で成功
- call_count=3, result.success=True
- 「最大リトライ数まで使い切って成功」を検証

```python
@pytest.mark.asyncio
async def test_max_retries_two_fail_twice_then_succeed_on_third(
    self, config: EmbeddingClientConfig
) -> None:
    config.max_retries = 2
    client = EmbeddingClient(config)
    call_count = 0

    async def _fake_post(*a: object, **kw: object) -> MagicMock:
        nonlocal call_count
        call_count += 1
        if call_count <= 2:
            raise httpx.HTTPStatusError(
                "fail", request=MagicMock(), response=MagicMock(status_code=500)
            )
        mock_resp = MagicMock()
        mock_resp.content = orjson.dumps({"embedding": [0.1]})
        mock_resp.raise_for_status = MagicMock()
        return mock_resp

    mock_http = AsyncMock(spec=httpx.AsyncClient)
    mock_http.post = _fake_post
    client._http = mock_http
    client._enabled = True

    result = await client.fetch("hello")
    assert result.success is True
    assert call_count == 3
```

**Test 2: `test_max_retries_one_all_fail_exactly_two_attempts`**
