## Assumptions

1. `apply_config()` のフィールド更新ロジック (`scripts/shared/llm_client.py:120-122`):
   ```python
   for attr, kwarg in self._HOT_CONFIG_FIELDS:
       if (value := args.get(kwarg)) is not None:
           self._apply_one(self, attr, kwarg, value)
   ```
   `False is not None` → True なので、`False` を渡した場合でもフィールドが更新される。

2. `_HOT_CONFIG_FIELDS` に含まれる boolean フラグ:
   ```python
   ("_llm_stream_retry_on_heartbeat_timeout", "stream_retry_on_heartbeat_timeout"),
   ("_llm_stream_retry_on_malformed_chunk", "stream_retry_on_malformed_chunk"),
   ```

3. `TestLLMClientApplyConfig._make_client()` のデフォルト値:
   - `_llm_stream_retry_on_heartbeat_timeout = True` (LLMClient default)
   - `_llm_stream_retry_on_malformed_chunk = False` (LLMClient default)

4. 境界値テストの目的: `if value:` ではなく `if value is not None:` が使われていることを
   `False` を渡した際に正しく更新されることで検証する。

5. 既存テスト `test_apply_config_sse_params` は `sse_heartbeat_timeout`, `sse_malformed_retry`,
   `sse_reconnect_max` のみカバー。boolean フラグは未テスト。

## Design

### 追加テスト一覧 (TestLLMClientApplyConfig.test_apply_config_none_args_are_no_op の後に追加)

**Test 1: `test_apply_config_stream_retry_on_heartbeat_timeout_false`**
- `_make_client()` のデフォルト: `_llm_stream_retry_on_heartbeat_timeout = True`
- `apply_config(stream_retry_on_heartbeat_timeout=False)` で `False` を設定
- `_llm_stream_retry_on_heartbeat_timeout == False` を検証
- これにより `if value is not None:` が `False` を正しく処理することを確認

```python
def test_apply_config_stream_retry_on_heartbeat_timeout_false(self) -> None:
    client = self._make_client()
    assert client._llm_stream_retry_on_heartbeat_timeout is True  # default
    client.apply_config(stream_retry_on_heartbeat_timeout=False)
    assert client._llm_stream_retry_on_heartbeat_timeout is False
```

**Test 2: `test_apply_config_stream_retry_on_malformed_chunk_true`**
- `_make_client()` のデフォルト: `_llm_stream_retry_on_malformed_chunk = False`
- `apply_config(stream_retry_on_malformed_chunk=True)` で `True` を設定
- `_llm_stream_retry_on_malformed_chunk == True` を検証

```python
def test_apply_config_stream_retry_on_malformed_chunk_true(self) -> None:
    client = self._make_client()
    assert client._llm_stream_retry_on_malformed_chunk is False  # default
    client.apply_config(stream_retry_on_malformed_chunk=True)
    assert client._llm_stream_retry_on_malformed_chunk is True
```
