# agent/commands/ 14 ファイル改修計画書

## Goal

`agent/commands/` 配下 14 ファイルから `except Exception`・直接 `print()`・
raw `dict`/`tuple[bool,str]` 契約・`.get()` 曖昧参照・暗黙フォールバックを除去し、
`OutputPort` + DTO + 専用例外による fail-fast 設計に置き換える。

## Assumptions

1. **plugin command の contract 維持**: `plugin_registry.iter_commands()` は `dict[str, tuple[Callable[..., Any], bool]]` を返し、handler は `(ctx: AgentContext, args: str)` で呼ばれる。`CommandRequest` DTO は built-in command 専用とし、plugin command の handler 署名は変更しない。
2. **`cmd_config.py:232` の `except Exception`** は `_build_db_cfg()` の設定ロード失敗を捕捉。具体例外は `ValueError | RuntimeError`（`shared/config_loader.py` が `ValueError` を送出）として限定する。
3. **`cmd_config.py:332` の `except Exception`** は `/reload` 失敗の fallback。`except (OSError, ValueError, ConfigReloadValidationError)` に限定し、`ConfigReloadValidationError` は 02_instruction 計画の新設例外クラスを使用する。
4. **`parse_flag_int()` の `None` 返却**: command handler が `None` を受け取って暗黙補正する箇所を特定し、`CommandValidationError` へ変更する。`utils.py` の `parse_flag_int()` 自体のシグネチャは後方互換維持とし、handler 側で `None` チェックを厳密化する。
5. **formatter.py の `print_*` API**: 段階的廃止。`OutputPort` を新設してから各 command を移行し、最後に `formatter.py` の `print_*` を `OutputPort` の委譲に置き換える。
6. **`tuple[bool, str]` の call sites**: `cmd_session.py:115` と `cmd_context.py:114` の `_, message =` パターンは 02_instruction 計画の `SessionRestoreResult`/`UndoResult` DTO への追従として処理する（本計画では call site の更新のみ実施）。

## Design

### 新設ファイル概要

**`agent/commands/enums.py`**:
```python
class CommandKind(str, Enum):
    MEMORY = "memory"; SESSION = "session"; CONTEXT = "context"
    DB = "db"; DEBUG = "debug"; INGEST = "ingest"; MCP = "mcp"
    NOTES = "notes"; TOOLING = "tooling"; CONFIG = "config"

class MemoryAction(str, Enum):
    LIST = "list"; SEARCH = "search"; SHOW = "show"
    ADD = "add"; DELETE = "delete"; PIN = "pin"; UNPIN = "unpin"; PRUNE = "prune"

class DbAction(str, Enum):
    STATS = "stats"; HEALTH = "health"; CHECKPOINT = "checkpoint"
    VACUUM = "vacuum"; PURGE = "purge"; RECOVER = "recover"; LIST = "list"

class McpAction(str, Enum):
    STATUS = "status"; INSTALL = "install"; PROBE = "probe"
```

**`agent/commands/exceptions.py`**:
```python
class CommandParseError(ValueError): ...
class CommandValidationError(ValueError): ...
class CommandDispatchError(RuntimeError): ...
class CommandRenderingError(RuntimeError): ...
class UnknownSubcommandError(CommandDispatchError):
    def __init__(self, sub: str, valid: tuple[str, ...]) -> None: ...
class UnknownPresetError(CommandValidationError): ...
class UnknownTierError(CommandValidationError): ...
```

**`agent/commands/output_port.py`**:
```python
class OutputPort(Protocol):
    def write(self, text: str) -> None: ...
    def write_table(self, headers: list[str], rows: list[list[str]]) -> None: ...
    def write_error(self, text: str) -> None: ...
    def write_success(self, text: str) -> None: ...

class CliOutputPort:
    """Wraps stdout print; replaces formatter.print_* API."""
    def write(self, text: str) -> None: print(text)
    def write_table(...) -> None: ...
    ...
```

**`agent/commands/models.py`** (主要 ViewModel):
```python
@dataclass(frozen=True)
class StatsViewModel:
    session_id: str; turns: int; tool_calls: int; tool_errors: int
    llm_retries: int; input_tokens: int | None; output_tokens: int | None
    debug_mode: bool; latency: dict[str, Any] | None  # presentation data; dict OK

@dataclass(frozen=True)
class ToolResultView:
    result_id: int; tool_name: str; summary: str | None
    args_masked: dict[str, Any]; is_error: bool

@dataclass(frozen=True)
class McpInstallRequest:
    server_name: str  # validated by installer_validation.validate_server_name()

@dataclass(frozen=True)
class McpInstallRenderModel:
    result: ScaffoldResult  # from McpInstallService
```

## Implementation Steps

各ステップは独立コミット可能。前ステップの `pytest` が pass してから次へ進む。

1. **[新設] enums.py / exceptions.py / models.py / output_port.py を作成**
   - 既存コードへの変更なし
   - コミット: `feat(commands): add enums, exceptions, models, output_port foundation`

2. **[formatter.py] OutputPort への置換**
   - `print_success()`, `print_error()`, `print_validation_error()`, `print_table()` 等を `OutputPort` メソッドに対応する実装に変更
   - `CliOutputPort` を `output_port.py` に新設し、既存の `formatter.print_*` の呼び出しを `OutputPort.write*()` に切り替え
   - コミット: `refactor(commands): replace formatter print_* with OutputPort`

3. **[registry.py] CommandDispatchError + print 廃止**
   - `_cmd_help()` の `print()` → `OutputPort.write()`
   - unknown command を `CommandDispatchError` で fail-fast
   - コミット: `refactor(commands/registry): OutputPort, CommandDispatchError for unknown commands`

4. **[cmd_config.py] StatsViewModel + except Exception 除去 + print 廃止**
   - `_collect_stats()` → `StatsViewModel` DTO
   - `except Exception as e:` (line 232) → `except (ValueError, RuntimeError) as e:`
   - `except Exception as e:` (line 332) → `except (OSError, ValueError, ConfigReloadValidationError) as e:`
   - 全 `print()` → `OutputPort.write*()`
   - コミット: `refactor(commands/cmd_config): StatsViewModel DTO, specific exceptions, OutputPort`

5. **[cmd_context.py] ContextViewModel + print 廃止 + service DTO 追従**
   - `state = collect_context_state(ctx)` の raw dict 依存 → `ContextStateView` DTO (02_instruction 計画) から参照
   - `msg.get("content")` → 型付き参照
   - `_, message = undo_last_turn()` → `result.message` (UndoResult DTO、02_instruction 追従)
   - 全 `print()` → `OutputPort.write*()`
   - コミット: `refactor(commands/cmd_context): ContextViewModel, OutputPort, DTO service contract`

6. **[cmd_db.py] DbCommandRequest + flags 厳密化 + print 廃止**
   - `parsed.flags.get("lang")` 等 → validator で `str | None` を厳密取得
   - service dict 戻り値 → `DbStats`/`DbHealth` DTO (02_instruction 計画) に追従
   - 全 `print()` → `OutputPort.write*()`
   - コミット: `refactor(commands/cmd_db): DbCommandRequest, typed flags, OutputPort`

7. **[cmd_memory.py] MemoryCommandRequest + print 廃止 + audit 移譲**
   - unknown sub → `UnknownSubcommandError`
   - `MemoryOpResult.action` → `MemoryAction` Enum
   - audit event 構築 → service/audit adapter へ移譲
   - 全 `print()` → `OutputPort.write*()`
   - コミット: `refactor(commands/cmd_memory): MemoryCommandRequest DTO, UnknownSubcommandError, OutputPort`

8. **[cmd_mcp.py] McpInstallRequest + print 廃止 + renderer 分離**
   - install 入力を `McpInstallRequest` DTO として検証
   - `print_next_steps()` の大量 `print()` → `McpInstallRenderModel` + renderer
   - 全 `print()` → `OutputPort.write*()`
   - コミット: `refactor(commands/cmd_mcp): McpInstallRequest DTO, renderer separation, OutputPort`

9. **[cmd_ingest / cmd_debug / cmd_notes / cmd_session] print 廃止 + DTO 追従**
   - 全 `print()` → `OutputPort.write*()`
   - `cmd_session.py:115` の `_, message =` → `result.message` (SessionRestoreResult、02 追従)
   - `cmd_notes.py` の unknown sub → `UnknownSubcommandError`
   - コミット: `refactor(commands): OutputPort migration for ingest/debug/notes/session`

10. **[cmd_tooling.py] ToolResultView DTO + invalid JSON fail-fast**

    - `orjson.loads(... or "{}")` → decode 失敗を `CommandValidationError` へ

|---|---|---|
