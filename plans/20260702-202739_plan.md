# Plan: Validate import boundaries after removing compatibility stubs and shims

## Goal

Confirm that deleting `agent/config.py`, `mcp/github/service.py`, removing package-level
service re-exports from `mcp/github/__init__.py`, and removing `McpServerConfig._cast_enums()`
leaves import boundaries valid and all tests passing.

## Scope

**In-Scope**:
- Run `uv run pytest` and confirm all tests pass.
- Run `PYTHONPATH=scripts uv run lint-imports` and confirm 0 architecture violations.
- Confirm `mcp/github/server.py` imports successfully after stub removal.
- Confirm `test_mcp_server_base.py` app module importability test passes.
- Confirm `shared/mcp_config.py` remains a shared-layer module with no upper-layer dependency.
- grep for any remaining references to `agent.config`, `mcp.github.service` (as a module), and
  package-level re-exports (`GitHubService`, `build_service`, `_GITHUB_TOKEN`) from
  `mcp.github.__init__`.

**Out-of-Scope**:
- Refactoring unrelated dependency boundaries.
- Changing MCP tool behavior.
- Changing agent config behavior.
- Introducing new tests beyond what is needed to cover changed import paths.

## Assumptions

1. The compatibility stubs (`agent/config.py`, `mcp/github/service.py`) and the `__init__.py`
   re-exports have already been identified as safe to delete; the goal here is validation only.
2. `uv run pytest` is the authoritative test runner for this project.
3. `PYTHONPATH=scripts uv run lint-imports` reads `.importlinter` and applies the declared
   contracts.
4. All callers of `agent.config` must be updated to import directly from
   `agent.config_dataclasses` or `agent.config_builders` before the stub can be deleted.
5. All callers of `mcp.github.service` must be updated to import from
   `mcp.github.service_dispatch`, `mcp.github.service_business`, or `mcp.github.service_init`
   before the stub can be deleted.
6. `McpServerConfig._cast_enums()` is called only from `__post_init__`; removing it is safe
   only when the config loader always passes correctly-typed enum values.
