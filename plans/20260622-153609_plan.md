## Assumptions

1. `_prompt_user_approval()` 実装 (`scripts/agent/tool_approval.py:86-94`):
   ```python
   async def _prompt_user_approval(risk: RiskLevel) -> bool:
       if risk == RiskLevel.HIGH:
           answer = (await asyncio.to_thread(input, "  Execute? [yes/no]: ")).strip().lower()
           return answer == "yes"
       answer = (await asyncio.to_thread(input, "  Execute? [y/N]: ")).strip().lower()
       return answer == "y"
   ```
   - `.strip().lower()` により大文字入力は正規化される
   - HIGH: `answer == "yes"` (full word)
   - MEDIUM: `answer == "y"` (single character)

2. 既存テストカバレッジ:
   - `test_medium_risk_y_approved`: "y" (小文字) → 承認 (テスト済み)
   - `test_high_risk_yes_approved`: "yes" (小文字) → 承認 (テスト済み)
   - `test_high_risk_y_is_insufficient`: "y" → 拒否 (テスト済み)
   - `test_medium_risk_empty_input_denied`: "" → 拒否 (テスト済み)
   - `test_medium_risk_input_with_whitespace_approved`: " y " → 承認 (テスト済み)

3. 未テストの境界:
   - "Y" (大文字) での medium 承認
   - "YES" (大文字) での high 承認
   - "yes" (full word) での medium 拒否 ("y" のみ有効)

4. `asyncio.to_thread` のパス: `patch("asyncio.to_thread", ...)` のモック形式を既存テストから踏襲。

## Design

### 追加テスト一覧 (TestCheckApproval.test_audit_logger_error_propagates の後に追加)

**Test 1: `test_medium_risk_uppercase_y_approved`**
- input = "Y" → `.strip().lower()` = "y" → `"y" == "y"` → True (承認)
- 大文字 "Y" が正しく受け付けられることを検証

```python
@pytest.mark.asyncio
async def test_medium_risk_uppercase_y_approved(self) -> None:
    ctx = _make_ctx()
    with patch("asyncio.to_thread", new=AsyncMock(return_value="Y")):
        result = await check_approval(ctx, "write_file", {"path": "/tmp/f.txt"})
    assert result is True
```

**Test 2: `test_high_risk_uppercase_yes_approved`**
- input = "YES" → `.strip().lower()` = "yes" → `"yes" == "yes"` → True (承認)
- 大文字 "YES" が正しく受け付けられることを検証

```python
@pytest.mark.asyncio
async def test_high_risk_uppercase_yes_approved(self) -> None:
    ctx = _make_ctx()
    with patch("asyncio.to_thread", new=AsyncMock(return_value="YES")):
        result = await check_approval(ctx, "delete_file", {"path": "/tmp/f.txt"})
    assert result is True
```

**Test 3: `test_medium_risk_full_word_yes_is_insufficient`**
- input = "yes" → `.strip().lower()` = "yes" → `"yes" == "y"` → False (拒否)
- MEDIUM リスクでは "yes" 全体の入力は有効でなく "y" のみ有効

```python
@pytest.mark.asyncio
async def test_medium_risk_full_word_yes_is_insufficient(self) -> None:
    ctx = _make_ctx()
    with patch("asyncio.to_thread", new=AsyncMock(return_value="yes")):
        result = await check_approval(ctx, "write_file", {"path": "/tmp/f.txt"})
    assert result is False
```
