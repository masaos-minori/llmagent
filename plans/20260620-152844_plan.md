# Plan: Fix `use_memory_layer` Default Inconsistency and Add Startup Visibility

## Goal

Correct the single documentation line that misreports the `use_memory_layer` default as `True`,
and expose memory-layer enabled/disabled state in the startup banner.

## Confirmed Implementation Default

- **Source:** `scripts/shared/config/config_dataclasses.py:298`
  ```python
  use_memory_layer: bool = False
  ```
- **Corroborated by:** `scripts/shared/config/config_builders.py:202`
  (`bool(cfg.get("use_memory_layer", False))`),
  `config/agent.toml:151`, `config/context.toml:17`, `config/memory.toml:2` (all `false`)
- **Default is `False`.**
