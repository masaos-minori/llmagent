# [agent][session] Add a fallback title when session title generation fails

## Summary
Session title generation runs asynchronously and can fail without user feedback.

## Problem
When title generation fails, the session may stay empty or keep a weak default title.

## Scope
- session title generation flow
- fallback title behavior
- session list UX

## Task
- add a fallback title when LLM title generation fails
- use a simple deterministic fallback, such as part of the first user input
- make empty-title sessions less likely

## Expected Result
Sessions still get useful titles even when title generation fails.

## Acceptance Criteria
- [ ] fallback title is created on generation failure
- [ ] empty titles become less common
- [ ] failure behavior is documented
