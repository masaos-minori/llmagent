# Update Agent Data Layer documentation to scoped DB commands

## Priority
Medium

## Labels
docs, data-layer, db, rag, cleanup

## Dependencies
- Remove flat DB alias section from CLI command reference

## Background
`05_agent_09_data-layer.md` describes RAG document access using flat commands such as `/db urls` and `/db clean`.

## Task
Update data-layer documentation to use scoped DB commands.

## Implementation Requirements
- Update `05_agent_09_data-layer.md`.
- Replace:
  - `/db urls` with `/db rag urls`
  - `/db clean` with `/db rag clean <url>`
  - `/db rebuild-fts` with `/db rag rebuild-fts`
- Where `/db stats` is mentioned, clarify whether the intended command is:
  - `/db rag stats`
  - `/db session stats`
  - or `/db help` for scoped options
- Update Agent-side document access patterns to use scoped commands.

## Acceptance Criteria
- Data-layer docs no longer mention flat `/db urls` or `/db clean`.
- RAG document operations are documented under `/db rag ...`.
- Session DB operations are documented under `/db session ...`.
