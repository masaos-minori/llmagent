# Add migration note for `/debug audit` removal

## Priority
Low

## Labels
docs, migration, debug, audit, breaking-change

## Dependencies
- Update documentation to remove `/debug audit`

## Background
`/debug audit` is removed without a backward-compatible alias. Users need a concise migration note pointing them to `/audit`.

## Task
Add a migration note documenting the removal of `/debug audit`.

## Implementation Requirements
Add a short migration note containing:

Removed command:
- `/debug audit`

Reason:
- Audit log browsing is handled by the dedicated `/audit` command.
- `/debug` should only control debug mode and log verbosity.

Replacement:
- `/audit`
- `/audit tail N`
- `/audit turn <task_id>`
- `/audit tool <name>`

## Acceptance Criteria
- Migration note clearly states `/debug audit` is removed.
- No backward-compatible alias is documented.
- Replacement commands are clear.

## Source Context
The previous shortcut duplicated `/audit tail`-style behavior inside the debug command.
