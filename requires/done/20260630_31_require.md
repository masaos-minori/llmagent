Medium

- `tests/test_tool_registry_counts.py`

- Add a test that imports `mcp.github.tools.TOOL_LIST`.
- Assert that `TOOL_LIST` has exactly 21 tools.
- Assert every `TOOL_LIST` tool name is registered in `ToolRegistry`.
- Assert every GitHub tool maps to `server_key="github"`.

- Tests pass.
