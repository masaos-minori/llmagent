# Remove MCP install-specific imports from `cmd_mcp.py`

## Priority
High

## Labels
agent, commands, mcp, cleanup

## Dependencies
- Remove `/mcp install` dispatch from MCP command handler

## Background
`cmd_mcp.py` imports installer-specific classes for the `/mcp install` wizard. Once `/mcp install` is removed, these imports become dead code.

## Task
Remove all installer-only imports and lazy imports from `cmd_mcp.py`.

## Implementation Requirements
- Remove this import from `cmd_mcp.py`:
  - `CliInstallQA`
  - `McpInstallService`
- Remove the lazy import of `validate_server_name` from the deleted `_cmd_mcp_install()` path.
- Update the module docstring to remove references to:
  - `McpInstallService`
  - `/mcp install wizard`
- Ensure the file only depends on MCP status-related services.

## Acceptance Criteria
- `cmd_mcp.py` does not reference:
  - `McpInstallService`
  - `CliInstallQA`
  - `validate_server_name`
  - `_cmd_mcp_install`
- Lint/type checks do not report unused imports.
- `/mcp` status behavior remains unchanged.

## Source Context
`cmd_mcp.py` imports `CliInstallQA` and `McpInstallService` only for the install wizard, and `_cmd_mcp_install()` lazily imports `validate_server_name`.
