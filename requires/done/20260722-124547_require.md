## Background

Agent documentation needs organization to ensure consistency between implementation and docs, eliminate duplicates, and remove references to deleted features.

## Problem

Multiple inconsistencies exist between current implementation and documentation:
- Slash command list may not match actual commands
- Deleted command descriptions may remain
- WorkflowEngine may be described as optional when it's required
- Old direct execution fallback explanations may remain
- DiagnosticStore/session_diagnostics descriptions may not match current implementation
- Deleted diagnostics.jsonl descriptions may remain
- LLMTurnRunner/ToolLoopGuard/partial completion descriptions may not match current behavior
- CLIView/OutputTag/startup validation display descriptions may not match implementation
- AgentContext/AppServices initialization flow may not match current implementation

## Reason for Change
