# Update documentation to remove `/debug audit`

## Priority
Medium

## Labels
docs, agent, commands, debug, audit, breaking-change

## Dependencies
- Remove `/debug audit` branch from debug command handler
- Update `/debug` command description in CommandRegistry

## Background
Agent documentation currently describes `/debug audit` and `/audit` as audit log browsing options. After removal, audit browsing must be documented only under `/audit`.

## Task
Remove `/debug audit` from Agent documentation.

## Implementation Requirements
Update relevant documentation:
- Remove `/debug audit` from Debug command sections.
- Describe `/debug` only as:
  - debug mode toggle
  - log level control via `verbose` and `normal`
- Keep `/audit [tail N|turn <id>|tool <name>]` as the audit browsing command.
- Update any CommandRegistry mixin responsibility descriptions if they mention audit under Debug.

## Acceptance Criteria
- Docs no longer present `/debug audit` as an available command.
- Audit log browsing is documented only under `/audit`.
- Debug and Audit responsibilities are clearly separated.

## Source Context
`cmd_debug.py` currently owns the `/debug audit` shortcut, while `cmd_audit.py` owns the proper audit browsing command.
