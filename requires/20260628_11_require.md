# Delete `_NotesMixin` and `cmd_notes.py`

## Priority
High

## Labels
agent, commands, notes, cleanup

## Dependencies
- Remove `/note` command registration from CommandRegistry

## Background
`cmd_notes.py` implements `_NotesMixin`, including handlers for `/note add`, `/note list`, `/note delete`, `/note pin`, `/note unpin`, and `/note search`.

## Task
Delete the note command implementation.

## Implementation Requirements
- Delete `cmd_notes.py` if it is no longer imported.
- Remove all note command handlers:
  - `_note_add`
  - `_note_list`
  - `_note_delete`
  - `_note_pin`
  - `_note_unpin`
  - `_note_search`
  - `_cmd_note`
- Remove note-specific usage and validation messages.

## Acceptance Criteria
- `_NotesMixin` no longer exists in runtime command code.
- No runtime code imports `agent.commands.cmd_notes`.
- `/note` has no remaining execution path.

## Source Context
`cmd_notes.py` dispatches all note subcommands through `_cmd_note()`, so deleting only `add/list/delete` would leave a partially functional `/note` group. The command group should be removed entirely.
