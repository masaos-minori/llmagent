# Remove flat DB alias section from CLI command reference

## Priority
High

## Labels
docs, agent, commands, db, breaking-change

## Dependencies
- Update `/db` help description in CommandRegistry

## Background
`05_agent_07_cli-and-commands.md` currently contains a “Flat forms (compatibility aliases)” section documenting flat DB commands such as `/db urls`, `/db clean`, and `/db rebuild-fts`.

## Task
Remove the flat DB alias section from the CLI command reference.

## Implementation Requirements
- Update `05_agent_07_cli-and-commands.md`.
- Remove the `Flat forms (compatibility aliases)` DB section.
- Remove user-facing documentation for:
  - `/db urls`
  - `/db clean <url>`
  - `/db rebuild-fts`
  - `/db recover`
- Keep and document only:
  - `/db rag ...`
  - `/db session ...`
- Ensure the note below the DB tables no longer says flat `/db <subcmd>` forms are compatibility aliases.

## Acceptance Criteria
- CLI command reference no longer presents flat DB aliases as available commands.
- `/db rag ...` and `/db session ...` are documented as the canonical forms.
- The DB command reference matches `cmd_db.py`.
