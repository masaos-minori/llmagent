# [MCP][Shell] Implement shell_run_bg or remove all references to it

## Summary
`shell_run_bg` appears in documentation references but is not implemented in the shell MCP server.

## Problem
The documented tool list and the implemented tool list do not match.

## Scope
- `mcp/shell/server.py`
- shell tool documentation
- tool metadata

## Task
Choose one option:

### Option A
Remove all references to `shell_run_bg` from docs and tool metadata.

### Option B
Implement `shell_run_bg` with:
- background execution
- job ID tracking
- status retrieval
- stop/cancel support
- audit logging
- the same allowlist and safety rules as `shell_run`

## Expected Behavior
- no documented shell tool should be missing from implementation
- no implemented shell tool should be undocumented

## Acceptance Criteria
- [ ] documentation matches implementation exactly
- [ ] `shell_run_bg` is either fully implemented or fully removed
- [ ] no call path returns `Unknown tool` for a documented shell tool
