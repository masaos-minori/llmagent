# Update `/debug` command description in CommandRegistry

## Priority
High

## Labels
agent, commands, help, debug, cleanup

## Dependencies
- Remove `/debug audit` branch from debug command handler

## Background
The command registry currently describes `/debug` as supporting an `audit` subcommand. After removing `/debug audit`, `/help` must not advertise it.

## Task
Update the `/debug` command description in `registry.py`.

## Implementation Requirements
- Update the `/debug` `CommandDef` description in `registry.py`.
- Remove `audit` and `audit=tail log` from the description.
- Suggested description:
  - `[verbose|normal]  Toggle debug; subcommands: verbose/normal=log level`
  - or `Toggle debug mode; subcommands: verbose|normal`

## Acceptance Criteria
- `/help` no longer mentions `/debug audit`.
- `/debug` help text matches the actual implementation.
- `/audit` remains separately documented.

## Source Context
`registry.py` currently includes `audit` in the `/debug` command description.
