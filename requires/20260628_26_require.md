# Rename `_IngestMixin` after removing `/ingest`

## Priority
Medium

## Labels
agent, commands, refactor, cleanup

## Dependencies
- Remove `_cmd_ingest()` from the Agent command mixin

## Background
After `/ingest` is removed, `_IngestMixin` no longer represents its responsibilities accurately. The remaining handlers are for export, compaction, and RAG search.

## Task
Rename the mixin and optionally the file to match the remaining responsibilities.

## Implementation Requirements
Choose and apply a clear name, for example:
- `_RagExportMixin`
- `cmd_rag_export.py`

Update all references:
- `registry.py` imports
- `CommandRegistry` inheritance list
- tests importing `_IngestMixin`
- file/module docstrings
- command group comments

## Acceptance Criteria
- No command mixin claims to own `/ingest`.
- Remaining responsibilities are clear:
  - `/export`
  - `/compact`
  - `/rag`
- All imports and tests use the new mixin name.
- `/rag`, `/export`, and `/compact` behavior remains unchanged.

## Source Context
`cmd_ingest.py` currently contains `/export`, `/compact`, `/rag`, and `/ingest` handlers in the same mixin.
