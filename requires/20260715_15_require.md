# Add regression coverage for command, config, Memory, and policy cleanup

## Priority

Medium

## Type

Tests

## Primary Files

- `tests/test_command_registry.py`
- `tests/test_repl.py`
- `tests/test_cmd_memory.py`
- `tests/test_config_memory.py`
- `tests/test_config_reload.py`
- `tests/test_tool_policy.py`
- `tests/test_tool_approval.py`
- `tests/test_memory_status.py`
- `tests/test_memory_retriever.py`

## Do Not Modify in This Issue

- Runtime implementation files, except minimal test fixtures if unavoidable
- Documentation files

## Background

Several legacy paths are being removed. Regression coverage is needed to prevent them from being reintroduced.

## Test Areas

### Command metadata and REPL

- [ ] Completion includes all command names from `_COMMANDS`.
- [ ] Completion includes `/exit` according to the selected design.
- [ ] Dispatch and help output remain consistent.
- [ ] Ctrl-C behavior is tested according to the selected design.

### Memory commands

- [ ] `/memory rebuild` is rejected.
- [ ] `/memory import-jsonl` is rejected.
- [ ] `/memory help` does not mention JSONL rebuild.
- [ ] `/memory check-consistency` does not mention JSONL.
- [ ] `/memory search` requires embedding-backed hybrid retrieval.
- [ ] `/memory status` does not show disabled or FTS-only modes.

### Config

- [ ] `use_memory_layer` is rejected.
- [ ] `memory_jsonl_dir` is rejected.
- [ ] `memory_embed_enabled` is rejected.
- [ ] GitOps fields follow the selected schema behavior.
- [ ] Failed reload does not partially mutate runtime config.
- [ ] `ConfigLoadError` includes the original reason.

### Tool policy

- [ ] GitHub allowlist accepts a listed repository.
- [ ] GitHub allowlist rejects an unlisted repository.
- [ ] Empty GitHub allowlist denies writes.
- [ ] `gitops_push_blocked` blocks the intended tools.
- [ ] GitOps force/protected branch behavior follows the selected design.

### Memory core

- [ ] Hybrid retriever does not return FTS-only mode.
- [ ] Missing embeddings are reported as degraded or unavailable.
- [ ] Vector index unavailability is observable.
- [ ] JSONL is not written during ingestion.

## Acceptance Criteria

- [ ] Tests fail if removed Memory commands return.
- [ ] Tests fail if removed Memory config keys are accepted.
- [ ] Tests fail if FTS-only mode is returned.
- [ ] Tests fail if GitHub write policy becomes fail-open.
- [ ] Test ownership remains separate from implementation issues.
