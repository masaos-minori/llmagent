# Remove note command tests

## Priority
Medium

## Labels
tests, agent, commands, notes, cleanup

## Dependencies
- Delete `_NotesMixin` and `cmd_notes.py`

## Background
`test_agent_cmd_notes.py` directly tests `_NotesMixin` and all note subcommands. Once `/note` is removed, these tests are obsolete.

## Task
Delete note command behavior-lock tests.

## Implementation Requirements
- Delete `test_agent_cmd_notes.py`.
- Remove all test imports of `_NotesMixin`.
- Remove tests for:
  - `/note add`
  - `/note list`
  - `/note delete`
  - `/note pin`
  - `/note unpin`
  - `/note search`
  - unknown `/note` subcommands
- Do not remove memory command tests.

## Acceptance Criteria
- No test imports `agent.commands.cmd_notes`.
- No test directly instantiates `_NotesMixin`.
- Memory command tests remain intact.

## Source Context
`test_agent_cmd_notes.py` covers add/list/delete/pin/unpin/search behavior for `_NotesMixin`.
