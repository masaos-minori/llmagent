# Shared/DB の不整合と既知の問題

本ファイルは、`shared/` および `db/` レイヤーにおけるドキュメント間の既知の不整合、実装上の不具合、
未文書化の領域、未実装の機能、未定義の挙動をすべて記録する。

各項目は以下の形式に従う:
- **種別:** `ドキュメント不整合` / `実装上の不具合` / `未文書化` / `未実装` / `未定義` / `確認が必要`
