# Goal

Remove direct `print()` calls from `registry.py`'s `_cmd_help()` method by routing output
through `OutputPort`, and ensure unknown-command paths surface as `CommandDispatchError`.

# Scope

- `scripts/agent/commands/registry.py`

# Assumptions

1. `_cmd_help()` is the only method in `registry.py` that calls `print()` directly (lines 239-246).
2. `dispatch()` currently returns `False` for unrecognized commands; callers handle the `False` return — no `CommandDispatchError` is raised here (registry is not the right layer to raise; the REPL is). So this step only removes `print()` from `_cmd_help`.
3. `OutputPort` / `CliOutputPort` from `output_port.py` (Step 1) must be available.
4. Plugin dispatch via `_dispatch_plugin` is not changed.

# Implementation

## Target file

`scripts/agent/commands/registry.py`

## Procedure

1. Add `_out: CliOutputPort` field to `CommandRegistry.__init__` (default `CliOutputPort()`).
2. Replace every `print(...)` in `_cmd_help` with `self._out.write(...)`.
3. Run ruff + mypy.

## Method

Inject `CliOutputPort` as instance attribute. Tests can substitute a mock by passing
a different `OutputPort` implementor to `__init__`.

## Details

```python
from agent.commands.output_port import CliOutputPort, OutputPort

class CommandRegistry(...):
    def __init__(self, ctx: AgentContext, out: OutputPort | None = None) -> None:
        self._ctx = ctx
        self._out: OutputPort = out if out is not None else CliOutputPort()

    def _cmd_help(self) -> None:
        ctx = self._ctx
        n_tools = len(ctx.cfg.tool.tool_definitions)
        sid = (
            f"session {ctx.session.session_id}"
            if ctx.session.session_id
            else "no session"
        )
        self._out.write("Agent REPL — type a question and press Enter.")
        self._out.write("Conversation history is preserved within the session.")
        self._out.write("")
        self._out.write("Slash commands:")
        for cmd in _COMMANDS:
            self._out.write(f"  {cmd.name:<22} {cmd.help}")
        self._out.write("")
        self._out.write(
            f"Tools: {n_tools}  |  LLM: {ctx.cfg.llm.llm_url}  |  {sid}"
        )
```
