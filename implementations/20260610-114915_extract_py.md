## Goal

Introduce `ExtractionPolicy` dataclass to group extraction thresholds so they can be
overridden at call time without modifying module-level constants.
`extract_memories()` gains an optional `policy` parameter; default behavior is unchanged.

## Scope

- **Target file**: `scripts/agent/memory/extract.py`
- **Secondary**: `scripts/agent/memory/ingestion.py` â€” no change required (passes no policy, gets default)
- **Tests**: `tests/test_memory_extract.py` â€” add one test exercising non-default `max_entries`

## Assumptions

1. Module-level constants (`MIN_CONTENT_CHARS`, etc.) are kept as the authoritative defaults.
   `ExtractionPolicy` fields reference those constants so the defaults are defined in one place.
2. `_try_extract_from_assistant` and `_try_extract_from_user` need access to policy thresholds.
   Pass `policy` as an argument to each private function, or pass individual threshold values.
   Prefer passing the whole `policy` object to keep signatures clean.
3. `_SEMANTIC_KEYWORDS` and `_EPISODIC_FAILURE_KEYWORDS` remain module-level constants (multilingual
   support is out of scope).
4. `ExtractionPolicy` is exported from `extract.py`; callers import it from there.

## Implementation

### Target file

`scripts/agent/memory/extract.py`

### Procedure

1. Add `@dataclass` import.
2. Define `ExtractionPolicy` after the module-level constants:
   ```python
   @dataclass
   class ExtractionPolicy:
       min_content_chars: int = MIN_CONTENT_CHARS
       min_user_content_chars: int = MIN_USER_CONTENT_CHARS
       min_turns: int = MIN_TURNS
       max_entries: int = MAX_ENTRIES
   ```
3. Update `_try_extract_from_assistant` signature to accept `policy: ExtractionPolicy`.
   Replace direct `MIN_CONTENT_CHARS` references with `policy.min_content_chars`.
4. Update `_try_extract_from_user` signature to accept `policy: ExtractionPolicy`.
   Replace `MIN_USER_CONTENT_CHARS` with `policy.min_user_content_chars`.
5. Update `extract_memories()`:
   - Add `policy: ExtractionPolicy | None = None` parameter.
   - Set `_policy = policy or ExtractionPolicy()` at the top.
   - Replace `MIN_TURNS` with `_policy.min_turns`.
   - Replace `MAX_ENTRIES` with `_policy.max_entries`.
   - Pass `_policy` to `_try_extract_from_assistant` and `_try_extract_from_user`.
