# Implementation: mdq/mcp/service.py — Real SQLite-backed service

## Goal

Replace all placeholder stubs in MdqService with real SQLite FTS5-backed implementations.

## Scope

**In scope:**
- `_init_db()` — Create `sections` table and `sections_fts` FTS5 virtual table
- `search_docs()` — Delegate to search.search_docs (already structured, needs DB schema)
- `get_chunk(chunk_id)` — Query sections table by ID
- `refresh_index(paths)` — Compare file mtime, re-index changed files
- `stats()` — COUNT queries on sections table
- `grep_docs(pattern)` — SQLite REGEXP/LIKE pattern search

**Out of scope:**
- Indexer logic (handled in indexer.py)
- Parser logic (handled in parser.py)
- Search logic (handled in search.py)

## Assumptions

1. SQLite3 is available with FTS5 extension support.
2. DB parent directory `/opt/llm/db/` exists or can be created.
3. `search.search_docs()` already has the query logic; just needs the schema.

## Implementation

### Target file
`scripts/mcp/mdq/service.py`

### Procedure

1. Add `_init_db()` that creates tables on first use.
2. Implement `get_chunk()`, `refresh_index()`, `stats()`, `grep_docs()` with real SQLite queries.
3. Keep `_db_path` configurable for testing.

### Method

Use `sqlite3` module (standard library). Create tables only if they don't exist (`CREATE TABLE IF NOT EXISTS`). Use parameterized queries everywhere.

import sqlite3
from pathlib import Path
from typing import Any

def _init_db(self) -> None:
    """Create sections table and FTS5 virtual table if they don't exist."""
    db_dir = Path(self.db_path).parent
    db_dir.mkdir(parents=True, exist_ok=True)
    conn = sqlite3.connect(self.db_path)
    try:
        conn.execute("""
            CREATE TABLE IF NOT EXISTS sections (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                file_path TEXT NOT NULL,
                heading TEXT NOT NULL,
                content TEXT NOT NULL,
                file_mtime REAL NOT NULL
            )
        """)
        conn.execute("""
            CREATE VIRTUAL TABLE IF NOT EXISTS sections_fts USING fts5(
                content,
                file_path,
                heading,
                content_detached=true
            )
        """)
        # Trigger: keep FTS in sync with sections table
        conn.execute("""
            CREATE TRIGGER IF NOT EXISTS sections_ai AFTER INSERT ON sections
            BEGIN
                INSERT INTO sections_fts(rowid, content, file_path, heading)
                VALUES (new.id, new.content, new.file_path, new.heading);
            END
        """)
        conn.execute("""
            CREATE TRIGGER IF NOT EXISTS sections_ad AFTER DELETE ON sections
            BEGIN
                INSERT INTO sections_fts(sections_fts, rowid, content)
                VALUES ('delete', old.id, old.content);
            END
        """)
        conn.execute("""
            CREATE TRIGGER IF NOT EXISTS sections_au AFTER UPDATE ON sections
            BEGIN
                INSERT INTO sections_fts(sections_fts, rowid, content)
                VALUES ('delete', old.id, old.content);
                INSERT INTO sections_fts(rowid, content, file_path, heading)
                VALUES (new.id, new.content, new.file_path, new.heading);
            END
        """)
        conn.commit()
    finally:
        conn.close()

def _get_db_connection(self) -> sqlite3.Connection:
    """Return a connection to the mdq database."""
    conn = sqlite3.connect(self.db_path)
    conn.row_factory = sqlite3.Row
    return conn

async def get_chunk(self, req: GetChunkRequest) -> str:
    """Retrieve a Markdown chunk by its ID."""
    conn = self._get_db_connection()
    try:
        row = conn.execute(
            "SELECT heading, content FROM sections WHERE id = ?",
            (req.chunk_id,),
        ).fetchone()
        if row is None:
            return f"Chunk {req.chunk_id} not found"
        return f"## {row['heading']}\n\n{row['content']}"
    finally:
        conn.close()

async def refresh_index(self, req: RefreshIndexRequest) -> str:
    """Incrementally refresh the index for changed files."""
    from mcp.mdq.indexer import index_paths as _index_paths
    return await _index_paths(self, req)

async def stats(self, req: StatsRequest) -> str:
    """Return document/chunk count and index metadata."""
    conn = self._get_db_connection()
    try:
        chunk_count = conn.execute("SELECT COUNT(*) as cnt FROM sections").fetchone()["cnt"]
        doc_count = conn.execute("SELECT COUNT(DISTINCT file_path) as cnt FROM sections").fetchone()["cnt"]
        return f"Documents: {doc_count}, Chunks: {chunk_count}"
    finally:
        conn.close()

async def grep_docs(self, req: GrepDocsRequest) -> str:
    """Search Markdown chunks with a regex pattern."""
    import re
    conn = self._get_db_connection()
    try:
        pattern = req.pattern
        compiled = re.compile(pattern)
        rows = conn.execute("SELECT id, heading, content FROM sections").fetchall()
        matches = []
        for row in rows:
            if compiled.search(row["content"]) or compiled.search(row["heading"]):
                matches.append(f"Chunk {row['id']}: {row['heading']}\n{row['content'][:200]}")
        if not matches:
            return "No matches found."
        return "\n---\n".join(matches)
    finally:
        conn.close()
