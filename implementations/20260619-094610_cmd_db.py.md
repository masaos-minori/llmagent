# Implementation: Make _db_clean and _db_list_urls Async and Rewire to MCP

## Goal

Replace direct `DbMaintenanceService` calls in `/db clean` and `/db urls` with async MCP tool calls to `rag_delete_document` and `rag_list_documents`. The command handlers must become `async def` to support `await`.

## Scope

- `scripts/agent/commands/cmd_db.py`
  - `_cmd_db()` → `async def _cmd_db()`
  - `_db_clean()` → `async def _db_clean()`: call `rag_delete_document` via MCP
  - `_db_list_urls()` → `async def _db_list_urls()`: call `rag_list_documents` via MCP
  - Update dispatch dict to `await` async sub-handlers
- `CommandRegistry` dispatch path — verify async handler support or add it

Out of scope:
- `_db_stats`, `_db_rebuild_fts`, `_db_health`, `_db_checkpoint`, `_db_vacuum`, `_db_purge`, `_db_recover`, `_db_consistency` — remain sync

## Assumptions

1. `CommandRegistry.dispatch()` must be updated to detect async handlers via `inspect.iscoroutinefunction` and `await` them if needed. Verify current dispatch implementation before implementing.
2. The agent REPL has access to `ToolExecutor.execute()` or an equivalent MCP call path. The MCP server key is `rag_pipeline` (port 8010).
3. `ToolExecutor.execute(tool_name, args)` is the correct dispatch path for MCP tool calls from the agent layer. Verify the actual call signature by reading `shared/tool_executor.py` or `agent/tool_runner.py`.
4. Error handling: if `rag-pipeline-mcp` is unreachable, catch the connection error and write a user-facing error message.
5. `_db_stats` may keep `DbStats.docs`/`DbStats.chunks` from `DbMaintenanceService.stats()` (pending decision in Step 3); no async change needed there.

## Implementation

### Target file

`scripts/agent/commands/cmd_db.py`

### Procedure

1. Look up how `CommandRegistry` dispatches `/db` and whether async is already supported.
2. Make `_cmd_db`, `_db_clean`, `_db_list_urls` async.
3. Update the dispatch dict to `await` async lambdas, or refactor to a coroutine-aware dispatch.
4. Rewrite `_db_clean` body to call `rag_delete_document` via MCP.
5. Rewrite `_db_list_urls` body to call `rag_list_documents` via MCP and format the result.

### Method

Minimal change: make only the two affected handlers async; keep the rest sync. Use the same MCP call pattern as existing async handlers in `cmd_ingest.py` or similar.

### Details

**_cmd_db dispatch — async-aware pattern:**
```python
async def _cmd_db(self, args: str) -> None:
    ...
    dispatch: dict[str, Any] = {
        "help": self._db_help,
        "stats": self._db_stats,
        "urls": lambda: self._db_list_urls(rest),
        "clean": lambda: self._db_clean(rest),
        ...
    }
    handler = dispatch.get(subcmd)
    if handler:
        result = handler()
        if inspect.isawaitable(result):
            await result
    else:
        ...
```

**_db_clean — MCP call:**
```python
async def _db_clean(self, rest: str) -> None:
    url = rest.strip()
    if not url:
        self._out.write_validation_error("/db clean <url>")
        return
    try:
        result = await self._call_mcp_tool("rag_delete_document", {"url": url})
        self._out.write(result)
    except Exception as e:
        self._out.write_error(f"rag-pipeline-mcp unavailable: {e}")
```

**_db_list_urls — MCP call:**
```python
async def _db_list_urls(self, rest: str) -> None:
    parsed = parse_command_args(rest.split())
    lang_raw = parsed.flags.get("lang")
    lang = str(lang_raw) if lang_raw in ("ja", "en") else None
    limit_raw = parsed.flags.get("limit")
    limit = int(limit_raw) if limit_raw and str(limit_raw).isdigit() else 20
    args: dict[str, Any] = {"limit": limit}
    if lang:
        args["lang"] = lang
    try:
        result = await self._call_mcp_tool("rag_list_documents", args)
        self._out.write(result)
    except Exception as e:
        self._out.write_error(f"rag-pipeline-mcp unavailable: {e}")
```

**`_call_mcp_tool` helper** — find the correct call path by reading how the agent dispatches MCP tools in other command handlers (e.g., `cmd_mcp.py` or `tool_runner.py`). Do not implement a new HTTP client; reuse the existing MCP dispatch path.

**Required import:** `import inspect`
