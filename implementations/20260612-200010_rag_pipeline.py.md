# Goal

Replace `_cfg: dict[str, Any]` with `PipelineConfig` DTO, fix the `except Exception`
in `_get_cfg()`, clean up `__all__` re-exports, and add schema validation for
LLM response bodies.

# Scope

- `scripts/rag/pipeline.py`

# Assumptions

1. `PipelineConfig` from `rag.models` (Step 2-3 prerequisite).
2. `_get_cfg()` currently uses `except Exception as e:` to catch TOML load errors;
   replace with `except (FileNotFoundError, tomllib.TOMLDecodeError, ValueError)`.
3. `__all__` currently re-exports `get_embedding` and `sanitize_document`; remove
   these re-exports (callers import directly from `rag.repository` / `rag.utils`).
4. `str(body.get("context", ""))` → validate `body["context"]` is `str`, raise
   `EmbeddingSchemaError` if not.
5. `body.get("selected_hits", [])` → validate it is a `list`, raise
   `PipelineValidationError` if not.
6. `_cfg: dict[str, Any] | None = None` module-level var → replaced by a lazy
   `_pipeline_config: PipelineConfig | None = None` with typed factory function.

# Implementation

## Target file

`scripts/rag/pipeline.py`

## Procedure

1. Import `PipelineConfig` from `rag.models`, `EmbeddingSchemaError` /
   `PipelineValidationError` from `rag.exceptions`.
2. Change `_cfg: dict[str, Any] | None = None` → `_pipeline_config: PipelineConfig | None = None`.
3. Rewrite `_get_cfg()` to return `PipelineConfig` with specific exception handling:
   ```python
   def _get_pipeline_config() -> PipelineConfig:
       global _pipeline_config
       if _pipeline_config is None:
           try:
               raw = ConfigLoader().load("rag_pipeline.toml")
           except (FileNotFoundError, ValueError) as e:
               raise PipelineValidationError(f"Cannot load pipeline config: {e}") from e
           _pipeline_config = _build_pipeline_config(raw)
       return _pipeline_config
   ```
4. Remove `get_embedding` and `sanitize_document` from `__all__` and from the
   import-based re-export section.
5. Add isinstance checks before `str(body.get("context", ""))` and
   `body.get("selected_hits", [])`.
6. Run ruff + mypy.

## Method

Module-level config var type change + exception narrowing + re-export cleanup.
