# Implementation: Extend /health to report stall-related degradation

Source plan: `plans/20260625-135720_plan.md` (req #19)

## Goal

Extend the `health()` endpoint in `scripts/eventbus/app.py` to return DB query latency, DLQ loop delay, total event count, and a `degraded_reasons` list, allowing operators to distinguish total failure from severe runtime degradation.

## Scope

- Add `_dlq_last_run: float` module-level variable, updated in `_dlq_loop()` after each iteration
- Extend `health()` response to include: `db_latency_ms`, `dlq_last_run_age_s`, `event_count`, `degraded_reasons`
- Change `status` to three-valued: `ok` / `degraded` / `unavailable`
- DB health check moves to `asyncio.to_thread` (consistent with req #14)
- Out of scope: Prometheus metrics export, new config fields

## Assumptions

1. req #14 (asyncio.to_thread) is implemented; DB latency measurement wraps the offloaded call
2. DLQ loop age threshold: `_DLQ_INTERVAL * 2 = 120 s`
3. DB latency threshold: `100 ms`
4. `_dlq_last_run == 0.0` means the DLQ loop has not yet completed one cycle (startup state); do NOT mark as degraded in this case
5. `event_count` uses `SELECT COUNT(*) FROM events` via the DB lock

## Implementation

### Target file

`scripts/eventbus/app.py`

### Procedure

1. Add `import time` (if not present)
2. Add `_dlq_last_run: float = 0.0` module-level variable
3. Update `_dlq_loop()` to set `_dlq_last_run = time.monotonic()` after each iteration
4. Rewrite `health()` to measure DB latency, compute DLQ age, fetch event count, build `degraded_reasons`

### Method

DB latency: measure `time.monotonic()` before/after the `asyncio.to_thread` DB check call. DLQ age: compute `time.monotonic() - _dlq_last_run` (skip if `_dlq_last_run == 0.0`). Event count: one additional `SELECT COUNT(*) FROM events` inside the DB check callable.

### Details

**Module-level addition:**
```python
_dlq_last_run: float = 0.0
```

**_dlq_loop() update:**
```python
async def _dlq_loop() -> None:
    global _dlq_last_run
    while True:
        try:
            ...
        except Exception:
            logger.exception("dlq_loop error")
        _dlq_last_run = time.monotonic()
        await asyncio.sleep(_DLQ_INTERVAL)
```

**health() rewrite:**
```python
@app.get("/health")
async def health() -> dict[str, Any]:
    def _check_db() -> tuple[str, float, int]:
        t0 = time.monotonic()
        try:
            _db.execute("SELECT 1")  # type: ignore[union-attr]
            count_row = _db.execute("SELECT COUNT(*) FROM events").fetchone()  # type: ignore[union-attr]
            latency_ms = (time.monotonic() - t0) * 1000
            return "ok", latency_ms, int(count_row[0])
        except Exception:
            latency_ms = (time.monotonic() - t0) * 1000
            return "unavailable", latency_ms, 0

    db_status, db_latency_ms, event_count = await asyncio.to_thread(_check_db)
    dlq_task_status = (
        "running" if (_dlq_task is not None and not _dlq_task.done()) else "stopped"
    )

    degraded_reasons: list[str] = []
    if db_latency_ms > 100:
        degraded_reasons.append("db_slow")
    now = time.monotonic()
    if _dlq_last_run > 0.0 and (now - _dlq_last_run) > _DLQ_INTERVAL * 2:
        degraded_reasons.append("dlq_loop_delayed")

    if db_status == "unavailable" or dlq_task_status == "stopped":
        overall = "unavailable"
    elif degraded_reasons:
        overall = "degraded"
    else:
        overall = "ok"

    return {
        "status": overall,
        "db": db_status,
        "db_latency_ms": round(db_latency_ms, 2),
        "dlq_task": dlq_task_status,
        "dlq_last_run_age_s": round(now - _dlq_last_run, 1) if _dlq_last_run > 0.0 else None,
        "event_count": event_count,
        "degraded_reasons": degraded_reasons,
    }
```
