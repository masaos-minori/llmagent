## Step 1: Strengthen existing fallback in _generate_session_title()

**Status:** Partially implemented (fallback exists but doesn't handle empty input)

**Target file:** `scripts/agent/commands/cmd_session.py`

**Current behavior (lines 39-46):**
```python
async def _generate_session_title(self, first_input: str) -> None:
    try:
        title = await self._title_service.generate(first_input)
        self._ctx.session.set_title(title)
    except SessionTitleGenerationError as e:
        logger.warning("Session title generation failed: %s", e)
        fallback_title = first_input[:32] if len(first_input) > 32 else first_input
        self._ctx.session.set_title(fallback_title)
```

**Changes needed:**

```python
async def _generate_session_title(self, first_input: str) -> None:
    try:
        title = await self._title_service.generate(first_input)
        if title and title.strip():
            self._ctx.session.set_title(title.strip())
        else:
            raise SessionTitleGenerationError("LLM returned empty title")
    except SessionTitleGenerationError as e:
        logger.info("Session title generation failed, using fallback: %s", e)
        # NEW: Handle empty/whitespace-only first input
        clean_input = first_input.strip() if first_input else ""
        if not clean_input:
            fallback_title = "(New Session)"
        elif len(clean_input) > 32:
            fallback_title = clean_input[:32] + "..."
        else:
            fallback_title = clean_input
        self._ctx.session.set_title(fallback_title)
```

**Completion criterion:** All failure paths produce a non-empty title; empty first input yields "(New Session)".

**Status:** Not implemented

title = sr.title if sr.title is not None else ""

**Changes needed in `_session_list()`:**

```python
async def _session_list(self) -> None:
    sessions = self._ctx.session.list_sessions()
    current_id = self._ctx.session.id if self._ctx.session else None
    
    rows = []
    for sr in sessions:
        is_current = sr.session_id == current_id
        
        # NEW: Check pending state for current session
        if is_current and self._ctx.session and self._ctx.session.is_title_pending():
            title = "(generating...)"
        elif sr.title is not None:
            title = sr.title
        else:
            title = "(no title)"  # NEW: explicit placeholder
        
        rows.append([
            str(sr.session_id),
            title,
            sr.last_activity.strftime("%Y-%m-%d %H:%M") if sr.last_activity else "",
            "*" if is_current else "",
        ])
    
    self._out.write_table(["Session ID", "Title", "Last Activity", ""], rows)
```

**Completion criterion:** Session list never shows blank title for current session; pending titles show "(generating...)".
