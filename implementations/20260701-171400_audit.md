# Implementation: Rewrite scripts/mcp/audit.py to emit JSON-lines format

## Goal

Replace the key=value AUDIT log format in `_audit_log()` with structured JSON-lines
so that MCP server audit logs are consistent with the agent-side JSON-lines audit format.

## Scope

- Target file: scripts/mcp/audit.py
- Only `_audit_log()` is changed; callers are not modified.

## Assumptions

1. The `action` parameter maps to `tool` field in the JSON output.
2. `detail` field is kept as a single string (not parsed into sub-fields).
3. Timestamp uses `time.time()` for epoch seconds (UTC implied by epoch).
4. Empty optional fields (`server_key`, `error_type`) are omitted from JSON output when empty.
5. `event` is always `"mcp_tool_exec"` and `source` is always `"mcp_server"`.
6. The output format is one JSON object per log call, terminated by the logger's newline.

## Implementation

### Target file: scripts/mcp/audit.py

#### Procedure

1. Add `import json` and `import time` to the imports.
2. Rewrite the `server_logger.info(...)` call body:
   - Build a `dict` with required fields: `event`, `source`, `ts`, `session`, `request`, `tool`, `target`, `outcome`.
   - Add `detail` to the dict only when non-empty.
   - Add `server_key` to the dict only when non-empty.
   - Add `error_type` to the dict only when non-empty.
   - Call `json.dumps(record, ensure_ascii=False)` and pass to `server_logger.info()`.
3. Update the docstring to reflect JSON-lines format.

#### Method

Direct rewrite of the function body. No interface changes (signature stays the same).

#### Details

New implementation pattern:
```python
import json
import time

def _audit_log(...) -> None:
    """Emit one JSON-lines audit record for an MCP tool execution."""
    record: dict[str, object] = {
        "event": "mcp_tool_exec",
        "source": "mcp_server",
        "ts": time.time(),
        "session": session_id or "-",
        "request": request_id or "-",
        "tool": action,
        "target": target,
        "outcome": outcome,
    }
    if detail:
        record["detail"] = detail
    if server_key:
        record["server_key"] = server_key
    if error_type:
        record["error_type"] = error_type
    server_logger.info(json.dumps(record, ensure_ascii=False))
```

Note: use `dict[str, object]` type annotation (Python 3.10+ compatible without `Dict`).
Do not use f-string formatting; use `json.dumps` only.
