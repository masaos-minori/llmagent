# Goal

Refactor `cmd_memory.py` to: replace unknown-subcommand silent fallback with
`UnknownSubcommandError`, use `MemoryAction` enum for action strings, and route
all `print()` calls through `OutputPort`.

# Scope

- `scripts/agent/commands/cmd_memory.py`

# Assumptions

1. `UnknownSubcommandError` and `MemoryAction` are in `agent/commands/exceptions.py` and `agent/commands/enums.py` (Step 1 prerequisite).
2. `OutputPort` from `output_port.py` (Step 1).
3. The local `MemoryOpResult` dataclass stays in this file (it's command-layer-specific); its `action: str` field changes to `action: MemoryAction | str` (StrEnum is a str subtype, so assignments remain valid).
4. `_emit_memory_audit` constructs a JSON payload — `result.action` becomes a StrEnum value which serializes as its string value via `orjson` (StrEnum → str is valid).
5. `_cmd_memory` line 89: `print_validation_error(f"Unknown subcommand: {sub!r}...")` → raise `UnknownSubcommandError(sub, tuple(dispatch.keys()))`.
6. The caller in `dispatch()` (registry.py) calls `_cmd_memory` without try/except — `UnknownSubcommandError` propagates to `AgentREPL` which should handle it gracefully. This is intentional fail-fast behavior.
7. `self._out` is available via `CommandRegistry` MRO.

# Implementation

## Target file

`scripts/agent/commands/cmd_memory.py`

## Procedure

1. Import `MemoryAction` from `agent.commands.enums`.
2. Import `UnknownSubcommandError` from `agent.commands.exceptions`.
3. Change `MemoryOpResult.action: str` to `action: MemoryAction | str`.
4. Replace `print_success(...)` / `print_validation_error(...)` imports with `self._out` calls.
5. Replace all bare `print(...)` with `self._out.write(...)`.
6. Replace `print(f"  [memory] No entries found")` etc. with `self._out.write(...)`.
7. In `_cmd_memory`: replace `print_validation_error(f"Unknown subcommand...")` with `raise UnknownSubcommandError(sub, tuple(dispatch.keys()))`.
8. In `_memory_pin`, `_memory_delete`: pass `MemoryAction.PIN` / `MemoryAction.DELETE` etc. instead of string literals.

## Method

Direct substitution for `print()` calls. Enum adoption for action constants.
Fail-fast for unknown subcommand.
