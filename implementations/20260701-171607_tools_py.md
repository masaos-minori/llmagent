# Implementation: Rename _MCP_TOOLS → TOOL_LIST in 9 tools.py files

## Goal

Rename the list variable `_MCP_TOOLS` to `TOOL_LIST` in all 9 MCP server tools.py
files so that every module exports the canonical public name.

## Scope

- Target files (all under scripts/mcp/):
  - mcp/git/tools.py (line 14)
  - mcp/mdq/tools.py (line 20)
  - mcp/rag_pipeline/tools.py (line 10)
  - mcp/shell/tools.py (line 10)
  - mcp/cicd/tools.py (line 10)
  - mcp/web_search/tools.py (line 10)
  - mcp/file/read_tools.py (line 47)
  - mcp/file/write_tools.py (line 10)
  - mcp/file/delete_tools.py (line 10)
- Only the variable name is changed; list contents are untouched.

## Assumptions

- `mcp/github/tools.py` already exports `TOOL_LIST` (confirmed); skip it.
- Each file has exactly one `_MCP_TOOLS` declaration.
- No file contains a re-export alias (`_MCP_TOOLS = TOOL_LIST`) that needs special handling.
- Type annotations on the line (e.g., `list[dict[str, Any]]`, `list[McpTool]`) are preserved as-is.

## Implementation

### Procedure

For each target file, apply a single rename:

```
_MCP_TOOLS: <type> = [   →   TOOL_LIST: <type> = [
_MCP_TOOLS = [            →   TOOL_LIST = [
```

No other lines in tools.py files reference `_MCP_TOOLS` (the list is defined once and
imported by server.py). Verify with grep after each file edit.

### Method

Use replace_all on each file: replace `_MCP_TOOLS` with `TOOL_LIST`.
Check that the replacement count is exactly 1 per file before saving.

### Details

- mcp/git/tools.py: type is `list[McpTool]`; rename only the variable name.
- mcp/mdq/tools.py: type is `list[MCPToolSchema]`; rename only the variable name.
- All others: type is `list[dict[str, Any]]` or untyped.
- After all renames: `rg "_MCP_TOOLS" scripts/mcp/*/tools.py scripts/mcp/file/*_tools.py`
  must return zero matches.

## Validation plan

| Step | Command | Expected outcome |
|---|---|---|
| Grep check | `rg -n "_MCP_TOOLS" scripts/mcp/` | Zero matches in tools.py files (server.py may still have them until Step 2) |
| Import test | `python -c "from mcp.git.tools import TOOL_LIST; print(len(TOOL_LIST))"` | Non-zero integer printed |
| Lint | `pre-commit run --files scripts/mcp/git/tools.py scripts/mcp/mdq/tools.py scripts/mcp/rag_pipeline/tools.py scripts/mcp/shell/tools.py scripts/mcp/cicd/tools.py scripts/mcp/web_search/tools.py scripts/mcp/file/read_tools.py scripts/mcp/file/write_tools.py scripts/mcp/file/delete_tools.py` | Pass |
