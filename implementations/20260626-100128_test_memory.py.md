# Implementation: Add regression tests for Persistent Semantic Memory key scenarios

Steps covered: Plan 20260626-100128 — Steps 1-6

---

## Goal

Add regression tests in `tests/agent/memory/` covering extraction quality, local-only guarantee, cross-session recall, context-boundary filtering, and dedup policy — pinning all key behaviors from plans 90-95.

---

## Scope

- **In scope**:
  - `tests/agent/memory/test_extract.py`: RULE/DECISION/FAILURE/CONVERSATION classification
  - `tests/agent/memory/test_retriever.py`: branch boundary tests
  - `tests/agent/memory/test_injection.py`: cross-session recall, min_importance filter
  - `tests/agent/memory/test_embedding_client.py`: local_only policy
  - `tests/agent/memory/test_ingestion.py`: dedup per source type
- **Out of scope**: performance benchmarks; large synthetic data

---

## Assumptions

- Plans 90-95 implemented before this step.
- In-memory SQLite for fast tests.
- Test pattern: one scenario per test function.

---

## Implementation

### Target files
5 test files in `tests/agent/memory/`

### Procedure
1. `test_extract.py`: Add 4 classification tests:
   - `"You must always use uv run pytest"` → `RULE`
   - `"We decided to use SQLite because trade-off is clear"` → `DECISION`
   - `"Fixed the regression by resetting root cause"` → `FAILURE`
   - `"That is a good question about the config"` → `CONVERSATION`

2. `test_retriever.py`: Add branch boundary test:
   - Insert memory with `branch="feat/x"`. Query with `branch="feat/y"` → not returned.
   - Insert memory with `branch=""`. Query with any branch → always returned.

3. `test_injection.py`: Add cross-session recall test:
   - Persist a DECISION memory with `importance=0.8`. Re-initialize injection service.
   - Call `on_session_start()` → assert memory is in injected context.
   - Also test: `min_importance=0.5` filters out `importance=0.3` entries.

4. `test_embedding_client.py`: Add local-only tests:
   - `local_only=True` + `http://external-api.com:8080` → `ValueError`.
   - `local_only=True` + `http://localhost:8080` → no error.
   - `local_only=False` + any URL → no error.

5. `test_ingestion.py`: Add dedup threshold tests:
   - CONVERSATION: similar text (similarity ~0.87) → deduped (threshold 0.85).
   - RULE: similar text (similarity ~0.95) → NOT deduped (threshold 0.98).

### Method
Pytest unit tests with in-memory SQLite.
