# Implementation: scripts/rag/cache.py

## Goal

Add `threading.RLock` to `SemanticCache` so that `lookup()`, `put()`, and `prune()` are safe under concurrent access from multiple async tasks sharing a single pipeline instance.

## Scope

- `scripts/rag/cache.py` — add `threading.RLock`; wrap method bodies with `with self._lock:`. No other files changed.

## Assumptions

1. All callers are within a single Python process but may run from multiple async tasks or threads.
2. `threading.RLock` is preferred over `asyncio.Lock`: operations are CPU-bound (no I/O), and using `asyncio.Lock` would require making methods `async`, breaking the `CacheService` Protocol.
3. `threading.RLock` (re-entrant) is used because `put()` calls `prune()` while holding the lock — re-entrant prevents deadlock.
4. `CacheService` Protocol remains unchanged (sync methods).
5. `prune()` must also be protected for external callers (even if currently only called from `put()`).

## Implementation

### Target file

`scripts/rag/cache.py`

### Procedure

1. Add `import threading` after `import logging`.
2. Add `self._lock: threading.RLock = threading.RLock()` to `SemanticCache.__init__()` after existing fields.
3. Wrap `lookup()` body in `with self._lock:`.
4. Wrap `put()` body in `with self._lock:`.
5. Wrap `prune()` body in `with self._lock:`.
6. Run ruff → mypy → tests.

### Method

Direct edit; minimal diff. `with self._lock:` wraps the full body of each method. Because `put()` calls `prune()` (while holding the lock), `threading.RLock` (not `threading.Lock`) is required to avoid deadlock on re-entry.

### Details

**`__init__` addition:**
```python
self._lock: threading.RLock = threading.RLock()
```

**`lookup()` after:**
```python
def lookup(self, embedding: list[float], history_context: str = "") -> str | None:
    with self._lock:
        if self._dim is not None and len(embedding) != self._dim:
            ...
        ...
        return None
```

**`put()` after:**
```python
def put(self, embedding: list[float], history_context: str, context_str: str) -> None:
    with self._lock:
        if self._dim is None:
            ...
        self._entries.append(...)
        self.prune()
```

**`prune()` after:**
```python
def prune(self) -> None:
    with self._lock:
        if len(self._entries) > self._max_size:
            self._entries = self._entries[-self._max_size :]
```
