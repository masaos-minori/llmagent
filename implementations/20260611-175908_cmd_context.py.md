# Goal

Refactor `cmd_context.py` to remove direct `print()` calls, replace `msg.get("content") or ""`
with type-correct access, and route output through `OutputPort`.

# Scope

- `scripts/agent/commands/cmd_context.py`

# Assumptions

1. `OutputPort` / `CliOutputPort` from `output_port.py` (Step 1 prerequisite).
2. `_print_token_line()` uses `print_kv_list` from `formatter.py` — replace with `self._out.write_kv(...)`.
3. `_cmd_history()` line 151: `content = msg.get("content") or ""` — `msg` is a `LLMMessage` TypedDict with `content: str | list[...]`; the `or ""` is a silent fallback for None. Replace with explicit None check: `content = msg.get("content") if msg.get("content") is not None else ""`.
4. `_cmd_context()` lines 104-107 use bare `print()` for budget breakdown loop — replace with `self._out.write(...)`.
5. `_cmd_clear()` line 118: `print(f" {result.message}")` — replace with `self._out.write_success(result.message)`.
6. `_cmd_system()` lines 164-165, 169: bare `print()` — replace with `self._out.write(...)`.
7. The `_print_token_line` helper uses `print_kv_list` import from formatter — replace direct calls with `self._out.write_kv(...)`.
8. `NothingToUndoError` and `undo_last_turn` are lazy imports — keep them lazy.

# Implementation

## Target file

`scripts/agent/commands/cmd_context.py`

## Procedure

1. Remove `from agent.commands.formatter import print_kv_list, print_no_data, print_success, print_validation_error`.
2. Add `from agent.commands.output_port import OutputPort` (type annotation only; `_out` provided by MRO from `CommandRegistry`).
3. Replace all `print_kv_list(...)` → `self._out.write_kv(...)`.
4. Replace all `print_success(...)` → `self._out.write_success(...)`.
5. Replace all `print_no_data(...)` → `self._out.write_no_data(...)`.
6. Replace all `print_validation_error(...)` → `self._out.write_validation_error(...)`.
7. Replace all bare `print(...)` → `self._out.write(...)`.
8. Fix `content = msg.get("content") or ""` → `content = msg.get("content") if msg.get("content") is not None else ""`.

## Method

Direct replacement — all `print_*` and `print()` calls are line-level substitutions.
`self._out` is available via MRO since `CommandRegistry.__init__` sets it.
