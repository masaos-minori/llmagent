# Implementation: Instrument `_execute_standard()` for serialization timing

## Goal

Every round of tool calls executed via `_execute_standard()` produces a `round_exec` audit event recording timing, mode (parallel/serial), side-effect status, and the triggering tool.

## Scope

- `scripts/agent/tool_runner.py` — add timing instrumentation to `_execute_standard()`

**Out of scope:** DAG-mode instrumentation (separate path), audit event function (separate doc).

## Assumptions

1. `write_round_exec()` already exists in `tool_audit.py` before this change.
2. The `AgentContext` object `ctx` is passed to `_execute_standard()` (confirmed — line 186).
3. `uuid4` is available for generating `round_id`.

## Implementation

### Target file

`scripts/agent/tool_runner.py`

### Procedure

1. Import `write_round_exec` from `agent.tool_audit` and `uuid4` from `uuid`.
2. At the start of `_execute_standard()`, record `round_id` and start time.
3. After the parallel/serial decision, record `has_side_effect` and `trigger_tool`.
4. After round completion, call `write_round_exec()` with elapsed time.

### Method

Surgical edits to `_execute_standard()`.

### Details

**Current signature and start (line 186-195):**


**Desired code:**

```python
async def _execute_standard(
    ctx: AgentContext,
    approved_calls: list[dict],
    turn: int,
) -> list[Any]:
    """Run approved calls in parallel, or serially when side effects are detected."""
    round_id = str(uuid4())
    t0 = time.perf_counter()
    trigger_tool: str | None = None
    has_side_effect = any(
        (tc["function"]["name"] if not trigger_tool else None)
        and (trigger_tool := tc["function"]["name"] or True)
        and is_side_effect(tc["function"]["name"])
        for tc in approved_calls
    )
    use_serial = ctx.cfg.tool.serial_tool_calls or has_side_effect
    mode = "serial" if use_serial else "parallel"
```

Wait — the walrus operator approach is fragile. Use a simple for-loop:

```python
    round_id = str(uuid4())
    t0 = time.perf_counter()
    has_side_effect = False
    trigger_tool: str | None = None
    for tc in approved_calls:
        if is_side_effect(tc["function"]["name"]):
            has_side_effect = True
            trigger_tool = tc["function"]["name"]
            break
    use_serial = ctx.cfg.tool.serial_tool_calls or has_side_effect
    mode = "serial" if use_serial else "parallel"
```

**After the round completes (before return), add:**

```python
    elapsed_ms = (time.perf_counter() - t0) * 1000
    write_round_exec(
        ctx,
        round_id=round_id,
        tool_count=len(approved_calls),
        mode=mode,
        has_side_effect=has_side_effect,
        trigger_tool=trigger_tool,
        elapsed_ms=elapsed_ms,
    )
```
