# Implementation: stat_embed_skip counter in ingestion.py

## Goal
Add `stat_embed_skip: int` counter to `MemoryIngestionService` that increments whenever
`_persist_entry()` stores a memory entry without an embedding (embedding failed), and log
the count in the `on_session_stop()` summary.

## Scope
- File: `scripts/agent/memory/ingestion.py`
- Modify `MemoryIngestionService.__init__()`, `_persist_entry()`, `on_session_stop()`

## Assumptions
- `_persist_entry()` is the canonical write path for manual writes; embedding failure
  is detected via `embed_result.success is False`
- `_persist_entry_with_dedup()` also calls `_embed_client.fetch()`; its embed failure
  path should also be counted (both paths go through the same embed client)
- `stat_embed_skip` is instance-level (session-scoped), not persisted to DB
- Counter starts at 0 on construction; not reset between calls within a session

## Implementation

### Target file
`scripts/agent/memory/ingestion.py`

### Procedure
1. Add `self.stat_embed_skip: int = 0` to `__init__()`
2. Increment in `_persist_entry()` when `embed_result.success` is False
3. Increment in `_persist_entry_with_dedup()` when `embed_result.success` is False
4. Log counter in `on_session_stop()` summary

### Method
Four-step edit.

### Details

**Step 1 — `__init__()`:**
Add after the existing instance variables:
```python
self.stat_embed_skip: int = 0
```

**Step 2 — `_persist_entry()` (manual write path):**
```python
async def _persist_entry(self, entry: MemoryEntry) -> None:
    """..."""
    embed_result = await self._embed_client.fetch(entry.content)
    embedding = embed_result.embedding if embed_result.success else None
    if not embed_result.success:
        self.stat_embed_skip += 1
        logger.info(
            "memory.embed_skip: stored without embedding "
            "(reason=%s) memory_id=%r",
            embed_result.error_kind,
            entry.memory_id,
        )
    await self._jsonl.write(entry)
    self._store.upsert(entry, embedding=embedding)
    logger.info(
        "memory.write memory_id=%r type=%s importance=%.2f",
        entry.memory_id, entry.memory_type, entry.importance,
    )
```

**Step 3 — `_persist_entry_with_dedup()` (session-stop write path):**
```python
async def _persist_entry_with_dedup(self, entry: MemoryEntry) -> None:
    """..."""
    embed_result = await self._embed_client.fetch(entry.content)
    embedding = embed_result.embedding if embed_result.success else None
    if not embed_result.success:
        self.stat_embed_skip += 1
        logger.info(
            "memory.embed_skip: stored without embedding "
            "(reason=%s) memory_id=%r",
            embed_result.error_kind,
            entry.memory_id,
        )
    if self._should_skip_dedup(embed_result, entry.memory_id):
        return
    # ... rest unchanged
```

**Step 4 — `on_session_stop()` summary:**
After the for loop:
```python
logger.info(
    "MemoryIngestionService.on_session_stop: persisted %d entries; %d embed_skipped",
    len(entries),
    self.stat_embed_skip,
)
```
(replace the existing `logger.info("MemoryIngestionService.on_session_stop: persisted entries")`)
