# Implementation Procedure: scripts/rag/ingestion/ingester.py

## Goal

Extend `RagIngester` in `ingester.py` to support an optional strict artifact validation mode
(Plan 142134 Phase 3). Specifically:
- Add `_strict_artifact_validation: bool` attribute in `__init__`
- Change `_validate_artifact` signature to accept `strict: bool = False`
- Apply strict presence check for `schema_version`, `artifact_type`, `created_by` when
  `strict=True`
- Update both call sites to pass `self._strict_artifact_validation`

## Scope

**In scope:**
- `RagIngester.__init__`: add `_strict_artifact_validation` attr
- `RagIngester._validate_artifact`: extend signature and logic
- `ingest_url_group`: update call site
- `_embed_and_store`: update call site

**Out of scope:**
- Any other method in `ingester.py`
- `crawler.py` or `chunk_splitter.py`

## Assumptions

1. `_validate_artifact` is currently a `@staticmethod`; it stays static â€” `strict` is passed
   explicitly by callers, no `self` reference needed inside the method.
2. The method currently checks `artifact_type` against `expected_type` and returns `bool`.
   Adding `strict` does not change the return type or existing checks.
3. `cfg.get("strict_artifact_validation", False)` retrieves the new config key safely with a
   `False` default if the key is absent (backward compatibility).
4. `ingest_url_group` and `_embed_and_store` are the only two call sites â€” confirm by running
   `grep -n "_validate_artifact" scripts/rag/ingestion/ingester.py` before editing.

## Implementation

### Target file

`scripts/rag/ingestion/ingester.py`

### Procedure

1. Run `grep -n "_validate_artifact" scripts/rag/ingestion/ingester.py` to locate all call
   sites.
2. Read lines around `_validate_artifact` definition and each call site.
3. Apply the following changes:

**`__init__`:**
```python
self._strict_artifact_validation: bool = bool(
    cfg.get("strict_artifact_validation", False)
)
```

**`_validate_artifact` signature and logic:**
```python
@staticmethod
def _validate_artifact(
    payload: ChunkJsonRaw,
    expected_type: str,
    *,
    strict: bool = False,
) -> bool:
    if strict:
        for required in ("schema_version", "artifact_type", "created_by"):
            if required not in payload:
                return False
    # existing artifact_type mismatch check (unchanged)
    if payload.get("artifact_type") != expected_type:
        return False
    return True
```

**Call sites (both identical change):**
```python
# before:
self._validate_artifact(first_data, "chunk")
# after:
self._validate_artifact(first_data, "chunk", strict=self._strict_artifact_validation)
```

4. Run `ruff check scripts/rag/ingestion/ingester.py` and `mypy scripts/rag/ingestion/ingester.py`.
