# Implementation: scripts/db/create_schema.py — Remove _migrate_* functions (DDL-only cleanup)

## Goal

## Assumptions

1. `db/schema_sql.py` の DDL には `chunk_type`, `source_file` (`chunks` テーブル), `undone` (`tool_results`), FK `REFERENCES sessions(session_id)` (`tool_results`), `workflow_id` (`tasks`) が既に含まれている。含まれていない場合は本フェーズより前に DDL を修正する別コミットが必要。
2. 本番 DB は全て移行済みであり、`_migrate_*` 関数は実質的なデッドコードである。
3. `sqlite3` (例外型) および `sys` (`sys.exit`) のインポートは削除後も引き続き必要。
4. `db.store_protocols.get_embedding_dims` および `db.helper.SQLiteHelper` のインポートは引き続き必要。削除対象は migration helper の呼び出しのみ。
