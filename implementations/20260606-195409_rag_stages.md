# Implementation: RAG Pipeline Stage Decomposition

## Goal

Introduce a `PipelineStage` Protocol and extract the existing MQE → Search → RRF → Rerank → Augment pipeline steps in `rag/pipeline.py` into individual stage classes under `rag/stages/`.

## Scope

**In:**
- `PipelineStage` Protocol in `scripts/rag/stage.py`
- `PipelineContext` dataclass to pass state between stages
- Stage classes: `MqeStage`, `SearchStage`, `RrfStage`, `RerankStage`, `AugmentStage`
- Optional `RefineStage` (if `use_refiner` is True)
- Each stage wraps its logic in an OTel span using `shared/otel_tracer.py`
- `RagPipeline.run()` orchestrates stages via `PipelineContext`

**Out:**
- Moving stage classes into a separate package (keep in `rag/stages/` directory, not a sub-package)
- Parallel stage execution (stages run sequentially, same as today)
- `/rag search` CLI command (add in a follow-up; out of scope for refactoring)

## Assumptions

- `scripts/rag/pipeline.py`: MQE is in-place (expand queries method); Search = `search_queries()`; RRF = `RagScorer.rrf_merge()`; Rerank = `rerank_candidates()`; Augment = `_format_chunks()`
- `scripts/shared/otel_tracer.py`: `build_tracer()` returns a tracer; use `tracer.start_as_current_span(name)` context manager
- Existing tests `test_rag_pipeline.py` will serve as behavior-lock tests during refactoring

## Implementation

### 1. `scripts/rag/stage.py` — Protocol and context

```python
"""rag/stage.py
PipelineStage Protocol and shared PipelineContext dataclass.
"""
from __future__ import annotations
import dataclasses
from typing import Any, Protocol


@dataclasses.dataclass
class PipelineContext:
    """Mutable state passed between pipeline stages."""
    query: str
    history_context: str = ""
    queries: list[str] = dataclasses.field(default_factory=list)
    search_results: list[Any] = dataclasses.field(default_factory=list)  # list[list[RagHit]]
    merged: list[Any] = dataclasses.field(default_factory=list)          # list[RagHit]
    reranked: list[Any] = dataclasses.field(default_factory=list)        # list[RagHit]
    augment_result: str = ""


class PipelineStage(Protocol):
    """Synchronous or async pipeline stage — run modifies ctx in-place."""

    async def run(self, ctx: PipelineContext, **kwargs: Any) -> None: ...
```

### 2. `scripts/rag/stages/` — individual stage files

Create `scripts/rag/stages/__init__.py` (empty) and one file per stage.

**`scripts/rag/stages/mqe.py`** — query expansion:
```python
class MqeStage:
    def __init__(self, cfg, llm) -> None: ...
    async def run(self, ctx: PipelineContext, **kwargs) -> None:
        # Moves MQE logic from RagPipeline._expand_queries() / run()
        ctx.queries = await _run_mqe(ctx.query, self._cfg, self._llm)
```

**`scripts/rag/stages/search.py`** — embedding + BM25 search:
```python
class SearchStage:
    async def run(self, ctx: PipelineContext, db=None, **kwargs) -> None:
        # Moves logic from RagPipeline.search_queries()
        ctx.search_results = await _search_all_queries(ctx.queries, db, self._cfg)
```

**`scripts/rag/stages/fusion.py`** — RRF merge:
```python
class FusionStage:
    async def run(self, ctx: PipelineContext, **kwargs) -> None:
        ctx.merged = RagScorer.rrf_merge(ctx.search_results, k=self._rrf_k)
```

**`scripts/rag/stages/rerank.py`** — Cross-Encoder:
```python
class RerankStage:
    async def run(self, ctx: PipelineContext, **kwargs) -> None:
        ctx.reranked = await _rerank(ctx.query, ctx.merged, self._cfg, self._llm)
```

**`scripts/rag/stages/augment.py`** — format output:
```python
class AugmentStage:
    async def run(self, ctx: PipelineContext, **kwargs) -> None:
        ctx.augment_result = _format_chunks(ctx.reranked)
```

### 3. `scripts/rag/pipeline.py` — update `run()` to use stages

```python
from rag.stage import PipelineContext
from rag.stages.mqe import MqeStage
from rag.stages.search import SearchStage
from rag.stages.fusion import FusionStage
from rag.stages.rerank import RerankStage
from rag.stages.augment import AugmentStage

async def run(self, query: str, db, history_context: str = "") -> ...:
    ctx = PipelineContext(query=query, history_context=history_context)
    tracer = self._tracer
    stages: list[PipelineStage] = [
        MqeStage(self._cfg, self._llm),
        SearchStage(self._cfg),
        FusionStage(self._cfg),
        RerankStage(self._cfg, self._llm),
        AugmentStage(),
    ]
    for stage in stages:
        with tracer.start_as_current_span(type(stage).__name__):
            await stage.run(ctx, db=db)
    # Preserve the existing return signature for backward compat:
    return ctx.queries, ctx.search_results, ctx.merged, ctx.reranked
```
