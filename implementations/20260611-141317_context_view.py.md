# Implementation: agent/services/context_view.py — ContextStateView DTO + remove token fallback

## Goal

1. Change `collect_context_state()` return type from `dict` to `ContextStateView` DTO.
2. Remove the `total_chars // 4` token fallback when `hist_mgr is None`.
3. Make `hist_mgr is None` a `ContextStateBuildError` (fail-fast).
4. Remove `str(m.get("content") or "")` in `budget_breakdown()` with explicit typed access.

## Scope

**Target file**: `scripts/agent/services/context_view.py`

In scope:
- `collect_context_state()` return type: `dict` → `ContextStateView`
- `hist_mgr is None` → raise `ContextStateBuildError`
- `total_chars // 4` fallback → removed
- `str(m.get("content") or "")` in `budget_breakdown()` → replaced

Out of scope:
- `budget_breakdown()` Enum changes (plan notes this as optional)
- `_format_memory_status()` (no change required)

## Assumptions

1. `ContextStateView` is `@dataclass(frozen=True)` in `models.py` with fields:
   `total_chars, compress_limit, n_msgs, sys_preview, compress_count, token_is_exact,
   token_estimate, token_limit, tokenize_configured, mem_status, git_branch, git_commit, breakdown`.
2. `ContextStateBuildError(RuntimeError)` is in `exceptions.py`.
3. `hist_mgr is None` is a programming error in production; it can occur in tests where
   mock contexts don't set `hist_mgr`. Tests must mock `hist_mgr` properly.
4. `git_info` from `get_repo_info()` returns `{"branch": str, "commit": str, "message": str} | None`.
   `git_branch` and `git_commit` are `str | None`.
5. Callers of `collect_context_state()` (in `cmd_context.py`) that currently access
   dict keys must be updated to access DTO fields (e.g., `state["total_chars"]` → `state.total_chars`).
6. `budget_breakdown()` return type remains `dict[str, int]` — it is presentation data.

### Procedure

**Update `budget_breakdown()`** — remove `str(m.get("content") or "")`:
```python
def budget_breakdown(messages: list[LLMMessage]) -> dict[str, int]:
    counts: dict[str, int] = {"system": 0, "history": 0, "tool_results": 0}
    for m in messages:
        role = m.get("role", "")
        content = m.get("content") or ""
        text = content if isinstance(content, str) else str(content)
        tool_calls = m.get("tool_calls") or []
        if role == "system":
            counts["system"] += len(text)
        elif role == "tool":
            counts["tool_results"] += len(text)
        elif role == "assistant":
            counts["history"] += len(text)
            if tool_calls:
                counts["tool_results"] += len(orjson.dumps(tool_calls))
        else:
            counts["history"] += len(text)
    return counts
```

**Update `collect_context_state()`**:
```python
from agent.services.exceptions import ContextStateBuildError
from agent.services.models import ContextStateView

def collect_context_state(ctx: AgentContext) -> ContextStateView:
    if ctx.services.hist_mgr is None:
        raise ContextStateBuildError("hist_mgr is not configured")
    total_chars = ctx.services.hist_mgr.count_chars(ctx.conv.history)
    system_msgs = [m for m in ctx.conv.history if m["role"] == "system"]
    sys_preview = str(system_msgs[0].get("content", ""))[:80] if system_msgs else ""
    compress_count = ctx.services.hist_mgr.stat_compress_count
    token_is_exact = ctx.stats.stat_input_tokens is not None
    token_estimate = ctx.services.hist_mgr.count_tokens(
        ctx.conv.history, ctx.stats.stat_input_tokens
    )
    git_info = get_repo_info()
    return ContextStateView(
        total_chars=total_chars,
        compress_limit=ctx.cfg.llm.context_char_limit,
        n_msgs=len(ctx.conv.history),
        sys_preview=sys_preview,
        compress_count=compress_count,
        token_is_exact=token_is_exact,
        token_estimate=token_estimate,
        token_limit=ctx.cfg.llm.context_token_limit,
        tokenize_configured=bool(ctx.cfg.llm.tokenize_url),
        mem_status=_format_memory_status(ctx),
        git_branch=git_info["branch"] if git_info else None,
        git_commit=git_info["commit"] if git_info else None,
        breakdown=budget_breakdown(ctx.conv.history),
    )
```

Update `cmd_context.py` to access DTO fields instead of dict keys.

### Method

`Edit` tool for `context_view.py`. Update `cmd_context.py` with `Edit` tool.

### Details

- The `git_str` key in the old dict is replaced by separate `git_branch` and `git_commit` fields.
  The command handler constructs the display string from these fields.
- Remove the `else` branch for `hist_mgr is None` from `collect_context_state()` entirely.
- `token_estimate` is `int | None`; it remains `None` if `count_tokens()` returns `None`.
