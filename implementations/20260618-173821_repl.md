# Implementation: Unify `repl.py` diagnostics to use `DiagnosticStore`

## Goal

Change `_persist_session_diagnostics()` in `repl.py` to also write the runtime diagnostics summary to `DiagnosticStore`, in addition to the existing `diagnostics.jsonl` write. Eventually the JSONL path may be removed, but initially both paths coexist.

## Scope

- `scripts/agent/repl.py` — update `_persist_session_diagnostics()`

**Out of scope:** Removing the JSONL file path.

## Assumptions

1. `DiagnosticStore` is importable from `agent.diagnostic_store`.
2. The JSONL file is the current source of truth; DiagnosticStore is an additional queryable channel.

## Implementation

### Target file

`scripts/agent/repl.py`

### Procedure

1. Import `DiagnosticStore`.
2. At the end of `_persist_session_diagnostics()`, after the JSONL write, call `self._diagnostic_store.save()` with the summary JSON.

### Method

Add import and a few lines at the end of the method.

### Details

**Current code (repl.py:208-211):**
```python
db_cfg = build_db_config()
diag_path = Path(db_cfg.session_db_path).parent / "diagnostics.jsonl"
with open(diag_path, "a") as f:
    f.write(json.dumps(summary) + "\n")
```

**Desired code:**
```python
db_cfg = build_db_config()
diag_path = Path(db_cfg.session_db_path).parent / "diagnostics.jsonl"
with open(diag_path, "a") as f:
    f.write(json.dumps(summary) + "\n")

# Also persist to queryable DiagnosticStore
try:
    self._diagnostic_store.save(
        session_id,
        kind="session_summary",
        content=json.dumps(summary),
    )
except (RuntimeError, sqlite3.Error) as e:
    logger.debug("DiagnosticStore.save failed: %s", e)
```

Add `self._diagnostic_store = DiagnosticStore()` in `ReplAgent.__init__()`.
