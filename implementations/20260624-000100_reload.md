# Implementation: Enhance /reload Scope and Fix Configuration Shadowing

## Goal

1. Make `/reload` reflect `security.toml` changes at runtime (currently `security_profile` and `security_lockdown_enabled` are loaded from TOML but not applied to `ctx.cfg`)
2. Document HTTP config resolution precedence in code

## Scope

**In:**
- `scripts/agent/services/config_reload.py` — add `security_profile` + `security_lockdown_enabled` to `_reload_approval_settings()`
- `scripts/shared/config_loader.py` — add HTTP precedence comment to `_BASE_CONFIG_FILES`
- `scripts/agent/commands/cmd_config.py` — fix `_cmd_reload()` docstring

**Out:**
- Restructuring `_BASE_CONFIG_FILES` ordering
- Centralizing HTTP config into a single file (deferred)

## Assumptions

1. `SecurityProfile` is an enum in `shared/mcp_config.py`; `_get_str()` returns the raw string from TOML, which maps directly to `SecurityProfile(v)` constructor
2. `security_lockdown_enabled` is `False` by default and toggling it is safe at runtime
3. The `_apply_bool` / `_get_bool` helpers are already in scope at `_reload_approval_settings()` — no new imports needed
4. HTTP precedence comment is informational only — no code change to resolution logic needed

## Implementation

### Target file: `scripts/agent/services/config_reload.py`

**Procedure:** Add security fields handling to `_reload_approval_settings()`.

**Method:** Insert two new if-blocks after the existing `memory_retention_days` block.

**Details:**
```python
def _reload_approval_settings(
    self,
    ctx: AgentContext,
    new_cfg: dict[str, Any],
) -> None:
    """Update approval, tool, and memory config fields when present in new_cfg."""
    self._reload_approval_config(ctx, new_cfg)
    if (lst := _get_list(new_cfg, "allowed_tools")) is not None:
        ctx.cfg.tool.allowed_tools = list(lst)
    if (v := _get_int(new_cfg, "memory_retention_days")) is not None:
        ctx.cfg.memory.memory_retention_days = v
    # security.toml fields — hot-reloadable
    if (v := _get_str(new_cfg, "security_profile")) is not None:
        try:
            from shared.mcp_config import SecurityProfile
            ctx.cfg.tool.security_profile = SecurityProfile(v)
        except ValueError:
            pass  # invalid enum value — leave current
    if (vb := _get_bool(new_cfg, "security_lockdown_enabled")) is not None:
        ctx.cfg.tool.security_lockdown_enabled = vb
```
