# Implementation: mixin_base.py — Promote _reset_session_stats to module function

## Goal

Extract `MixinBase._reset_session_stats()` to a module-level function `reset_session_stats(ctx)`
so it can be imported by new service modules (`session_restore.py`) without coupling to MixinBase.

## Scope

- `scripts/agent/commands/mixin_base.py`: add module-level `reset_session_stats(ctx)` function;
  keep `MixinBase._reset_session_stats` as a thin delegate for backward compatibility.
- `scripts/agent/commands/cmd_context.py`: call `reset_session_stats(ctx)` instead of `self._reset_session_stats(ctx)`.
- `scripts/agent/commands/cmd_session.py`: same update.

## Assumptions

1. `_reset_session_stats` is called in exactly two places: `_ContextMixin._cmd_clear()` and `_SessionMixin._load_session()`.
2. The method signature `(ctx: AgentContext) -> None` is unchanged.
3. Keeping the `MixinBase._reset_session_stats` wrapper avoids breaking any external callers or tests that use the method via `MixinBase`.

## Implementation

### Target file

`scripts/agent/commands/mixin_base.py`

### Procedure

1. Before the `MixinBase` class definition, add module-level function:

```python
def reset_session_stats(ctx: AgentContext) -> None:
    """Reset all per-session counters to zero.

    Used by _cmd_clear, _load_session, and session_restore service.
    """
    ctx.stats.stat_turns = 0
    ctx.stats.stat_tool_calls = 0
    ctx.stats.stat_tool_errors = 0
    ctx.stats.stat_latency = {}
    ctx.stats.stat_semantic_cache_hits = 0
    if ctx.services.llm is not None:
        ctx.services.llm.stat_retries = 0
```

2. Change `MixinBase._reset_session_stats` to delegate:

```python
def _reset_session_stats(self, ctx: AgentContext) -> None:
    reset_session_stats(ctx)
```

3. Update `cmd_context.py`: `self._reset_session_stats(ctx)` → `reset_session_stats(ctx)`
   (add `from agent.commands.mixin_base import reset_session_stats`).

4. Update `cmd_session.py`: same change.

### Method

Edit `mixin_base.py`, then update two callers.
