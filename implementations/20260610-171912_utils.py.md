# Implementation: rag/utils.py ‚Äî Fail-Fast Validation

## Goal

Remove `except Exception` from `validate_url` (it never throws); strengthen `_validate_float_list` to reject every non-numeric element and explicitly reject NaN/inf values.

---

## Scope

**Target file:** `scripts/rag/utils.py`

**In:**
- `validate_url`: remove try/except entirely ‚Äî `urlparse` never raises on a string input
- `_validate_float_list`: iterate all elements, not just the first; add `math.isfinite` check for NaN/inf
- `floats_to_blob`: rely on the strengthened `_validate_float_list`

**Out:**
- `normalize_unicode` ‚Äî no changes needed
- No changes to callers

---

## Assumptions

1. `urllib.parse.urlparse` does not raise on a `str` argument ‚Äî the `except Exception` in the current code is unreachable dead code.
2. Rejecting NaN/inf is correct ‚Äî `sqlite-vec` stores IEEE 754 float32, so NaN/inf embeddings would produce undefined similarity scores.
3. Full-list validation O(N) cost is acceptable ‚Äî embeddings are 384-dim (small), and this runs once per query.

---

## Implementation

### Target file
`scripts/rag/utils.py`

### Procedure

1. Import `math` at the top.
2. Rewrite `validate_url` without try/except ‚Äî call `urlparse` directly and return the boolean check.
3. In `_validate_float_list`, replace the single-element check with a full-list loop that verifies `isinstance(v, int | float)` and `math.isfinite(v)` for each element.
4. Keep `floats_to_blob` unchanged (it calls `_validate_float_list` then `struct.pack`).

### Method

```python
import math
import struct
import unicodedata
from urllib.parse import urlparse

def validate_url(url: str) -> bool:
    parsed = urlparse(url)
    return parsed.scheme in ("http", "https") and bool(parsed.netloc)

def _validate_float_list(values: list[float]) -> None:
    if not isinstance(values, list):
        raise TypeError(
            f"floats_to_blob expects list[float], got {type(values).__name__}",
        )
    if not values:
        raise ValueError("floats_to_blob received an empty list.")
    for i, v in enumerate(values):
        if not isinstance(v, int | float):
            raise ValueError(
                f"floats_to_blob: element {i} must be numeric, got {type(v).__name__}",
            )
        if not math.isfinite(v):
            raise ValueError(
                f"floats_to_blob: element {i} is not finite ({v!r})",
            )
```

### Details

- Removing try/except from `validate_url` eliminates dead code and makes the function always return `bool` without swallowing unexpected errors.
- The full-element check is O(N) but N=384 (E5 model dimension) ‚Äî negligible overhead.
- `math.isfinite(v)` returns `False` for both `math.nan` and `math.inf` / `-math.inf`.
- `struct.pack` with NaN would succeed (IEEE 754 allows NaN bytes) but would corrupt the sqlite-vec index silently ‚Äî rejecting early is correct.
