## Scope

- Target: `scripts/agent/factory.py`
- Remove `_transport_state_to_lifecycle()` — after `stdio_lifecycle.py` uses `LifecycleState` natively
- Remove `from agent.stdio_lifecycle import ... TransportState` import (no longer needed)
- In `get_transport_state()`, call `self._stdio_mgr.get_transport_state()` directly (now returns `LifecycleState`)
- Change `stdio_procs: dict` to `stdio_procs: dict[str, StdioTransport]`
- `_build_tool_executor()` parameter `stdio_procs: dict` → `dict[str, StdioTransport]`

## Assumptions

1. `stdio_lifecycle.py` has been updated to use `LifecycleState` (dependency: `stdio_lifecycle_py.md`).
2. `_ServerLifecycleRouter` is the only implementation of `LifecycleManagerProtocol`; no other callers depend on `TransportState`.
3. `build_agent_context()` interface remains unchanged (called by `repl.py`).

## Implementation

### Target file

`scripts/agent/factory.py`

### Procedure

1. Remove the import line: `from agent.stdio_lifecycle import StdioServerLifecycleManager, TransportState`.
   - Add: `from agent.stdio_lifecycle import StdioServerLifecycleManager`.
2. Delete `_transport_state_to_lifecycle()` function.
3. In `_ServerLifecycleRouter.get_transport_state()`: change `raw = self._stdio_mgr.get_transport_state(server_key)` and remove `return _transport_state_to_lifecycle(raw)` — return `self._stdio_mgr.get_transport_state(server_key)` directly (now returns `LifecycleState`).
4. Change `stdio_procs: dict = {}` in `build_agent_context()` to `stdio_procs: dict[str, StdioTransport] = {}`.
5. Change `_build_tool_executor(ctx, http, stdio_procs: dict)` parameter type to `stdio_procs: dict[str, StdioTransport]`.

### Method

Cleanup of the adapter that existed only because `TransportState ≠ LifecycleState`. After unification, no mapping is needed.

### Details

```python
# Before
from agent.stdio_lifecycle import StdioServerLifecycleManager, TransportState

def _transport_state_to_lifecycle(state: TransportState) -> LifecycleState: ...

def get_transport_state(self, server_key: str) -> LifecycleState:
    raw = self._stdio_mgr.get_transport_state(server_key)
    return _transport_state_to_lifecycle(raw)

# After
from agent.stdio_lifecycle import StdioServerLifecycleManager

def get_transport_state(self, server_key: str) -> LifecycleState:
    cfg = self._server_configs.get(server_key)
    if cfg is None:
        return LifecycleState.UNKNOWN
    if cfg.transport == "stdio":
        return self._stdio_mgr.get_transport_state(server_key)
    return LifecycleState.UNKNOWN
```
