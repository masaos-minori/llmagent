# Implementation: formatter.py (new) + registry.py CommandDef extension

## Goal

1. Create `scripts/agent/commands/formatter.py` with shared output helpers used by all command mixins.
2. Extend `CommandDef` in `registry.py` with `subcommands: list[SubcommandSpec]` so subcommand
   help can be auto-generated from the spec.

## Scope

- `scripts/agent/commands/formatter.py` — new file
- `scripts/agent/commands/registry.py` — add `SubcommandSpec` and `subcommands` field to `CommandDef`
- No existing command files are changed in this step

## Assumptions

1. `formatter.py` is pure output helpers; no imports from `agent.context` or business logic.
2. `CommandDef.subcommands` defaults to empty list — backward-compatible, no existing code breaks.
3. `print_table` accepts `headers: list[str]` and `rows: list[list[str]]`, prints a plain-text
   table with fixed-width columns computed from max column width.
4. `print_kv_list` accepts `pairs: list[tuple[str, str]]` and prints `  key : value` aligned.

### Details

`formatter.py`:
```python
"""agent/commands/formatter.py
Shared output helpers for command mixins.

All display output from command handlers should go through these functions
to ensure consistent success/error/table formatting across commands.
"""
from __future__ import annotations


def print_success(msg: str) -> None:
    print(f"  {msg}")


def print_error(msg: str) -> None:
    print(f"  [error] {msg}")


def print_no_data(msg: str) -> None:
    print(f"  {msg}")


def print_validation_error(msg: str) -> None:
    print(f"  [usage] {msg}")


def print_table(headers: list[str], rows: list[list[str]]) -> None:
    if not rows:
        return
    widths = [max(len(h), max(len(r[i]) for r in rows)) for i, h in enumerate(headers)]
    sep = "  ".join(h.ljust(w) for h, w in zip(headers, widths))
    print(sep)
    print("-" * len(sep))
    for row in rows:
        print("  ".join(cell.ljust(w) for cell, w in zip(row, widths)))


def print_kv_list(pairs: list[tuple[str, str]], key_width: int = 22) -> None:
    for k, v in pairs:
        print(f"  {k:<{key_width}}: {v}")
```

`registry.py` additions:
```python
@dataclass
class SubcommandSpec:
    """Declarative metadata for one subcommand."""
    name: str
    help: str

@dataclass
class CommandDef:
    name: str
    prefix: bool
    is_async: bool
    handler: str
    help: str
    subcommands: list[SubcommandSpec] = field(default_factory=list)
```
