# Implementation: jsonl_store.py — Remove append() and source_type fallback

## Goal

Remove the synchronous `append()` method from `JsonlMemoryStore` and the `source_type` silent
fallback in `_entry_from_dict()`. The async `write()` becomes the single JSONL write path.

## Scope

- `scripts/agent/memory/jsonl_store.py`: remove `append()` method; remove `source_type` fallback in `_entry_from_dict()`.
- `tests/test_memory_jsonl.py`: migrate all `store.append(...)` calls to `asyncio.run(store.write(...))` or `await store.write(...)` with pytest-asyncio.

## Assumptions

1. `JsonlMemoryStore.append()` is called only in `tests/test_memory_jsonl.py` (grep confirmed; zero production callers).
2. The async `write()` method is already correct and uses `asyncio.Lock` for serialization.
3. `MemoryEntry.__post_init__` already validates `source_type` and raises `ValueError` for unknown values; removing the fallback in `_entry_from_dict()` delegates to this existing validation.
4. When `_entry_from_dict()` encounters a bad `source_type`, `MemoryEntry(...)` raises `ValueError`, which is caught by the `except Exception` block and returns `None` (skip behavior is maintained, but reason is now explicit).

## Implementation

### Target files

- `scripts/agent/memory/jsonl_store.py`
- `tests/test_memory_jsonl.py`

### Procedure

**jsonl_store.py:**

1. In `_entry_from_dict()`, remove the `try/except ValueError` block that falls back to `SourceType.CONVERSATION`:
   - Remove: `try: source_type = SourceType(raw_source) except ValueError: logger.warning(...); source_type = SourceType.CONVERSATION`
   - Replace with: pass `source_type=d.get("source_type", "conversation")` directly into `MemoryEntry()`; let `__post_init__` validate and raise.
2. Remove the entire `append()` method (lines 117–123).

**tests/test_memory_jsonl.py:**

3. For each `store.append(entry)` call, replace with an async equivalent:
   - If the test is already `async def`, replace with `await store.write(entry)`.
   - If the test is `def` (sync), add `@pytest.mark.asyncio` and make it `async def`, then use `await store.write(entry)`.

### Method

- Edit `jsonl_store.py`: two targeted removals.
- Edit `tests/test_memory_jsonl.py`: replace all `append` calls with `write`.

### Details

**_entry_from_dict() before:**
```python
raw_source = d.get("source_type", "conversation")
try:
    source_type = SourceType(raw_source)
except ValueError:
    logger.warning(
        f"Unknown source_type={raw_source!r}; falling back to conversation"
    )
    source_type = SourceType.CONVERSATION
return MemoryEntry(
    ...
    source_type=source_type,
    ...
)
```

**_entry_from_dict() after:**
```python
return MemoryEntry(
    ...
    source_type=d.get("source_type", "conversation"),
    ...
)
# MemoryEntry.__post_init__ raises ValueError for unknown source_type;
# caught by outer except Exception → returns None (skip)
```

**append() removal:** delete the entire method:
```python
def append(self, entry: MemoryEntry) -> None:
    """Append one entry as a single JSONL line."""
    self._path.parent.mkdir(parents=True, exist_ok=True)
    line: bytes = orjson.dumps(asdict(entry)) + b"\n"
    with self._path.open("ab") as f:
        f.write(line)
    logger.debug(f"JSONL appended memory_id={entry.memory_id!r}")
```
