# Implementation: Improve Ingestion and Query Diagnostics — CLI View + Cmd Ingest

## Goal

Wire `debug_fn` through the REPL `/rag search --debug` path so `write_debug_rag()` is called with pipeline debug data.

## Scope

- `scripts/agent/commands/cmd_ingest.py` — pass `debug_fn` to `pipeline.augment()`, show fallback reasons in `_print_rag_timings`
- `scripts/agent/cli_view.py` — extend `write_debug_rag()` to also render stage results (fallback reasons)

## Assumptions

1. `RagPipeline.last_stage_results` is already populated after each `run()` call (line 263 of pipeline.py).
2. `write_debug_rag()` signature expects `dict` with keys: `queries`, `all_results`, `merged`, `reranked`.
3. The MCP debug path already works (uses `RagDebugResponse` with full data).

## Proposed Changes

### 1. Wire `debug_fn` in `_cmd_rag()`

```python
async def _cmd_rag(self, args):
    # ... existing parse logic ...
    
    # Capture debug data when --debug is set
    debug_data: dict[str, Any] = {}
    if debug:
        def capture_fn(queries, all_results, merged, reranked):
            debug_data["queries"] = queries
            debug_data["all_results"] = all_results
            debug_data["merged"] = merged
            debug_data["reranked"] = reranked

    context = await pipeline.augment(query, debug_fn=capture_fn if debug else None)
    
    # ... existing output logic ...
    
    if debug:
        self._print_rag_debug(pipeline, debug_data)
```

### 2. Replace `_print_rag_timings()` with `_print_rag_debug()`

```python
def _print_rag_debug(self, pipeline: RagPipeline, debug_data: dict) -> None:
    """Print full RAG pipeline debug output including timings and stage results."""
    self._out.write("\n--- Stage timings ---")
    for stage_name, elapsed in pipeline.last_timings.items():
        self._out.write(f"  {stage_name}: {elapsed * 1000:.1f} ms")
    
    if pipeline.last_stage_results:
        self._out.write("\n--- Stage results ---")
        for sr in pipeline.last_stage_results:
            status_icon = {"success": "✓", "fallback": "~", "failure": "✗"}.get(sr["status"], "?")
            line = f"  {status_icon} {sr['stage_name']}: {sr['status']} ({sr['elapsed_seconds'] * 1000:.1f} ms)"
            if sr.get("fallback_reason"):
                line += f" — {sr['fallback_reason']}"
            self._out.write(line)
```
