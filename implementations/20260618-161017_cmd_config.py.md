# Implementation: cmd_config.py — fix /reload to use load_all()

## Goal

`_cmd_reload()` currently loads only `common.toml` and `agent.toml` (2 of 12 base files),
causing startup/reload inconsistency. Fix: call `ConfigLoader().load_all()` instead.

## Scope

- `scripts/agent/commands/cmd_config.py` — change `_cmd_reload()` to use `load_all()`

## Change

```python
# Before
source_files = ["common.toml", "agent.toml"]
new_cfg = ConfigLoader().load(*source_files)
result.source_files = source_files
self._out.write(f"Config reloaded from: {', '.join(source_files)}")

# After
new_cfg = ConfigLoader().load_all()
self._out.write("Config reloaded (all base config files)")
```

`result.source_files` is set to the list of all base config files from `_BASE_CONFIG_FILES`
by importing it, or simply omit `result.source_files` if the attribute is optional.
