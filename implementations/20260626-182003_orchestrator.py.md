## Implementation

### Target file
`scripts/agent/orchestrator.py` (or `scripts/agent/llm_turn_runner.py` â€” verify with grep)

### Procedure
1. Before calling `history_manager.compress(ctx.conv.history)`, snapshot current DB message_ids: `msg_ids = ctx.session.fetch_message_ids()` (add this helper to session_message_repo).
2. After assigning `ctx.conv.history = compressed`, if `result.compressed_indices`:
   - Compute `cutoff_message_id = msg_ids[result.compressed_indices[-1]]`
   - Extract summary content from compressed history (first system message with `[Conversation summary]` prefix)
   - Extract retained messages (all messages after the summary in compressed history that are NOT system)
   - Call `ctx.session.replace_compressed_messages(cutoff_message_id, summary_content, retained_messages)`

### Details

**Additional helper needed in `session_message_repo.py`:**

```python
def fetch_message_ids(self, session_id: int) -> list[int]:
    """Return message_ids in insertion order for the given session."""
    with SQLiteHelper("session").open(row_factory=True) as db:
        rows = db.fetchall(
            "SELECT message_id FROM messages WHERE session_id = ? ORDER BY message_id",
            (session_id,),
        )
    return [r["message_id"] for r in rows]
```

**Wiring pseudocode:**

```python
# Before compress():
msg_ids = ctx.session.fetch_message_ids(ctx.session.session_id) if ctx.session.session_id else []

compressed, result = await ctx.history_manager.compress(ctx.conv.history)
ctx.conv.history = compressed

# Wire to DB:
if result.summary_added and result.compressed_indices and msg_ids:
    try:
        cutoff_idx = result.compressed_indices[-1]
        if cutoff_idx < len(msg_ids):
            cutoff_id = msg_ids[cutoff_idx]
            # Find summary msg (first system msg with [Conversation summary])
            summary_msg = next(
                (m for m in compressed if m["role"] == "system"
                 and str(m.get("content", "")).startswith("[Conversation summary]")),
                None,
            )
            if summary_msg:
                retained = [m for m in compressed if m is not summary_msg and m["role"] != "system"]
                ctx.session.replace_compressed_messages(
                    cutoff_id, str(summary_msg["content"]), retained
                )
    except Exception:
        logger.warning("Failed to persist compression to DB; in-memory state diverges", exc_info=True)
```
