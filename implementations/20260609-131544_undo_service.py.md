# Implementation: undo_service.py — Extract undo logic from cmd_context

## Goal

Create `agent/services/undo_service.py` with `undo_last_turn(ctx)` extracted from
`_ContextMixin._cmd_undo()`, so the command only calls the service and prints the result.

## Scope

- `scripts/agent/services/undo_service.py`: new file.
- `scripts/agent/commands/cmd_context.py`: `_cmd_undo()` becomes a service call + print.

## Assumptions

1. The full undo logic (find last user idx, strip `_memory_injected` markers, trim history,
   decrement `stat_turns`, call `ctx.session.undo_last_turn()`) moves to the service.
2. Returns `(success: bool, message: str)` — command just prints the message.
3. `AgentContext` is imported via `TYPE_CHECKING` to avoid circular imports.

## Implementation

### Target files

- `scripts/agent/services/undo_service.py` (new)
- `scripts/agent/commands/cmd_context.py`

### Procedure

**undo_service.py:**

```python
"""agent/services/undo_service.py
Undo service — rolls back the last user+assistant turn.
"""
from __future__ import annotations
import logging
from typing import TYPE_CHECKING

if TYPE_CHECKING:
    from agent.context import AgentContext

logger = logging.getLogger(__name__)


def undo_last_turn(ctx: AgentContext) -> tuple[bool, str]:
    """Roll back the last user+assistant turn from history and DB.

    Strips preceding memory injection markers (_memory_injected=True),
    trims ctx.conv.history, decrements stat_turns, and calls session.undo_last_turn().
    Returns (True, "Last turn undone.") on success, (False, "Nothing to undo.") when
    no user message is found.
    """
    last_user_idx = next(
        (
            i
            for i in range(len(ctx.conv.history) - 1, -1, -1)
            if ctx.conv.history[i]["role"] == "user"
        ),
        None,
    )
    if last_user_idx is None:
        return False, "Nothing to undo."
    cut_idx = last_user_idx
    while cut_idx > 0 and ctx.conv.history[cut_idx - 1].get("_memory_injected"):
        cut_idx -= 1
    removed = len(ctx.conv.history) - cut_idx
    ctx.conv.history = ctx.conv.history[:cut_idx]
    ctx.stats.stat_turns = max(0, ctx.stats.stat_turns - 1)
    ctx.session.undo_last_turn()
    logger.info(f"Undo: removed {removed} messages from history")
    return True, "Last turn undone."
```

**cmd_context.py — replace `_cmd_undo`:**

```python
def _cmd_undo(self) -> None:
    from agent.services.undo_service import undo_last_turn  # noqa: PLC0415 — lazy import
    _, message = undo_last_turn(self._ctx)
    print(message)
```

Create service file, then slim down `_cmd_undo` in `cmd_context.py`.
