# Goal

Replace `list[Any]` fields in `PipelineContext`, remove the `observers` pattern,
and tighten `run(**kwargs: Any)` to typed context parameter.

# Scope

- `scripts/rag/stage.py`
- All stage classes (`MqeStage`, `SearchStage`, etc.) whose `run()` signatures must
  be updated if `kwargs` is removed

# Assumptions

1. `RawHit` dataclass from Step 3-1 prerequisite.
2. `search_results: list[Any]` → `list[list[RawHit]]` (each sub-list is results for
   one query variant from MQE expansion).
3. `observers: list[Any]` + `add_observer()` are removed. Any caller adding observers
   must be updated to not use that API.
4. `run(**kwargs: Any)` → `run(ctx: PipelineContext) → None`. Each stage receives
   the full context and modifies it in-place. The `ctx` argument replaces kwargs.
5. Check callers of `stage.run(**kwargs)` and update them to `stage.run(ctx)`.

# Implementation

## Target file

`scripts/rag/stage.py`

## Procedure

1. Change `search_results: list[Any]` → `search_results: list[list[RawHit]]`.
2. Remove `observers: list[Any]` field and `add_observer()` method.
3. Check if `observers` is called anywhere; update callers.
4. Change `run(**kwargs: Any)` abstract method → `run(ctx: PipelineContext) -> None`.
5. Update each concrete stage class to match the new signature.
6. Run ruff + mypy.

## Method

Field type change + abstract method signature update.

## Details

```python
@dataclasses.dataclass
class PipelineContext:
    query: str
    cfg: dict  # replaced with PipelineConfig in Step 4-4
    search_results: list[list[RawHit]] = dataclasses.field(default_factory=list)
    # remove observers
```
