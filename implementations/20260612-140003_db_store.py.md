# Goal

Replace the three `assert cur.lastrowid is not None` in `store.py` with explicit
`RuntimeError`, and change all eight `dict[str, Any]` return type annotations to
typed DTOs (`DocumentRow`, `SessionRow`, `MessageRow`).

# Scope

- `scripts/db/store.py`

# Assumptions

1. `DocumentRow`, `SessionRow`, `MessageRow` defined in `db/models.py` (Step 1 prerequisite).
2. The Protocol methods (`doc_get`, `doc_list`, `session_list`, `message_list`) and
   the concrete implementation methods both need updating.
3. `assert cur.lastrowid is not None` appears at lines 206, 272, 288. Each gets
   converted to a guarded `if` + `RuntimeError`.
4. The SQL queries already return all needed fields; only the return type annotation
   and construction change.
5. `doc_get()` returns `DocumentRow | None`; others return `list[DocumentRow]` etc.
6. Callers of these methods are in the agent layer via Protocol — they use `AgentStore`
   / `AgentSession` wrappers, not direct access. No caller changes needed for this step.

```python
def doc_get(self, url: str) -> DocumentRow | None:
    row = self._db.fetchall(
        "SELECT url, lang, chunk_count, fetched_at FROM documents WHERE url = ?",
        (url,),
    )
    if not row:
        return None
    r = row[0]
    return DocumentRow(
        url=str(r["url"]),
        lang=r["lang"],
        chunk_count=int(r["chunk_count"]),
        fetched_at=str(r["fetched_at"]),
    )
```
