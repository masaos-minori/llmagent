### Step 2: Add error recording and repeated failure detection method

Add after `_truncate_with_meta()` function (before `attach_auth_middleware`):
```python
def _record_tool_error(self, tool_name: str) -> None:
    """Record a tool error timestamp and warn if threshold exceeded.

    Uses a sliding window of _error_window_sec seconds. Warns when
    _error_threshold or more errors occur for the same tool within the window.
    """
    now = time.time()  # needs time import
    # Clean old entries outside the window
    cutoff = now - self._error_window_sec
    self._tool_errors = [t for t in self._tool_errors if t > cutoff]
    self._tool_errors.append(now)

    if len(self._tool_errors) >= self._error_threshold:
        logger.warning(
            "Repeated tool failures detected: %s failed %d times in %.0fs window",
            tool_name,
            len(self._tool_errors),
            self._error_window_sec,
        )
```
