# Implementation: injection.py — prefix unification and unused arg removal

## Goal

1. Remove the unused `session_id` parameter from `on_session_start`.
2. Unify snippet prefix: `on_session_start` currently uses hardcoded `"[Memory]"`;
   change it to use `self._policy.format_prefix_semantic` (same as `on_user_prompt`).

## Scope

- **Target file**: `scripts/agent/memory/injection.py`
- **Secondary**: `scripts/agent/memory/services.py` — facade must accept `session_id` and discard it
  (since `repl.py` passes `session_id` and we cannot change repl.py in this step)
- **Tests**: `tests/test_memory_layer.py` — `TestOnSessionStart` class

## Assumptions

1. `repl.py` calls `memory.on_session_start(session_id)` where `session_id` is `int | None`.
   The `services.py` facade absorbs this arg; `injection.py` no longer needs it.
2. `InjectionPolicy.format_prefix_semantic` default is `"[Semantic memory]"`.
   After this change, session-start snippets will use that prefix instead of `"[Memory]"`.
   This is intentional — consistent with query-time injection.
3. No other callers of `on_session_start` pass `session_id` directly to `MemoryInjectionService`.

## Implementation

### Target file

`scripts/agent/memory/injection.py`

### Procedure

1. Change `on_session_start(self, session_id: int | None) -> list[str]` to `on_session_start(self) -> list[str]`.
2. Replace `f"[Memory] {e.summary or e.content[:100]}"` with
   `f"{self._policy.format_prefix_semantic} {e.summary or e.content[:100]}"`.

Also update `services.py` facade:
- `on_session_start(self, session_id: int | None)` → calls `self.injection.on_session_start()` (drops session_id).

### Method

Direct textual edit.

### Details

In `injection.py`:
```python
# Before:
def on_session_start(self, session_id: int | None) -> list[str]:
    ...
    snippets = [f"[Memory] {e.summary or e.content[:100]}" for e in entries]

# After:
def on_session_start(self) -> list[str]:
    ...
    snippets = [
        f"{self._policy.format_prefix_semantic} {e.summary or e.content[:100]}"
        for e in entries
    ]
```

In `services.py` facade:
```python
# Before:
def on_session_start(self, session_id: int | None) -> list[str]:
    return self.injection.on_session_start(session_id)

# After:
def on_session_start(self, session_id: int | None) -> list[str]:
    return self.injection.on_session_start()   # session_id not forwarded
```
