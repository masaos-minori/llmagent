validation in `ingester.py` (Plan 142134 Phase 1).

## Scope

**In scope:**
- Add two `NotRequired[str]` fields to `ChunkJsonRaw` in `pipeline_utils.py`

**Out of scope:**

1. `ChunkJsonRaw` already uses `NotRequired` for optional fields (e.g., `chunk_id_override`).

   callers that already spread `**payload`.
