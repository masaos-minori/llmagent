# Implementation: agent/commands/cmd_workflow.py — Fail-closed approval with explicit ID

## Goal

Update `_cmd_approve` and `_cmd_reject` to fail closed when multiple pending approvals exist and no explicit `approval_id` is provided. Add UUID-based `approval_id` parsing. Emit audit events on resolution.

## Scope

**In**: Count check before resolution. UUID detection in arg. Fail-closed path for multi-pending without ID. Audit event emission. Update imports.

**Out**: Changes to `approval_ops.py`, `command_defs_list.py`, tests.

## Assumptions

1. `count_pending_approvals()` and `find_approval_by_id()` are added to `approval_ops.py` first.
2. UUID detection: `arg.strip().split()[0]` looks like a UUID if it matches `^[0-9a-f-]{36}$`.
3. Audit event uses `ctx.services_required.audit_logger` (same pattern as other commands).
4. Audit event fields: `approval_id`, `task_id`, `workflow_id`, `session_id`, `decision`, `reason`.
5. `workflow_id` comes from the task record — need `store.get_task_by_id(task_id)` to fetch it.

### Method

**`_parse_approval_arg()` helper:**
```python
import re
_UUID_RE = re.compile(r'^[0-9a-f]{8}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{12}$')

def _parse_approval_arg(arg: str) -> tuple[str | None, str | None]:
    """Return (approval_id, reason) by splitting UUID prefix from free-text reason."""
    parts = arg.strip().split(None, 1)
    if parts and _UUID_RE.match(parts[0]):
        approval_id = parts[0]
        reason = parts[1] if len(parts) > 1 else None
        return approval_id, reason
    return None, arg.strip() or None
```

**Updated `_cmd_approve`:**
```python
def _cmd_approve(self, arg: str) -> None:
    from agent.workflow import StateStore
    from agent.workflow.approval_ops import (
        count_pending_approvals, find_approval_by_id,
        find_latest_pending_approval, resolve_approval,
    )
    store = StateStore()
    try:
        explicit_id, reason = _parse_approval_arg(arg)
        count = count_pending_approvals(store._db)

        if count == 0:
            self._out.write_validation_error("No pending approval.")
            return

        if count > 1 and explicit_id is None:
            self._out.write_validation_error(
                f"{count} pending approvals exist. Use: /approve <approval_id> [reason]"
            )
            return

        if explicit_id is not None:
            approval = find_approval_by_id(store._db, explicit_id)
            if approval is None or approval.status != "pending":
                self._out.write_validation_error(f"Approval {explicit_id!r} not found or not pending.")
                return
            task_id = approval.task_id
        else:
            result = find_latest_pending_approval(store._db)
            assert result is not None  # count == 1
            task_id, approval = result

        resolve_approval(store._db, approval.approval_id, "approved", reason)
        self._emit_approval_audit(store, approval.approval_id, task_id, "approved", reason)
    finally:
        store.close()

    self._ctx.turn.pending_approval_id = None
    self._ctx.workflow.approval_pending = False
    self._ctx.turn.pending_approval_task_id = task_id
    self._out.write(f"Approved: {approval.approval_id} — workflow will resume on next turn")
```

**`_emit_approval_audit()` helper:**
```python
def _emit_approval_audit(self, store, approval_id, task_id, decision, reason):
    if self._ctx.services_required.audit_logger is None:
        return
    task = store.get_task_by_id(task_id)
    self._ctx.services_required.audit_logger.log(
        event_type="workflow_approval",
        data={
            "approval_id": approval_id,
            "task_id": task_id,
            "workflow_id": task.workflow_id if task else None,
            "session_id": self._ctx.session.session_id,
            "decision": decision,
            "reason": reason,
        },
    )
```
