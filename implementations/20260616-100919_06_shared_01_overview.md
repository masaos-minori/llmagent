# Implementation: docs/06_shared_01_overview.md

## Goal

Create a top-level overview of the shared/ layer covering purpose, scope, module inventory,
import direction constraints, prerequisites, and links to detailed chapters.

## Scope

- Content from: `06_spec_shared.md` §1-5 (purpose, scope, background, prerequisites, constraints)
- Content from: `06_shared.md` index/module table, import direction note
- Output: `docs/06_shared_01_overview.md`
- Not covered: per-module API details (→ 03), type definitions (→ 02), DB layer (→ 04/05)

## Assumptions

- Audience: engineer unfamiliar with shared/ layer or AI that has not read it
- shared/ is the lowest layer; all other layers depend on it
- `.importlinter` enforces import direction at CI

## Implementation

### Target file

`docs/06_shared_01_overview.md`

### Procedure

1. State purpose (one sentence): provide common infrastructure for agent, mcp, rag, db
2. Scope: list all submodules (config_loader, logger, types, llm_types, tool_constants,
   route_resolver, mcp_config, tool_executor, plugin_registry, otel_tracer, token_counter,
   git_helper, formatters, action_result, events, protocols/shell)
3. Import direction diagram: shared/ → external libs only; agent/mcp/rag/db → shared/
4. Prerequisites table (Python 3.13+, orjson, httpx, pydantic, tomllib; OTel optional; GitPython optional)
5. Constraints table (JSON via orjson, HTTP via httpx, config TOML only, English logs)
6. Module reference table: file → responsibility (one line each)
7. Links to detailed chapters (02, 03, 04, 05)

### Method

- Use heading structure: Purpose / Scope / Import Direction / Prerequisites / Constraints / Modules / Chapters
- No API signatures (those belong in 03); no type field tables (those belong in 02)
- Module reference table replaces 06_shared.md index table (canonically)

### Details

- Import direction enforced by `.importlinter` — state this explicitly
- orjson.dumps() returns bytes; requires .decode() — note in constraints
- shared/ from `agent/memory/store.py` and `db/` is downstream, not upstream
