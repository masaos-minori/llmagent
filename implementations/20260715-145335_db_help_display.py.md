## Goal

Delete `scripts/agent/commands/db_help_display.py` in full — it becomes fully dead
code once `cmd_db.py` (its sole consumer) is deleted, and its help text is `/db`-specific
(not applicable to `/session`).

## Scope

### In scope
- Delete the file `scripts/agent/commands/db_help_display.py` (63 lines:
  `DbHelpDisplay` class with `_db_help()` and `_db_help_session()`).

  `grep -rn "DbHelpDisplay\|_db_help\b\|_db_help_session" scripts/ tests/`; the only
