# Goal

Update `cmd_tooling.py` so that its internal `_to_tool_result_view()` adapter accepts
`ToolResultRow` (typed DTO from Step 4) instead of `dict[str, Any]`.

# Scope

- `scripts/agent/commands/cmd_tooling.py`

# Assumptions

1. `ToolResultRow` is now returned by `tool_results.get()` and `tool_results.list_recent()`
   (Step 4 prerequisite).
2. The existing `ToolResultView` display DTO in `agent/commands/models.py` is kept —
   cmd_tooling renders via `ToolResultView`, not `ToolResultRow` directly.
3. `_to_tool_result_view(entry: dict[str, Any])` → change to accept `ToolResultRow`.
   Field mapping:
   - `entry["id"]` → `entry.id`
   - `entry["tool_name"]` → `entry.tool_name`
   - `entry.get("summary")` → `entry.summary`
   - `entry.get("args_masked")` → `entry.args_masked`
   - `entry.get("is_error", False)` → `entry.is_error`
4. `_decode_args` helper currently receives `str | None`; `ToolResultRow.args_masked` is
   always `str` (defaulting to `""`), so the `if not raw` guard still handles empty string.
5. `_tool_show()` accesses `raw.get("full_text", "")` — after the change, this becomes
   `raw.full_text` (always str via ToolResultRow default).
6. Import for `ToolResultRow` comes from `db.models` (not `agent.commands.models`).
   Import-linter: `agent` layer CAN import `db` layer — no violation.

# Implementation

## Target file

`scripts/agent/commands/cmd_tooling.py`

## Procedure

1. Add import: `from db.models import ToolResultRow`.
2. Remove `from typing import Any` if it becomes unused after the change.
3. Change `_to_tool_result_view` signature and body:
   ```python
   def _to_tool_result_view(entry: ToolResultRow) -> ToolResultView:
       return ToolResultView(
           result_id=entry.id,
           tool_name=entry.tool_name,
           summary=entry.summary,
           args_masked=_decode_args(entry.args_masked),
           is_error=entry.is_error,
       )
   ```
4. In `_tool_show()`, change `full_text = str(raw.get("full_text", ""))` → `full_text = raw.full_text`.
5. Run ruff + mypy.

## Method

Type signature change and field access update. No logic change.
