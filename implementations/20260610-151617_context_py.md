## Implementation

### Target file

`scripts/agent/context.py`

### Procedure

1. Change `self.services: Any = None` to `self.services: AppServices | None = None`.
2. Move `AppServices` import from `TYPE_CHECKING` block to the top-level import (it is now needed at runtime for the type annotation).
3. Add `@property services_required(self) -> AppServices` that raises `RuntimeError("services not initialized")` when `self.services is None`. Callers that assert non-None can switch to this property.
4. Remove `from typing import Any` if `TurnState.background_tasks` can use `set[asyncio.Task[object]]` â€” otherwise keep with `# type: ignore[type-arg]  # Task generic not available at runtime` justification.
5. Verify no circular imports: `context.py` â†’ `factory.py` is forbidden by import-linter; `AppServices` class must not import from `factory.py`.

### Method

Minimal change: type annotation update + property addition. No structural refactoring.

### Details

```python
# Before
self.services: Any = None

# After
self.services: AppServices | None = None

@property
def services_required(self) -> AppServices:
    if self.services is None:
        raise RuntimeError("AgentContext.services not initialized â€” call build_agent_context() first")
    return self.services
```

All existing `assert ctx.services is not None` and `assert ctx.services.tools is not None` guards remain intact; callers migrate to `services_required` incrementally.
