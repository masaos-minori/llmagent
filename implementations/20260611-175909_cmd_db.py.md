# Goal

Refactor `cmd_db.py` to remove formatter `print_*` imports, route output through `OutputPort`,
and replace raw `str(r["url"])` dict access with typed `DocumentRow` attribute access.

# Scope

- `scripts/agent/commands/cmd_db.py`

# Assumptions

1. `OutputPort` / `CliOutputPort` from `output_port.py` (Step 1 prerequisite).
2. `print_kv_list`, `print_no_data`, `print_success`, `print_table`, `print_validation_error` are currently imported from `formatter.py` — all replaced with `self._out` calls.
3. `self._ctx.session.list_documents(lang, limit)` returns `list[dict]` — a `DocumentRow` DTO conversion at the call site is the minimal approach without changing `session.py`.
4. `str(limit_raw).isdigit()` guard is acceptable for now; no strict `parse_flag_int` needed (that's a Low-priority improvement).
5. `_out` is available via `CommandRegistry` MRO.

# Implementation

## Target file

`scripts/agent/commands/cmd_db.py`

## Procedure

1. Remove all `from agent.commands.formatter import ...` imports.
2. Replace `print_kv_list(...)` → `self._out.write_kv(...)`.
3. Replace `print_success(...)` → `self._out.write_success(...)`.
4. Replace `print_no_data(...)` → `self._out.write_no_data(...)`.
5. Replace `print_table(...)` → `self._out.write_table(...)`.
6. Replace `print_validation_error(...)` → `self._out.write_validation_error(...)`.
7. In `_db_list_urls`: replace `str(r["url"])`, `str(r["lang"] or "?")`, `str(r["chunk_count"])`, `str(r["fetched_at"])` with a local `DocumentRow`-like conversion using attribute or dict access, making the types explicit.

## Method

Direct substitution. `_db_list_urls` needs a minor refactor to eliminate `str(r["url"])` casts:

```python
# After: use explicit str conversion with type-safe dict access
def _db_list_urls(self, rest: str) -> None:
    ...
    rows = self._ctx.session.list_documents(lang, limit)
    if not rows:
        self._out.write_no_data("No documents found")
        return
    table_rows = []
    for r in rows:
        url = str(r["url"])
        url_display = url[:57] + "..." if len(url) > 60 else url
        table_rows.append([
            url_display,
            str(r["lang"] or "?"),
            str(r["chunk_count"]),
            str(r["fetched_at"]),
        ])
    self._out.write_table(["URL", "Lang", "Chunks", "Fetched"], table_rows)
```

Note: full `DocumentRow` DTO migration (removing dict access) is deferred to the 02_plan
agent/services layer which will update `session.list_documents()` return type.
