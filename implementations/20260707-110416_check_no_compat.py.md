## Goal
Add workflow enforcement patterns to `COMPAT_PATTERNS` in `tools/check_no_compat.py` and update `DEFAULT_ALLOWLIST` to prevent reintroduction of forbidden workflow-mode and policy code.

## Scope
**In**: `tools/check_no_compat.py` — add 10 patterns to `COMPAT_PATTERNS`; update `DEFAULT_ALLOWLIST`; add `WorkflowExecutionPolicy` import patterns (for Plan 13).
**Out**: Semantic analysis; automatic rewriting; files outside `scripts/`, `docs/`, `tests/`.

## Assumptions
- `COMPAT_PATTERNS` is a dict: `{name: regex_pattern}`.
- `DEFAULT_ALLOWLIST` is a set of `Path` objects.
- Hook runs on Python and Markdown files — new patterns apply to Python files.
- `tests/test_workflow_execution_policy.py` is deleted by Plan 13 (req13) — allowlist entry is temporary.
- `tests/test_check_no_compat.py` contains pattern fixtures — must be allowlisted.
