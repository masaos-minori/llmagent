# Implementation: Clean Up Layering Violations in db/maintenance.py

## Goal

Remove `rotate_rag_db()` from `db/maintenance.py` (a session-DB module) and relocate it to `RagMaintenanceService`. Clarify that `vacuum_db()` is generic and does not violate layering.

## Scope

**In:**
- `scripts/agent/services/rag_maintenance_service.py` ‚Äî add `rotate_rag_db(archive_dir=None) -> Path` method
- `scripts/db/maintenance.py` ‚Äî remove `rotate_rag_db()`; inline its logic in `rotate_db()` via `_archive_db_file()`
- `scripts/db/__init__.py` ‚Äî remove `rotate_rag_db` from exports
- `tests/test_db_maintenance.py` ‚Äî update import and test references for `rotate_rag_db`

**Out:**
- Moving `vacuum_db()` ‚Äî it's generic; no change
- Moving `_vacuum_db()` / `recover_corruption()` ‚Äî dual-purpose; no change
- Moving `rotate_session_db()` ‚Äî correctly scoped to session module; no change

## Assumptions

1. `_archive_db_file()` is a generic utility (WAL-safe SQLite backup) ‚Äî safe to call from `RagMaintenanceService`
2. `rotate_db()` (archives both RAG + session) stays in `db/maintenance.py` but will inline the RAG archival via `_archive_db_file()` directly ‚Äî no upward dependency
3. Tests that import `rotate_rag_db` from `db.maintenance` will be updated to import from `agent.services.rag_maintenance_service`
4. `db/__init__.py` backward-compat concern: `rotate_rag_db` is not used by any agent code at runtime ‚Äî only tests import it

## Implementation

### Target file: `scripts/agent/services/rag_maintenance_service.py`

**Procedure:** Add `rotate_rag_db()` method with lazy imports to avoid circular dependencies.

**Method:** Add new method using lazy imports for `build_db_config` and `_archive_db_file`.

**Details:**
```python
def rotate_rag_db(self, archive_dir: str | Path | None = None) -> Path:
    """Archive rag.sqlite with a timestamp suffix; returns the archive path."""
    from db.config import build_db_config  # noqa: PLC0415 ‚Äî lazy to avoid circular dep
    from db.maintenance import _archive_db_file  # noqa: PLC0415 ‚Äî lazy to avoid circular dep
    db_cfg = build_db_config()
    return _archive_db_file(Path(db_cfg.rag_db_path), archive_dir)
```

### Target file: `scripts/db/maintenance.py`

**Procedure:** Remove `rotate_rag_db()` function; inline its logic in `rotate_db()`.

**Method:** Replace the `rotate_db()` function body to call `_archive_db_file()` directly instead of `rotate_rag_db()`.

**Details:**
Before:
```python
def rotate_rag_db(archive_dir: str | Path | None = None) -> Path:
    """Archive rag.sqlite to archive_dir with a timestamp suffix; returns the archive path."""
    db_cfg = build_db_config()

def rotate_db(archive_dir: str | Path | None = None) -> tuple[Path, Path]:
```
