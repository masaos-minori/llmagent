## Goal

Change `AgentContext.services` field type from `Any` to `AppServices | None` in `scripts/agent/context.py`. This is Phase 1 of the ctx.services type migration plan.

## Scope

**In**: `scripts/agent/context.py` lines 185–186 only — remove the workaround comment and change `services: Any` to `services: AppServices | None`.

**Out**: All other files, including `orchestrator.py`, `llm_turn_runner.py`, `commands/`, `repl.py` — covered by separate implementation docs.

## Assumptions

1. `AppServices` is defined in the same file (`context.py`) — no new import required.
2. `Any` is still needed at line 76 (`set[asyncio.Task[Any]]`), so `from typing import TYPE_CHECKING, Any` must remain.
3. After this change, mypy will surface `union-attr` errors in all files that access `ctx.services.xxx` without a None guard. Those are resolved in the orchestrator.py and llm_turn_runner.py implementation docs.
