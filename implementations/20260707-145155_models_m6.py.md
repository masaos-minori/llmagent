## Implementation

### Target file

`scripts/agent/services/models.py`

### Procedure

#### Step 1: Add ProcessInfoSnapshot dataclass

```python
@dataclass(frozen=True)
class ProcessInfoSnapshot:
    """Read-only snapshot of an HTTP MCP subprocess's runtime state."""

    server_key: str
    managed: bool
    pid: int | None
    pgid: int | None
    running: bool
    last_exit_code: int | None
    stderr_log: str
```

Add `from dataclasses import dataclass` if not already present (it should be, for
`McpProbeResult`).

### Method

- Pure data class; no logic, no imports beyond `dataclasses`.
- `frozen=True` prevents mutation after construction.

### Details

- `server_key`: identifies which server this snapshot belongs to.
- `managed: bool`: `True` if the server is in `_http_procs`; `False` for persistent.
- `pid`, `pgid`: process IDs; `None` before H-8 populates `_http_pgids`.
- `running`: `proc.poll() is None` at snapshot time.
- `last_exit_code`: `proc.poll()` for already-exited processes; `None` if running.
- `stderr_log`: path to the per-server log file (H-1); `""` before H-1 is merged.
