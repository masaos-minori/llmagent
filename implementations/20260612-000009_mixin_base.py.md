# Goal

Remove the `_reset_session_stats` compatibility wrapper from `MixinBase`; callers
should call the module-level `reset_session_stats(ctx)` function directly.

# Scope

- `scripts/agent/commands/mixin_base.py`

# Assumptions

1. `reset_session_stats(ctx)` module-level function already exists in `mixin_base.py`
   (line 18) and contains the actual implementation.
2. `_reset_session_stats(self, ctx)` at line 43 is a one-line wrapper that delegates
   to `reset_session_stats(ctx)`. It was kept for backward compatibility.
3. Callers of `_reset_session_stats`: search for `self._reset_session_stats` in the
   codebase. All found call sites must be updated to call `reset_session_stats(ctx)` directly.
4. After removal, run `grep -rn "_reset_session_stats" scripts/` → 0 hits to confirm
   all callers were updated.

# Implementation

## Target file

`scripts/agent/commands/mixin_base.py` and any files calling `self._reset_session_stats`

## Procedure

1. Run `grep -rn "_reset_session_stats" scripts/` to find all call sites.
2. For each call site `self._reset_session_stats(ctx)`, replace with
   `reset_session_stats(ctx)` and ensure `reset_session_stats` is imported.
3. Delete the `_reset_session_stats` method from `MixinBase`.
4. Run ruff + mypy.

## Method

Inline the single delegation: remove the wrapper, update all callers.

## Details

In `mixin_base.py`, delete:
```python
def _reset_session_stats(self, ctx: AgentContext) -> None:
    """Delegate to module-level reset_session_stats for backward compatibility."""
    reset_session_stats(ctx)
```

For any caller file that imports `MixinBase` or uses `self._reset_session_stats`:
```python
# Before:
self._reset_session_stats(ctx)
# After:
from agent.commands.mixin_base import reset_session_stats
reset_session_stats(ctx)
```
(Or, if the caller already imports from mixin_base, just add `reset_session_stats` to the import.)

# Validation plan

- `grep -rn "_reset_session_stats" scripts/` → 0 hits
- `uv run ruff check scripts/agent/commands/mixin_base.py`
- `uv run mypy scripts/agent/commands/mixin_base.py`
- `uv run pytest --ignore=tests/test_create_schema.py -v -q`
