# Implementation Plan: Workflow Loader (Step 4)

## Goal

Implement `workflow_loader.py` to load and validate workflow definitions from `config/workflows/*.json`.

## Scope

**In:**
- `scripts/agent/workflow/workflow_loader.py`: JSON loading, schema validation, error handling

**Out:**
- File system discovery of workflow files (uses fixed path per plan Assumption 6)
- Schema definition (embedded in loader as minimal required fields check)

## Assumptions

1. Workflow files are located at `config/workflows/*.json` relative to repo root.
2. `ConfigLoader._resolve_path()` does not join with config_dir for Path objects — loader uses direct path resolution.
3. JSON Schema validation checks: version (str), name (str), stages (list of {id, timeout_sec, retry}).
4. Missing or invalid files raise `WorkflowLoadError` with descriptive message.

### Procedure

#### Create `scripts/agent/workflow/workflow_loader.py`

```python
#!/usr/bin/env python3
"""workflow/workflow_loader.py — Load and validate workflow definitions from JSON."""

from __future__ import annotations

import json
import logging
from pathlib import Path
from typing import Any

from agent.workflow.models import RetryPolicy, StageDef, WorkflowDef

logger = logging.getLogger(__name__)

_REQUIRED_FIELDS: tuple[str, ...] = ("version", "name", "stages")
_STAGE_REQUIRED_FIELDS: tuple[str, ...] = ("id", "timeout_sec")


class WorkflowLoadError(Exception):
    """Raised when workflow JSON is missing, invalid, or schema-violating."""


def _resolve_config_path() -> Path:
    """Resolve the config/workflows/ directory path relative to this module."""
    return Path(__file__).resolve().parent.parent.parent / "config" / "workflows"


def load_workflow(name: str = "default") -> WorkflowDef:
    """Load a workflow definition by name from config/workflows/{name}.json.

    Raises WorkflowLoadError if file missing, invalid JSON, or schema violation.
    """
    config_path = _resolve_config_path() / f"{name}.json"

    if not config_path.exists():
        raise WorkflowLoadError(f"Workflow file not found: {config_path}")

    try:
        raw = json.loads(config_path.read_text(encoding="utf-8"))
    except json.JSONDecodeError as e:
        raise WorkflowLoadError(f"Invalid JSON in {config_path}: {e}") from e

    return _validate_workflow_def(raw, str(config_path))


def _validate_workflow_def(raw: dict[str, Any], source: str) -> WorkflowDef:
    """Validate raw JSON dict against required schema and return WorkflowDef."""
    # Check top-level required fields
    for field in _REQUIRED_FIELDS:
        if field not in raw:
            raise WorkflowLoadError(f"Missing required field '{field}' in {source}")

    version = raw["version"]
    if not isinstance(version, str):
        raise WorkflowLoadError(f"'version' must be string, got {type(version).__name__}")

    name = raw["name"]
    if not isinstance(name, str):
        raise WorkflowLoadError(f"'name' must be string, got {type(name).__name__}")

    stages_raw = raw["stages"]
    if not isinstance(stages_raw, list):
        raise WorkflowLoadError(f"'stages' must be array, got {type(stages_raw).__name__}")

    stages: list[StageDef] = []
    for i, stage_raw in enumerate(stages_raw):
        if not isinstance(stage_raw, dict):
            raise WorkflowLoadError(f"stage[{i}] must be object in {source}")

        for field in _STAGE_REQUIRED_FIELDS:
            if field not in stage_raw:
                raise WorkflowLoadError(
                    f"Missing required field '{field}' in stages[{i}] of {source}"
                )

        retry_raw = stage_raw.get("retry", {})
        if not isinstance(retry_raw, dict):
            retry_raw = {}

        retry_policy = RetryPolicy(
            max_attempts=int(retry_raw.get("max_attempts", 3)),
            backoff_multiplier=float(retry_raw.get("backoff_multiplier", 2.0)),
            retryable_errors=tuple(retry_raw.get("retryable_errors", [])),
        )

        stage = StageDef(
            id=str(stage_raw["id"]),
            timeout_sec=int(stage_raw["timeout_sec"]),
            retry=retry_policy,
        )
        stages.append(stage)

    global_timeout = int(raw.get("global_timeout_sec", 300))
    on_halt = str(raw.get("on_halt", "rollback"))

    return WorkflowDef(
        version=version,
        name=name,
        stages=stages,
        global_timeout_sec=global_timeout,
        on_halt=on_halt,
    )
```

### Details

- `_resolve_config_path()` uses `Path(__file__).resolve().parent.parent.parent` to reach repo root from `scripts/agent/workflow/`
- Validation is strict: missing required fields raise `WorkflowLoadError` with source file path
- `retry` sub-object defaults applied if missing (max_attempts=3, backoff_multiplier=2.0)
- No external JSON Schema library dependency — manual validation keeps imports minimal
