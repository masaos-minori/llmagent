## Step 5: `tests/test_shell_mcp_service.py` — `/health` 形式テスト追加

既存テストクラスの末尾に追加。`TestClient` と `monkeypatch` を使用してサービス初期化をバイパスする:

```python
def test_health_response_includes_sandbox_backend_in_details(monkeypatch) -> None:
    from fastapi.testclient import TestClient
    from mcp.shell import server as shell_server

    # _service の sandbox_backend をモックして初期化不要にする
    class _FakeService:
        sandbox_backend = "none"
        ready = True
        dependencies: dict = {}

    monkeypatch.setattr(shell_server, "_service", _FakeService())
    client = TestClient(shell_server.app)
    resp = client.get("/health")
    assert resp.status_code == 200
    body = resp.json()
    assert "sandbox_backend" not in body              # ルートレベルにない
    assert "sandbox_backend" in body.get("details", {})  # details に存在
    assert body["details"]["sandbox_backend"] in ("none", "firejail")
```

**注意:** `_service` が module-level singleton かどうかを事前確認。`server.py` の `_service` がモジュール変数であれば `monkeypatch.setattr(shell_server, "_service", ...)` が使える。

## Step 6: `tests/test_agent_mcp_status.py` — `_get_http_status` テスト追加

`_get_http_status` のシグネチャ: `async def _get_http_status(self, probe: httpx.AsyncClient, url: str) -> tuple[McpAvailability, str]`

`respx` が利用可能かどうかを `import respx` で確認してから使う。利用不可なら `unittest.mock.AsyncMock` で代替する。

```python
import pytest
import httpx
from agent.services.mcp_status import McpStatusService
from agent.services.enums import McpAvailability


class TestGetHttpStatusSandboxBackend:
    @pytest.mark.asyncio
    async def test_reads_sandbox_backend_from_details(self) -> None:
        """_get_http_status reads sandbox_backend from details dict."""
        service = McpStatusService.__new__(McpStatusService)

        class _FakeResponse:
            status_code = 200
            def json(self):
                return {
                    "status": "ok",
                    "ready": True,
                    "dependencies": {},
                    "details": {"sandbox_backend": "firejail"},
                }

        class _FakeClient:
            async def get(self, url: str):
                return _FakeResponse()

        avail, sandbox = await service._get_http_status(_FakeClient(), "http://localhost:8009")
        assert avail == McpAvailability.OK
        assert sandbox == "firejail"

    @pytest.mark.asyncio
    async def test_returns_empty_when_sandbox_backend_missing_from_details(self) -> None:
        service = McpStatusService.__new__(McpStatusService)

        class _FakeResponse:
            status_code = 200
            def json(self):
                return {"status": "ok", "ready": True, "dependencies": {}, "details": {}}

        class _FakeClient:
            async def get(self, url: str):
                return _FakeResponse()

        avail, sandbox = await service._get_http_status(_FakeClient(), "http://localhost:8009")
        assert avail == McpAvailability.OK
        assert sandbox == ""
```

**注意:** `McpStatusService.__new__(McpStatusService)` で `__init__` をスキップして lightweight インスタンスを生成。`_get_http_status` は `self` を使わないため問題なし。
