# Implementation: Replace raw SQL in route handlers with DB facade calls

Source plan: `plans/20260625-135640_plan.md` (req #15, app.py portion)

## Goal

Remove all raw `_db.execute(...)` / `_db.commit()` / `.fetchall()` calls from `scripts/eventbus/app.py` route handlers and replace them with the named DB facade functions from `scripts/eventbus/db.py`.

## Scope

- Replace raw SQL in: `health()`, `publish()`, `replay()`, `subscribe()`, `dlq_list()`, `dlq_requeue()`, `_get_seq()`, `_dlq_loop()`
- The `asyncio.to_thread` wrapping from req #14 is assumed to already be in place; this change updates the callable bodies
- Out of scope: adding new facade functions (req #15 db.py doc), connection strategy (req #16)

## Assumptions

1. req #14 (asyncio.to_thread wrapping) is implemented first; this change modifies the callables inside `asyncio.to_thread`
2. All six facade functions from `db.py` are available: `check_db`, `insert_event`, `get_seq`, `fetch_events_since`, `fetch_dlq`, `requeue_event`
3. `_get_seq()` in `app.py` becomes redundant after this change and is removed

## Implementation

### Target file

`scripts/eventbus/app.py`

### Procedure

1. Add import: `from eventbus.db import check_db, insert_event, get_seq, fetch_events_since, fetch_dlq, requeue_event`
2. Replace `health()` callable body with `check_db(_db)`
3. Replace `publish()` callable body with `insert_event(_db, ...)`
4. Replace `replay()` callable body with `fetch_events_since(_db, since_seq)`
5. Replace `subscribe()` poll callable body with `fetch_events_since(_db, current_seq, topic or None)`
6. Replace `dlq_list()` callable body with `fetch_dlq(_db)`
7. Replace `dlq_requeue()` callable body with `requeue_event(_db, event_id)`
8. Remove the now-redundant `_get_seq()` function definition from `app.py`

### Method

Each route handler's `asyncio.to_thread` callable is updated in place. The outer async structure (thread wrapping, response construction) remains unchanged.

### Details

**health()** — before and after pattern:
```python
# Before (after req #14)
def _check() -> str:
    try:
        _db.execute("SELECT 1")
        return "ok"
    except Exception:
        return "unavailable"

# After (req #15)
def _check() -> bool:
    return check_db(_db)  # type: ignore[arg-type]
db_ok = await asyncio.to_thread(_check)
db_status = "ok" if db_ok else "unavailable"
```

**publish():**
```python
# After
def _insert() -> tuple[int, bool]:
    return insert_event(_db, event_id, topic, payload_str, producer, published_at)  # type: ignore[arg-type]
seq, _ = await asyncio.to_thread(_insert)
```

**replay():**
```python
rows = await asyncio.to_thread(fetch_events_since, _db, since_seq)
```

**subscribe() poll:**
```python
rows = await asyncio.to_thread(fetch_events_since, _db, current_seq, topic or None)
```

**dlq_list():**
```python
rows = await asyncio.to_thread(fetch_dlq, _db)
```

**dlq_requeue():**
```python
found = await asyncio.to_thread(requeue_event, _db, event_id)
if not found:
    raise HTTPException(status_code=404, detail="event not found")
```

**Remove `_get_seq()` from app.py** — this function is now `get_seq()` in `db.py`.
