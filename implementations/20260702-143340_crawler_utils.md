# Implementation Procedure: `_validate_target_url()` + `parse_targets_file()`

**Target file:** `scripts/rag/ingestion/crawler_utils.py`

---

## Goal

Add two helper functions to `crawler_utils.py`:

1. `_validate_target_url(url: str) -> bool` — URL scheme validator that accepts `http`, `https`, and `file` schemes (unlike the existing `validate_url()` in `rag/utils.py` which only accepts `http`/`https`).
2. `parse_targets_file(path: Path) -> list[tuple[str, str]]` — reads a TOML targets file and returns validated `(url, lang)` pairs.

These functions are used exclusively by the new `--targets-file` path in `crawler.py`. The existing `parse_target_urls()` function is not modified.

```python
def _validate_target_url(url: str) -> bool:
    """Return True when url has an accepted scheme (http, https, or file).

    Unlike validate_url() in rag.utils, this function also accepts file://
    URIs used by the --targets-file crawl path.
    """
    scheme = urlparse(url).scheme
    return scheme in {"http", "https", "file"}
```

#### Step 3: Add `parse_targets_file()`

Insert immediately after `_validate_target_url()`, before `def url_to_slug(...)`.

```python
def parse_targets_file(path: Path) -> list[tuple[str, str]]:
    """Read a TOML targets file and return validated (url, lang) pairs.

    The file must contain a ``target_urls`` key with a list of [url, lang]
    2-element lists, matching the format used in config/rag_pipeline.toml.

    Raises:
        FileNotFoundError: when the file does not exist.
        ValueError: when an entry has an unsupported URL scheme or an invalid
            lang value.
    """
    raw_text = path.read_text(encoding="utf-8")
    data = tomllib.loads(raw_text)
    target_raw: list[list[str]] = data.get("target_urls", [])
    result: list[tuple[str, str]] = []
    for i, entry in enumerate(target_raw):
        if not isinstance(entry, list | tuple) or len(entry) != _TARGET_URL_ENTRY_LENGTH:
            raise ValueError(
                f"targets-file entry [{i}] must be a 2-element list [url, lang]"
            )
        url, lang = str(entry[0]), str(entry[1])
        if not _validate_target_url(url):
            raise ValueError(
                f"targets-file entry [{i}]: unsupported URL scheme in {url!r}"
                " (must be http, https, or file)"
            )
        if lang not in _VALID_HINT_LANGS:
            raise ValueError(
                f"targets-file entry [{i}]: unsupported lang {lang!r}"
                f" (must be one of {sorted(_VALID_HINT_LANGS)})"
            )
        result.append((url, lang))
    return result
```

### Method details

**`_validate_target_url(url: str) -> bool`**

- Uses `urlparse(url).scheme` — same pattern as the existing `validate_url()` in `rag/utils.py`
- Accepts `"http"`, `"https"`, `"file"` — returns `False` for all other schemes (e.g., `ftp`, `ssh`)
- Private function: prefixed with `_`, not exported

**`parse_targets_file(path: Path) -> list[tuple[str, str]]`**

- Calls `path.read_text()` — raises `FileNotFoundError` naturally if the file does not exist (no wrapping needed; caller converts to `parser.error()`)
- Uses `tomllib.loads()` — `tomllib.load()` requires a binary file handle; `loads()` with pre-read text is simpler and consistent with the rest of the codebase
- `data.get("target_urls", [])` — returns an empty list when the key is absent, resulting in an empty (but valid) `list[tuple[str, str]]`
- Validates each entry using `_validate_target_url()` and `_VALID_HINT_LANGS` — raises `ValueError` with a clear positional message (`entry [i]`)
- Returns `list[tuple[str, str]]` — same type as `parse_target_urls()`
