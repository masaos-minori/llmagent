# Implementation: rag/pipeline.py — per-stage latency measurement in run()

## Goal

Populate `self.last_timings` in `RagPipeline.run()` with per-stage wall-clock seconds so that callers (e.g., `/rag search --debug`) can display pipeline performance breakdown.

## Scope

- `scripts/rag/pipeline.py` — modify `run()` to wrap each stage call with `time.perf_counter()`
- No changes to stage classes themselves

## Assumptions

1. `self.last_timings: dict[str, float]` already exists on `RagPipeline` (initialized to `{}`).
2. Key = `stage.__class__.__name__` (e.g., `"MqeStage"`, `"SearchStage"` etc.).
3. Value = elapsed seconds as `float`.
4. `last_timings` is reset at the start of each `run()` call to avoid stale data.
5. `import time` is already present or needs to be added.

## Implementation

### Target file

`scripts/rag/pipeline.py`

### Procedure

1. Read the file.
2. Add `import time` if not already imported.
3. Modify the `run()` method stage loop.
4. Run ruff + mypy.

### Method

Add `perf_counter` timing around each `await stage.run(ctx, db=db)` call inside `run()`.

### Details

**`run()` method — current implementation:**
```python
async def run(self, query, db, history_context=""):
    try:
        ctx = PipelineContext(query=query, history_context=history_context)
        stages: list = [
            MqeStage(...),
            SearchStage(...),
            FusionStage(...),
            RerankStage(...),
            AugmentStage(),
        ]
        for stage in stages:
            await stage.run(ctx, db=db)
        ...
```

**After change:**
```python
async def run(self, query, db, history_context=""):
    try:
        ctx = PipelineContext(query=query, history_context=history_context)
        self.last_timings = {}  # reset per run
        stages: list = [
            MqeStage(self._cfg.__dict__, self._llm),
            SearchStage(self._cfg.__dict__),
            FusionStage(self._cfg.__dict__),
            RerankStage(self._cfg.__dict__, self._llm),
            AugmentStage(),
        ]
        for stage in stages:
            t0 = time.perf_counter()
            await stage.run(ctx, db=db)
            self.last_timings[stage.__class__.__name__] = time.perf_counter() - t0
        ...
```

Add `import time` at the top of the file (after existing stdlib imports, before third-party).
