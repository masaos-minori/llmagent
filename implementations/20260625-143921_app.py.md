# Implementation: app.py — POST /events/{event_id}/ack endpoint (req #25)

## Goal

Add `POST /events/{event_id}/ack` endpoint that sets `acked_at` and optionally advances the consumer offset when `consumer_id` is provided.

## Scope

- Add `POST /events/{event_id}/ack?consumer_id=X` route handler
- Call `ack_event()` from `db.py` (req #25)
- If `consumer_id` is provided, call `write_offset()` to advance the consumer's offset
- Does NOT change the nack endpoint (handled by req #24)

## Assumptions

- req #28 (schema) and req #25 (db.py ack_event) are implemented first
- Offset update on ack is the primary path for offset advancement (req #29 change)
- `asyncio.to_thread()` is used for both `ack_event()` and `write_offset()` (both are sync I/O)
- `acked_at` is set to UTC ISO-8601 format: `datetime.now(UTC).strftime("%Y-%m-%dT%H:%M:%SZ")`

## Implementation

### Target file

`scripts/eventbus/app.py`

### Procedure

1. Add import for `ack_event` from `eventbus.db`
2. Add `POST /events/{event_id}/ack` route handler with optional `consumer_id` query parameter
3. The handler calls `ack_event()` and, if `consumer_id` is set, fetches the event's seq and calls `write_offset()`

### Details

**Updated import line:**

```python
from eventbus.db import ack_event, nack_event, open_db
```

**New ack endpoint:**
```python
@app.post("/events/{event_id}/ack")
async def ack_event_endpoint(
    event_id: str,
    consumer_id: str = Query(default=""),
) -> dict[str, Any]:
    from datetime import UTC, datetime
    from eventbus.offsets import write_offset  # noqa: PLC0415

    now = datetime.now(UTC).strftime("%Y-%m-%dT%H:%M:%SZ")

    def _ack_and_offset() -> dict[str, Any]:
        assert _db is not None
        acked = ack_event(_db, event_id, now)
        seq: int | None = None
        if consumer_id and acked:
            row = _db.execute(
                "SELECT seq FROM events WHERE event_id = ?", (event_id,)
            ).fetchone()
            if row:
                seq = int(row["seq"])
                assert _cfg is not None
                write_offset(_cfg.offsets_dir, consumer_id, seq)
        return {"acked": acked, "seq": seq}

    result = await asyncio.to_thread(_ack_and_offset)
    if not result["acked"]:
        # Either not found or already acked — still 200 for idempotency
        pass
    return {"event_id": event_id, "acked": result["acked"]}
```

Note: returning 200 even if already acked preserves idempotency. If the caller needs to distinguish "first ack" from "already acked", check the `acked` field in the response body.
