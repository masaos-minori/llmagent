# Goal

Remove `except Exception` from `store.py`, replace `dict[str, int]` return of
`check_consistency()` with `ConsistencyReport` DTO, and adapt `add()`/`upsert()` for
frozen `MemoryEntry` using `dataclasses.replace()`.

# Scope

- `scripts/agent/memory/store.py`

# Assumptions

1. `ConsistencyReport` from `agent.memory.models` (Step 1 prerequisite).
2. `MemoryConsistencyError` from `agent.memory.exceptions` (Step 1).
3. `MemoryEntry` is now frozen (Step 2 prerequisite) — direct `entry.field = value` fails.
4. `count_vec()`, `count_entries()`, `count_prunable()`: current `except Exception` returns 0
   sentinel — plan says "remove sentinel, let exception propagate". However, these are
   "Diagnostic use only" (per docstring) — the plan says raise instead of return 0.
   For `count_vec()`, the DB table may not exist in all environments (vec0 not loaded).
   Decision: `count_vec()` raises `sqlite3.OperationalError` (not silently returns 0).
5. `_stamp_entry(entry, now)` helper uses `dataclasses.replace()` to create a new
   MemoryEntry with `created_at`/`updated_at` set.
6. `check_consistency()` exceptions: outer `except sqlite3.OperationalError`.
   The `-1` sentinel for `memories_fts` is replaced with a separate `except` that raises
   `MemoryConsistencyError`.
7. Callers of `check_consistency()` that access `result["memories"]` etc. must update
   to `result.memories` etc. — handled in Step 12.

# Implementation

## Target file

`scripts/agent/memory/store.py`

## Procedure

1. Import `dataclasses`, `ConsistencyReport`, `MemoryConsistencyError`,
   `MemoryStorageError` (for future use).
2. Add `_stamp_entry(entry, now)` helper using `dataclasses.replace()`.
3. Rewrite `add()` to call `_stamp_entry(entry, now)` before INSERT.
4. Rewrite `upsert()` to call `_stamp_entry(entry, _now_iso())` before INSERT.
5. Rewrite `count_vec()` to remove `except Exception`.
6. Rewrite `count_entries()` to remove `except Exception`.
7. Rewrite `count_prunable()` to remove `except Exception`.
8. Rewrite `check_consistency()` to return `ConsistencyReport` and remove `except Exception`.

## Method

```python
import dataclasses

def _stamp_entry(entry: MemoryEntry, now: str) -> MemoryEntry:
    updates: dict[str, str] = {}
    if not entry.created_at:
        updates["created_at"] = now
    if not entry.updated_at:
        updates["updated_at"] = now
    return dataclasses.replace(entry, **updates) if updates else entry

def add(self, entry: MemoryEntry, embedding: list[float] | None = None) -> None:
    now = _now_iso()
    stamped = _stamp_entry(entry, now)
    with SQLiteHelper("session").open(write_mode=True) as db:
        with db.begin_immediate():
            db.execute(self._INSERT_SQL, self._build_row_params(stamped))
            ...

def count_vec(self) -> int:
    with SQLiteHelper("session").open() as db:
        rows = db.fetchall("SELECT COUNT(*) FROM memories_vec")
        return int(rows[0][0]) if rows else 0
    # No except — sqlite3.OperationalError propagates to caller

def check_consistency(self) -> ConsistencyReport:
    with SQLiteHelper("session").open() as db:
        rows = db.fetchall("SELECT COUNT(*) FROM memories")
        memories = int(rows[0][0]) if rows else 0
        try:
            fts_rows = db.fetchall(
                "SELECT COUNT(*) FROM memories_fts WHERE memories_fts MATCH '*'"
            )
            fts = int(fts_rows[0][0]) if fts_rows else 0
        except sqlite3.OperationalError:
            fts_rows2 = db.fetchall("SELECT COUNT(*) FROM memories_fts")
            fts = int(fts_rows2[0][0]) if fts_rows2 else 0
    vec = self.count_vec()
    return ConsistencyReport(memories=memories, fts=fts, vec=vec)
```

## Details

`_build_row_params` is called with `stamped` (not `entry`), so it picks up the new timestamps.
