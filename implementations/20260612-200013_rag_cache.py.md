# Goal

Replace `self._entries: list[dict[str, Any]]` with `list[CacheEntry]` and convert
all `entry["embedding"]` / `entry["context_str"]` dict accesses to attribute access.

# Scope

- `scripts/rag/cache.py`

# Assumptions

1. `CacheEntry` frozen dataclass from `rag.models` (Step 2-3 prerequisite).
2. `CacheEntry(embedding=..., context_str=...)` replaces
   `{"embedding": ..., "context_str": ...}`.
3. `entry["embedding"]` → `entry.embedding`.
4. `entry["context_str"]` → `entry.context_str`.
5. Constructor config fields (`max_size`, `threshold`, `dimension`) remain as
   `__init__` parameters (no separate config DTO needed for this simple class).

# Implementation

## Target file

`scripts/rag/cache.py`

## Procedure

1. Import `CacheEntry` from `rag.models`.
2. Change `self._entries: list[dict[str, Any]]` → `list[CacheEntry]`.
3. Replace dict construction `{"embedding": ..., "context_str": ...}` with
   `CacheEntry(embedding=..., context_str=...)`.
4. Replace `entry["embedding"]` → `entry.embedding`.
5. Replace `entry["context_str"]` → `entry.context_str`.
6. Remove `from typing import Any` if no longer used.
7. Run ruff + mypy.

## Method

Type annotation change + dict literal → dataclass construction + attribute access.
