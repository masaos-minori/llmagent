# Implementation: scripts/shared/config_loader.py

## Goal

Add dedicated exception classes `ConfigMissingError`, `ConfigParseError`, `ConfigReadError` to `config_loader.py`. Update `_load_single()` to raise these typed exceptions instead of generic `ValueError`. Update `load_all()` to catch only `ConfigMissingError` so parse and read errors propagate to callers.

## Scope

- Target: `scripts/shared/config_loader.py`
- Add 3 exception classes
- Update `_load_single()` raise types (4 except clauses)
- Update `load_all()` catch target from `ValueError` to `ConfigMissingError`
- No caller changes outside this file

## Assumptions

1. All new exceptions inherit from `ValueError` so existing `except ValueError` in callers (`config_builders.py:50`, `rag/pipeline.py:78`) continue to catch them.
2. `load()` already propagates `_load_single()` exceptions directly — no change needed.
3. `load_all()` iterates `_BASE_CONFIG_FILES`; missing files should be skipped (debug log); parse/read errors should propagate.

## Implementation

### Target file
`scripts/shared/config_loader.py`

### Procedure
1. Insert 3 exception classes before `class ConfigLoader` (line 34).
2. Update 4 `except` clauses in `_load_single()` (lines 85-92) to raise the new types.
3. Update `load_all()` `except` clause (line 57) from `ValueError` to `ConfigMissingError`.

### Method
Insert after imports, before `class ConfigLoader`:

```python
class ConfigMissingError(ValueError):
    """Config file does not exist."""

class ConfigParseError(ValueError):
    """Config file exists but cannot be parsed."""

class ConfigReadError(ValueError):
    """Config file exists but cannot be read (permission, I/O)."""
```

In `_load_single()`, replace each except:
- `FileNotFoundError` → raise `ConfigMissingError`
- `tomllib.TOMLDecodeError` → raise `ConfigParseError`
- `orjson.JSONDecodeError` → raise `ConfigParseError`
- `OSError` → raise `ConfigReadError`
- non-dict JSON check → raise `ConfigParseError`

In `load_all()`, change:
```python
# Before
except ValueError:
# After
except ConfigMissingError:
    logger.debug("Config file not found: %s", name)
```

### Details
- Keep existing error message strings unchanged — only change exception type.
- `load()` public method: no change (already propagates from `_load_single()`).
- `load_all()` must NOT catch `ConfigParseError` or `ConfigReadError` — these must reach the caller.
