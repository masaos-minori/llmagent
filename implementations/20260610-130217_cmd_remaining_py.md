# Implementation: cmd_config.py / cmd_tooling.py / cmd_ingest.py / cmd_mcp.py / cmd_debug.py

## Goal

Apply formatter delegation and parser migration to the remaining five command files.
Remove config fallback `{}` in `cmd_ingest.py`. Add behavior-lock tests for the three
previously uncovered files (cmd_tooling, cmd_mcp, cmd_debug).

## Scope

- `scripts/agent/commands/cmd_config.py` — `_print_*` → formatter
- `scripts/agent/commands/cmd_tooling.py` — output → formatter; `parse_command_args`
- `scripts/agent/commands/cmd_ingest.py` — config fallback deletion; output → formatter
- `scripts/agent/commands/cmd_mcp.py` — subcommand spec; output → formatter
- `scripts/agent/commands/cmd_debug.py` — structured result; output → formatter
- `tests/test_agent_cmd_tooling.py` — new behavior-lock tests
- `tests/test_agent_cmd_mcp.py` — new behavior-lock tests
- `tests/test_agent_cmd_debug.py` — new behavior-lock tests

## Assumptions

1. `cmd_config.py` `_print_*` methods (`_print_llm_settings`, `_print_rag_config`, etc.)
   are refactored to use `formatter.print_kv_list()`. The formatter replaces individual
   `print(f"  key : value")` calls.
2. `cmd_ingest.py:108-110`: the `except Exception as exc: rag_cfg_dict = {}` fallback is
   deleted. Config load failure raises, which the REPL's outer handler reports.
3. `cmd_tooling.py` and `cmd_mcp.py` behavior-lock tests are written BEFORE refactoring
   to ensure no regressions.
4. `cmd_debug.py` currently has ~3 subcommands (toggle, audit, verbose/normal);
   these are mapped to `parse_command_args` subcommand spec.
5. `deploy/deploy.sh` does not need updating (no files added/removed in scripts/).

## Implementation

### Procedure (per file)

**cmd_config.py**:
1. Replace each `print(f"  key : value")` block in `_print_*` methods with
   `formatter.print_kv_list(pairs)`.
2. Keep `_print_*` methods but they now build `pairs` list and call formatter.

**cmd_tooling.py**:
1. Replace `args.strip().split()` parsing with `parse_command_args`.
2. Replace `print(...)` with `formatter.print_table / print_success / print_no_data`.

**cmd_ingest.py**:
1. Delete lines 108-110 (`except Exception as exc: rag_cfg_dict = {}`).
2. Replace inline `print(...)` with `formatter.*` calls.

**cmd_mcp.py**:
1. Replace `if/elif` subcommand dispatch with `parse_command_args` + dict dispatch.
2. Replace output `print(...)` with formatter calls.

**cmd_debug.py**:
1. Replace inline `print(...)` with `formatter.*` calls.
2. Subcommand dispatch via `parse_command_args`.

### Method

Direct textual edit. Write behavior-lock tests first for tooling/mcp/debug.
