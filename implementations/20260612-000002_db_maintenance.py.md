# Goal

Replace `except Exception` with specific exception types in `db/maintenance.py`, and change
`checkpoint_wal` and `purge_old_sessions` to return typed `WalCheckpointCounts` / `PurgeCounts`
DTOs instead of `dict[str, int]`.

# Scope

- `scripts/db/maintenance.py`

# Assumptions

1. `WalCheckpointCounts` and `PurgeCounts` are defined in `agent/services/models.py`
   (Step 1 prerequisite).
2. `SQLiteHelper.checkpoint()` raises `ValueError` (bad mode) or `RuntimeError` (conn not open);
   `sqlite3.OperationalError` can be raised by `conn.execute()`. The outer context manager
   `SQLiteHelper.open()` can raise `sqlite3.OperationalError` or `ValueError`.
3. `SQLiteHelper.vacuum()` calls `conn.execute("VACUUM")` which raises `sqlite3.OperationalError`
   or `RuntimeError` (conn not open).
4. `_run_integrity_check` `except Exception` (line 190): replace with
   `except (sqlite3.OperationalError, ValueError, RuntimeError)`.
5. `_vacuum_db` `except Exception` (line 218): replace with
   `except (sqlite3.OperationalError, RuntimeError)`.
6. `checkpoint_wal()` currently calls `db.checkpoint(mode)` and returns its `dict[str, int]`
   result directly. Change to wrap in `WalCheckpointCounts`.
7. `purge_old_sessions()` returns `{"age_deleted": ..., "count_deleted": ...}` at line 118.
   Change to return `PurgeCounts(age_deleted=age_deleted, count_deleted=count_deleted)`.
8. Import path: `from agent.services.models import PurgeCounts, WalCheckpointCounts`.
   This creates an `agent` → `db` direction which VIOLATES import-linter contracts
   (db must not import from agent). Instead, move `WalCheckpointCounts` and `PurgeCounts`
   to `db/models.py` (new file) and import from there; `agent/services/models.py` re-exports
   for callers in the agent layer.
   **Revised approach**: define `WalCheckpointCounts` and `PurgeCounts` in a new
   `scripts/db/models.py` file. `agent/services/models.py` imports and re-exports them.

## Details

### db/models.py (new file)

```python
"""db/models.py
Typed DTOs for db-layer operation results.
"""
from __future__ import annotations
from dataclasses import dataclass


@dataclass(frozen=True)
class WalCheckpointCounts:
    """Result of a WAL checkpoint (busy, log_size=pages_in_wal, pages_checkpointed)."""
    busy: int
    log_size: int
    pages_checkpointed: int


@dataclass(frozen=True)
class PurgeCounts:
    """Result of a session retention purge (age_deleted, count_deleted)."""
    age_deleted: int
    count_deleted: int
```

Note: `agent/services/models.py` Step 1 doc placed `WalCheckpointCounts` and `PurgeCounts`
there. If that step was applied first, replace those definitions with imports from `db.models`:
```python
from db.models import PurgeCounts, WalCheckpointCounts
```
