## Goal

Add five tests to `tests/test_ingester.py` covering embedding retry logic and `IngestUrlResult.n_embed_failed` propagation:
1. Retry success: first attempt fails, second succeeds → chunk inserted, `_embed_and_store` returns `(True, True)`.
2. All retries exhausted: every attempt fails → chunk not inserted, returns `(False, False)`.
3. Network error on first attempt, success on retry → chunk inserted.
4. Dimension mismatch on first attempt, success on retry → chunk inserted.
5. `IngestUrlResult.n_embed_failed` is correctly incremented when embedding fails across a full `ingest_url_group()` call.

## Scope

**In-Scope:**
- `tests/test_ingester.py` — append five test methods to `TestEmbedAndStore` (tests 1–4) and `TestIngestUrlGroup` (test 5)
- Add one module-level helper `_make_ingester_with_retry(tmp_path, embed_retry)` to support configurable retry counts
- Mock `rag.ingestion.ingester.time.sleep` in all retry tests to prevent real delays

**Out-of-Scope:**
- Modifying `RagIngester`, `_get_embedding`, `_embed_and_store`, or any production file
- Changing existing tests or helpers

## Assumptions

1. `_get_embedding` uses `self._embed_retry` (set from `config["embed_retry"]` as `int`). The existing `_make_ingester` hardcodes `"1"`. Tests requiring N > 1 retries use the new `_make_ingester_with_retry` helper.
2. Between retries `time.sleep(min(2**attempt, 10))` is called. It must be patched via `patch("rag.ingestion.ingester.time.sleep")` in all retry tests to keep the suite fast.
3. `_embed_and_store` returns `(False, False)` when `_get_embedding` returns `None`. It returns `(True, True)` on success.
4. `_ingest_chunk_files` sets `embed_failed += 1` when `chunk_ok=False` AND `embed_ok=False`. `IngestUrlResult.n_embed_failed` is populated from this count.
5. For `ingest_url_group()` end-to-end tests, patching `"rag.ingestion.ingester.SQLiteHelper"` with a `_FakeSQLiteHelper` instance is sufficient to isolate DB writes.
6. `httpx.RequestError` requires a `request` argument; use `httpx.RequestError("simulated network error")` (no request kwarg needed for the constructor in httpx >= 0.23).

### Method

```python
def _make_ingester_with_retry(tmp_path: Path, embed_retry: int) -> RagIngester:
    (tmp_path / "chunk").mkdir(parents=True, exist_ok=True)
    (tmp_path / "registered").mkdir(parents=True, exist_ok=True)
    return RagIngester(
        config={
            "rag_src_dir": str(tmp_path),
            "embed_url": "http://localhost:8003/embedding",
            "embed_retry": str(embed_retry),
            "embed_workers": "1",
        }
    )
```

### Details

#### test_retry_success

**Class:** `TestEmbedAndStore`  
**Purpose:** Verify that when the first HTTP post returns an empty embedding and the second returns a valid one, `_embed_and_store` succeeds and the chunk is inserted.

```python
def test_retry_success(self, tmp_path: Path) -> None:
    conn, fake_db = _make_db()
    doc_id = self._insert_parent_doc(conn)
    path = _write_chunk(tmp_path / "chunk", "c.txt")
    ingester = _make_ingester_with_retry(tmp_path, embed_retry=2)

    bad_resp = MagicMock()
    bad_resp.raise_for_status.return_value = None
    bad_resp.content = orjson.dumps({"embedding": []})  # empty → ValueError

    with (
        patch("rag.ingestion.ingester.time.sleep"),
        patch.object(
            ingester._client, "post",
            side_effect=[bad_resp, _fake_embed_resp()],
        ),
        patch("rag.ingestion.ingester.SQLiteHelper", return_value=fake_db),
    ):
        result = ingester._embed_and_store(doc_id, path)

    assert result == (True, True)
    count = conn.execute("SELECT COUNT(*) FROM chunks").fetchone()[0]
    assert count == 1
```

Key assertions:
- `result == (True, True)` — retry succeeded
- `chunks` row count is 1 — DB write completed

#### test_ingest_url_group_embed_failed_count

**Class:** `TestIngestUrlGroup`  
**Purpose:** Verify that `IngestUrlResult.n_embed_failed` is set to the number of chunks for which embedding failed, via a full `ingest_url_group()` call.

```python
def test_ingest_url_group_embed_failed_count(self, tmp_path: Path) -> None:
    conn, fake_db = _make_db()
    url = "https://example.com/doc"
    chunk_dir = tmp_path / "chunk" / "example.com" / "doc"
    _write_chunk(chunk_dir, "c1.txt", dataclasses.replace(_DEFAULT_CHUNK, url=url))
    _write_chunk(chunk_dir, "c2.txt", dataclasses.replace(_DEFAULT_CHUNK, url=url, chunk_index=1))
    ingester = _make_ingester(tmp_path)  # embed_retry=1

    bad_resp = MagicMock()
    bad_resp.raise_for_status.return_value = None
    bad_resp.content = orjson.dumps({"embedding": []})  # all embeddings fail

    with (
        patch.object(ingester._client, "post", return_value=bad_resp),
        patch("rag.ingestion.ingester.SQLiteHelper", return_value=fake_db),
    ):
        results = ingester.ingest_url_group([url])

    assert len(results) == 1
    r = results[0]
    assert r.url == url
    assert r.n_embed_failed == 2  # both chunks failed embedding
    assert r.n_success == 0
```

Key assertions:
- `r.n_embed_failed == 2` — both chunks counted as embed failures
- `r.n_success == 0` — no chunk was successfully inserted
