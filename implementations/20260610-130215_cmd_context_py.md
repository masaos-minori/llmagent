# Implementation: cmd_context.py ‚Äî remove ctx direct mutation, service delegation

## Goal

1. `/clear`: replace `ctx.conv.history = ctx.conv.history[:1]` with a call to a new
   `clear_conversation(ctx, new_session: bool)` helper in `agent/services/conversation_service.py`.
2. `/system`: replace direct `ctx.conv.system_prompt_name` / `system_prompt_content` /
   `history[0]` mutation with a call to `switch_system_prompt(ctx, name)`.
3. Move `_print_token_line()` output to use `formatter.print_kv_list()`.

## Scope

- `scripts/agent/commands/cmd_context.py` ‚Äî primary
- `scripts/agent/services/conversation_service.py` ‚Äî new file with `clear_conversation` and
  `switch_system_prompt` functions
- `tests/test_agent_cmd_context.py` ‚Äî verify tests still pass

## Assumptions

1. `clear_conversation(ctx, new_session)` encapsulates:
   - `ctx.conv.history = ctx.conv.history[:1]`
   - `reset_session_stats(ctx)` (from mixin_base)
   - if `new_session`: `ctx.session.start()`
2. `switch_system_prompt(ctx, name)` encapsulates:
   - validation: name in ctx.cfg.tool.system_prompts
   - `ctx.conv.system_prompt_name = name`
   - `ctx.conv.system_prompt_content = ctx.cfg.tool.system_prompts[name]`
   - update `ctx.conv.history[0]` system message
3. The service functions take `AgentContext` directly (not injected via DI container).
4. `/history` `int()` parsing is updated to use `parse_command_args`.
5. `deploy/deploy.sh` needs a new `cp` line for `conversation_service.py`.

## Implementation

### Target files

`scripts/agent/services/conversation_service.py` (new), `scripts/agent/commands/cmd_context.py`

### Procedure

1. Create `conversation_service.py` with `clear_conversation` and `switch_system_prompt`.
2. In `cmd_context.py`:
   - Import `clear_conversation`, `switch_system_prompt` from the new service.
   - Replace direct ctx mutations in `_cmd_clear()`.
   - Replace direct ctx mutations in `_cmd_system()`.
   - Update `_cmd_history()` int parsing to use `parse_command_args`.
   - Update `_print_token_line()` to use `formatter.print_kv_list`.

### Method

Direct textual edit; new file for conversation_service.py.
