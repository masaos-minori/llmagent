### Method

```python
def create_workflow_schema() -> None:
    """Create workflow.sqlite tables."""
    from db.workflow_schema import init_schema
    from db.config import build_db_config
    cfg = build_db_config()
    Path(cfg.workflow_db_path).parent.mkdir(parents=True, exist_ok=True)
    init_schema()
    logger.info("Workflow schema created successfully.")
```

### Details
- Import `init_schema` inline to avoid circular imports at module level.
- Directory creation (`mkdir`) mirrors the `__main__` block in `workflow_schema.py`.
- `logger` is already defined in `create_schema.py` — no new import needed.
- `Path` must be imported at top of file if not already present.
