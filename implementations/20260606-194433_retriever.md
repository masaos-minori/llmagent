# Implementation: Memory Retriever Configurable Scoring

## Goal

Replace hard-coded retrieval constants in `agent/memory/retriever.py` with values injected from `MemoryConfig`, enabling tuning without code changes.

## Scope

**In:**
- Add `memory_fts_limit`, `memory_rrf_k`, `memory_recency_days` fields to `MemoryConfig` in `agent/config.py`
- Update `MemoryRetriever.__init__()` to accept and store these values
- Replace module-level constants `_FTS_CANDIDATE_LIMIT`, `_RRF_K`, `_RECENCY_DAYS` with instance attributes

**Out:**
- Changing the default values (keep existing defaults: 50 / 60 / 7.0)
- Adding new retrieval algorithms

## Assumptions

- `scripts/agent/config.py`: `MemoryConfig` dataclass with `__post_init__` validation
- `scripts/agent/memory/retriever.py`: `MemoryRetriever` class, module-level constants `_FTS_CANDIDATE_LIMIT = 50`, `_RRF_K = 60`, `_RECENCY_DAYS = 7.0`
- `scripts/agent/factory.py` or `memory/layer.py` constructs `MemoryRetriever`; must pass new args

## Implementation

### 1. `scripts/agent/config.py` — add fields to `MemoryConfig`

```python
@dataclass
class MemoryConfig:
    # ... existing fields ...
    memory_fts_limit: int = 50          # max FTS5 candidates before rescoring
    memory_rrf_k: int = 60              # RRF constant
    memory_recency_days: float = 7.0    # recency window in days

    def __post_init__(self) -> None:
        # add validation
        if self.memory_fts_limit < 1:
            raise ValueError(f"memory_fts_limit must be >= 1, got {self.memory_fts_limit}")
        if self.memory_rrf_k < 1:
            raise ValueError(f"memory_rrf_k must be >= 1, got {self.memory_rrf_k}")
        if self.memory_recency_days <= 0:
            raise ValueError(f"memory_recency_days must be > 0, got {self.memory_recency_days}")
```

Also update `_build_memory_config()` to read the new fields:
```python
memory_fts_limit=int(cfg.get("memory_fts_limit", 50)),
memory_rrf_k=int(cfg.get("memory_rrf_k", 60)),
memory_recency_days=float(cfg.get("memory_recency_days", 7.0)),
```

### 2. `scripts/agent/memory/retriever.py` — use injected values

Change `MemoryRetriever.__init__()`:

```python
def __init__(
    self,
    *,
    fts_limit: int = 50,
    rrf_k: int = 60,
    recency_days: float = 7.0,
) -> None:
    self._fts_limit = fts_limit
    self._rrf_k = rrf_k
    self._recency_days = recency_days
```

Replace every use of `_FTS_CANDIDATE_LIMIT`, `_RRF_K`, `_RECENCY_DAYS` with `self._fts_limit`, `self._rrf_k`, `self._recency_days`.

Keep module-level constants as defaults (or remove them if unused after the change).

### 3. Construction site (wherever `MemoryRetriever()` is instantiated)

Pass the config values:
```python
retriever = MemoryRetriever(
    fts_limit=cfg.memory.memory_fts_limit,
    rrf_k=cfg.memory.memory_rrf_k,
    recency_days=cfg.memory.memory_recency_days,
)
```

```bash
uv run mypy scripts/

```

Confirm:
