# Implementation: Add logger.info in FusionStage.run() for use_rrf=False path

## Goal
Emit `logger.info("FusionStage: dedup-only mode (use_rrf=False)")` when `_dedup_hits()`
is used, making the active fusion mode visible in INFO-level logs without `--debug`.

## Scope
- File: `scripts/rag/stages/fusion.py`
- Method: `run()` — one `logger.info` call added to the `else` branch
- Requires adding `import logging` and `logger = logging.getLogger(__name__)` if not present

## Assumptions
- Current `run()` (lines 14-18): `if self._use_rrf: rrf_merge; else: _dedup_hits`
- No `logger` is currently defined in `fusion.py` — must add module-level logger
- INFO level is appropriate (configuration choice, not an error)
- No warning for `use_rrf=False` + `use_mqe=True` combination in this step
  (that is a separate enhancement; the plan marks it as future / UNK-01)

## Implementation

### Target file
`scripts/rag/stages/fusion.py`

### Procedure
1. Add `import logging` and `logger = logging.getLogger(__name__)` at module level
2. In `run()`, add `logger.info(...)` before `ctx.merged = _dedup_hits(...)`

### Method
Two edits: logger setup + info call.

### Details

**Current `run()` body (lines 14-18):**
```python
    def run(self, ctx: PipelineContext) -> None:
        if self._use_rrf:
            ctx.merged = RagScorer.rrf_merge(ctx.search_results, rrf_k=self._rrf_k)
        else:
            ctx.merged = _dedup_hits(ctx.search_results)
```

**Updated `run()` body:**
```python
    def run(self, ctx: PipelineContext) -> None:
        if self._use_rrf:
            ctx.merged = RagScorer.rrf_merge(ctx.search_results, rrf_k=self._rrf_k)
        else:
            logger.info("FusionStage: dedup-only mode (use_rrf=False)")
            ctx.merged = _dedup_hits(ctx.search_results)
```

**New module-level logger (add after imports):**
```python
logger = logging.getLogger(__name__)
```
