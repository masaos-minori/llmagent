# Implementation: Tests - Add eventbus_db_path tests

## Goal

Add tests for eventbus_db_path field and SQLiteHelper("eventbus") connection.

## Scope

- tests/test_db_config.py: add tests for eventbus_db_path field
- tests/test_sqlite_helper.py: add tests for SQLiteHelper("eventbus") connection

## Assumptions

1. Test directory is /opt/llm/db (same as other DBs) — may need to adjust if different
2. Existing test patterns should be followed

## Implementation

### Target files

- `tests/test_db_config.py`
- `tests/test_sqlite_helper.py`

### Procedure

#### test_db_config.py additions

1. Add test for DbConfig.eventbus_db_path default value
2. Add test for build_db_config() resolution of eventbus_db_path

#### test_sqlite_helper.py additions

1. Add test for SQLiteHelper("eventbus") connection
2. Add test verifying no vec extension loaded for eventbus

### Method

- Follow existing test patterns exactly (same style as workflow tests)

### Details

```python
# In test_db_config.py:

def test_eventbus_db_path_default() -> None:
    """DbConfig.eventbus_db_path should have the default value."""
    cfg = DbConfig(
        rag_db_path="/opt/llm/db/rag.sqlite",
        session_db_path="/opt/llm/db/session.sqlite",
    )
    assert cfg.eventbus_db_path == "/opt/llm/db/eventbus.sqlite"

def test_build_db_config_eventbus() -> None:
    """build_db_config() should resolve eventbus_db_path from common.toml."""
    cfg = build_db_config()
    assert cfg.eventbus_db_path == "/opt/llm/db/eventbus.sqlite"


# In test_sqlite_helper.py:

def test_sqlite_helper_eventbus_target() -> None:
    """SQLiteHelper('eventbus') should resolve to eventbus DB path."""
    with SQLiteHelper("eventbus") as helper:
        assert helper.DB_PATH == "/opt/llm/db/eventbus.sqlite"
        assert helper._default_load_vec is False

def test_sqlite_helper_eventbus_no_vec() -> None:
    """SQLiteHelper('eventbus') should not load vec extension by default."""
    with SQLiteHelper("eventbus") as helper:
        helper.open(write_mode=False, row_factory=False)
        assert helper._default_load_vec is False
```
