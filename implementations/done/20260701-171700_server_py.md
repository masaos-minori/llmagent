# Implementation: Update _MCP_TOOLS imports and usages in 9 server.py files

## Goal

Update all `from mcp.<name>.tools import _MCP_TOOLS` import lines and every usage of
`_MCP_TOOLS` within each server.py file to use `TOOL_LIST` instead.

## Scope

- Target files (all under scripts/mcp/):
  - mcp/git/server.py (import line 32; usages at lines 72, 117)
  - mcp/mdq/server.py (import line 42)
  - mcp/rag_pipeline/server.py (import line 40; usages at lines 148, 174)
  - mcp/shell/server.py (import line 38)
  - mcp/cicd/server.py (import line 39; usages at lines 100, 159)
  - mcp/web_search/server.py (import line 31; usages at lines 165, 189)
  - mcp/file/read_server.py (import line 56)
  - mcp/file/write_server.py (import line 43)
  - mcp/file/delete_server.py (import line 37; usages at lines 125, 147)

## Assumptions

- tools.py files have already been renamed (see `20260701-171607_tools_py.md`).
  Running server.py update before tools.py rename will cause ImportError.
- Each server.py has exactly the import lines listed; the line numbers are approximate
  and must be verified by reading the file before editing.
- All usages of `_MCP_TOOLS` within each server.py are in list comprehensions or
  direct assignment (`mcp_tools = _MCP_TOOLS`).

## Implementation

### Procedure

For each target file:
1. Replace import: `from mcp.<name>.tools import _MCP_TOOLS` → `from mcp.<name>.tools import TOOL_LIST`
2. Replace all usages of `_MCP_TOOLS` in the file body with `TOOL_LIST`.
3. Verify with grep that zero `_MCP_TOOLS` references remain in the file.

### Method

Use replace_all on each file: `_MCP_TOOLS` → `TOOL_LIST`.
The replacement count should equal (1 import + N usages) per file.

### Details

Known usage patterns in server.py files:
- List comprehension: `[{**t, "server_key": "x"} for t in _MCP_TOOLS]`
  → `[{**t, "server_key": "x"} for t in TOOL_LIST]`
- Direct assignment: `mcp_tools = _MCP_TOOLS`
  → `mcp_tools = TOOL_LIST`

After all updates, run:
```bash
rg "_MCP_TOOLS" scripts/mcp/
```
Expected: zero matches across all scripts/mcp/ source files.
