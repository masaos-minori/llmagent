# Implementation: agent/workflow/attempt_ops.py + artifact_ops.py + idempotency_ops.py + state_store.py — Persist new fields

## Goal

Update the four workflow ops files to persist `error_kind`, `error_detail`, `workflow_id`, and `attempt_number` in the relevant tables.

## Scope

**In**: Function signature changes in `finish_attempt()`, `record_artifact()`, `begin_stage_if_new()`. Update `state_store.py` duplicate methods.

**Out**: Changes to `schema_sql.py`, `workflow_engine.py`, tests.

## Assumptions

1. `schema_sql.py` is updated first (new columns exist before ops functions try to write them).
2. All new parameters have `None` defaults for backward compatibility with callers that don't pass them.
3. `state_store.py` has duplicate `start_attempt()`/`finish_attempt()` methods that must also be updated.

## Implementation

### Target file
`scripts/agent/workflow/attempt_ops.py`, `scripts/agent/workflow/artifact_ops.py`, `scripts/agent/workflow/idempotency_ops.py`, `scripts/agent/workflow/state_store.py`

### Procedure
1. Update `finish_attempt()` signature in `attempt_ops.py`.
2. Update `record_artifact()` signature in `artifact_ops.py`.
3. Update `begin_stage_if_new()` in `idempotency_ops.py` to insert `workflow_id`.
4. Update duplicate methods in `state_store.py`.

### Method

**`attempt_ops.py` — `finish_attempt()` update:**
```python
def finish_attempt(
    db: SQLiteHelper,
    attempt_id: str,
    status: str,
    error_msg: str | None = None,
    error_kind: str | None = None,   # NEW: "timeout"|"halt"|"exception"
    error_detail: str | None = None, # NEW: full error detail
) -> None:
    db.execute(
        "UPDATE attempts SET status=?, ended_at=?, error_msg=?, error_kind=?, error_detail=? WHERE attempt_id=?",
        (status, _now(), error_msg, error_kind, error_detail, attempt_id),
    )
    db.commit()
```

**`artifact_ops.py` — `record_artifact()` update:**
```python
def record_artifact(
    db: SQLiteHelper,
    task_id: str,
    stage_id: str,
    uri: str,
    workflow_id: str | None = None,     # NEW
    attempt_number: int | None = None,  # NEW
) -> ArtifactRef:
    artifact_id = str(uuid.uuid4())
    now = _now()
    db.execute(
        """
        INSERT INTO artifacts (artifact_id, task_id, stage_id, uri, created_at, workflow_id, attempt_number)
        VALUES (?, ?, ?, ?, ?, ?, ?)
        """,
        (artifact_id, task_id, stage_id, uri, now, workflow_id, attempt_number),
    )
    db.commit()
    return ArtifactRef(
        artifact_id=artifact_id,
        task_id=task_id,
        stage_id=stage_id,
        uri=uri,
        created_at=now,
    )
```
