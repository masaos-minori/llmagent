# Goal

Replace raw dict access (`r["session_id"]`, `r["title"]`, etc.) in `cmd_session.py`
with `SessionRow` typed DTO attribute access.

# Scope

- `scripts/agent/commands/cmd_session.py`

# Assumptions

1. `SessionRow` is defined in `agent/services/models.py` (Step 1 prerequisite).
2. `ctx.session.list_sessions(limit)` currently returns `list[dict]`.
   `cmd_session.py` must convert each dict to `SessionRow` at the call site.
   (Changing `session.py` to return `SessionRow` directly is a larger change — use
   a local conversion helper `_to_session_row(r: dict) -> SessionRow` here.)
3. Current dict access at the list command (around line 68-84):
   ```python
   rows = self._ctx.session.list_sessions(limit)
   ...
   title = r["title"] if r["title"] is not None else ""
   f"{r['session_id']:>4}{'*' if r['is_current'] else ' '}"
   r["created_at"]
   ```
4. `SessionRow.title` is `str | None` — use `sr.title or ""` is acceptable here since
   empty string and None are both "no title" for display purposes. Or use
   `sr.title if sr.title is not None else ""` for explicit None handling.
5. `_session_load_safe` and `_session_delete` deal with `arg` (int session_id), not
   list rows — those do not need SessionRow changes.

# Implementation

## Target file

`scripts/agent/commands/cmd_session.py`

## Procedure

1. Import `SessionRow` from `agent.services.models`.
2. Add module-level helper:
   ```python
   def _to_session_row(r: dict) -> SessionRow:
       return SessionRow(
           session_id=int(r["session_id"]),
           title=r.get("title"),
           created_at=str(r.get("created_at", "")),
           is_current=bool(r.get("is_current", False)),
       )
   ```
3. In the list command handler (around line 68), convert rows:
   ```python
   rows = self._ctx.session.list_sessions(limit)
   if not rows:
       self._out.write_no_data("No sessions found")
       return
   session_rows = [_to_session_row(r) for r in rows]
   table_rows = [
       [
           f"{sr.session_id:>4}{'*' if sr.is_current else ' '}",
           (title := sr.title if sr.title is not None else "")[:29] + "..."
           if len(title) > 32 else title,
           sr.created_at,
       ]
       for sr in session_rows
   ]
   self._out.write_table(["ID  ", "Title", "Created"], table_rows)
   ```
4. Run ruff + mypy.

## Method

Adapter pattern at call site. `session.list_sessions()` interface unchanged.
