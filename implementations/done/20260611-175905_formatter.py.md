# Goal

Replace direct `print()` calls in `formatter.py` with delegation to `CliOutputPort`,
and expose a module-level singleton so existing callers continue to work unchanged.

# Scope

- `scripts/agent/commands/formatter.py` — refactor existing `print_*` functions to delegate to `CliOutputPort`
- No changes to callers in this step (backward-compatible shim approach)

# Assumptions

1. Existing callers (`cmd_db.py`, `cmd_session.py`, etc.) import `print_success`, `print_error`, etc. as module-level names — those names must remain importable after this change.
2. `CliOutputPort` is defined in `output_port.py` (Step 1 prerequisite).
3. A module-level `_default_out = CliOutputPort()` singleton is sufficient; no global-state concern since this is a CLI tool.

# Implementation

## Target file

`scripts/agent/commands/formatter.py`

## Procedure

1. Import `CliOutputPort` from `agent.commands.output_port`.
2. Create module-level `_default_out: CliOutputPort = CliOutputPort()`.
3. Replace each `print_*` function body with a delegation call to `_default_out`.
4. Keep all function names and signatures identical.
5. Run ruff + mypy.

## Method

Shim delegation — all existing callers keep importing the same names.
The actual output behavior moves to `CliOutputPort`, which tests can replace.

## Details

```python
"""agent/commands/formatter.py
Shared output helpers for command mixins — thin shims over OutputPort.
"""
from __future__ import annotations

from agent.commands.output_port import CliOutputPort

_default_out: CliOutputPort = CliOutputPort()


def print_success(msg: str) -> None:
    _default_out.write_success(msg)


def print_error(msg: str) -> None:
    _default_out.write_error(msg)


def print_no_data(msg: str) -> None:
    _default_out.write_no_data(msg)


def print_validation_error(msg: str) -> None:
    _default_out.write_validation_error(msg)


def print_table(headers: list[str], rows: list[list[str]]) -> None:
    _default_out.write_table(headers, rows)


def print_kv_list(pairs: list[tuple[str, str]], key_width: int = 22) -> None:
    _default_out.write_kv(pairs, key_width)
```
