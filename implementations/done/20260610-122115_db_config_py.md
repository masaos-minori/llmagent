# Implementation: db/config.py — remove embed_url, make sqlite_vec_so optional

## Goal

Remove `embed_url` from `DbConfig` (it belongs to the embedding service layer, not the DB layer)
and make `sqlite_vec_so` optional (empty string allowed; validation moved to call site).
Remove the `embed_url` key from `build_db_config()`.

## Scope

- `scripts/db/config.py` — primary target
- Callers of `DbConfig` that pass `embed_url`: `build_db_config()` (same file), any tests

## Assumptions

1. `embed_url` is already read in `agent/config.py` (`EmbeddingClientConfig.embed_url`).
   Removing it from `DbConfig` does not break the embedding path.
2. `sqlite_vec_so` is only needed when the vec extension must be loaded (RAG path).
   Session DB operations do not need it. Allowing empty string here lets `SQLiteHelper`
   decide at `open()` time whether to load the extension.
3. `build_db_config()` currently reads `embed_url` from `common.toml`; that line is deleted.
4. `__post_init__` validation for `sqlite_vec_so` non-emptiness is removed.
   The existence check (`Path(self.sqlite_vec_so).exists()`) is also removed (guarded by empty check).

## Implementation

### Target file

`scripts/db/config.py`

### Procedure

1. Delete the `embed_url: str` field from `DbConfig`.
2. Remove the `if not self.embed_url: raise ValueError(...)` block from `__post_init__`.
3. Remove the `if not self.sqlite_vec_so: raise ValueError(...)` block from `__post_init__`.
4. Remove the `if not Path(self.sqlite_vec_so).exists(): raise ValueError(...)` block.
5. Remove `embed_url=cfg.get("embed_url", "")` from `build_db_config()`.
6. Run ruff + mypy.

### Method

Direct textual edit.

### Details

Resulting `DbConfig` fields:
```python
@dataclass
class DbConfig:
    rag_db_path: str
    session_db_path: str
    sqlite_vec_so: str          # empty string = vec extension not required
    sqlite_timeout: int = 30
    sqlite_busy_timeout_ms: int = 30000
    embedding_dims: int = 384
```

`__post_init__` keeps:
- `rag_db_path` non-empty check
- `session_db_path` non-empty check
- parent directory existence checks for both paths
- `sqlite_timeout >= 1` check
- `embedding_dims >= 1` check

Removes all `sqlite_vec_so` and `embed_url` validation.

`build_db_config()` keeps all fields except `embed_url`.
