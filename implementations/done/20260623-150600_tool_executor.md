**変更後:**

`_restart_after_timeout()` は fire-and-forget で再起動を試みる (呼び出し元の待機を遅延させない)。

### `_restart_after_timeout()` 追加 (StdioTransport クラス内):

```python
    async def _restart_after_timeout(self) -> None:
        """Kill the timed-out process and restart it to prevent buffer contamination."""
        logger.info("Restarting stdio server after timeout: key=%r", self._server_key)
        if self._proc is not None:
            try:
                self._proc.kill()
                await asyncio.wait_for(self._proc.wait(), timeout=2.0)
            except (TimeoutError, OSError):
                pass
        try:
            await self.start()
            logger.info("stdio server restarted: key=%r", self._server_key)
        except Exception as e:  # noqa: BLE001
            logger.error("Failed to restart stdio server %r: %s", self._server_key, e)
```

## 変更 4: `_execute_with_cache()` — キャッシュスタンピード防止

`ToolExecutor.__init__()` に `self._inflight: dict[str, asyncio.Future[ToolCallResult]] = {}` を追加する。

### 変更箇所: `_execute_with_cache()` (lines 602-632)

**変更後:**
```python
    async def _execute_with_cache(
        self,
        tool_name: str,
        args: dict[str, Any],
    ) -> ToolCallResult:
        cache_key = f"{tool_name}:{_json_dumps(args)}"
        cached = self._cache.get(cache_key)
        if cached is not None:
            age = time.time() - cached.cached_at
            if age < self._cache_ttl:
                self._cache.move_to_end(cache_key)
                self.stat_cache_hits += 1
                logger.info("Tool cache hit: %s (age=%.0fs)", tool_name, age)
                return ToolCallResult(
                    output=cached.output,
                    is_error=cached.is_error,
                    request_id="",
                    server_key="",
                    error_type="tool" if cached.is_error else "",
                )
            del self._cache[cache_key]

        # In-flight deduplication: await existing Future if same key is executing
        if cache_key in self._inflight:
            logger.debug("Tool cache in-flight hit: %s", tool_name)
            return await asyncio.shield(self._inflight[cache_key])

        fut: asyncio.Future[ToolCallResult] = asyncio.get_event_loop().create_future()
        self._inflight[cache_key] = fut
        try:
            result = await self._raw_execute(tool_name, args)
            if not result.is_error:
                self._cache[cache_key] = CacheEntry(
                    output=result.output, is_error=result.is_error, cached_at=time.time()
                )
                if self._cache_max_size > 0 and len(self._cache) > self._cache_max_size:
                    evicted_key, _ = self._cache.popitem(last=False)
                    logger.debug("Tool cache LRU evict: %r", evicted_key)
            fut.set_result(result)
        except Exception as e:
            fut.set_exception(e)
            raise
        finally:
            self._inflight.pop(cache_key, None)
        return result
```
