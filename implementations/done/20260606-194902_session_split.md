# Implementation: Split session.py Repository Classes into Separate Files

## Goal

Move `SessionMessageRepository`, `DocumentRepository`, and `NoteRepository` from `scripts/agent/session.py` into separate files, keeping `AgentSession` as a thin facade.

## Scope

**In:**
- Create `scripts/agent/session_message_repo.py` containing `SessionMessageRepository`
- Create `scripts/agent/document_repo.py` containing `DocumentRepository`
- Create `scripts/agent/note_repo.py` containing `NoteRepository`
- Update `scripts/agent/session.py` to import and re-export from the new modules

**Out:**
- Changing the public API of any class (interface preserved exactly)
- Moving `AgentSession` itself to a new file

## Assumptions

- `scripts/agent/session.py` contains four classes in order: `SessionMessageRepository`, `DocumentRepository`, `NoteRepository`, `AgentSession`
- `AgentSession` delegates to the three repository classes internally
- `AGENTS.md` lists `agent/session.py` as a test-covered module; tests must continue to pass
- `deploy/deploy.sh` copies `scripts/` to `/opt/llm/scripts/`; new files must be added to the copy list

## Implementation

### Step A — Create `scripts/agent/session_message_repo.py`

Copy `SessionMessageRepository` class (and any module-level helpers it requires) into the new file. Include the module docstring:
```python
"""agent/session_message_repo.py
Session and message persistence repository.
"""
```

### Step B — Create `scripts/agent/document_repo.py`

Copy `DocumentRepository` class into the new file.

### Step C — Create `scripts/agent/note_repo.py`

Copy `NoteRepository` class into the new file.

### Step D — Update `scripts/agent/session.py`

Replace each class definition with an import. Keep backward-compat re-exports so existing callers `from agent.session import SessionMessageRepository` continue to work:

```python
"""agent/session.py
AgentSession facade — delegates to domain-specific repository modules.
"""
from agent.session_message_repo import SessionMessageRepository  # noqa: F401 — re-export
from agent.document_repo import DocumentRepository               # noqa: F401 — re-export
from agent.note_repo import NoteRepository                       # noqa: F401 — re-export

# AgentSession remains here (it uses all three repos)
class AgentSession:
    ...
```

### Step E — Update `deploy/deploy.sh`

Add copy lines for the three new files (follow existing cp pattern in the file).

### Step F — Update `routing.md` if needed

Add entries for the new modules under the "File / module layout" row if they represent significant standalone units.
