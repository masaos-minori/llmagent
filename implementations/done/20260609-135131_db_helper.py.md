# Implementation: db/helper.py — Replace ConfigLoader with build_db_config

## Goal

Replace the `ConfigLoader` direct usage in `_ensure_config()` with `build_db_config()`,
so `helper.py` no longer imports `shared.config_loader`.

## Scope

- `scripts/db/helper.py`: swap import; update `_ensure_config()`.

## Assumptions

1. `build_db_config()` is importable from `db.config`; no circular import risk since
   `db.config` does not import from `db.helper`.
2. The class-level cache pattern (`_config_loaded`) is KEPT — only the data source changes.
3. `_BUSY_TIMEOUT_MS` is now populated from `db_cfg.sqlite_busy_timeout_ms`.
4. If `build_db_config()` raises (e.g., validation error), the except block logs and leaves
   `_config_loaded = False` so a retry is possible.

## Implementation

### Target file

`scripts/db/helper.py`

### Procedure

**Replace import:**
```python
# Remove:
from shared.config_loader import ConfigLoader

# Add:
from db.config import build_db_config
```

**Replace _ensure_config() body:**
```python
@classmethod
def _ensure_config(cls) -> None:
    if cls._config_loaded:
        return
    try:
        db_cfg = build_db_config()
    except Exception as e:
        logger.warning(f"DbConfig load failed: {e}")
        return
    cls._RAG_PATH = db_cfg.rag_db_path
    cls._SESSION_PATH = db_cfg.session_db_path
    cls.SQLITE_VEC_SO = db_cfg.sqlite_vec_so
    cls._SQLITE_TIMEOUT = db_cfg.sqlite_timeout
    cls._BUSY_TIMEOUT_MS = db_cfg.sqlite_busy_timeout_ms
    cls._config_loaded = True
```
