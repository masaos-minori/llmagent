# Implementation: Memory Extract — Expand Scope to User Messages

## Goal

Extend `agent/memory/extract.py` to extract memory candidates from user messages in addition to assistant messages, so that persistent user rules, constraints, and explicit instructions are captured.

## Scope

**In:**
- Add `_try_extract_from_user()` function alongside the existing `_try_extract_from_assistant()`
- Update `extract_memories()` to call both and merge results
- Respect `MAX_ENTRIES` cap using importance-priority ordering

**Out:**
- Tool output extraction (deferred; `role="tool"` messages are raw data, risk of noise is high)
- Changing the `MAX_ENTRIES = 20` default value
- LLM-assisted extraction (remains rule-based only)

## Assumptions

- `scripts/agent/memory/extract.py`: public entry point is `extract_memories(history: list[LLMMessage]) -> list[MemoryEntry]`
- `LLMMessage.role` values: `"user"`, `"assistant"`, `"tool"`, `"system"`
- User messages that contain explicit rules, constraints, or policy statements should be captured as semantic memory
- User messages that are simple questions or short inputs should be skipped

## Implementation

### `scripts/agent/memory/extract.py`

Add a user-message extractor:

```python
# Minimum user message character count to consider for extraction
MIN_USER_CONTENT_CHARS = 60

def _try_extract_from_user(msg: LLMMessage, turn_index: int) -> MemoryEntry | None:
    """Extract a MemoryEntry from a user message when it contains rules or constraints.

    Only captures messages that contain explicit instruction/rule/policy language.
    Short questions and confirmations are ignored.
    """
    content = msg.get("content", "") if isinstance(msg, dict) else getattr(msg, "content", "")
    if not isinstance(content, str) or len(content) < MIN_USER_CONTENT_CHARS:
        return None
    # Only extract if semantic keywords are present — user rules/constraints only
    if not _SEMANTIC_KEYWORDS.search(content):
        return None
    summary = _make_summary(content)
    importance = _importance_from_content(content, is_semantic=True)
    return _make_entry(
        content=content,
        summary=summary,
        memory_type="semantic",
        source_type=SourceType.RULE,  # user-stated rules
        importance=importance,
        turn_id=str(turn_index),
    )
```

Update `extract_memories()` to include user messages:

```python
def extract_memories(history: list[LLMMessage], ...) -> list[MemoryEntry]:
    non_system = [m for m in history if _get_role(m) not in ("system",)]
    if len(non_system) < MIN_TURNS:
        return []

    candidates: list[MemoryEntry] = []
    for i, msg in enumerate(non_system):
        role = _get_role(msg)
        if role == "assistant":
            entry = _try_extract_from_assistant(msg, i)
        elif role == "user":
            entry = _try_extract_from_user(msg, i)
        else:
            entry = None
        if entry is not None:
            candidates.append(entry)

    # Sort by importance descending; apply MAX_ENTRIES cap
    candidates.sort(key=lambda e: e.importance, reverse=True)
    return candidates[:MAX_ENTRIES]
```

def test_user_rule_message_extracted_as_semantic():
    """User message containing 'always' keyword is extracted as semantic memory."""
    history = [
        make_msg("user", "Always use type annotations in all Python functions."),
        make_msg("assistant", "Understood, I will always use type annotations."),
    ]
    entries = extract_memories(history)
    assert any(e.memory_type == "semantic" and "annotation" in e.content for e in entries)

def test_short_user_message_not_extracted():
    """Short user messages below MIN_USER_CONTENT_CHARS are not extracted."""
    history = [
        make_msg("user", "ok"),
        make_msg("assistant", "Got it."),
    ]
    entries = extract_memories(history)
    assert len(entries) == 0

def test_max_entries_cap_uses_importance_priority():
    """When candidates exceed MAX_ENTRIES, highest-importance entries are kept."""
    # Create more than MAX_ENTRIES candidates
    history = [make_high_importance_msg(i) for i in range(25)]
    entries = extract_memories(history)
    assert len(entries) <= MAX_ENTRIES
    # All kept entries should have higher importance than dropped ones
