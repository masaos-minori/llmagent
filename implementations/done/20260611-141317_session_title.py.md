# Implementation: agent/services/session_title.py — SessionTitleResult DTO + remove fallback

## Goal

1. Replace `except Exception` with specific exception types in `generate()`.
2. Remove both `ctx.session.set_title(first_input[:50])` fallback calls.
3. Change return type from `None` to `SessionTitleResult`.
4. Raise `SessionTitleGenerationError` on failure.
5. Update `cmd_session._generate_session_title()` to catch `SessionTitleGenerationError`
   and log a warning.

## Scope

**Target files**:
- `scripts/agent/services/session_title.py`
- `scripts/agent/commands/cmd_session.py` (`_generate_session_title` method)

In scope:
- `generate()` return type: `None` → `SessionTitleResult`
- Remove `http is None` fallback; raise `SessionTitleGenerationError` instead
- `except Exception` → `except (httpx.HTTPStatusError, httpx.RequestError, orjson.JSONDecodeError, KeyError)`
- `_generate_session_title()` in `cmd_session.py` catches `SessionTitleGenerationError`

Out of scope:
- Other callers of `generate()`

## Assumptions

1. `SessionTitleResult` is `@dataclass(frozen=True)` with `title: str`.
2. `SessionTitleGenerationError(RuntimeError)` is in `exceptions.py`.
3. `ctx.services.http is None` path was a fallback; it now raises `SessionTitleGenerationError`.
   The caller (`_generate_session_title`) will log the warning.
4. `resp.json()` may raise `orjson.JSONDecodeError` if the LLM response is malformed.
5. `choices[0].get("message", {}).get("content", "")` path: if `title` is empty string,
   raise `SessionTitleGenerationError("LLM returned empty title")`.
6. `cmd_session._generate_session_title()` currently at line 30 calls `generate()` and
   does not catch exceptions (the task exception would propagate as an asyncio warning).

### Procedure

**session_title.py — full rewrite**:
```python
import httpx
import orjson
from agent.services.exceptions import SessionTitleGenerationError
from agent.services.models import SessionTitleResult

class SessionTitleService:
    async def generate(self, ctx: AgentContext, first_input: str) -> SessionTitleResult:
        if ctx.services.http is None:
            raise SessionTitleGenerationError("HTTP client not configured")
        prompt = _TITLE_PROMPT.format(text=first_input[:200])
        try:
            resp = await ctx.services.http.post(
                ctx.cfg.llm.llm_url,
                json={
                    "messages": [{"role": "user", "content": prompt}],
                    "temperature": ctx.cfg.llm.title_llm_temperature,
                    "max_tokens": ctx.cfg.llm.title_llm_max_tokens,
                    "stream": False,
                },
            )
            resp.raise_for_status()
            data = orjson.loads(resp.content)
            choices = data.get("choices", [])
            if not choices:
                raise SessionTitleGenerationError("LLM response has no choices")
            title = choices[0].get("message", {}).get("content", "").strip()
            if not title:
                raise SessionTitleGenerationError("LLM returned empty title")
            ctx.session.set_title(title)
            logger.info(f"Session title generated: {title!r}")
            return SessionTitleResult(title=title)
        except (httpx.HTTPStatusError, httpx.RequestError) as e:
            raise SessionTitleGenerationError(str(e)) from e
        except (orjson.JSONDecodeError, KeyError) as e:
            raise SessionTitleGenerationError(f"Response parse error: {e}") from e
```

**cmd_session.py — update `_generate_session_title()`**:

Before (lines 30-36):
```python
async def _generate_session_title(self, first_input: str) -> None:
    from agent.services.session_title import (
        SessionTitleService,  # noqa: PLC0415
    )
    await SessionTitleService().generate(self._ctx, first_input)
```

After:
```python
async def _generate_session_title(self, first_input: str) -> None:
    from agent.services.session_title import (
        SessionTitleService,  # noqa: PLC0415
    )
    from agent.services.exceptions import (
        SessionTitleGenerationError,  # noqa: PLC0415
    )
    try:
        await SessionTitleService().generate(self._ctx, first_input)
    except SessionTitleGenerationError as e:
        logger.warning("Session title generation failed: %s", e)
```

### Method

`Edit` tool. Update both files.

### Details

- `orjson.loads(resp.content)` replaces `resp.json()` to use orjson per coding conventions.
- `resp.content` is bytes; `orjson.loads` accepts bytes directly.
- The `set_title()` call moves inside `generate()` (successful path only), removed from
  the fallback which no longer exists.
