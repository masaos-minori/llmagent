### Details

**New `requeue_event()` function:**

```python
def requeue_event(conn: sqlite3.Connection, event_id: str) -> bool:
    """Return event from DLQ to live state. Increments dlq_requeue_count.

    Returns True if the event was found and requeued, False if not found.
    """
    cur = conn.execute(
        "UPDATE events SET dlq_requeue_count = dlq_requeue_count + 1, dlq_at = NULL"
        " WHERE event_id = ?",
        (event_id,),
    )
    conn.commit()
    return cur.rowcount > 0
```

Note: `app.py`'s `dlq_requeue` handler can be updated to call this facade instead of inline SQL, consistent with the DB facade pattern from req #15.
