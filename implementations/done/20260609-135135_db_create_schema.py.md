# Implementation: db/create_schema.py — Configurable log path + deprecation comment

## Goal

1. Replace the hardcoded `/opt/llm/logs/create_schema.log` with a value from config.
2. Add a deprecation comment to `_SAFE_MIGRATION_ERRORS`.

## Scope

- `scripts/db/create_schema.py`: update `Logger(...)` call; add comment to `_SAFE_MIGRATION_ERRORS`.

## Assumptions

1. `common.toml` has a `log_dir` key (or we fall back to `/opt/llm/logs`).
   If `build_db_config()` raises, fall back to the hardcoded path.
2. The deprecation comment signals intent without breaking existing behavior.

## Implementation

### Target file

`scripts/db/create_schema.py`

### Procedure

**Replace Logger instantiation:**
```python
# Before:
logger = Logger(__name__, "/opt/llm/logs/create_schema.log")

# After:
def _get_schema_log_path() -> str:
    try:
        from shared.config_loader import ConfigLoader
        cfg = ConfigLoader().load("common.toml")
        log_dir = cfg.get("log_dir", "/opt/llm/logs")
        return f"{log_dir}/create_schema.log"
    except Exception:
        return "/opt/llm/logs/create_schema.log"

logger = Logger(__name__, _get_schema_log_path())
```

**Add deprecation comment to _SAFE_MIGRATION_ERRORS:**
```python
# DEPRECATED: _SAFE_MIGRATION_ERRORS will be removed once versioned migration replaces
# IF NOT EXISTS + safe-skip approach. Do not add new entries here.
_SAFE_MIGRATION_ERRORS: tuple[str, ...] = (
    "duplicate column name",  # ALTER TABLE ADD COLUMN already applied
    "already exists",         # CREATE TRIGGER IF NOT EXISTS on an existing trigger
)
```
