# Implementation: installer.py — Remove thin facade, update callers

## Goal

Delete `scripts/mcp/installer.py` (thin re-export facade).
Update the two callers to import directly from the sub-modules.

## Scope

- `scripts/mcp/installer.py`: delete.
- `scripts/agent/services/mcp_install.py`: update import.
- `scripts/agent/commands/cmd_mcp.py`: update import.

## Assumptions

1. Both callers use `from mcp.installer import validate_server_name, ...`.
2. All exported names come from the four sub-modules:
   - `installer_validation.py`: `validate_server_name`, `name_to_module`, `name_to_class`
   - `installer_port.py`: `scan_used_ports`, `suggest_port`
   - `installer_templates.py`: `generate_*`, `tool_definition_snippet`
   - `installer_writer.py`: `install_mcp_server`

## Implementation

### Target files

- `scripts/mcp/installer.py` (delete via `git rm`)
- `scripts/agent/services/mcp_install.py`
- `scripts/agent/commands/cmd_mcp.py`

### Procedure

**mcp_install.py** — current import:
```python
from mcp.installer import (
    generate_agent_toml_mcp_snippet,
    install_mcp_server,
    name_to_module,
    suggest_port,
    tool_definition_snippet,
)
```
Replace with:
```python
from mcp.installer_port import suggest_port
from mcp.installer_templates import generate_agent_toml_mcp_snippet, tool_definition_snippet
from mcp.installer_validation import name_to_module
from mcp.installer_writer import install_mcp_server
```

**cmd_mcp.py** — current import:
```python
from mcp.installer import (
    validate_server_name,
)
```
Replace with:
```python
from mcp.installer_validation import validate_server_name
```

**Delete installer.py:** `git rm scripts/mcp/installer.py`
