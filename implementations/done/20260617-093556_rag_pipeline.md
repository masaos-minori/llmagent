# Implementation: Integrate plugin hooks into RagPipeline.run() (req 24 — Step 2)

## Goal

Call post-rerank plugin hooks after `RerankStage` completes in `RagPipeline.run()`, before `AugmentStage`, by delegating to the new `run_pipeline_stages()` helper.

## Scope

- `scripts/rag/pipeline.py` — modify `RagPipeline.run()` to invoke post-rerank hooks

## Assumptions

1. `run_pipeline_stages()` from `shared/plugin_registry.py` is already implemented (Step 1).
2. Post-rerank hooks run on `ctx.reranked` after `RerankStage` and before `AugmentStage`.
3. `hook_strict: bool = False` is added as an optional parameter to `run()` (backward-compatible).
4. `get_pipeline_post_stages()` is checked; hooks are called only when the list is non-empty.
5. The import of `run_pipeline_stages` and `get_pipeline_post_stages` is added from `shared.plugin_registry`.

## Implementation

### Target file

`scripts/rag/pipeline.py`

### Procedure

1. Add import: `from shared.plugin_registry import get_pipeline_post_stages, run_pipeline_stages`.
2. Add `hook_strict: bool = False` parameter to `RagPipeline.run()` signature.
3. In the `stages` list inside `run()`, separate `RerankStage` from `AugmentStage`.
4. After the loop that runs `RerankStage`, call `run_pipeline_stages()` if hooks are registered, updating `ctx.reranked`.
5. Then run `AugmentStage` separately.

### Method

Edit tool. Modify `RagPipeline.run()` method.

New `run()` signature and body:
```python
async def run(
    self,
    query: str,
    db: SQLiteHelper,
    history_context: str = "",
    hook_strict: bool = False,
) -> tuple[list[str], list[list[RawHit]], list[RagHit], list[RagHit]]:
    try:
        ctx = PipelineContext(query=query, history_context=history_context)
        self.last_timings = {}
        pre_augment_stages: list = [
            MqeStage(self._cfg.__dict__, self._llm),
            SearchStage(self._cfg.__dict__, self._http, self._embed_url),
            FusionStage(self._cfg.__dict__),
            RerankStage(self._cfg.__dict__, self._llm),
        ]
        for stage in pre_augment_stages:
            t0 = time.perf_counter()
            await stage.run(ctx, db=db)
            self.last_timings[stage.__class__.__name__] = time.perf_counter() - t0

        # Post-rerank plugin hooks (before AugmentStage)
        if get_pipeline_post_stages():
            t0 = time.perf_counter()
            ctx.reranked = await run_pipeline_stages(
                ctx.reranked, query, strict=hook_strict
            )
            self.last_timings["PluginHooks"] = time.perf_counter() - t0

        augment_stage = AugmentStage()
        t0 = time.perf_counter()
        await augment_stage.run(ctx, db=db)
        self.last_timings[augment_stage.__class__.__name__] = time.perf_counter() - t0

        self.last_fetch_result = TwoStageFetchResult(
            hits=ctx.reranked,
            min_score_applied=self._cfg.rag_min_score,
            max_chunks_per_doc=self._cfg.max_chunks_per_doc,
        )
        return ctx.queries, ctx.search_results, ctx.merged, ctx.reranked
    finally:
        self._on_clear()
```
