# Implementation: Add logger.info for refiner fallback in pipeline.py augment()

## Goal
Emit `logger.info` in `RagPipeline.augment()` when the refiner falls back to raw chunks,
making the event visible at INFO log level without requiring `--debug`.

## Scope
- File: `scripts/rag/pipeline.py`
- Method: `augment()` — insert after the `StageResult` append for "Refiner"
- One `if refined.text is None: logger.info(...)` block

## Assumptions
- `StageResult` append is at lines 486-493; `if refined.text is not None: return` is at 494-495
- The `logger.warning` in `refine_context()` fires at the failure site (different scope);
  the new `logger.info` in `augment()` fires at the pipeline decision point — complementary, not duplicate
- INFO level (not WARNING) is appropriate: fallback is a designed graceful degradation
- The log message pattern follows existing pipeline log style:
  `"augment: refiner fallback (reason=%s); using raw chunks", refined.reason`

## Implementation

### Target file
`scripts/rag/pipeline.py`

### Procedure
1. Locate the Refiner block in `augment()` (lines ~481-495)
2. Insert the `logger.info` block after `self.last_stage_results.append(...)` and before
   `if refined.text is not None:`

### Method
Single edit — insert 3 lines.

### Details

**Current code (lines 486-495):**
```python
            self.last_stage_results.append(
                StageResult(
                    stage_name="Refiner",
                    status=refiner_status,
                    elapsed_seconds=elapsed,
                    fallback_reason=refined.reason,
                )
            )
            if refined.text is not None:
                return refined.text
```

**Updated code:**
```python
            self.last_stage_results.append(
                StageResult(
                    stage_name="Refiner",
                    status=refiner_status,
                    elapsed_seconds=elapsed,
                    fallback_reason=refined.reason,
                )
            )
            if refined.text is None:
                logger.info(
                    "augment: refiner fallback (reason=%s); using raw chunks",
                    refined.reason,
                )
            if refined.text is not None:
                return refined.text
```
