# Implementation: config_reload.py — add source_files to ConfigReloadOutcome

## Goal

Add `source_files: list[str]` field to `ConfigReloadOutcome` so callers can record
and display which config files were loaded during a `/reload`.

## Scope

`scripts/agent/services/config_reload.py` — `ConfigReloadOutcome` dataclass only.

## Assumptions

1. `source_files` is an additive optional field with `default_factory=list`.
2. No existing callers need to change — they can set `result.source_files` after receiving the object.

## Implementation

### Target file

`scripts/agent/services/config_reload.py`

### Procedure

Add `source_files` field to the `ConfigReloadOutcome` dataclass at line ~136.

### Method

Additive, one-line change. No behavior change.

### Details

```python
@dataclass
class ConfigReloadOutcome:
    """Structured report of what changed after a /reload."""

    applied: list[str] = field(default_factory=list)
    needs_restart: list[str] = field(default_factory=list)
    skipped: list[str] = field(default_factory=list)
    source_files: list[str] = field(default_factory=list)
```
