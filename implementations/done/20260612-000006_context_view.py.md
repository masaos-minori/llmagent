# Goal

Replace silent `or ""` None-coalesce and `str(content)` unconditional cast in
`context_view.py` with explicit type-safe access, and change `budget_breakdown()`
to return `ContextBudget` DTO instead of `dict[str, int]`.

# Scope

- `scripts/agent/services/context_view.py`

# Assumptions

1. `ContextBudget` is defined in `agent/services/models.py` (Step 1 prerequisite).
2. `LLMMessage` is `TypedDict` with `content: str | list[...] | None` — the `or ""`
   pattern treats `""` and `None` as equivalent (losing information). Replace with
   explicit `isinstance(content_raw, str)` check.
3. `m.get("role", "")` → `m.get("role")` with `role = m.get("role") or ""` is acceptable;
   `role` is always a non-empty string in valid history.
4. `system_msgs[0].get("content", "")` → use `content_raw = system_msgs[0].get("content");
   sys_preview = content_raw[:80] if isinstance(content_raw, str) else ""`
5. `budget_breakdown()` return type changes from `dict[str, int]` to `ContextBudget`.
   Callers: `cmd_context._cmd_context()` iterates over `breakdown.items()` — must update
   to use `ContextBudget` fields directly or via `dataclasses.asdict()`.
   `collect_context_state()` passes `budget_breakdown()` result to `ContextStateView.breakdown`
   which is typed `dict[str, int]` — must change `ContextStateView.breakdown` type to
   `ContextBudget` as well.
6. `ContextStateView.breakdown: dict[str, int]` → `ContextBudget` (models.py change needed
   in the same step).

# Implementation

## Target file

`scripts/agent/services/context_view.py`, `scripts/agent/services/models.py` (ContextStateView.breakdown field)

## Procedure

1. Import `ContextBudget` from `agent.services.models`.
2. In `budget_breakdown()`:
   - Change return type annotation to `ContextBudget`.
   - Replace the `counts` dict with a direct `ContextBudget` construction. Accumulate into
     local `int` variables `system`, `history`, `tool_results`, then return
     `ContextBudget(system=system, history=history, tool_results=tool_results)`.
   - Fix line 28-29:
     ```python
     content_raw = m.get("content")
     text = content_raw if isinstance(content_raw, str) else ""
     ```
     Remove the now-redundant `isinstance(content, str) else str(content)` check.
3. In `collect_context_state()`:
   - Fix line 67:
     ```python
     content_raw = system_msgs[0].get("content") if system_msgs else None
     sys_preview = content_raw[:80] if isinstance(content_raw, str) else ""
     ```
4. In `models.py`, change `ContextStateView.breakdown: dict[str, int]` →
   `breakdown: ContextBudget`.
5. In `cmd_context._cmd_context()`, update the iteration over `breakdown`:
   - Old: `for cat, n in breakdown.items()`
   - New: `for cat, n in dataclasses.asdict(state.breakdown).items()` OR explicitly:
     ```python
     breakdown = state.breakdown
     for cat, n in [
         ("system", breakdown.system),
         ("history", breakdown.history),
         ("tool_results", breakdown.tool_results),
     ]:
     ```
6. Run ruff + mypy.

## Method

The `ContextBudget` DTO makes the categories explicit and removes the stringly-typed
`dict[str, int]`. The `cmd_context` iteration is updated to use explicit field access.
