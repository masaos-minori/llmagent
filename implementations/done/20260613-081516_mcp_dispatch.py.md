## Implementation

### Target file

`scripts/mcp/dispatch.py` (primary), 12 additional server files

### Procedure

1. Add `DispatchResult` frozen dataclass to `mcp/dispatch.py`.
2. Change `dispatch_tool()` return type and all `return ..., True/False` to `DispatchResult(...)`.
3. Update `mcp/server.py`: base `dispatch()` signature + `run_stdio()` unpack.
4. For each of 11 servers: change `_dispatch_xxx_tool()` signature, update `call_tool()`
   endpoint, update `MCPServer.dispatch()` subclass.
5. Update `installer_templates.py` template string.
6. Add `from mcp.dispatch import DispatchResult` imports where needed.

### Method

Mechanical substitution: `tuple[str, bool]` → `DispatchResult`; `result, is_error = await ...`
→ `r = await ...; return CallToolResponse(result=r.output, is_error=r.is_error)`.

### Details

**`mcp/dispatch.py` additions:**
```python
from dataclasses import dataclass

@dataclass(frozen=True)
class DispatchResult:
    """Typed result from MCPServer.dispatch() and dispatch_tool()."""
    output: str
    is_error: bool
```

**`dispatch_tool()` return changes:**
```python
# Before                                         # After
return "Tool name must be ...", True     →  return DispatchResult(output="Tool name must be ...", is_error=True)
return f"Unknown tool: {name}", True     →  return DispatchResult(output=f"Unknown tool: {name}", is_error=True)
return result, False                     →  return DispatchResult(output=result, is_error=False)
return f"Validation error: {e}", True   →  return DispatchResult(output=f"Validation error: {e}", is_error=True)
```

**`mcp/server.py:run_stdio()` (line 174):**
```python
# Before
raw_result, is_error = await self.dispatch(name, dict(req.get("args", {})))
tr = _truncate_with_meta(raw_result)
# After
dispatch_result = await self.dispatch(name, dict(req.get("args", {})))
tr = _truncate_with_meta(dispatch_result.output)
is_error = dispatch_result.is_error
```

**Per-server pattern (example: cicd/server.py):**
```python
# _dispatch_cicd_tool
async def _dispatch_cicd_tool(name: str, args: ToolArgs) -> DispatchResult:
    return await dispatch_tool(_service.get_dispatch_table(), name, args)

# call_tool endpoint
r = await _dispatch_cicd_tool(req.name, req.args)
return CallToolResponse(result=r.output, is_error=r.is_error)

# CiCdMCPServer.dispatch
async def dispatch(self, name: str, args: dict[str, Any]) -> DispatchResult:
    return await _dispatch_cicd_tool(name, args)
```

**Import change per server:**
```python
# Before
from mcp.dispatch import dispatch_tool
# After
from mcp.dispatch import DispatchResult, dispatch_tool
```
