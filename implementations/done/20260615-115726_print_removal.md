# Implementation: Phase A1 — print() OutputPort/CLIView 移行

## Goal

`scripts/` 内の library/service モジュールに残る `print()` 呼び出し 15 件を
OutputPort または CLIView 経由に移行し、UI 依存を presentation 層に集約する。

## Scope

**In**:
- `agent/tool_output.py` (8 件) — emit_* 関数群
- `agent/repl.py` (5 件) — shutdown/unknown_command/warn 等
- `agent/services/export_formatter.py` (2 件) — export 結果表示

**Out**: `shared/plugin_registry.py` の print("pong") は docstring 内コード例のため除外

## Assumptions

1. `agent/tool_output.py` の emit_* 関数は `tool_approval.py` と `tool_runner.py` の
   2 箇所のみから import されている（確認済み）。
2. `agent/repl.py` は既に `self._view = CLIView(...)` を使っている。print() は
   `self._view.write_warning()` / `self._view.write()` に置き換え可能。
3. `export_formatter.py` の print() は呼び出し元 `cmd_ingest.py` の `self._out`
   (OutputPort) に委譲するか、`WriterPort` 引数を追加する。
4. 変更後に `ruff check` と `mypy` でエラーなしを確認する。

## Implementation

### Target files and procedures

**1. `agent/tool_output.py`**

`emit_*` 関数は現在 `print()` を直接呼ぶ。`output: OutputPort` 引数を追加し、
`output.write(...)` に変換する。

```python
# 変更前
def emit_tool_call(name: str, args_json: str) -> None:
    print(f"  [tool] {name}({args_json})")

# 変更後
def emit_tool_call(name: str, args_json: str, output: OutputPort) -> None:
    output.write(f"  [tool] {name}({args_json})")
```

全 6 関数 (emit_tool_call, emit_tool_result, emit_approval_prompt, emit_denied,
emit_plan_blocked, emit_skipped) に `output: OutputPort` 引数を追加する。

**2. `agent/tool_approval.py` と `agent/tool_runner.py` (呼び出し側2箇所)**

`emit_*` 呼び出しに `output` 引数を追加する。既存の `ctx` や `_out` から
`OutputPort` を渡す。呼び出し側がどの `OutputPort` を持っているかを確認してから変更。

**3. `agent/repl.py`**

`self._view` (CLIView) を使って print() を置き換える:
- `print()` (空行) → `self._view.write("")` または `self._view.write_blank_line()`
- `print("\nShutdown requested, exiting...")` → `self._view.write(...)`
- `print(f"Unknown command: {line}  (type /help for commands)")` → `self._view.write(...)`
- 複数行 print → `self._view.write_warning(...)` 等

**4. `agent/services/export_formatter.py`**

```python
# 変更前
print(content)
print(f"Exported {n_messages} messages to {path} ({len(content)} chars)")

# 変更後 (OutputPort を引数で受け取る)
def write_export(content: str, path: str | None, n_messages: int, output: OutputPort) -> None:
    if path is None:
        output.write(content)
    else:
        Path(path).write_text(content, encoding="utf-8")
        output.write_success(f"Exported {n_messages} messages to {path} ({len(content)} chars)")
```

## Validation plan

1. `grep -rn "^\s*print(" scripts/ | grep -v "__pycache__\|cli_view\|output_port"` → 0 件
2. `ruff check scripts/` → 0 errors
3. `uv run mypy scripts/` → no new errors
4. `uv run pytest -q` → no regressions
