# Implementation: services.py — MemoryServices facade methods

## Goal

Add facade methods (`on_session_start`, `on_session_stop`, `on_user_prompt`) to `MemoryServices`
so that `repl.py` and `orchestrator.py` can call them without AttributeError.
Convert `MemoryServices` from a plain dataclass to a class with `__init__` and delegate methods.

## Scope

- **Target file**: `scripts/agent/memory/services.py`
- **Secondary**: `scripts/agent/factory.py` — verify construction code still works after dataclass removal
- **Tests**: `tests/test_memory_layer.py`, `tests/test_agent_factory.py`
- **Not in scope**: `repl.py`, `orchestrator.py` — they call these methods directly; no changes needed

## Assumptions

1. `factory.py` constructs `MemoryServices` with keyword args. Switching from `@dataclass` to a
   regular `__init__` that accepts the same four kwargs is backward-compatible.
2. `injection.MemoryInjectionService.on_session_start` currently takes `session_id` as a positional arg.
   The facade must accept `session_id` and forward it (Step 2 later removes it from `injection.py`).
3. `ingestion.MemoryIngestionService.on_session_stop` is async. The facade `on_session_stop` must also be async.
4. `injection.MemoryInjectionService.on_user_prompt` is async. The facade `on_user_prompt` must also be async.
5. `shared.types.LLMMessage` is the type for history entries.

## Implementation

### Target file

`scripts/agent/memory/services.py`

### Procedure

1. Remove `@dataclass` decorator.
2. Add `__init__(self, injection, ingestion, store, retriever)` with type annotations.
3. Assign each arg to `self.<name>`.
4. Add three facade methods:
   - `on_session_start(self, session_id: int | None) -> list[str]`
   - `async on_session_stop(self, session_id, history, turn_id) -> None`
   - `async on_user_prompt(self, query, session_id) -> list[str]`
5. Add necessary imports: `LLMMessage` from `shared.types`.

### Method

Direct textual edit (no LibCST needed — file is small, no comments/docstrings to preserve).

### Details

```python
"""agent/memory/services.py
MemoryServices — facade over the three memory service instances.

Replaces MemoryLayer as the AppServices.memory type.
"""
from __future__ import annotations

from agent.memory.ingestion import MemoryIngestionService
from agent.memory.injection import MemoryInjectionService
from agent.memory.retriever import HybridRetriever
from agent.memory.store import MemoryStore
from shared.types import LLMMessage


class MemoryServices:
    """Facade over memory sub-services; injected into AppServices.memory."""

    def __init__(
        self,
        injection: MemoryInjectionService,
        ingestion: MemoryIngestionService,
        store: MemoryStore,
        retriever: HybridRetriever,
    ) -> None:
        self.injection = injection
        self.ingestion = ingestion
        self.store = store
        self.retriever = retriever

    def on_session_start(self, session_id: int | None) -> list[str]:
        return self.injection.on_session_start(session_id)

    async def on_session_stop(
        self,
        session_id: int | None,
        history: list[LLMMessage],
        turn_id: str | None = None,
    ) -> None:
        await self.ingestion.on_session_stop(session_id, history, turn_id)

    async def on_user_prompt(
        self,
        query: str,
        session_id: int | None,
    ) -> list[str]:
        return await self.injection.on_user_prompt(query, session_id)
```
