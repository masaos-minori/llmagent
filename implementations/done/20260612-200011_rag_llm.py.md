# Goal

Replace `cfg: dict[str, Any]` parameters in `rag/llm.py` with typed config DTOs,
add schema validation to `_extract_chat_content()`, and replace
`c.get("title") or c.get("url", "")` with typed `RankedHit` attribute access.

# Scope

- `scripts/rag/llm.py`

# Assumptions

1. `MqeConfig`, `RerankConfig` from `rag.models` (Step 2-3 prerequisite).
2. `RankedHit` dataclass from Step 3-1 prerequisite.
3. `EmbeddingSchemaError` from `rag.exceptions` (Step 2-1 prerequisite).
4. `_extract_chat_content(data: dict[str, Any])` currently does
   `choices[0].get("message", {}).get("content")` — add isinstance checks at
   each level and raise `EmbeddingSchemaError` on schema mismatch.
5. `_mqe_prompt(query, context, cfg: dict)` → `(cfg: MqeConfig)`.
6. `_rerank_prompt(query, candidates: list[RagHit], cfg: dict)` → `(candidates: list[RankedHit], cfg: RerankConfig)`.
7. `c.get("title") or c.get("url", "")` → `c.title if c.title else c.url`.
8. `self._cfg: dict[str, Any]` constructor → `PipelineConfig` (or `RagLLMConfig`
   subset DTO).
9. `score_map: dict[str, Any] = orjson.loads(...)` → add schema validation
   (must be dict[str, float]).

# Implementation

## Target file

`scripts/rag/llm.py`

## Procedure

1. Change `_mqe_prompt(cfg: dict[str, Any])` → `(cfg: MqeConfig)`.
2. Change `_rerank_prompt(candidates: list[RagHit], cfg: dict[str, Any])` →
   `(candidates: list[RankedHit], cfg: RerankConfig)`.
3. In `_extract_chat_content(data: dict[str, Any])`:
   - Add: `choices = data.get("choices"); if not isinstance(choices, list) or not choices: raise EmbeddingSchemaError(...)`
   - Add: `first = choices[0]; if not isinstance(first, dict): raise EmbeddingSchemaError(...)`
   - Add: `message = first.get("message"); if not isinstance(message, dict): raise EmbeddingSchemaError(...)`
   - Add: `content = message.get("content"); if not isinstance(content, str): raise EmbeddingSchemaError(...)`
4. Replace `c.get("title") or c.get("url", "")` → `c.title if c.title else c.url`.
5. Replace `score_map: dict[str, Any]` with explicit validation that parsed value
   is `dict[str, float]`.
6. Change `self._cfg: dict[str, Any]` to appropriate typed config.
7. Run ruff + mypy.

## Method

Parameter type changes + schema validation additions.
