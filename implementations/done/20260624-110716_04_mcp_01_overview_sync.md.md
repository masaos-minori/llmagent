# Implementation: Sync Overview/Catalog Cross-References

## Goal

Ensure overview doc (`04_mcp_01`) server paragraphs match the catalog (`04_mcp_04`) content, and that `04_mcp_04` notes its authoritative role.

## Scope

**In:**
- `docs/04_mcp_01_mcp_overview.md` — update server overview paragraphs
- `docs/04_mcp_04_server_catalog.md` — add authoritative note + cross-reference to `04_mcp_01`

**Out:** No new servers added, no infrastructure changes.

## Assumptions

1. `04_mcp_04` is the authoritative server list.
2. `04_mcp_01` provides overview descriptions that may have drifted from catalog.

## Implementation

### Target file

`docs/04_mcp_01_mcp_overview.md`, `docs/04_mcp_04_server_catalog.md`

### Procedure

1. Read the server list in `docs/04_mcp_04_server_catalog.md` (headings: grep for `##` under "Servers").
2. Read the corresponding server descriptions in `docs/04_mcp_01_mcp_overview.md`.
3. For each server in `04_mcp_04`, check if `04_mcp_01` has a corresponding entry.
4. Update stale or missing entries in `04_mcp_01`.
5. Add note at top of `04_mcp_04`: "This is the authoritative server catalog. For overview descriptions, see `04_mcp_01`."

### Method

`grep -n "^##" docs/04_mcp_04_server_catalog.md` to get server list → Read both docs → Edit patches.

### Details

**Authoritative note for `04_mcp_04` (after frontmatter/intro):**
```markdown
> **Note:** This document is the authoritative server catalog. For system-level overview descriptions of each server, see `04_mcp_01_mcp_overview.md`.
```

**Sync approach for `04_mcp_01`:**
- For each server section heading in `04_mcp_04`, verify a paragraph exists in `04_mcp_01`.
- If paragraph is absent: add a one-paragraph summary based on `04_mcp_04` description.
- If paragraph is stale (mentions "stub", old feature set): update to match `04_mcp_04`.

## Validation plan

| Check | Command | Expected |
|---|---|---|
| Authoritative note in 04_mcp_04 | `grep -n "authoritative.*catalog\|canonical.*server" docs/04_mcp_04_server_catalog.md` | found |
| All servers covered in 04_mcp_01 | `diff <(grep "^##" docs/04_mcp_04_server_catalog.md) <(grep "^##" docs/04_mcp_01_mcp_overview.md)` | minimal diff |
| No code changes | `git diff scripts/` | empty |
