# Implementation: repl.py — strip startup methods; delegate to StartupOrchestrator

## Goal

Remove all startup orchestration methods from `AgentREPL`; replace with calls to
`StartupOrchestrator`. REPL keeps only UI loop, command dispatch, and output display.

## Scope

## Updated run() method

```python
async def run(self) -> None:
    """Start the interactive REPL.

    Delegates startup orchestration to StartupOrchestrator:
    component init, MCP server spawning, service health checks, and prompt setup.
    """
    from agent.startup import StartupOrchestrator
    startup = StartupOrchestrator(self._ctx, self._view)
    self._cmds, self._orchestrator = await startup.run()
    await self._run_repl_loop()
```
