# Implementation: Expose Memory Embedding and Circuit-Breaker State in Runtime Views

## Goal

Make memory-layer embedding enabled/disabled, circuit-breaker state, and retrieval mode (hybrid vs FTS-only) visible in a normal operator-facing command without requiring log analysis.

## Scope

**In-Scope:**
- Add `get_status()` to `EmbeddingClient` returning an `EmbeddingClientStatus` dataclass: `(enabled: bool, circuit_open: bool, fail_count: int, circuit_reset_in_sec: float | None)`
- Add `last_retrieval_mode: str` tracking to `HybridRetriever` ("hybrid" when KNN was used, "fts_only" when embedding was skipped or failed)
- Add `/memory status` subcommand displaying: embedding enabled, circuit state, retrieval mode
- Emit a `logger.warning` when retrieval degrades to FTS-only due to circuit open or disabled embed (already emitted by `EmbeddingClient` on circuit open; add a warning in `HybridRetriever`)

**Out-of-Scope:**
- Changing embedding API behavior or fallback semantics
- Redesigning retrieval scoring or ranking
- Adding circuit reset via CLI (operator can restart to reset)

## Assumptions

1. `EmbeddingClient` is accessible from `MemoryServices.retriever.embed_client` or similar path.
2. `HybridRetriever` holds a reference to `EmbeddingClient` (checked by reading retriever.py).
3. "circuit_open" vs "circuit_closed" is sufficient; operators don't need exact failure timestamp.
4. `last_retrieval_mode` is reset per search call — shows last-used mode, not aggregate.

## Implementation

### Target files

- `scripts/agent/memory/embedding_client.py` — add `get_status()` + `EmbeddingClientStatus` dataclass
- `scripts/agent/memory/retriever.py` — add `last_retrieval_mode: str = "unknown"` + update in `search()`
- `scripts/agent/commands/cmd_memory.py` — add `/memory status` subcommand

### Procedure

#### Step 1: Add `EmbeddingClientStatus` + `get_status()` to `embedding_client.py`

```python
@dataclass(frozen=True)
class EmbeddingClientStatus:
    enabled: bool
    circuit_open: bool
    fail_count: int
    resets_in_sec: float | None  # None if circuit closed

def get_status(self) -> EmbeddingClientStatus:
    circuit_open = self._is_circuit_open()
    resets_in = None
    if circuit_open and self._circuit_opened_at is not None:
        elapsed = time.monotonic() - self._circuit_opened_at
        resets_in = max(0.0, self._config.circuit_reset_sec - elapsed)
    return EmbeddingClientStatus(
        enabled=self._enabled,
        circuit_open=circuit_open,
        fail_count=self._fail_count,
        resets_in_sec=resets_in,
    )
```

#### Step 2: Add `last_retrieval_mode` to `HybridRetriever`

- Init: `self.last_retrieval_mode: str = "unknown"`
- In `search()`: set to `"hybrid"` when KNN succeeds, `"fts_only"` when fallback is used
- When setting `"fts_only"`: also emit `logger.info("[memory] retrieval degraded to FTS-only")`

#### Step 3: Add `/memory status` to `_MemoryMixin`

```python
def _memory_status(self, mem: MemoryServices | None) -> None:
    if mem is None:
        self._out.write("  [memory] Memory layer: disabled (use_memory_layer=false)")
        return
    embed_status = mem.retriever.embed_client.get_status()
    circuit_detail = ""
    if embed_status.circuit_open and embed_status.resets_in_sec is not None:
        circuit_detail = f" (resets in {embed_status.resets_in_sec:.0f}s)"
    rows = [
        ["Memory layer", "enabled"],
        ["Embedding enabled", "Yes" if embed_status.enabled else "No"],
        ["Circuit", ("OPEN" + circuit_detail) if embed_status.circuit_open else "closed"],
        ["Consecutive failures", str(embed_status.fail_count)],
        ["Last retrieval mode", mem.retriever.last_retrieval_mode],
    ]
    self._out.write_table(["Field", "Value"], rows)
```

- Register in `_cmd_memory` dispatch: `"status"` → `lambda: _memory_status(mem if sub_set else None)`
- Update help text

#### Step 4: Handle `/memory status` when memory disabled

Unlike other subcommands, `status` should be accessible even when `mem is None`. Adjust `_cmd_memory` dispatch order: check `status` before the `if mem is None` guard.

#### Step 5: Tests

- Test: `get_status()` with circuit closed → `circuit_open=False`
- Test: `get_status()` with circuit open → `circuit_open=True, resets_in_sec>0`
- Test: `/memory status` with memory disabled → shows "disabled" message
- Test: `/memory status` with memory enabled, circuit open → shows "OPEN"

- Additive dataclass + method on `EmbeddingClient`
