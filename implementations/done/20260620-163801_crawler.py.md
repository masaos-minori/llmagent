# Implementation: Update module docstrings in crawler.py, chunk_splitter.py, ingester.py

## Goal
Explicitly state that `.txt` output files contain JSON payloads (not plain text)
in the module docstrings of all three ingestion pipeline scripts.

## Scope
- Files: `scripts/rag/ingestion/crawler.py`, `chunk_splitter.py`, `ingester.py`
- Docstring-only changes — one clarifying line added to each
- No behavior change

## Assumptions
- `crawler.py` docstring (lines 1-7): says "saves extracted text and code blocks to rag-src/"
  — needs explicit `.txt = JSON` note
- `chunk_splitter.py` docstring (lines 1-9): already says "(JSON files)" but not prominently
  — add explicit NOTE line
- `ingester.py` docstring (lines 1-7): already says "(JSON files)" but not prominently
  — add explicit NOTE line

## Implementation

### Target files
1. `scripts/rag/ingestion/crawler.py`
2. `scripts/rag/ingestion/chunk_splitter.py`
3. `scripts/rag/ingestion/ingester.py`

### Procedure
One edit per file — add explicit format note to module docstring.

### Method
Three docstring edits.

### Details

**crawler.py — current docstring:**
```python
"""Crawler.py
BFS web crawler that saves extracted text and code blocks to rag-src/.
Entry point: python Crawler.py [--url URL ...] [--lang {en,ja}]

Pipeline position: Crawler.py → ChunkSplitter.py → RagIngester.py

```

**crawler.py — updated docstring:**
```python
"""Crawler.py
BFS web crawler that saves extracted text and code blocks to rag-src/.
Entry point: python Crawler.py [--url URL ...] [--lang {en,ja}]

Output: rag-src/{timestamp}-{slug}.txt — JSON payload (not plain text).
Fields: url, title, lang, fetched_at, content, code_blocks, etag, last_modified.

Pipeline position: Crawler.py → ChunkSplitter.py → RagIngester.py
"""
```

---

**chunk_splitter.py — current docstring:**
```python
"""ChunkSplitter.py
Reads rag-src/*.txt (JSON files generated by Crawler.py), splits text and code
into chunks, and saves them to rag-src/chunk/{stem}-{idx:04d}.txt.

Output file format: JSON
  {url, title, lang, source_file, chunk_index, chunk_type, content}
  chunk_type: "text" or "code"

Pipeline position: Crawler.py → ChunkSplitter.py → RagIngester.py
```

**chunk_splitter.py — updated docstring:**
```python
"""ChunkSplitter.py
Reads rag-src/*.txt (JSON files generated by Crawler.py), splits text and code
into chunks, and saves them to rag-src/chunk/{stem}-{idx:04d}.txt.

NOTE: All .txt files (input and output) contain JSON payloads, not plain text.
Output file format: JSON
  {url, title, lang, source_file, chunk_index, chunk_type, content}
  chunk_type: "text" or "code"

Pipeline position: Crawler.py → ChunkSplitter.py → RagIngester.py
```

---

**ingester.py — current docstring:**
```python
"""RagIngester.py
Reads rag-src/chunk/*.txt (JSON files generated by ChunkSplitter.py),
generates embeddings, and ingests them into SQLite (documents / chunks / chunks_vec).
Processed files are moved to rag-src/registered/.

Pipeline position: Crawler.py → ChunkSplitter.py → RagIngester.py
"""
```

**ingester.py — updated docstring:**
```python
"""RagIngester.py
Reads rag-src/chunk/*.txt (JSON files generated by ChunkSplitter.py),
generates embeddings, and ingests them into SQLite (documents / chunks / chunks_vec).
Processed files are moved to rag-src/registered/.

NOTE: All .txt files consumed here contain JSON payloads, not plain text.
Parse with orjson.loads() or json.loads(); do not treat as UTF-8 text.

Pipeline position: Crawler.py → ChunkSplitter.py → RagIngester.py
"""
```
