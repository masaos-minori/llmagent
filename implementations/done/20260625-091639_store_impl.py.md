# Implementation: scripts/db/store_impl.py (tool_call_id in message_save/message_list)

## Goal

Update `SQLiteSessionStore.message_save()` to persist `tool_call_id` in the INSERT, and `message_list()` to SELECT and return `tool_call_id` in `MessageRow`. This aligns the store implementation with the updated `MessageRow` DTO and Protocol.

## Scope

- Target: `scripts/db/store_impl.py`
- `message_save()`: add `tool_call_id` parameter + include in SQL INSERT
- `message_list()`: SELECT `tool_call_id` + populate `MessageRow.tool_call_id`

## Assumptions

1. `messages` table already has `tool_call_id TEXT` column (in `schema_sql.py`).
2. `message_save()` is called by `session_message_repo.py` which already handles `tool_call_id` separately — this change ensures the store-level API is consistent.
3. `SQLiteHelper.execute()` returns `sqlite3.Cursor`.
4. `message_list()` currently SELECTs `role, content, tool_calls` — adding `tool_call_id` as 4th column.

## Implementation

### Target file
`scripts/db/store_impl.py`

### Procedure
1. In `message_save()`: add `tool_call_id: str | None = None` parameter; add `tool_call_id` to SQL INSERT column list and values tuple.
2. In `message_list()`: add `tool_call_id` to SELECT columns; populate `MessageRow(tool_call_id=r[3])`.

### Method

`message_save()`:
```python
def message_save(
    self,
    session_id: int,
    role: str,
    content: str,
    tool_calls: str | None,
    tool_call_id: str | None = None,  # add
) -> None:
    self._db.execute(
        "INSERT INTO messages (session_id, role, content, tool_calls, tool_call_id)"
        " VALUES (?, ?, ?, ?, ?)",
        (session_id, role, content, tool_calls, tool_call_id),  # add tool_call_id
    )
```

`message_list()`:
```python
def message_list(self, session_id: int) -> list[MessageRow]:
    rows = self._db.fetchall(
        "SELECT role, content, tool_calls, tool_call_id FROM messages"  # add tool_call_id
        " WHERE session_id = ? ORDER BY message_id",
        (session_id,),
    )
    return [
        MessageRow(
            role=str(r[0]),
            content=str(r[1]) if r[1] is not None else "",
            tool_calls=r[2],
            tool_call_id=r[3] if r[3] is not None else None,  # add
        )
        for r in rows
    ]
```

### Details
- The SQL INSERT column order must match the VALUES tuple order exactly.
- `r[3]` is the 4th column in the SELECT result (0-indexed).
