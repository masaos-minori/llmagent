## Goal

**Out**:

## Assumptions

1. `orchestrator.py` is only called after `factory.build_agent_context()` completes, so `ctx.services` is non-None at every call site. Using `services_required` (which raises `RuntimeError` on None) is safe.
2. `AppServices.hist_mgr`, `AppServices.audit_logger`, `AppServices.llm`, `AppServices.memory` are all non-optional fields — the `is None` guards at lines 358 and 396 are dead code and can be removed when changing to `services_required`.
3. Line 615–616: `ctx.services.llm is not None` guard before `stat_partial_completions += 1` is dead code (LLMClient is non-optional in AppServices). Remove the guard and access unconditionally.
