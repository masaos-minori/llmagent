# Implementation: Git/GitHub Dangerous Operation Guards

## Goal

Add a `GitOpsPolicy` configuration and enforcement mechanism that:
1. Blocks push/PR/merge operations unless explicitly permitted
2. Enforces `allowed_repositories` at the approval layer
3. Defines an `ArtifactEvent` type for future artifact flow notifications

## Scope

**In:**
- Add `GitOpsPolicy` dataclass to `scripts/agent/config.py` (`ApprovalConfig` or new `GitOpsConfig`)
- Enforce `allowed_repositories` in `scripts/agent/tool_approval.py`
- Define `ArtifactEvent` TypedDict in `scripts/shared/events.py`
- Add `force_push_blocked: bool` and `push_requires_allowed_repo: bool` flags

**Out:**
- Separate Worker classes (not in current codebase architecture)
- Actual event bus / pub-sub infrastructure (only type definition in scope)
- Changing GitHub MCP server (enforcement stays agent-side in approval flow)

## Assumptions

- `scripts/agent/config.py`: `ApprovalConfig` already has `approval_github_allowed_repos: list[str]`; this is currently not enforced by the GitHub MCP server itself
- `scripts/agent/tool_approval.py`: determines tool approval; can add `_check_github_repo()` logic
- GitHub write tools: `github_push_files`, `github_create_or_update_file`, `github_delete_file`, `github_merge_pull_request`, `github_create_pull_request`, `github_create_branch`

## Implementation

### 1. `scripts/agent/config.py` — add gitops config fields to `ApprovalConfig`

```python
@dataclass
class ApprovalConfig:
    # ... existing fields ...

    # Git/GitHub operation policy
    gitops_push_blocked: bool = False       # block all push operations globally
    gitops_force_push_blocked: bool = True  # block github_push_files with force=True
    gitops_protected_branches: list[str] = field(
        default_factory=lambda: ["main", "master"]
    )  # already in approval_high_risk_branches; may merge or keep separate
```

### 2. `scripts/agent/tool_approval.py` — enforce `allowed_repositories`

Add a helper that checks `approval_github_allowed_repos`:

```python
def _check_github_repo_allowed(
    tool_name: str,
    args: dict[str, Any],
    cfg: ApprovalConfig,
) -> bool:
    """Return True if the GitHub repo in args is in the allowed list.

    Returns True when allowed_repos is empty (no restriction) or when
    the repo is explicitly listed.
    """
    if not cfg.approval_github_allowed_repos:
        return True  # empty = unrestricted
    if not tool_name.startswith("github_"):
        return True  # non-GitHub tools are not restricted here
    repo = args.get("repo") or args.get("owner_and_repo") or args.get("repository", "")
    if not repo:
        return True  # no repo arg; let other checks decide
    return any(
        repo == allowed or repo.endswith(f"/{allowed.split('/')[-1]}")
        for allowed in cfg.approval_github_allowed_repos
    )
```

Call this check in the main approval evaluation. If the repo is not allowed, return a hard-deny (no interactive prompt; reject immediately with a clear error message).

Also add a check for `gitops_push_blocked`:

```python
_GITHUB_WRITE_TOOLS = frozenset({
    "github_push_files",
    "github_create_or_update_file",
    "github_delete_file",
    "github_merge_pull_request",
    "github_create_pull_request",
    "github_update_pull_request",
    "github_create_branch",
})

if cfg.gitops_push_blocked and tool_name in _GITHUB_WRITE_TOOLS:
    return ApprovalResult.DENIED, "gitops_push_blocked is set; write operations are disabled"
```

### 3. `scripts/shared/events.py` — define ArtifactEvent

```python
"""shared/events.py
Typed event definitions for agent lifecycle and artifact notifications.
"""
from typing import TypedDict


class ArtifactEvent(TypedDict, total=False):
    """Emitted when a repo artifact is created or updated."""
    event_type: str       # "artifact.updated" | "artifact.created" | "artifact.deleted"
    repo: str             # "owner/repo"
    branch: str           # branch name
    commit: str           # commit SHA or empty
    path: str             # file path or empty (for whole-branch events)
    pr_number: int        # PR number or 0
    session_id: int       # agent session that triggered the event
    timestamp: str        # ISO-8601 UTC
```

This type is a pure data definition — no event bus yet. Future work will emit and consume `ArtifactEvent` values.

def test_github_push_blocked_when_gitops_flag_set():
    cfg = make_approval_cfg(gitops_push_blocked=True)
    result, reason = evaluate_approval("github_push_files", {}, cfg)
    assert result == ApprovalResult.DENIED
    assert "gitops_push_blocked" in reason

def test_github_allowed_repos_rejects_unlisted_repo():
    cfg = make_approval_cfg(approval_github_allowed_repos=["myorg/allowed-repo"])
    result, _ = evaluate_approval("github_push_files", {"repo": "myorg/other-repo"}, cfg)
    assert result == ApprovalResult.DENIED

def test_github_allowed_repos_empty_permits_all():
    cfg = make_approval_cfg(approval_github_allowed_repos=[])
    result, _ = evaluate_approval("github_push_files", {"repo": "any/repo"}, cfg)
    assert result != ApprovalResult.DENIED  # empty = unrestricted
