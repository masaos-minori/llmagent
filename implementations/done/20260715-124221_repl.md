# Implementation Procedure: Unify Command Metadata as Single Source of Truth

## Goal

Eliminate duplicate slash command lists by making `_COMMANDS` the single source of truth for dispatch, `/help`, and tab completion. Define `/exit` as an explicit REPL-reserved command.

## Scope

### In scope
- `agent/repl.py`: Replace `SLASH_COMMANDS` with dynamic generation from `_COMMANDS` + REPL-reserved commands
- `agent/commands/registry.py`: Update `/help` output to include `/exit` as REPL control command

### Out of scope
- `agent/commands/cmd_memory.py` and `memory_*` modules
- `agent/config_*` files
- `agent/tool_policy.py`
- `agent/tool_approval.py`
- Documentation files
- Test files

## Assumptions

1. `_COMMANDS` in `CommandRegistry` contains all 23 built-in MCP tool commands (`/help`, `/config`, `/stats`, `/context`, `/plan`, `/undo`, `/reload`, `/compact`, `/mcp`, `/session`, `/clear`, `/export`, `/history`, `/system`, `/db`, `/set`, `/memory`, `/debug`, `/audit`, `/approve`, `/reject`, `/plugin`, `/skill`, `/mdq`).
2. `AgentREPL.SLASH_COMMANDS` currently contains 19 hardcoded commands, missing 6 from `_COMMANDS`: `/plan`, `/memory`, `/audit`, `/plugin`, `/skill`, `/mdq`. Conversely, `/exit` is in SLASH_COMMANDS but not in _COMMANDS.
3. `/exit` is handled specially in `repl.py:_should_exit()` before `CommandRegistry.dispatch()`. No overlap or conflict exists.
4. Tab completion needs to include both `_COMMANDS` commands and REPL-reserved commands.
5. `/help` output should include `/exit` as a REPL control command.

## Implementation

### Target file

`agent/repl.py`

### Procedure

#### Step 1: Add REPL-reserved command constant

Add a new module-level constant in `repl.py`:

```python
_REPL_RESERVED_COMMANDS = frozenset(["/exit"])
```

#### Step 2: Add helper functions

Add three helper functions in `repl.py` (module-level):

```python
def builtin_command_names() -> set[str]:
    """Return names of all built-in commands from _COMMANDS."""
    from agent.commands.command_defs_list import _COMMANDS
    return frozenset(cmd.name for cmd in _COMMANDS)

def reserved_repl_command_names() -> frozenset[str]:
    """Return names of REPL-reserved commands."""
    return _REPL_RESERVED_COMMANDS

def completion_command_names() -> frozenset[str]:
    """Return all commands available for tab completion."""
    return builtin_command_names() | reserved_repl_command_names()
```

Use lazy import inside `builtin_command_names()` to avoid circular imports at module level. Return `frozenset` instead of `set` for immutability.

#### Step 3: Replace SLASH_COMMANDS class attribute

Replace the hardcoded `SLASH_COMMANDS` class attribute with a cached property:

```python
from functools import cached_property

@cached_property
def SLASH_COMMANDS(self) -> frozenset[str]:
    """Tab completion candidates derived from _COMMANDS + REPL-reserved commands."""
    return completion_command_names()
```

Using `cached_property` avoids regenerating the set on every access while keeping it dynamic enough for potential future changes.

#### Step 4: Update /help output

Modify the `_cmd_help` method in `registry.py` to add a note about `/exit` being a REPL control command (not a dispatchable command). This requires updating the help text display logic after iterating over `_COMMANDS`.

### Method

- Use `functools.cached_property` for caching the computed set per instance
- Use `frozenset` for immutability of command name sets
- Use lazy import inside `builtin_command_names()` to defer the import until function call time, avoiding circular imports
- Maintain backward compatibility: `SLASH_COMMANDS` is still accessed as an attribute, so no changes needed to `CLIView.__init__`

### Details

**Type safety considerations:**
- `SLASH_COMMANDS` was previously a `list[str]`, now becomes `frozenset[str]`. Verify `CLIView.__init__` signature accepts `Iterable[str]` or update accordingly.
- All helper functions return `frozenset[str]` for immutability guarantees.

**Import considerations:**
- The lazy import in `builtin_command_names()` prevents circular import issues between `repl.py` and `agent/commands/command_defs_list.py`.
- `cached_property` is imported from `functools` — standard library, no additional dependencies.

**Backward compatibility:**
- `CLIView.__init__` accesses `SLASH_COMMANDS` as an attribute — no change needed since the cached property returns the same type (iterable of strings).
- Existing code that references `AgentREPL.SLASH_COMMANDS` will continue to work without modification.
