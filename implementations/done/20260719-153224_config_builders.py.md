## Scope

**In-Scope:**
- Remove `_FORBIDDEN_KEYS` constant (lines 254-266) and its check block (lines 267-271)
- Remove `github_server_url` individual check (lines 272-275)
- Clean up unused imports if applicable

**Out-of-Scope:**
- Changes to other source files (handled in separate documents)
- Changes to Pydantic dataclass definitions

3. The remaining code after removal (line 276 onward) does not depend on `_FORBIDDEN_KEYS` or `found`.
