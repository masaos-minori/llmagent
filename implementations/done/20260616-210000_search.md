# Implementation: mdq/mcp/search.py — FTS5-backed search

## Goal

Replace placeholder search with real SQLite FTS5 query and formatted results.

## Scope

**In scope:**
- `search_docs()` — Ensure FTS5 table exists, run MATCH query, format results
- `_search_docs_structured()` — Query sections_fts, return SearchDocsResult

**Out of scope:**
- Table creation (handled in service.py)
- Index population (handled by triggers in service.py)

## Assumptions

1. `sections_fts` FTS5 table exists and is populated via triggers.
2. FTS5 MATCH operator works for BM25 search.
3. `SearchDocsResult` from `rag.models_result` is available.

## Implementation

### Target file
`scripts/mcp/mdq/search.py`

### Procedure

1. Ensure FTS5 table exists before querying (check service._init_db was called).
2. Build query with optional filters (path_prefix, tag_filter, heading_prefix).
3. Return formatted results.

### Method

Use FTS5 MATCH with query string. Add WHERE clauses for path_prefix filtering.

### Details

```python
import logging
from typing import TYPE_CHECKING

from mcp.mdq.models import SearchDocsRequest

if TYPE_CHECKING:
    from mcp.mdq.service import MdqService

logger = logging.getLogger(__name__)


async def search_docs(service: "MdqService", req: SearchDocsRequest) -> str:
    """Search indexed Markdown sections by query; returns formatted results."""
    result = _search_docs_structured(service, req)
    if not result.results:
        return f"No results found for: {req.query!r}"
    lines = [f"Search results for: {req.query!r} ({result.total} found)"]
    for i, r in enumerate(result.results, 1):
        lines.append(f"{i}. {r}")
    return "\n".join(lines)


def _search_docs_structured(
    service: "MdqService", req: SearchDocsRequest
) -> dict:
    """Run FTS5 search; return structured result dict."""
    if not req.query or not req.query.strip():
        return {"query": req.query, "results": [], "total": 0}
    
    logger.info("MDQ search query: %s", req.query)
    
    import sqlite3
    
    conn = service._get_db_connection()
    try:
        limit = getattr(req, "limit", 10) or 10
        
        # Build query with optional path_prefix filter
        if req.path_prefix:
            rows = conn.execute(
                """SELECT s.file_path, s.heading, s.content 
                   FROM sections_fts f
                   JOIN sections s ON f.rowid = s.id
                   WHERE sections_fts MATCH ?
                   AND s.file_path LIKE ?
                   ORDER BY rank
                   LIMIT ?""",
                (req.query, f"{req.path_prefix}%", limit),
            ).fetchall()
        else:
            rows = conn.execute(
                """SELECT s.file_path, s.heading, s.content 
                   FROM sections_fts f
                   JOIN sections s ON f.rowid = s.id
                   WHERE sections_fts MATCH ?
                   ORDER BY rank
                   LIMIT ?""",
                (req.query, limit),
            ).fetchall()
        
        results = []
        for row in rows:
            results.append(f"[{row['file_path']}] {row['heading']}: {row['content'][:150]}")
            
    except sqlite3.OperationalError as e:
        logger.warning("MDQ FTS5 search failed (table may not exist): %s", e)
        results = []
    finally:
        conn.close()
    
    return {"query": req.query, "results": results, "total": len(results)}
```
