```markdown
#### `use_tool_dag`

**Default:** `false`

When `true`, tool calls in a single turn may execute in parallel (DAG-ordered, respecting explicit dependencies).

- `false` (default): all tool calls execute sequentially in order
- `true`: parallel execution enabled; respects `depends_on` DAG annotations

**Security note:** Parallel execution increases concurrency surface area. Audit all tool definitions before enabling. See `05_agent_16` §DAG Security.

For performance considerations, see `05_agent_06` §DAG Execution.
```
