## Implementation

### models.py — DTO 追加

```python
from dataclasses import dataclass

@dataclass(frozen=True)
class LatencySnapshot:
    """Per-tool latency stats (tool_name -> avg_ms)."""
    data: dict[str, float]

@dataclass(frozen=True)
class MaskedArgs:
    """Decoded and masked tool arguments for display (arg_name -> masked_value)."""
    data: dict[str, str]
```

### models.py — フィールド型更新

```python
# Before:
latency: dict[str, Any] | None = None
args_masked: dict[str, Any]

# After:
latency: LatencySnapshot | None = None
args_masked: MaskedArgs
```
