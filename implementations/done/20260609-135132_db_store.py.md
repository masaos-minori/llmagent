# Implementation: db/store.py — Replace ConfigLoader with build_db_config for embedding dims

## Goal

Replace the `ConfigLoader` usage in `get_embedding_dims()` with `build_db_config().embedding_dims`.

## Scope

- `scripts/db/store.py`: update `get_embedding_dims()`; remove `ConfigLoader` import.

## Assumptions

1. `build_db_config()` may raise (e.g., in test environments with no config files).
   The existing fallback to `384` must be preserved.
2. `get_embedding_dims()` is called frequently; `build_db_config()` reads a TOML file each time.
   This is acceptable for now; caching is a Phase 2 concern.

## Implementation

### Target file

`scripts/db/store.py`

### Procedure

**Replace import and function:**
```python
# Remove:
from shared.config_loader import ConfigLoader

# Add:
from db.config import build_db_config

def get_embedding_dims() -> int:
    """Return embedding dimensions from DbConfig; fallback to 384."""
    try:
        return build_db_config().embedding_dims
    except Exception:
        return 384
```
