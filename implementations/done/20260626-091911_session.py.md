## Implementation

### Target file
`scripts/agent/session.py`

### Procedure
1. Read `scripts/agent/session.py` to understand the existing state management API.
2. If `_handle_partial_completion()` does not exist, add:
   ```python
   def _handle_partial_completion(
       self,
       task_id: str,
       completed_steps: list[str],
       error: Exception,
   ) -> None:
       self._store.upsert_task(
           task_id=task_id,
           status="partial",
           completed_steps=completed_steps,
           error_type=type(error).__name__,
           error_msg=str(error),
       )
       logger.info(
           "partial completion recorded: task=%s steps=%d error=%s",
           task_id, len(completed_steps), type(error).__name__,
       )
   ```
3. If a partial version already exists, audit and align it with the above signature.

### Method
Single method addition/unification. No schema changes needed if `status="partial"` is already a valid task status.
