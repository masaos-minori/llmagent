# Goal

Fix two issues in `scripts/mcp/server.py`:
1. Narrow `except Exception` (last-resort stdio handler) to specific types.
2. Replace `str(req.get("name", ""))` with isinstance-validated access.

# Scope

- `scripts/mcp/server.py` only

# Assumptions

1. The `except Exception` at line 188 is the last-resort handler in the stdio
   transport loop. The preceding catches already cover:
   - `orjson.JSONDecodeError`
   - `(ValueError, OSError)`
   - `RuntimeError`
   Remaining realistic types from `dispatch()` and related logic:
   `TypeError`, `AttributeError`, `KeyError`, `sqlite3.OperationalError`,
   `sqlite3.DatabaseError`. Add `import sqlite3` to the imports.

2. `str(req.get("name", ""))` is at the JSON external boundary
   (`req = orjson.loads(line)`). A non-str `name` should raise `ValueError`
   so the enclosing `except (ValueError, OSError)` catch handles it and returns
   an error response.

# Implementation

## Target file

`scripts/mcp/server.py`

## Procedure

1. Add `import sqlite3` after the existing `import` block (before third-party
   imports, or after — ruff will sort).

2. Line 162: replace `str(req.get("name", ""))`:
   ```python
   # Before
   name = str(req.get("name", ""))

   # After
   name_raw = req.get("name", "")
   if not isinstance(name_raw, str):
       raise ValueError(
           f"Request 'name' must be str, got {type(name_raw).__name__}"
       )
   name = name_raw
   ```

3. Line 188: replace `except Exception as e:`:
   ```python
   # Before
   except Exception as e:
       # Last-resort handler: stdio transport must always write a response.
       logger.error(f"run_stdio unexpected error: {e}")
       result = f"Internal server error: {e}"
       is_error = True

   # After
   except (
       TypeError,
       AttributeError,
       KeyError,
       sqlite3.OperationalError,
       sqlite3.DatabaseError,
   ) as e:
       # Last-resort handler: stdio transport must always write a response.
       logger.error(f"run_stdio unexpected error: {e}")
       result = f"Internal server error: {e}"
       is_error = True
   ```

4. Run ruff + mypy.

## Method

One import addition + two targeted replacements.
