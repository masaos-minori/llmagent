# Implementation: scripts/db/config.py

## Goal

Change `build_db_config()` to load DB configuration exclusively from `common.toml` via `ConfigLoader().load("common.toml")` instead of `ConfigLoader().load_all()`. This makes the DB config source explicit and prevents accidental override from other config files.

## Scope

- Target: `scripts/db/config.py`
- Change `load_all()` call on line 56 to `load("common.toml")`
- Update docstring to reflect the new load method
- No other files changed

## Assumptions

1. All DB config keys (`rag_db_path`, `session_db_path`, `sqlite_timeout`, `sqlite_busy_timeout_ms`, `sqlite_vec_so`, `embedding_dims`) exist in `common.toml`.
2. `workflow_db_path` is absent from `common.toml` but the `cfg.get("workflow_db_path", "/opt/llm/db/workflow.sqlite")` fallback handles it.
3. `ConfigLoader().load("common.toml")` raises `ValueError` (or `ConfigMissingError` after plan 5 is applied) if `common.toml` is missing — this is intentional and preferred over silent empty dict.
4. Callers of `build_db_config()` already handle `ValueError` / `RuntimeError`.

## Implementation

### Target file
`scripts/db/config.py`

### Procedure
1. On line 56, replace `ConfigLoader().load_all()` with `ConfigLoader().load("common.toml")`.
2. Update docstring on line 55 to reflect `common.toml` direct load.

### Method

```python
# Before
def build_db_config() -> DbConfig:
    """Construct DbConfig from common.toml via ConfigLoader.load_all()."""
    cfg = ConfigLoader().load_all()

# After
def build_db_config() -> DbConfig:
    """Construct DbConfig from common.toml; raises ValueError if common.toml is missing or malformed."""
    cfg = ConfigLoader().load("common.toml")
```

### Details
- All remaining lines of `build_db_config()` (`DbConfig(...)` construction) are unchanged.
- No import changes needed — `ConfigLoader` is already imported on line 16.
