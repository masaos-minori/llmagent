# Implementation: services.py — New MemoryServices dataclass + factory.py + context.py update

## Goal

Create `scripts/agent/memory/services.py` with a `MemoryServices` dataclass that holds
the three memory services. Update `factory.py` and `context.py` to use `MemoryServices`
instead of `MemoryLayer`.

## Scope

- `scripts/agent/memory/services.py`: new file with `MemoryServices` dataclass.
- `scripts/agent/factory.py`: replace `_build_memory_layer()` with `_build_memory_services()`.
- `scripts/agent/context.py`: change `AppServices.memory` type from `MemoryLayer | None` to `MemoryServices | None`.

## Assumptions

1. `MemoryServices` holds `injection`, `ingestion`, `store`, `retriever` as direct references.
2. `factory.py` builds the same objects as before but returns `MemoryServices` instead of `MemoryLayer`.
3. `context.py` only changes the type annotation of `AppServices.memory`; no other fields change.
4. This step is a preparation step — `cmd_memory.py` still uses `MemoryLayer`; it will be migrated in step 8. Therefore `layer.py` is NOT deleted in this step.
5. `deploy/deploy.sh` must be updated to include `scripts/agent/memory/services.py` in the copy list.

## Implementation

### Target files

- `scripts/agent/memory/services.py` (new)
- `scripts/agent/factory.py`
- `scripts/agent/context.py`
- `deploy/deploy.sh`

### Procedure

**services.py — new file:**
```python
"""agent/memory/services.py
MemoryServices — container for the three memory service instances.

Replaces MemoryLayer as the AppServices.memory type.
"""
from __future__ import annotations

from dataclasses import dataclass

from agent.memory.ingestion import MemoryIngestionService
from agent.memory.injection import MemoryInjectionService
from agent.memory.retriever import HybridRetriever
from agent.memory.store import MemoryStore


@dataclass
class MemoryServices:
    """Container for memory sub-services; injected into AppServices.memory."""

    injection: MemoryInjectionService
    ingestion: MemoryIngestionService
    store: MemoryStore
    retriever: HybridRetriever
```

**factory.py — rename and update _build_memory_layer:**

Replace `_build_memory_layer(ctx, http) -> MemoryLayer | None` with:
```python
def _build_memory_services(
    ctx: AgentContext,
    http: httpx.AsyncClient,
) -> MemoryServices | None:
    if not ctx.cfg.memory.use_memory_layer:
        return None
    from agent.memory.embedding_client import EmbeddingClient, EmbeddingClientConfig
    from agent.memory.ingestion import DedupPolicy, MemoryIngestionService
    from agent.memory.injection import InjectionPolicy, MemoryInjectionService
    from agent.memory.jsonl_store import JsonlMemoryStore
    from agent.memory.retriever import HybridRetriever
    from agent.memory.services import MemoryServices
    from agent.memory.store import MemoryStore

    embed_cfg = EmbeddingClientConfig(
        embed_url=ctx.cfg.rag.embed_url,
        timeout=ctx.cfg.memory.memory_embed_timeout_sec,
    )
    embed_client = EmbeddingClient(embed_cfg, http, enabled=ctx.cfg.memory.memory_embed_enabled)
    retriever = HybridRetriever(
        fts_limit=ctx.cfg.memory.memory_fts_limit,
        rrf_k=ctx.cfg.memory.memory_rrf_k,
        recency_days=ctx.cfg.memory.memory_recency_days,
    )
    store = MemoryStore()
    jsonl = JsonlMemoryStore(f"{ctx.cfg.memory.memory_jsonl_dir}/memories.jsonl")
    injection = MemoryInjectionService(
        policy=InjectionPolicy(
            max_semantic=ctx.cfg.memory.memory_max_inject_semantic,
            max_episodic=ctx.cfg.memory.memory_max_inject_episodic,
            min_importance=ctx.cfg.memory.memory_min_importance,
        ),
        retriever=retriever,
        embed_client=embed_client,
    )
    ingestion = MemoryIngestionService(
        store=store,
        jsonl=jsonl,
        retriever=retriever,
        embed_client=embed_client,
        dedup_policy=DedupPolicy(threshold=ctx.cfg.memory.memory_dedup_threshold),
        branch=ctx.cfg.memory.memory_branch if hasattr(ctx.cfg.memory, "memory_branch") else "",
        max_content_chars=ctx.cfg.memory.memory_max_content_chars,
    )
    _build_audit_logger(ctx).info("MemoryServices initialised (use_memory_layer=True)")
    return MemoryServices(
        injection=injection,
        ingestion=ingestion,
        store=store,
        retriever=retriever,
    )
```

Update the call site in `build_agent_context()`:
```python
memory = _build_memory_services(ctx, http)
```

Update the `TYPE_CHECKING` import at the top of `factory.py`:
```python
if TYPE_CHECKING:
    from agent.memory.services import MemoryServices
```

**context.py — update AppServices.memory type:**

Change:
```python
from agent.memory.layer import MemoryLayer
...
memory: MemoryLayer | None,
...
self.memory = memory
```

To:
```python
if TYPE_CHECKING:
    from agent.memory.services import MemoryServices
...
memory: MemoryServices | None,
...
self.memory = memory
```

**deploy/deploy.sh — add services.py to copy list:**

Add `scripts/agent/memory/services.py` to the deploy script alongside the other memory files.

### Method

New file creation + targeted edits to factory.py and context.py.

### Details

`ingestion.py` currently takes `project` and `repo` args. In `factory.py` these were passed
as empty strings (from MemoryLayer constructor defaults). After refactoring, simply omit them
(they default to `""`).

`InjectionPolicy` no longer has `dedup_window` (removed in step 1), so no need to pass it.
