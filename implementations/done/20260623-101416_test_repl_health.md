## Goal

Add four new test methods to `TestAuditSecurityDefaults` in `tests/test_repl_health.py` covering the two new security checks added to `audit_security_defaults()`:
1. `cicd.workflow_allowlist` empty + `production_mode=True` → `RuntimeError`
2. `cicd.workflow_allowlist` empty + `production_mode=False` → warning only
3. `github.allow_force_push=True` → warning
4. `github.require_pr_review=False` → warning

## Scope

**In-Scope:**
- `tests/test_repl_health.py` — append four test methods to `class TestAuditSecurityDefaults`

**Out-of-Scope:**
- No changes to any production file
- No new test classes
- No changes to existing tests or `_make_ctx()`

## Assumptions

1. The existing `_make_ctx()` helper (lines 303-340) returns a context where `ShellConfig.load`, `SqliteConfig.load`, and `GitConfig.load` are NOT patched; tests that don't care about those checks must patch them to `side_effect=OSError` to isolate the check under test.
2. `CicdConfig` is imported from `mcp.cicd.models`; it must be patched at `agent.repl_health.CicdConfig` — but since it uses a lazy `from mcp.cicd.models import CicdConfig` inside the function body, it must be patched at `mcp.cicd.models.CicdConfig.load` or by patching the import path. The safest approach matches the existing pattern: `patch("agent.repl_health.CicdConfig", ...)` — but because CicdConfig is imported lazily inside the function, use `patch("mcp.cicd.models.CicdConfig.load", ...)` instead.
3. Similarly, `GitHubConfig.load` for the write settings check must be patched at `mcp.github.models_config.GitHubConfig.load` since it is also a lazy import inside the function.
4. A `CicdConfig` with `workflow_allowlist=[]` represents the fail-open empty case.
5. A `GitHubConfig` with `allow_force_push=True` or `require_pr_review=False` triggers individual warnings.

#### test_cicd_empty_workflow_allowlist_raises_in_production

```python
def test_cicd_empty_workflow_allowlist_raises_in_production(self) -> None:
    """cicd.workflow_allowlist empty + production_mode=True → RuntimeError."""
    from mcp.cicd.models import CicdConfig as CiCd

    ctx = self._make_ctx()
    empty_cicd = CiCd(workflow_allowlist=[])
    with (
        patch("mcp.cicd.models.CicdConfig.load", return_value=empty_cicd),
        patch("agent.repl_health.ShellConfig.load", side_effect=OSError),
        patch("agent.repl_health.SqliteConfig.load", side_effect=OSError),
        patch("agent.repl_health.GitConfig.load", side_effect=OSError),
    ):
        with pytest.raises(RuntimeError, match="Production mode requires explicit workflow_allowlist"):
            audit_security_defaults(ctx, production_mode=True)
```

def test_cicd_empty_workflow_allowlist_warns_in_dev(self) -> None:
    """cicd.workflow_allowlist empty + production_mode=False → warning only."""
    from mcp.cicd.models import CicdConfig as CiCd

    ctx = self._make_ctx()
    empty_cicd = CiCd(workflow_allowlist=[])
    with (
        patch("mcp.cicd.models.CicdConfig.load", return_value=empty_cicd),
        patch("agent.repl_health.ShellConfig.load", side_effect=OSError),
        patch("agent.repl_health.SqliteConfig.load", side_effect=OSError),
