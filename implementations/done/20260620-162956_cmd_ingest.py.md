# Implementation: Add refiner fallback hint in _cmd_rag() in cmd_ingest.py

## Goal
After printing context in `_cmd_rag()`, emit a user-visible `[warn]` hint line
when the refiner fell back to raw chunks, so operators see the event without `--debug`.

## Scope
- File: `scripts/agent/commands/cmd_ingest.py`
- Method: `_cmd_rag()` — insert between `self._out.write(context)` (line 149) and
  `if debug:` (line 151)
- Only when `pipeline.last_stage_results` contains a `Refiner` entry with `status == "fallback"`

## Assumptions
- `pipeline.last_stage_results` is `list[StageResult]` where `StageResult` is a `TypedDict`
  — access via `sr["stage_name"]` and `sr["status"]` (confirmed from stage.py)
- `sr["fallback_reason"]` is `str | None`; safe to use in f-string (None renders as "None"
  which is acceptable, but in practice refiner always sets a non-None reason on fallback)
- The hint should NOT appear when `context` is empty (no results found) — place the block inside
  the `else:` branch after `self._out.write(context)`, or guard with `if context:`
- Actually the current structure is:
  ```python
  if not context:
      self._out.write("No results found.")
  else:
      self._out.write(context)
  ```
  Insert the refiner hint check in the `else:` block (after `self._out.write(context)`) OR
  outside the if/else (always check, context may be empty while refiner ran)
  → safest: insert outside the if/else but before the `if debug:` block, so it fires
    regardless of whether context was empty (operators still want to know refiner fell back)

## Implementation

### Target file
`scripts/agent/commands/cmd_ingest.py`

### Procedure
1. Locate lines 145-152 in `_cmd_rag()`
2. Insert refiner fallback check after `else: self._out.write(context)` block (line 149),
   before `if debug:` (line 151)

### Method
Single edit — insert 8 lines.

### Details

**Current code (lines 145-152):**
```python
        context = await pipeline.augment(query, debug_fn=debug_fn)
        if not context:
            self._out.write("No results found.")
        else:
            self._out.write(context)

        if debug:
            self._print_rag_debug(pipeline.last_timings, pipeline.last_stage_results)
```

**Updated code:**
```python
        context = await pipeline.augment(query, debug_fn=debug_fn)
        if not context:
            self._out.write("No results found.")
        else:
            self._out.write(context)

        refiner_fb = next(
            (
                sr
                for sr in pipeline.last_stage_results
                if sr["stage_name"] == "Refiner" and sr["status"] == "fallback"
            ),
            None,
        )
        if refiner_fb:
            self._out.write(
                f"[warn] refiner fallback: {refiner_fb['fallback_reason']}"
                " — raw chunks used (run with --debug for stage details)"
            )

        if debug:
            self._print_rag_debug(pipeline.last_timings, pipeline.last_stage_results)
```
