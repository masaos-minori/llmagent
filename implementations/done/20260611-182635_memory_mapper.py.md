# Goal

Remove `dict[str, Any]` silent defaults in `mapper.py` by replacing `.get(field, default)`
for required fields with direct access (KeyError → MemorySchemaError), while keeping
optional fields with explicit None handling.

# Scope

- `scripts/agent/memory/mapper.py`

# Assumptions

1. `MemorySchemaError` from `agent.memory.exceptions` (Step 1 prerequisite).
2. Required fields (always present in DB schema): `memory_id`, `content`, `memory_type`,
   `source_type`, `importance`, `created_at`, `updated_at`.
3. Optional fields (nullable in DB): `session_id`, `turn_id`, `summary`, `tags`, `project`,
   `repo`, `branch`, `pinned`.
4. `float(d.get("importance", 0.5))` → `float(d["importance"])` — DB column is NOT NULL with
   DEFAULT 0.5, so always present. If missing, raise MemorySchemaError.
5. `d.get("source_type", "conversation")` → `d["source_type"]` — source_type is NOT NULL.
6. `tags` is JSON-encoded string in DB, always present. Empty JSON `[]` is valid.
7. MemoryEntry's `__post_init__` validates ranges/format — mapper only needs to ensure
   fields are present and correctly typed before constructing.

# Implementation

## Target file

`scripts/agent/memory/mapper.py`

## Procedure

1. Import `MemorySchemaError` from `agent.memory.exceptions`.
2. Wrap `d["field"]` accesses in try/except KeyError → raise MemorySchemaError.
3. Remove `float(d.get("importance", 0.5))` → `float(d["importance"])` with KeyError handling.
4. Remove `d.get("source_type", "conversation")` → `d["source_type"]`.
5. Keep optional fields (`session_id`, `turn_id`, `project`, `repo`, `branch`, `summary`, `pinned`)
   with `d.get("field")` (returns None if absent, which is valid).

## Method

Replace `.get(field, default)` with direct access for required fields, wrapped in a helper
that converts KeyError → MemorySchemaError with a clear message.

## Details

```python
from agent.memory.exceptions import MemorySchemaError

def _require(d: dict, key: str) -> object:
    try:
        return d[key]
    except KeyError:
        raise MemorySchemaError(f"Memory row missing required field: {key!r}")

def row_to_entry(row: sqlite3.Row | dict[str, Any]) -> MemoryEntry:
    d = dict(row)
    tags_raw = d.get("tags", "[]")
    ...  # tags parsing unchanged
    try:
        importance = float(_require(d, "importance"))
    except (TypeError, ValueError) as e:
        raise MemorySchemaError(f"Invalid importance value: {d.get('importance')!r}") from e
    return MemoryEntry(
        memory_id=str(_require(d, "memory_id")),
        memory_type=str(_require(d, "memory_type")),
        source_type=str(_require(d, "source_type")),
        session_id=d.get("session_id"),
        turn_id=d.get("turn_id"),
        project=d.get("project") or "",
        repo=d.get("repo") or "",
        branch=d.get("branch") or "",
        content=str(_require(d, "content")),
        summary=d.get("summary") or "",
        tags=tags,
        importance=importance,
        pinned=bool(d.get("pinned", 0)),
        created_at=d.get("created_at") or "",
        updated_at=d.get("updated_at") or "",
    )
```
