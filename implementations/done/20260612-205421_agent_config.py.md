# Implementation: agent/config.py — Narrow except Exception + remove redundant str()

## Goal

Narrow the `except Exception` in `load_config()` to specific types, and remove two redundant `str()` calls in `_build_llm_config()` and `_build_memory_config()` where the `cfg.get()` default already guarantees a `str`.

## Scope

**In:** `scripts/agent/config.py`
**Out:** No other files change.

## Assumptions

1. `ConfigLoader().load_all()` reads TOML files from disk; it can raise `OSError` (file not found/permission) and `ValueError` or `TypeError` (TOML parse / type mismatch). These cover all realistic failure modes.
2. `cfg.get("tokenize_url", "")` has a `str` default `""`, so `cfg.get()` never returns `None` here. The `str()` call is redundant but harmless; removing it simplifies the code without changing behavior.
3. `cfg.get("memory_jsonl_dir", "/opt/llm/memory")` has a `str` default; same reasoning.

## Implementation

### Target file

`scripts/agent/config.py`

### Procedure

1. In `load_config()`, change `except Exception as e:` → `except (OSError, ValueError, TypeError) as e:`.
2. In `_build_llm_config()`, remove the `str()` wrapping `cfg.get("tokenize_url", "")`.
3. In `_build_memory_config()`, remove the `str()` wrapping `cfg.get("memory_jsonl_dir", "/opt/llm/memory")`.

### Method

Targeted in-place edits. Three independent changes in one file.

### Details

**load_config() (line ~29):**
```python
# Before
except Exception as e:
    raise ConfigLoadError(f"Config load failed: {e}") from e

# After
except (OSError, ValueError, TypeError) as e:
    raise ConfigLoadError(f"Config load failed: {e}") from e
```

**_build_llm_config() (line ~517):**
```python
# Before
tokenize_url=str(cfg.get("tokenize_url", "")),

# After
tokenize_url=cfg.get("tokenize_url", ""),
```

**_build_memory_config() (line ~578):**
```python
# Before
memory_jsonl_dir=str(cfg.get("memory_jsonl_dir", "/opt/llm/memory")),

# After
memory_jsonl_dir=cfg.get("memory_jsonl_dir", "/opt/llm/memory"),
```

**Note on mypy:** After removing `str()`, mypy will see `cfg.get("tokenize_url", "")` returning `str` (since the default is `str`). If mypy raises an error about `str | Any`, use an explicit cast: `cast(str, cfg.get("tokenize_url", ""))` — but only if needed. Check first.
