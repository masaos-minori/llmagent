# Implementation: Shared Type Safety — Literal Strings for McpServerConfig and LLMMessage

## Goal

Strengthen static type checking in `shared/mcp_config.py` and `shared/types.py` by replacing bare `str` annotations with `Literal` types for fields with constrained value sets.

## Scope

**In:**
- `McpServerConfig.transport`: `str` → `Literal["http", "stdio"]`
- `McpServerConfig.startup_mode`: `str` → `Literal["persistent", "ondemand", "subprocess"]`
- `McpServerConfig.healthcheck_mode`: `str` → `Literal["http", "process", "ping_tool", ""]`
- `LLMMessage.role`: `str` → `Literal["user", "assistant", "tool", "system"]`
- Add `deprecation warning` to `_build_mcp_servers()` legacy fallback path

**Out:**
- Changing `McpServerConfig._validate_*()` runtime checks (they remain identical)
- Adding `Literal` to any other fields

## Assumptions

- `scripts/shared/mcp_config.py`: `McpServerConfig` dataclass; `_validate_transport()` / `_validate_startup_mode()` / `_resolve_healthcheck_mode()` do runtime validation
- `scripts/shared/types.py`: `LLMMessage(TypedDict)` with `role: str`
- mypy version supports `Literal` from `typing` (Python 3.13)
- `LLMMessage` is created via dict literals throughout the codebase; `Literal["user", ...]` is compatible with string dict construction

## Implementation

### `scripts/shared/mcp_config.py`

```python
from typing import Any, Literal

@dataclass
class McpServerConfig:
    transport: Literal["http", "stdio"]            # was: str
    url: str
    cmd: list[str]
    openrc_service: str
    startup_mode: Literal["persistent", "ondemand", "subprocess"] = "persistent"  # was: str
    healthcheck_mode: Literal["http", "process", "ping_tool", ""] = ""             # was: str
    # ... rest unchanged ...
```

In `_build_mcp_servers()`, add a deprecation warning to the legacy URL-constants fallback:

```python
import warnings

def _build_mcp_servers(cfg: dict[str, Any]) -> dict[str, McpServerConfig]:
    raw: dict[str, Any] = cfg.get("mcp_servers", {})
    if raw:
        return { ... }  # unchanged
    # Legacy fallback
    warnings.warn(
        "mcp_servers config section missing; falling back to legacy URL constants. "
        "Add [mcp_servers] to config/mcp_servers.toml.",
        DeprecationWarning,
        stacklevel=2,
    )
    return { ... }  # unchanged legacy path
```

### `scripts/shared/types.py`

```python
from typing import Literal, Protocol, TypedDict, runtime_checkable

class LLMMessage(TypedDict, total=False):
    role: Literal["user", "assistant", "tool", "system"]  # was: str
    content: str | None
    tool_calls: list[dict]
    tool_call_id: str
    name: str
```
