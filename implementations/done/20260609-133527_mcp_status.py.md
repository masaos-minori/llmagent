# Implementation: mcp_status.py — Split McpServerStatus.status and move format_table

## Goal

1. Split `McpServerStatus.status` (concatenated string) into `availability: str` + `health: str`.
2. Delete `McpStatusService.format_table()` from the service; move it to `cmd_mcp.py`.

## Scope

- `scripts/agent/services/mcp_status.py`: restructure `McpServerStatus`; remove `format_table`.
- `scripts/agent/commands/cmd_mcp.py`: add `_format_mcp_table()` function; update `_cmd_mcp_status`.
- `tests/` that reference `McpServerStatus.status`: update to `availability`/`health`.

## Assumptions

1. `format_table()` is called only from `cmd_mcp.py._cmd_mcp_status()`.
2. The combined display `f"{availability}/{health}"` preserves the existing output format.
3. `_tier_label_for_server` stays in `mcp_status.py` (pure data helper).

## Implementation

### Target files

- `scripts/agent/services/mcp_status.py`
- `scripts/agent/commands/cmd_mcp.py`

### Procedure

**mcp_status.py — restructure McpServerStatus:**

```python
@dataclass
class McpServerStatus:
    key: str
    transport: str
    startup_mode: str
    auth: str
    write: str
    role: str
    availability: str   # "OK" | "HTTP NNN" | "FAIL (...)" | "STOPPED" | ...
    health: str         # "HEALTHY" | "DEGRADED" | "UNAVAILABLE"
    endpoint: str
```

**mcp_status.py — update probe_all():**

Replace:
```python
status = f"{status}/{health_label}"
results.append(McpServerStatus(..., status=status, ...))
```
With:
```python
results.append(McpServerStatus(
    ...,
    availability=availability,
    health=health_label,
    ...
))
```
(`availability` is the string from `_get_http_status` / `_get_stdio_status`; `health_label` stays as-is.)

**mcp_status.py — delete format_table().**

**cmd_mcp.py — add _format_mcp_table() and update _cmd_mcp_status:**

```python
def _format_mcp_table(rows: list[McpServerStatus]) -> str:
    col = "{:<14} {:<6} {:<11} {:<5} {:<12} {:<12} {:<16} {}"
    lines = [col.format("SERVER","TRANS","MODE","AUTH","WRITE","ROLE","STATUS","ENDPOINT/CMD"), "-"*99]
    for r in rows:
        lines.append(col.format(
            r.key, r.transport, r.startup_mode, r.auth, r.write, r.role,
            f"{r.availability}/{r.health}", r.endpoint
        ))
    return "\n".join(lines)
```

Update `_cmd_mcp_status`:
```python
print(_format_mcp_table(rows))  # was: svc.format_table(rows)
```

### Method

Edit `mcp_status.py` (field rename + method deletion), then edit `cmd_mcp.py` (add helper + update call).
