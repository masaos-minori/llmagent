# Implementation: agent/repl.py — Replace assert + narrow except Exception

## Goal

Replace 2 `assert` statements with `RuntimeError` guards and narrow 4 `except Exception` clauses to specific exception types in `scripts/agent/repl.py`.

## Scope

**In:** `scripts/agent/repl.py`
**Out:** No other files change.

## Assumptions

1. The two asserts guard `ctx.services.tools` and `ctx.services.lifecycle` in `_start_mcp_servers()`; both are set by the factory.
2. `_get_chunk_count()` queries SQLite and can raise `sqlite3.Error` or `OSError`. The function returns `"?"` on error.
3. `_start_mcp_servers()` starts HTTP subprocess and stdio MCP servers. Failures are non-fatal (logged and printed); `(OSError, RuntimeError)` covers subprocess / transport errors.
4. `run()` finally block catches any error from `memory.on_session_stop()`. Memory ingestion can raise `(RuntimeError, sqlite3.Error, OSError)` based on the ingestion code analysis.
5. `repl.py:305` `except Exception:` swallows and logs the error without re-raising; narrowing must preserve this behaviour.

## Implementation

### Target file

`scripts/agent/repl.py`

### Procedure

1. Replace assert in `_start_mcp_servers()` (lines 199–200) with RuntimeError guards.
2. Narrow `_get_chunk_count()` `except Exception as e:` → `except (sqlite3.Error, OSError)`.
3. Narrow `_start_mcp_servers()` HTTP branch `except Exception as e:` → `except (OSError, RuntimeError)`.
4. Narrow `_start_mcp_servers()` stdio branch `except Exception as e:` → `except (OSError, RuntimeError)`.
5. Narrow `run()` finally `except Exception:` → `except (RuntimeError, sqlite3.Error, OSError)`.

### Method

Targeted in-place edits. `import sqlite3` must be present at the top of the file; verify and add if missing.

### Details

**Assert replacements (lines 199–200):**
```python
# Before
assert ctx.services.tools is not None
assert ctx.services.lifecycle is not None

# After
if ctx.services.tools is None:
    raise RuntimeError("tools service not initialized")
if ctx.services.lifecycle is None:
    raise RuntimeError("lifecycle service not initialized")
```

**_get_chunk_count (line ~100):**
```python
# Before
except Exception as e:
    logger.debug(f"Failed to get chunk count: {e}")
    return "?"

# After
except (sqlite3.Error, OSError) as e:
    logger.debug(f"Failed to get chunk count: {e}")
    return "?"
```

**_start_mcp_servers HTTP branch (line ~205):**
```python
# Before
except Exception as e:
    logger.error(f"Failed to start HTTP subprocess MCP server {key!r}: {e}")
    print(f"[warn] HTTP subprocess MCP server {key!r} failed to start: {e}")

# After
except (OSError, RuntimeError) as e:
    logger.error(f"Failed to start HTTP subprocess MCP server {key!r}: {e}")
    print(f"[warn] HTTP subprocess MCP server {key!r} failed to start: {e}")
```

**_start_mcp_servers stdio branch (line ~230):**
```python
# Before

    logger.error(f"Failed to start stdio MCP server {key!r}: {e}")
    print(f"[warn] stdio MCP server {key!r} failed to start: {e}")

# After
except (OSError, RuntimeError) as e:
    logger.error(f"Failed to start stdio MCP server {key!r}: {e}")
    print(f"[warn] stdio MCP server {key!r} failed to start: {e}")
```

**run() finally block memory stop (line ~305):**
```python

    logger.exception("Memory on_session_stop failed; session data may be incomplete")

```
