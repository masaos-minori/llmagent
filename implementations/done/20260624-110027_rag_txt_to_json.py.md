# Implementation Procedure: scripts/rag/ingestion/{crawler,chunk_splitter,ingester}.py

## Goal

RAG ingestion パイプラインの全出力ファイル拡張子を `.txt` → `.json` に変更する。

## Scope

**In:**
- `scripts/rag/ingestion/crawler.py` — 出力パス `.txt` → `.json`
- `scripts/rag/ingestion/chunk_splitter.py` — glob パターン + 出力パス `.txt` → `.json`
- `scripts/rag/ingestion/ingester.py` — glob パターン `.txt` → `.json`
- 関連ドキュメント 4 本

**Out:** JSON ペイロードスキーマの変更

## Migration Policy

後方互換 glob なし。既存 `.txt` ファイルは手動リネームまたは再クロール必要。移行コマンド:
```bash
for f in rag-src/*.txt rag-src/chunk/*.txt; do mv "$f" "${f%.txt}.json"; done
```
