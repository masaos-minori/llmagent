# Implementation: cmd_config.py — enhance _cmd_reload() output

## Goal

Replace the current minimal `/reload` output with a structured, informative report
showing what was reloaded from which files, what was applied, and what requires restart.

## Scope

`scripts/agent/commands/cmd_config.py` — `_cmd_reload()` method only.

## Assumptions

1. Source files are always `common.toml` and `agent.toml` — hardcoded as before.
2. The output goes to `self._out.write()`.
3. If `result.applied` is non-empty, list each applied component.
4. If `result.needs_restart` is non-empty, list each component requiring restart.
5. If both are empty, output "No changes detected."

## Implementation

### Target file

`scripts/agent/commands/cmd_config.py`

### Procedure

Replace the body of `_cmd_reload()` with the enhanced version.

### Method

Display-only change. Logic (ConfigLoader + ConfigReloadService) is unchanged.

### Details

```python
def _cmd_reload(self) -> None:
    from agent.services.config_reload import ConfigReloadService

    try:
        from shared.config_loader import ConfigLoader

        source_files = ["common.toml", "agent.toml"]
        new_cfg = ConfigLoader().load(*source_files)
        result = ConfigReloadService(self._ctx).apply_config_dict(new_cfg)
        result.source_files = source_files

        self._out.write(f"Config reloaded from: {', '.join(source_files)}")
        if result.applied:
            self._out.write("Applied (runtime):")
            for item in result.applied:
                self._out.write(f"  - {item}")
        if result.needs_restart:
            self._out.write("Restart required:")
            for item in result.needs_restart:
                self._out.write(f"  - {item}")
        if not result.applied and not result.needs_restart:
            self._out.write("No changes detected.")
        logger.info("Config reloaded: applied=%s, needs_restart=%s", result.applied, result.needs_restart)
    except OSError as e:
        logger.warning("Config reload I/O error: %s", e)
        self._out.write(f"Reload failed (I/O error): {e}")
    except ValueError as e:
        logger.warning("Config reload failed: %s", e)
        self._out.write(f"Reload failed: {e}")
```
