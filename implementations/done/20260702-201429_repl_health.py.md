1. Add `_collect_server_tool_lists()` after `_collect_server_tool_names()`:

```python
async def _collect_server_tool_lists(ctx: AgentContext) -> dict[str, list[str]]:
    """Probe all configured HTTP MCP servers and return per-server tool name lists.

    Returns {server_key: [tool_names]} for reachable servers.
    Unreachable servers are silently skipped (same pattern as _collect_server_tool_names).

    Note: This helper exists because _collect_server_tool_names() returns a merged set,
    while validate_routing_against_live() needs per-server tool lists for ownership drift detection.
    """
    if ctx.services.http is None:
        raise RuntimeError("http service not initialized")
    result: dict[str, list[str]] = {}
    for key, srv_cfg in ctx.cfg.mcp.mcp_servers.items():
        if srv_cfg.transport == "http":
            if not srv_cfg.url:
                continue
            try:
                resp = await ctx.services.http.get(
                    f"{srv_cfg.url}/v1/tools",
                    timeout=5.0,
                )
                if resp.status_code == HTTPStatus.OK:
                    result[key] = [t["name"] for t in resp.json().get("tools", [])]
                else:
                    msg = f"{key} /v1/tools returned HTTP {resp.status_code}"
                    logger.warning(msg)
            except (httpx.HTTPError, OSError) as e:
                msg = f"{key} unreachable at {srv_cfg.url}/v1/tools: {e}"
                logger.warning(msg)
    return result
```

2. Add `check_routing_drift_live()` after `check_routing_drift()`:

```python
async def check_routing_drift_live(ctx: AgentContext) -> list[str]:
    """Check live /v1/tools tool ownership against ToolRegistry at startup. Returns warning messages."""
    from shared.tool_registry import validate_routing_against_live  # noqa: PLC0415

    try:
        live_tool_lists = await _collect_server_tool_lists(ctx)
        if not live_tool_lists:
            logger.info("Live routing drift check skipped: no reachable HTTP servers")
            return []
        drift = validate_routing_against_live(live_tool_lists=live_tool_lists)
        warnings: list[str] = []
        for server_key, messages in drift.items():
            for msg in messages:
                full_msg = f"Live routing drift [{server_key}]: {msg}"
                logger.warning(full_msg)
                warnings.append(full_msg)
        return warnings
    except Exception as exc:  # noqa: BLE001
        logger.warning("Live routing drift check failed: %s", exc)
        return []
```

### Method

- Follow the same error-swallowing pattern as `_collect_server_tool_names()` and `check_routing_drift()`.
- Use lazy import (`# noqa: PLC0415`) for `validate_routing_against_live` to avoid circular imports.
- Log INFO when no reachable HTTP servers (not WARNING).
- Return empty list on any exception (non-fatal, consistent with existing pattern).

### Details

- `_collect_server_tool_lists()` is a private function (leading underscore) — only called by `check_routing_drift_live()`.
- The function iterates the same server list as `_collect_server_tool_names()` but builds a per-server dict instead of merging into a set.
- `check_routing_drift_live()` signature matches `check_routing_drift()` pattern: takes `AgentContext`, returns `list[str]`.
