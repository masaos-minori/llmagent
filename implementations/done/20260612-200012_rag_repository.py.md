# Goal

Replace untyped `_sd_tkn: Any` / `_sd_split_c: Any` with a typed
`SudachiTokenizer` wrapper class, convert `{**item, "rrf_score": 0.0}` dict
construction to `MergedHit` dataclass, and separate SQL row → `RawHit` conversion
into a dedicated function.

# Scope

- `scripts/rag/repository.py`

# Assumptions

1. `MergedHit`, `RawHit` dataclasses from Step 3-1 prerequisite.
2. `_sd_tkn` and `_sd_split_c` are sudachipy objects. The typed wrapper
   `SudachiTokenizer` can use `TYPE_CHECKING` import from sudachipy stubs or
   store them as `Any` internally but expose a typed `tokenize(text: str) -> list[str]`
   method.
3. `{**item, "rrf_score": 0.0}` → `MergedHit(chunk_id=item.chunk_id, ..., rrf_score=0.0)`.
4. `_row_to_raw_hit(row: sqlite3.Row) -> RawHit` extracts columns by name
   and constructs `RawHit`.
5. `hit.get("url", "")` → `hit.url` after dataclass conversion.

# Implementation

## Target file

`scripts/rag/repository.py`

## Procedure

1. Define `SudachiTokenizer` wrapper class in `repository.py` (or a new
   `rag/ingestion/_sudachi.py`):
   ```python
   class SudachiTokenizer:
       def __init__(self) -> None:
           import sudachipy  # noqa: PLC0415
           self._tkn = sudachipy.dictionary.Dictionary().create()
           self._mode = sudachipy.SplitMode.C
       def tokenize(self, text: str) -> list[str]:
           return [m.surface() for m in self._tkn.tokenize(text, self._mode)]
   ```
2. Replace `_sd_tkn: Any` / `_sd_split_c: Any` fields with `_tokenizer: SudachiTokenizer`.
3. Add `_row_to_raw_hit(row: sqlite3.Row) -> RawHit` function.
4. Replace `merged.append({**item, "rrf_score": 0.0})` with
   `merged.append(MergedHit(..., rrf_score=0.0))`.
5. Replace `hit.get("url", "")` with `hit.url`.
6. Run ruff + mypy.

## Method

Typed wrapper class + dict-to-dataclass conversion + helper function extraction.
