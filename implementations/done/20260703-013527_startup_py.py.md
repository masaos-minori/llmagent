# Implementation: scripts/agent/startup.py — Add [non-fatal] prefix to startup warning messages

**Plan source:** `plans/20260702-202919_plan.md` (Phase 3 (startup))
**Target file:** `scripts/agent/startup.py`

---

## Goal

startup.py の _check_services() および _start_servers() 内の write_warning 呼び出しに `[non-fatal]` プレフィックスを付加し、非致命的な起動時警告であることを明示する。また factory.py の plugin load 失敗警告にも同様の対応を行う。

---

## Scope

**In:**
- startup.py: _check_services() の write_warning 呼び出しに `[non-fatal]` プレフィックスを追加、コメント `# non-fatal in local profile` を付記
- startup.py: _start_servers() の HTTP subprocess MCP server 失敗時の write_warning に `[non-fatal]` プレフィックスを追加
- factory.py (Phase 4): _init_plugin_registry() の audit_logger.warning で strict_mode=False 時のプラグインロード失敗に `[non-fatal]` プレフィックスを追加

**Out:**
- repl_health.py の変更 (Phase 3 (repl_health) で対応済み)
- fatal エラーへの格上げ
- write_warning 自体のメソッドシグネチャ変更

---

## Assumptions

1. startup.py の _check_services() に write_warning 呼び出しが存在し、repl_health.py から渡されたメッセージをそのまま転送するか、独自メッセージを生成している
2. _start_servers() に HTTP subprocess MCP server 起動失敗時の write_warning が存在する
3. factory.py の _init_plugin_registry() に `audit_logger.warning` が存在し、strict_mode フラグが参照可能

---

## Implementation

### Target file

`scripts/agent/startup.py` (+ `scripts/agent/factory.py`)

### Procedure

#### startup.py

1. startup.py を読み込み、_check_services() と _start_servers() の write_warning 呼び出し箇所を確認する
2. _check_services() の各 write_warning のメッセージに `[non-fatal]` プレフィックスを追加し、行末に `# non-fatal in local profile` コメントを付ける
3. _start_servers() の HTTP subprocess MCP server 失敗時の write_warning メッセージに `[non-fatal]` プレフィックスを追加する
4. lint/mypy を確認する

#### factory.py (Phase 4)

5. factory.py を読み込み、_init_plugin_registry() の audit_logger.warning 呼び出し箇所を確認する
6. strict_mode=False の分岐内の audit_logger.warning メッセージに `[non-fatal]` プレフィックスを追加する
7. lint/mypy を確認する

### Method

Edit tool でコード変更 (startup.py と factory.py それぞれ)

### Details

startup.py _check_services() の変更パターン:
```python
self._view.write_warning(f"[non-fatal] {existing_message}")  # non-fatal in local profile
```

startup.py _start_servers() の変更パターン:
```python
self._view.write_warning(f"[non-fatal] {existing_mcp_server_message}")
```

factory.py の変更パターン:
```python
# strict_mode=False のブランチ内
audit_logger.warning(f"[non-fatal] {existing_plugin_load_message}")
```

実際のメッセージ文字列は各ファイルを読んで確認してから編集する。
