# Implementation: mcp_install.py — Remove print_next_steps from service

## Goal

Delete `McpInstallService.print_next_steps()` and move the output logic to `cmd_mcp.py`.

## Scope

- `scripts/agent/services/mcp_install.py`: delete `print_next_steps()`.
- `scripts/agent/commands/cmd_mcp.py`: inline the next-steps output in `_cmd_mcp_install()`.

## Assumptions

1. `print_next_steps()` is called only from `cmd_mcp._cmd_mcp_install()`.
2. The output content is preserved exactly — only the location moves.

## Implementation

### Target files

- `scripts/agent/services/mcp_install.py`
- `scripts/agent/commands/cmd_mcp.py`

### Procedure

**mcp_install.py:** Delete the entire `print_next_steps()` method.

**cmd_mcp.py — replace `svc.print_next_steps(result)` with inline helper:**

```python
def _print_mcp_install_next_steps(result: ScaffoldResult) -> None:
    """Print post-install checklist for the newly scaffolded MCP server."""
    print()
    print("Next steps:")
    print(f"  1. Edit scripts/mcp/{result.module}/server.py — implement _DISPATCH handlers")
    print()
    print("  2. Add tool definition to config/agent.toml (tool_definitions array):")
    for line in result.tool_snippet.splitlines():
        print(f"     {line}")
    print()
    print("  3. Add to config/agent.toml [mcp_servers]:")
    for line in (result.agent_toml_snippet or "").splitlines():
        print(f"     {line}")
    print()
    print("  4. Add to deploy/deploy.sh:")
    print(f'     cp -r "${{REPO_ROOT}}/scripts/mcp/{result.module}" "${{DEPLOY_SCRIPTS}}/mcp/"')
    print(f'     cp "${{REPO_ROOT}}/config/{result.module}_mcp_server.toml" "${{DEPLOY_CONFIG}}/"')
    print()
    print("  5. Add to deploy/setup_services.sh (service list and conf.d copy):")
    print(f"     {result.server_name}  (add to the for-loop service list)")
    if result.with_confd:
        print(f'     cp "${{REPO_ROOT}}/conf.d/{result.server_name}" "/etc/conf.d/{result.server_name}"')
    print()
    print("  6. Deploy and start:")
    print("     bash deploy/deploy.sh")
    print(f"     cp init.d/{result.server_name} /etc/init.d/{result.server_name}")
    print(f"     chmod +x /etc/init.d/{result.server_name}")
    print(f"     rc-update add {result.server_name} default")
    print(f"     rc-service {result.server_name} start")
```

In `_cmd_mcp_install()`, replace `svc.print_next_steps(result)` with `_print_mcp_install_next_steps(result)`.

### Method

Delete method from service; add module-level function to cmd_mcp.py.
