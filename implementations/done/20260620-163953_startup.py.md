# Implementation: Add RAG consistency check to startup.py _check_services()

## Goal
Emit `write_warning()` during agent startup when the RAG index is inconsistent
(chunks/FTS/vec out of sync), making consistency issues visible on every startup
without requiring a manual `/db consistency` run.

## Scope
- File: `scripts/agent/startup.py`
- Method: `_check_services()` — append RAG consistency check after existing readiness checks
- Guarded by `try/except Exception` so it never crashes startup if rag.sqlite is absent

## Assumptions
- `RagMaintenanceService` is importable from `agent.services.rag_maintenance_service`
  (already used in `cmd_db.py` — same import path)
- `RagMaintenanceService().consistency()` opens `rag.sqlite` via `SQLiteHelper("rag")`
  — will raise `FileNotFoundError` or `sqlite3.OperationalError` if rag.sqlite absent;
  caught by the `except Exception` guard
- On success (consistent): `logger.info("RAG consistency: OK")` only — no startup output
- On failure (inconsistent): one `self._view.write_warning()` per issue
- Startup must stay fast: 3 COUNT queries are O(1) on SQLite — acceptable

## Implementation

### Target file
`scripts/agent/startup.py`

### Procedure
1. Add `from agent.services.rag_maintenance_service import RagMaintenanceService`
   to imports (if not already present)
2. In `_check_services()`, append the consistency block at the end

### Method
Two edits: import addition and method body extension.

### Details

**New import (add to import block):**
```python
from agent.services.rag_maintenance_service import RagMaintenanceService
```

**New block at end of `_check_services()` (after existing readiness checks):**
```python
        try:
            consistent, issues = RagMaintenanceService().consistency()
            if consistent:
                logger.info("RAG consistency: OK")
            else:
                for issue in issues:
                    self._view.write_warning(f"[RAG] Consistency issue: {issue}")
        except Exception as e:  # noqa: BLE001 — skip if rag.sqlite absent or unreadable
            logger.debug("RAG consistency check skipped: %s", e)
```
