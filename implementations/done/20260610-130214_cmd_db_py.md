# Implementation: cmd_db.py — remove broad exceptions, apply formatter

## Goal

Remove all `except sqlite3.Error` / `except Exception` handlers that swallow errors
and print directly to the CLI. Route all output through `formatter.py`. Migrate
`--lang` / `--limit` parsing to `parse_command_args()`.

## Scope

- `scripts/agent/commands/cmd_db.py` — primary target
- `tests/test_agent_cmd_db.py` — update tests that expected silent failure

## Assumptions

1. After removing `except` blocks, exceptions propagate to the REPL's outer handler,
   which logs and prints a generic error. This is acceptable behavior (fail-fast).
2. `_db_stats()`, `_db_health()`, `_db_checkpoint()`, `_db_vacuum()`, `_db_rebuild_fts()`
   all have `except` blocks to remove.
3. `/db urls` currently uses `parse_flag_str(tokens, "--lang")` and `parse_flag_int(tokens, "--limit")`.
   Replace with `parse_command_args(tokens)` and read `args.flags["lang"]` / `args.flags["limit"]`.
4. Output `print()` calls are replaced with `formatter.print_kv_list()` / `formatter.print_table()` /
   `formatter.print_success()` / `formatter.print_error()`.
5. `sqlite3` import can be removed after deleting `except sqlite3.Error` blocks.

## Implementation

### Target file

`scripts/agent/commands/cmd_db.py`

### Procedure

1. Remove `import sqlite3` (no longer needed after except removal).
2. Add imports: `from agent.commands.formatter import print_error, print_no_data, print_success, print_table, print_kv_list`.
3. Add import: `from agent.commands.utils import parse_command_args`.
4. In `_db_stats()`: remove try/except; replace `print(f"documents : ...")` with `print_kv_list(...)`.
5. In `_db_health()`: remove try/except; replace individual `print(f"...")` with `print_kv_list(...)`.
6. In `_db_rebuild_fts()`: remove try/except; replace `print("FTS5 index rebuilt.")` with `print_success(...)`.
7. In `_db_checkpoint()`: remove try/except; replace output with `print_kv_list(...)`.
8. In `_db_vacuum()`: remove try/except; replace `print` with `print_success(...)`.
9. In `_db_urls()`: replace `parse_flag_str/int` with `parse_command_args`; replace print loop with `print_table(...)`.
10. In `_db_clean()`: replace `print(...)` with `print_success / print_no_data`.

### Method

Direct textual edit.
