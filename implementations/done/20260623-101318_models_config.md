## Goal

Change the Python defaults for `allow_force_push` and `require_pr_review` in `scripts/mcp/github/models_config.py` from the current unsafe values (`True`/`False`) to the safe values (`False`/`True`), and update the `from_dict()` fallback arguments to match.

## Scope

**In-Scope:**
- `scripts/mcp/github/models_config.py` lines 45-46 — dataclass field defaults
- `scripts/mcp/github/models_config.py` lines 61-62 — `from_dict()` fallback defaults

**Out-of-Scope:**
- No changes to `GitHubConfig.load()`, `GitHubService`, or any other method
- No new fields or validation logic
- No changes to test files (test changes are covered in the `test_repl_health.py` implementation doc)

## Assumptions

1. Python dataclass field defaults and `from_dict()` fallbacks must be changed together — they are the two places that set a value when the TOML key is absent.
2. Existing tests in `tests/test_github_mcp_service.py` do not rely on `allow_force_push=True` or `require_pr_review=False` as implicit defaults (confirmed via grep: no such assertions).
3. Any test or caller that explicitly sets `allow_force_push=True` in its config dict will be unaffected — the change only affects the no-key-present case.

## Implementation

### Target file
`scripts/mcp/github/models_config.py`

### Procedure

1. Locate line 45 (`allow_force_push: bool = True`) and line 46 (`require_pr_review: bool = False`).
2. Change both field defaults.
3. Locate lines 61-62 in `from_dict()`.
4. Change both fallback defaults.

### Method

Two `Edit` operations (one for field defaults, one for `from_dict` fallbacks), or a single `Edit` covering both locations.

### Details

**Current (lines 45-46):**
```python
    allow_force_push: bool = True
    require_pr_review: bool = False
```

**Replacement:**
```python
    allow_force_push: bool = False
    require_pr_review: bool = True
```

**Current (lines 61-62):**
```python
            allow_force_push=bool(d.get("allow_force_push", True)),
            require_pr_review=bool(d.get("require_pr_review", False)),
```

**Replacement:**
```python
            allow_force_push=bool(d.get("allow_force_push", False)),
            require_pr_review=bool(d.get("require_pr_review", True)),
```
