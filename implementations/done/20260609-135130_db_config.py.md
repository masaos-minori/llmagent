# Implementation: db/config.py — Add fields, relax path validation

## Goal

1. Add `sqlite_busy_timeout_ms: int = 30000` and `embedding_dims: int = 384` to `DbConfig`.
2. Change `__post_init__` path validation from "file must exist" to "parent directory must exist".
3. Update `build_db_config()` to populate the new fields.

## Scope

- `scripts/db/config.py`: add two fields; change `__post_init__`; update `build_db_config`.

## Assumptions

1. Existing callers of `DbConfig(...)` use keyword arguments or will break on positional — the new
   fields have defaults so existing code is unaffected.
2. Parent directory existence is the right check: DB files are auto-created by SQLite on first open.
3. `sqlite_vec_so` path still needs full existence check (it is a shared library, not created automatically).

## Implementation

### Target file

`scripts/db/config.py`

### Procedure

**Add fields to DbConfig:**
```python
@dataclass
class DbConfig:
    rag_db_path: str
    session_db_path: str
    sqlite_vec_so: str
    embed_url: str
    sqlite_timeout: int = 30
    sqlite_busy_timeout_ms: int = 30000
    embedding_dims: int = 384
```

**Update __post_init__ — replace file-existence checks with parent-directory checks:**
```python
def __post_init__(self) -> None:
    if not self.rag_db_path:
        raise ValueError("rag_db_path must not be empty")
    if not self.session_db_path:
        raise ValueError("session_db_path must not be empty")
    if not self.sqlite_vec_so:
        raise ValueError("sqlite_vec_so must not be empty")
    if not self.embed_url:
        raise ValueError("embed_url must not be empty")
    if self.sqlite_timeout < 1:
        raise ValueError(f"sqlite_timeout must be >= 1, got {self.sqlite_timeout}")
    # DB files are created automatically by SQLite; validate parent directory instead.
    for label, path_str in (("rag_db_path", self.rag_db_path), ("session_db_path", self.session_db_path)):
        parent = Path(path_str).parent
        if not parent.exists():
            raise ValueError(f"{label} parent directory does not exist: {parent}")
    # sqlite-vec is a shared library that must already exist.
    if not Path(self.sqlite_vec_so).exists():
        raise ValueError(f"sqlite_vec_so does not exist: {self.sqlite_vec_so}")
    if self.embedding_dims < 1:
        raise ValueError(f"embedding_dims must be >= 1, got {self.embedding_dims}")
```

**Update build_db_config():**
```python
def build_db_config() -> DbConfig:
    cfg = ConfigLoader().load_all()
    return DbConfig(
        rag_db_path=cfg.get("rag_db_path", ""),
        session_db_path=cfg.get("session_db_path", ""),
        sqlite_vec_so=cfg.get("sqlite_vec_so", ""),
        embed_url=cfg.get("embed_url", ""),
        sqlite_timeout=int(cfg.get("sqlite_timeout", 30)),
        sqlite_busy_timeout_ms=int(cfg.get("sqlite_busy_timeout_ms", 30000)),
        embedding_dims=int(cfg.get("embedding_dims", 384)),
    )
```
