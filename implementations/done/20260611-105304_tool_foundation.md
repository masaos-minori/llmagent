# Implementation: agent/tool_{enums,exceptions,models,output}.py — Foundation files

## Goal

Create 4 new foundation files that all other tool subsystem files will depend on.
These files introduce the Enum types, exception classes, DTO models, and output
port that replace raw strings, `bool` returns, `dict` usage, and direct `print()`
calls across the 8 target files.

## Scope

**Target files (all new)**:
- `scripts/agent/tool_enums.py`
- `scripts/agent/tool_exceptions.py`
- `scripts/agent/tool_models.py`
- `scripts/agent/tool_output.py`

No existing files are modified in this step.

## Assumptions

1. `from __future__ import annotations` is used throughout.
2. `ToolCallRequest.args` is typed as `dict[str, Any]` — this is the boundary type;
   internal code uses typed accessors rather than raw dict access.
3. `GuardDecision.message` is `str | None` — `None` means no exit message (PASS).
4. `tool_output.py` wraps `print()` so that test code can capture output.
5. `AuditUnavailableError` is defined but **never raised** — silent return is the
   confirmed behavior when audit logger is None.

## Implementation

### Target files

**`scripts/agent/tool_enums.py`**:
```python
"""agent/tool_enums.py
Enum types for the tool execution subsystem.
"""
from __future__ import annotations
from enum import Enum

class RiskLevel(str, Enum):
    NONE = "none"
    MEDIUM = "medium"
    HIGH = "high"

class OperationType(str, Enum):
    WRITE = "write"
    DELETE = "delete"
    EXECUTE = "execute"
    API_WRITE = "api_write"
    READ = "read"

class ApprovalDecisionType(str, Enum):
    AUTO = "auto"
    APPROVED = "approved"
    DENIED = "denied"
    DRY_RUN = "dry_run"
    PLAN_BLOCKED = "plan_blocked"

class GuardDecisionType(str, Enum):
    PASS = "pass"
    CYCLE = "cycle"
    DEDUP = "dedup"
    RETRY = "retry"
    ERROR_LIMIT = "error_limit"
```

**`scripts/agent/tool_exceptions.py`**:
```python
"""agent/tool_exceptions.py
Domain exceptions for the tool execution subsystem.
"""
from __future__ import annotations

class ToolArgumentsDecodeError(ValueError):
    """Raised when tool call arguments cannot be decoded as JSON."""

class ToolExecutorUnavailableError(RuntimeError):
    """Raised when ctx.services.tools is None."""

class PolicyViolationError(RuntimeError):
    """Raised when a pre-flight policy check denies the tool call."""
    def __init__(self, audit_decision: str, message: str) -> None:
        super().__init__(message)
        self.audit_decision = audit_decision

class ApprovalPreviewError(RuntimeError):
    """Raised when dry-run preview execution fails."""

class AuditUnavailableError(RuntimeError):
    """Raised when audit logger is required but not configured.
    NOTE: Currently never raised — silent return is the confirmed behavior.
    """

class LifecycleConfigurationError(RuntimeError):
    """Raised when a lifecycle operation is attempted with missing configuration."""
```

**`scripts/agent/tool_models.py`**:
```python
"""agent/tool_models.py
Immutable DTO models for the tool execution subsystem.
"""
from __future__ import annotations
from dataclasses import dataclass, field
from typing import Any
from agent.tool_enums import ApprovalDecisionType, GuardDecisionType, RiskLevel

@dataclass(frozen=True)
class ToolCallRequest:
    id: str
    name: str
    args: dict[str, Any]

@dataclass(frozen=True)
class ToolMeta:
    resource_scope: str
    requires_serial: bool
    is_write: bool

@dataclass(frozen=True)
class ToolExecutionResult:
    tc_id: str
    name: str
    args: dict[str, Any]
    full_text: str
    is_error: bool
    llm_text: str

@dataclass(frozen=True)
class ApprovalOutcome:
    tool_name: str
    risk_level: RiskLevel
    decision: ApprovalDecisionType
    escalation_reason: str = ""
    preview: str = ""

@dataclass(frozen=True)
class GuardDecision:
    type: GuardDecisionType
    message: str | None = None

    @property
    def blocks(self) -> bool:
        return self.type != GuardDecisionType.PASS
```

**`scripts/agent/tool_output.py`**:
```python
"""agent/tool_output.py
CLI output helpers for the tool execution subsystem.
Centralizes all print() calls so they can be intercepted in tests.
"""
from __future__ import annotations

def emit_tool_call(name: str, args_json: str) -> None:
    print(f"  [tool] {name}({args_json})")

def emit_tool_result(name: str, display: str) -> None:
    print(f"  [tool] {name} → {display}")

def emit_approval_prompt(risk: str, tool_name: str, preview: str) -> None:
    print(f"\n[{risk} risk] {tool_name}")
    print(f"  Preview: {preview}")

def emit_denied(tool_name: str, args_json: str) -> None:
    print(f"  args: {args_json}")

def emit_plan_blocked(tool_name: str, args_json: str) -> None:
    print(f"  [plan mode] Blocked: {tool_name}")
    print(f"  args: {args_json}")
```

### Method

`Write` tool for each new file. No imports from existing agent modules to avoid cycles.

### Details

- `tool_models.py` imports from `tool_enums.py` only — no agent/* imports.
- `tool_output.py` has no imports beyond stdlib — pure print wrappers.
- `dict[str, Any]` in `ToolCallRequest.args` / `ToolExecutionResult.args` is intentional;
  the boundary accepts `Any`-typed dicts from LLM responses.
