## Implementation

### Target file
`scripts/eventbus/db.py`

### Procedure
1. Read `scripts/eventbus/db.py` — find all DLQ query methods.
2. Verify each query uses indexed columns for filtering:
   ```python

   "SELECT * FROM dlq WHERE stream = ? AND status = ? ORDER BY created_at ASC LIMIT ? OFFSET ?"
   
   # Bad (full scan):
   "SELECT * FROM dlq ORDER BY id ASC"
   ```
3. For any DLQ query that does NOT use an indexed column for filtering, refactor to add the filter.
4. Run `EXPLAIN QUERY PLAN` in test to verify index is used (add assertion in test step 4).
