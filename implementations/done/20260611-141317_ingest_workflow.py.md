# Implementation: agent/services/ingest_workflow.py — IngestOutcome DTO + specific exceptions

## Goal

1. Replace `except Exception` (3 occurrences) with specific exception types.
2. Replace mutable `IngestResult` dataclass with `IngestOutcome` frozen DTO from `models.py`.
3. On stage failure, raise `IngestStageError` instead of setting `result.stage` and returning.

## Scope

**Target file**: `scripts/agent/services/ingest_workflow.py`

In scope:
- Replace all 3 `except Exception` blocks with specific types
- Replace `IngestResult` with `IngestOutcome`
- Raise `IngestStageError` on stage failure; remove in-place mutation pattern

Out of scope:
- `rag.ingestion.*` internal implementation
- `cmd_ingest.py` call site (it already handles `IngestResult`; update after this step)

## Assumptions

1. `IngestOutcome` is `@dataclass(frozen=True)` with:
   `stage: IngestStage; n_chunks: int = 0; messages: tuple[str, ...] = ()`
2. `IngestStageError(RuntimeError)` carries `stage: IngestStage` and `detail: str`.
3. `rag.ingestion.crawler.WebCrawler.crawl_site()` can raise `httpx.RequestError` and `OSError`.
   `crawl_file()` can raise `OSError`.
4. `ChunkSplitter.process_all()` can raise `OSError`, `ValueError`, `RuntimeError`.
5. `RagIngester.ingest_all()` can raise `OSError`, `RuntimeError`, `sqlite3.Error`.
6. `cmd_ingest.py` catches `IngestStageError` and renders stage-specific messages.
   Update `cmd_ingest.py` as part of this step.

### Procedure

Replace `IngestResult` inline class and the `run()` / `_crawl()` / `_split_and_ingest()` methods:

```python
import sqlite3
import httpx
from agent.services.enums import IngestStage
from agent.services.exceptions import IngestStageError
from agent.services.models import IngestOutcome

class IngestWorkflowService:
    async def run(self, target: str, lang: str = "ja", snippets_only: bool = False) -> IngestOutcome:
        messages: list[str] = []
        loop = asyncio.get_running_loop()
        messages = await self._crawl(target, lang, loop, messages)
        n_chunks, messages = await self._split_and_ingest(loop, snippets_only, messages)
        return IngestOutcome(stage=IngestStage.OK, n_chunks=n_chunks, messages=tuple(messages))

    async def _crawl(self, target, lang, loop, messages) -> list[str]:
        try:
            from rag.ingestion.crawler import WebCrawler as _WebCrawler  # noqa: PLC0415
            crawler = _WebCrawler()
            if target.startswith(("http://", "https://")):
                messages.append(f"[ingest] crawling {target} (lang={lang})...")
                await crawler.crawl_site(target, lang)
                messages.append("[ingest] crawl done")
            else:
                file_path = Path(target)
                if not file_path.exists():
                    raise IngestStageError(IngestStage.CRAWL, f"file not found: {file_path}")
                messages.append(f"[ingest] reading local file {file_path} (lang={lang})...")
                count = await loop.run_in_executor(None, crawler.crawl_file, file_path, lang)
                if count == 0:
                    raise IngestStageError(IngestStage.CRAWL, "failed to read local file")
                messages.append("[ingest] file read done")
        except IngestStageError:
            raise
        except (OSError, httpx.RequestError, ImportError) as e:
            logger.exception("Ingest crawl stage failed for %r", target)
            raise IngestStageError(IngestStage.CRAWL, str(e)) from e
        return messages

    async def _split_and_ingest(self, loop, snippets_only, messages) -> tuple[int, list[str]]:
        try:
            from rag.ingestion.chunk_splitter import ChunkSplitter as _Splitter  # noqa: PLC0415
            messages.append("[ingest] splitting chunks...")
            if snippets_only:
                from shared.config_loader import ConfigLoader  # noqa: PLC0415
                base_cfg = ConfigLoader().load("rag_pipeline.toml")
                base_cfg["md_index_enable"] = True
                splitter = _Splitter(config=base_cfg)
            else:
                splitter = _Splitter()
            n_chunks = await loop.run_in_executor(None, splitter.process_all)
            messages.append(f"[ingest] {n_chunks} chunks written")
        except IngestStageError:
            raise
        except (OSError, ValueError, RuntimeError, ImportError) as e:
            logger.exception("Ingest split stage failed")
            raise IngestStageError(IngestStage.SPLIT, str(e)) from e

        try:
            from rag.ingestion.ingester import RagIngester as _Ingester  # noqa: PLC0415
            messages.append("[ingest] ingesting to DB...")
            ingester = _Ingester()
            await loop.run_in_executor(None, ingester.ingest_all)
            messages.append("[ingest] done — RAG DB updated")
        except IngestStageError:
            raise
        except (OSError, RuntimeError, sqlite3.Error, ImportError) as e:
            logger.exception("Ingest ingest stage failed")
            raise IngestStageError(IngestStage.INGEST, str(e)) from e

        return n_chunks, messages
```

### Method

### Details

- Remove the mutable `IngestResult` dataclass entirely from the file.
- `except IngestStageError: raise` ensures that already-raised stage errors are not re-wrapped.
- `httpx` is already a dependency of the project; `import httpx` at module level is fine.
- `sqlite3` is stdlib; add to module-level imports.
