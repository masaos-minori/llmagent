# Implementation: db/maintenance.py — Remove ConfigLoader and private SQLiteHelper attrs

## Goal

1. Replace all `ConfigLoader().load("common.toml")` calls with `build_db_config()`.
2. Replace `SQLiteHelper._ensure_config()` + `SQLiteHelper._RAG_PATH` / `_SESSION_PATH`
   references with `build_db_config().rag_db_path` / `session_db_path`.

## Scope

- `scripts/db/maintenance.py`: remove `ConfigLoader` import; add `build_db_config` import;
  update `RetentionConfig.from_config()`, `_get_archive_dir()`, `rotate_rag_db()`,
  `rotate_session_db()`, `recover_corruption()`.

## Assumptions

1. `RetentionConfig.from_config()` reads `sqlite_retention_max_sessions` / `max_age_days`.
   These are NOT in `DbConfig` yet — use `ConfigLoader` fallback or add to DbConfig.
   Decision: add a `_load_retention_cfg()` helper that uses `ConfigLoader` directly for now,
   but remove the `SQLiteHelper` private attr access.
2. Archive dir is read from `common.toml` `sqlite_archive_dir`. Add to `build_db_config()` or
   keep `ConfigLoader` usage for this specific key. Decision: keep `ConfigLoader` for
   archive-dir-only call since `DbConfig` doesn't carry it yet.
3. `SQLiteHelper._ensure_config()` + `_RAG_PATH`/`_SESSION_PATH` references in maintenance.py
   are replaced by `build_db_config().rag_db_path` / `session_db_path`.
4. After this change, `maintenance.py` still imports `ConfigLoader` for `RetentionConfig`
   and `_get_archive_dir`. The goal is to remove the `SQLiteHelper` private-attr dependency
   and the duplicate `ConfigLoader` calls that mirror `build_db_config`.

## Implementation

### Target file

`scripts/db/maintenance.py`

### Procedure

**Add import and remove duplicate ConfigLoader usage:**
```python
# Add alongside existing imports:
from db.config import build_db_config
```

**Replace SQLiteHelper private attr access in rotate_rag_db, rotate_session_db, recover_corruption:**

```python
# Before:
SQLiteHelper._ensure_config()
return _archive_db_file(Path(SQLiteHelper._RAG_PATH), archive_dir)

# After:
db_cfg = build_db_config()
return _archive_db_file(Path(db_cfg.rag_db_path), archive_dir)
```

```python
# Before:
SQLiteHelper._ensure_config()
return _archive_db_file(Path(SQLiteHelper._SESSION_PATH), archive_dir)

# After:
db_cfg = build_db_config()
return _archive_db_file(Path(db_cfg.session_db_path), archive_dir)
```

```python
# Before (in recover_corruption):
SQLiteHelper._ensure_config()
db_path = Path(SQLiteHelper._RAG_PATH)

# After:
db_cfg = build_db_config()
db_path = Path(db_cfg.rag_db_path)
```

**Note:** `RetentionConfig.from_config()` and `_get_archive_dir()` keep their `ConfigLoader`
usage — these keys are not yet in `DbConfig`. The goal is only to remove
`SQLiteHelper._RAG_PATH` / `_SESSION_PATH` / `_ensure_config()` from maintenance.py.
