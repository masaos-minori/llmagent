# Plan: stat_embed_skip counter in ingestion.py

## Target
`implementations/20260620-072148_ingestion.py.md`

## Goal
Add `stat_embed_skip: int` counter to `MemoryIngestionService` that increments whenever `_persist_entry()` stores a memory entry without an embedding (embedding failed), and log the count in the `on_session_stop()` summary.

## Scope
- File: `scripts/agent/memory/ingestion.py`
- Modify: `MemoryIngestionService.__init__()`, `_persist_entry()`, `_persist_entry_with_dedup()`, `on_session_stop()`

## Assumptions
- `_persist_entry()` is the canonical write path for manual writes; embedding failure is detected via `embed_result.success is False`
- `_persist_entry_with_dedup()` also calls `_embed_client.fetch()`; its embed failure path should also be counted (both paths go through the same embed client)
- `stat_embed_skip` is instance-level (session-scoped), not persisted to DB
- Counter starts at 0 on construction; not reset between calls within a session

## Implementation Steps

### Step 1 — `__init__()`: Add counter
Add after existing instance variables:
```python
self.stat_embed_skip: int = 0
```

### Step 2 — `_persist_entry()` (manual write path): Increment on embed failure
After `embed_result = await self._embed_client.fetch(entry.content)` and before `await self._jsonl.write(entry)`:
```python
if not embed_result.success:
    self.stat_embed_skip += 1
    logger.info(
        "memory.embed_skip: stored without embedding "
        "(reason=%s) memory_id=%r",
        embed_result.error_kind,
        entry.memory_id,
    )
```

### Step 3 — `_persist_entry_with_dedup()` (session-stop write path): Increment on embed failure
After `embed_result = await self._embed_client.fetch(entry.content)` and before `if self._should_skip_dedup(...)`:
```python
if not embed_result.success:
    self.stat_embed_skip += 1
    logger.info(
        "memory.embed_skip: stored without embedding "
        "(reason=%s) memory_id=%r",
        embed_result.error_kind,
        entry.memory_id,
    )
```

### Step 4 — `on_session_stop()` summary: Log counter
Replace existing summary log line:
```python
logger.info(
    "MemoryIngestionService.on_session_stop: persisted %d entries; %d embed_skipped",
    len(entries),
    self.stat_embed_skip,
)
```
