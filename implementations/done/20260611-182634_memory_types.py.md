# Goal

Add `MemoryType` Enum to `types.py`, change `MemoryEntry.memory_type: str` to `MemoryType`,
and convert `MemoryEntry` to a frozen dataclass to enable safe use after construction.

# Scope

- `scripts/agent/memory/types.py`

# Assumptions

1. `MemoryType` is imported from `agent.memory.enums` (Step 1 prerequisite).
2. `MemoryEntry` becomes `@dataclass(frozen=True)`. Direct attribute assignments in `store.py`
   (`entry.created_at = now`, `entry.updated_at = ...`) will break and are fixed in Step 4.
3. `__post_init__` uses `object.__setattr__` for `source_type` coercion — this must use
   `object.__setattr__(self, "source_type", SourceType(self.source_type))` syntax
   which IS valid in frozen dataclasses for post_init coercion.
4. `MemoryQuery.memory_type: str | None` → `MemoryType | None`.
5. `MEMORY_TYPES: frozenset[str]` is kept as a derived constant for any external code
   that still uses it, but is now `frozenset(m.value for m in MemoryType)`.
6. All callers that pass `memory_type="semantic"` or `memory_type="episodic"` strings will
   still work because `MemoryType("semantic") == "semantic"` (StrEnum equality).

# Implementation

## Target file

`scripts/agent/memory/types.py`

## Procedure

1. Import `MemoryType` from `agent.memory.enums`.
2. Change `@dataclass` → `@dataclass(frozen=True)` on `MemoryEntry`.
3. Change `memory_type: str` field to `memory_type: MemoryType`.
4. Update `__post_init__` to coerce `memory_type` if str using `object.__setattr__`.
5. Update `MEMORY_TYPES` to be derived from enum: `frozenset(m.value for m in MemoryType)`.
6. Change `MemoryQuery.memory_type: str | None` → `MemoryType | None`.
7. Update `MemoryQuery.__post_init__` validation to use MemoryType.

## Method

Minimal edit: change dataclass decorator, field type, and coercion in `__post_init__`.

## Details

```python
from agent.memory.enums import MemoryType

MEMORY_TYPES: frozenset[str] = frozenset(m.value for m in MemoryType)

@dataclass(frozen=True)
class MemoryEntry:
    memory_id: str
    memory_type: MemoryType
    ...

    def __post_init__(self) -> None:
        # Coerce str → MemoryType
        if not isinstance(self.memory_type, MemoryType):
            try:
                object.__setattr__(self, "memory_type", MemoryType(self.memory_type))
            except ValueError:
                raise ValueError(
                    f"Invalid memory_type={self.memory_type!r}; must be one of {list(MemoryType)}"
                )
        # source_type coercion (was using object.__setattr__ before too)
        if not isinstance(self.source_type, SourceType):
            try:
                object.__setattr__(self, "source_type", SourceType(self.source_type))
            except ValueError:
                raise ValueError(...)
        ...

@dataclass
class MemoryQuery:
    query: str
    memory_type: MemoryType | None = None
    limit: int = 10

    def __post_init__(self) -> None:
        if not self.query.strip():
            raise ValueError("MemoryQuery.query must not be empty")
        if self.memory_type is not None and not isinstance(self.memory_type, MemoryType):
            try:
                object.__setattr__(self, "memory_type", MemoryType(self.memory_type))
            except ValueError:
                raise ValueError(
                    f"MemoryQuery.memory_type must be MemoryType or None; got {self.memory_type!r}"
                )
```
