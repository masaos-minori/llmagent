**Operator tracing example for `04_mcp_06` (new section "End-to-End Tool Call Tracing"):**

```markdown
## End-to-End Tool Call Tracing

To trace a failed tool call:

1. Find the `X-Request-Id` in the agent dispatch log:
   ```
   grep "tool_name=my_tool" agent.log | grep "X-Request-Id"
   ```
2. Search transport log for the same `X-Request-Id`:
   ```
   grep "X-Request-Id=<id>" transport.log
   ```
3. Search server audit log for the `X-Request-Id`:
   ```
   grep "X-Request-Id=<id>" server_audit.log
   ```
4. Check health state for `server_key` at that timestamp.
5. If health changed: check watchdog actions log for restart/failover.
```
