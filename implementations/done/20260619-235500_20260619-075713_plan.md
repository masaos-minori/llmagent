## Step 3: Update LLMTurnRunner._handle_llm_error()

**Status:** Not implemented

**Target file:** `scripts/agent/llm_turn_runner.py`

**Current behavior (line 112-120):** Delegates to `ErrorInjectionService`:
```python
async def _handle_llm_error(self, e: LLMTransportError, turn: int) -> TurnResult:
    from agent.error_injection_service import ErrorInjectionService
    error_service = ErrorInjectionService(self._ctx)
    summary = error_service.inject_mid_turn_error(e, turn)
    return TurnResult(action="fail", answer=summary, reason="llm_transport_error", exception=e)
```

**Changes needed:**

1. Replace delegation with direct diagnostic storage:
```python
async def _handle_llm_error(self, e: LLMTransportError, turn: int) -> TurnResult:
    summary = str(e) if hasattr(e, 'detail') else str(e)
    
    # Store in diagnostic channel only
    if hasattr(self._ctx, 'diagnostics') and self._ctx.diagnostics:
        self._ctx.diagnostics.save(
            kind="mid_turn_error",
            content={
                "error_type": type(e).__name__,
                "detail": summary,
                "turn": turn,
                "timestamp": datetime.utcnow().isoformat(),
            },
        )
    
    return TurnResult(action="fail", answer=summary, reason="llm_transport_error", exception=e)
```

**Completion criterion:** LLM errors reported via diagnostics; turn result carries error context.

**Status:** Partially implemented (partial completion path is diagnostic-only)

**Target file:** `scripts/agent/orchestrator.py`

**Current behavior:**
- `_handle_llm_transport_error()` line 361: Already uses `self._diagnostic_store.save(...)` for partial completions
- Line 383-384: For non-partial errors, calls `ctx.conv.history.pop()` — history-modifying operation

**Changes needed:**

1. Consolidate mid-turn errors into single diagnostic entry with error chain:
```python
def _consolidate_mid_turn_errors(self, session_id: str, errors: list[dict]) -> None:
    """Store multiple mid-turn errors as a single diagnostic entry."""
    if not errors:
        return
    self._diagnostic_store.save(
        session_id=session_id,
        kind="mid_turn_error",
        content={
            "error_chain": errors,
            "count": len(errors),
            "timestamp": datetime.utcnow().isoformat(),
        },
    )
```

2. Replace `ctx.conv.history.pop()` with diagnostic-only approach:
```python

    content={
        "action": "removed_user_message",
        "reason": "llm_transport_error_non_partial",
        "timestamp": datetime.utcnow().isoformat(),
    },
)
```

3. Pass error context as non-persistent system message for LLM recovery:
