- Append two new test methods to TestMcpServerConfigValidation.

#### Details

New test `test_http_transport_requires_url`:
```python
def test_http_transport_requires_url(self) -> None:
    with pytest.raises(ValueError, match="url"):
        McpServerConfig(transport="http", url="", cmd=[], auth_token="")
```

New test `test_ondemand_invalid_for_http`:
```python
def test_ondemand_invalid_for_http(self) -> None:
    with pytest.raises(ValueError):
        McpServerConfig(
            transport="http", url="http://localhost", cmd=[], auth_token="",
            startup_mode=StartupMode.ONDEMAND,
        )
```

Verify the error message pattern matches the current McpServerConfig validation logic before writing the match= string.
