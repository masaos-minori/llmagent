## Goal

Delete the two dead top-level keys `mqe_prompt_template` and `rerank_prompt_template` from the "System
prompts" section of `config/agent.toml`. These keys are parsed by `ConfigLoader.load_all()` but are not
read by any `AgentConfig` dataclass field or builder function — the effective, consumed copies of these
same key names live in `config/rag_pipeline_mcp_server.toml`. Removing the dead keys from
`config/agent.toml` prevents a reader from mistaking them for the effective control of MQE/rerank prompt
behavior.

## Scope

**In scope**:
- `config/agent.toml`: delete lines 169-170 (`mqe_prompt_template = "..."` and
  `rerank_prompt_template = "..."`) from the "System prompts" section.

**Out of scope** (confirmed no change needed):
- `config/rag_pipeline_mcp_server.toml` and its identically-named keys — unchanged, remain the effective
  setting.
- `scripts/mcp_servers/rag_pipeline/rag_pipeline_models.py`, `scripts/mcp_servers/rag_pipeline/rag_pipeline_service.py`,
  `scripts/rag/llm_prompts.py` — no code changes.

  set references either key name; no doc edit required.
- `docs/04_mcp_04_03_rag-pipeline-and-cicd.md`, `docs/03_rag_05_1-configuration-reference.md` — these
  already correctly document the keys as `RagPipelineConfig`/`rag_pipeline_mcp_server.toml` fields; no
