# Implementation: Add RRF config display to write_debug_rag()

## Goal

Show active `use_rrf` and `rrf_k` values in CLI debug output when RAG pipeline runs with `--debug`.

## Scope

- `scripts/agent/cli_view.py` — step 4 (write_debug_rag)
- `scripts/rag/pipeline.py` — augment() caller change (pass rrf_config to debug_fn)

## Assumptions

1. `write_debug_rag()` receives a `dict` with keys: `queries`, `all_results`, `merged`, `reranked`.
2. The caller in `pipeline.py:aughook()` passes these as keyword arguments unpacked into the dict.

## Implementation

### Step 4a: Add RRF config display to write_debug_rag() (cli_view.py line 127)

**Current code:**
```python
    def write_debug_rag(self, data: dict) -> None:
        """Render structured RAG pipeline debug data to stdout."""
        queries: list[str] = data.get("queries", [])
        all_results: list[list[dict]] = data.get("all_results", [])
        merged: list[dict] = data.get("merged", [])
        reranked: list[dict] = data.get("reranked", [])

        print(f"  [debug] MQE queries ({len(queries)}):")
```

**Replace with:**
```python
    def write_debug_rag(self, data: dict) -> None:
        """Render structured RAG pipeline debug data to stdout."""
        rrf_config: dict = data.get("rrf_config", {})
        print(
            f"  [debug] RRF config: use_rrf={rrf_config.get('use_rrf', True)} "
            f"rrf_k={rrf_config.get('rrf_k', 60)}"
        )

        queries: list[str] = data.get("queries", [])
        all_results: list[list[dict]] = data.get("all_results", [])
        merged: list[dict] = data.get("merged", [])
        reranked: list[dict] = data.get("reranked", [])

        print(f"  [debug] MQE queries ({len(queries)}):")
```

### Step 4b: Pass rrf_config to debug_fn in pipeline.py (pipeline.py line 374-375)

**Current code:**
```python
        if debug_fn is not None:
            debug_fn(queries, all_results, merged, reranked)
```

**Replace with:**
```python
        if debug_fn is not None:
            debug_fn(
                queries,
                all_results,
                merged,
                reranked,
                rrf_config={
                    "use_rrf": self._cfg.use_rrf,
                    "rrf_k": self._cfg.rrf_k,
                },
            )
```
