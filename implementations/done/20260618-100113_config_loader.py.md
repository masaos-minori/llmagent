# Implementation: shared/config_loader.py — add common.toml to _BASE_CONFIG_FILES

## Goal

Add `"common.toml"` to `_BASE_CONFIG_FILES` so `load_all()` includes DB paths,
embedding settings, and sqlite-vec path without explicit workaround calls.

## Change

- `scripts/shared/config_loader.py`: add `"common.toml"` at index 0 of `_BASE_CONFIG_FILES`
- `scripts/db/config.py`: `build_db_config()` uses `load_all()` instead of `load("common.toml")`
- `scripts/db/maintenance.py`: 3 x `load("common.toml")` → `load_all()`
- `scripts/rag/pipeline.py`: `load("common.toml", "agent.toml")` → `load_all()`
- `scripts/rag/llm_client.py`: `load("common.toml", "agent.toml")` → `load_all()`
- `scripts/rag/ingestion/ingester.py`: `load("common.toml", "rag_pipeline.toml")` → `{**load_all(), **load("rag_pipeline.toml")}`
- `scripts/agent/commands/cmd_ingest.py`: `load("common.toml", "rag.toml")` → `load_all()`
