# Goal

Refactor `cmd_config.py` to: replace `dict[str, Any]` return from `_collect_stats()` with
`StatsViewModel` DTO, narrow two `except Exception` clauses to specific types, and route
all `print()` calls through `OutputPort`.

# Scope

- `scripts/agent/commands/cmd_config.py`

# Assumptions

1. `StatsViewModel` is defined in `agent/commands/models.py` (Step 1 prerequisite).
2. `CliOutputPort` / `OutputPort` from `output_port.py` (Step 1).
3. Line 232: `except Exception` wraps `_build_db_cfg()` which can raise `ValueError` (config parse) or `RuntimeError`. Replace with `except (ValueError, RuntimeError)`.
4. Line 332: `except Exception` wraps `ConfigReloadService.apply_config_dict()` which can raise `OSError`, `ValueError`. The plan also references `ConfigReloadValidationError` — import it from `agent.services.config_reload` (it may not exist yet; if absent, use `ValueError` only and add a TODO comment).
5. `_ConfigMixin` inherits from `MixinBase` which provides `self._ctx`. The `_out` attribute from `registry.py` (Step 3) is accessible here via MRO since `CommandRegistry` has `self._out`.
6. The 44+ `print()` calls across `_print_llm_settings`, `_print_sse_settings`, etc. are all replaced with `self._out.write(...)`.

# Implementation

## Target file

`scripts/agent/commands/cmd_config.py`

## Procedure

1. Import `StatsViewModel` from `agent.commands.models`.
2. Import `OutputPort` from `agent.commands.output_port` (type annotation only).
3. Change `_collect_stats()` return type from `dict[str, Any]` to `StatsViewModel`; return a `StatsViewModel(...)` instead of a dict.
4. Update `_cmd_stats()` to use `stats.session_id`, `stats.turns`, etc. (attribute access instead of `stats["key"]`).
5. Replace all `print(...)` calls across all `_print_*` helper methods and `_cmd_*` methods with `self._out.write(...)`.
6. Narrow line 232 `except Exception` → `except (ValueError, RuntimeError)`.
7. Narrow line 332 `except Exception` → `except (OSError, ValueError)` (add TODO for ConfigReloadValidationError when available).

## Method

Attribute replacement throughout — no structural changes. Each `print(f"...")` becomes
`self._out.write(f"...")`. Multiline print blocks preserved as multiple `write()` calls.

        self._out.write(f"Restart required for: {', '.join(result.needs_restart)}")
