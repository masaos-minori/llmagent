### Details

**Add import (update existing db import line):**
```python
from eventbus.db import nack_event, open_db
```

**New nack endpoint (add after publish or near dlq handlers):**
```python
@app.post("/events/{event_id}/nack")
async def nack_event_endpoint(event_id: str) -> dict[str, Any]:
    def _nack() -> int:
        assert _db is not None
        return nack_event(_db, event_id)

    count = await asyncio.to_thread(_nack)
    if count < 0:
        raise HTTPException(status_code=404, detail="event not found")
    return {"event_id": event_id, "failure_count": count}
```

Note: inline DLQ promotion (`if count >= _cfg.max_retry`) is added in req #26. This handler just increments the count.

**Updated dlq_requeue (lines 228–237 in current app.py):**

Change:
```python
"UPDATE events SET retry_count = retry_count + 1, dlq_at = NULL WHERE event_id = ?",
```
To:
```python

```

Also update the log message:

