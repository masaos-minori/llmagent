# Implementation: H-10 — Remove tool_results from db architecture and schema doc

## Goal

Update `docs/90_shared_04_db_architecture_and_schema.md` to remove `tool_results` from the
directory tree listing, the four-DB table-list overview, and the detailed `### tool_results
table` schema section — reflecting both the DDL removal (H-9/H-10) and the full deletion of
`ToolResultStore` (H-9).

## Scope

**Target**: `docs/90_shared_04_db_architecture_and_schema.md`

**Not addressed by any prior doc in this rollout** — this file was not touched by the H-7 docs
update doc (`implementations/20260708-164730_h7_docs_updates.md`), which covered only
`docs/05_agent_02_runtime-architecture.md`, `docs/05_agent_04_state-and-persistence.md`, and
`docs/05_agent_09_data-layer.md`.

**Depends on**: `scripts/db/schema_sql.py`'s DDL removal (H-9) and `scripts/db/tool_results.py`'s
deletion (H-9) — this doc describes the FINAL end state of the whole rollout (table absent from
