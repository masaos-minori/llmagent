# Implementation: note_repo.py

## Goal

Replace all `except Exception` handlers with `sqlite3.Error` and remove `None`/`[]`/`False` fallback returns so that DB failures propagate to the caller.

## Scope

- Target: `scripts/agent/note_repo.py`
- Replace `except Exception` in `add_note()`, `list_notes()`, `delete_note()`, `get_all_note_contents()` with `except sqlite3.Error`
- Remove `return None` from `add_note()` on failure — raise instead
- Remove `return []` from `list_notes()`, `get_all_note_contents()` on failure — raise instead
- Change `add_note()` return type from `int | None` to `int`

## Assumptions

1. `AgentSession.add_note()` delegates here; it must handle `sqlite3.Error` after this change.
2. `cmd_notes.py` calls `session.add_note()` etc.; it must handle exceptions for user-facing error messages.
3. The `delete_note(note_id)` not-found case (`row is None`) remains `return False` — domain behavior, not a DB error.

## Implementation

### Target file

`scripts/agent/note_repo.py`

### Procedure

1. Add `import sqlite3` at the top.
2. In `add_note()`: remove `try/except`, change return type to `int`, `cur.lastrowid` is never `None` after successful INSERT (SQLite contract). Assert `note_id is not None` before returning.
3. In `list_notes()`: remove `try/except`; `sqlite3.Error` propagates.
4. In `delete_note()`: remove `try/except`; `sqlite3.Error` propagates. Keep `return False` for not-found.
5. In `get_all_note_contents()`: remove `try/except`; `sqlite3.Error` propagates.

### Method

Identical to `document_repo.py` pattern. Only `add_note()` changes return type.

### Details

```python
import sqlite3

class NoteRepository:
    def add_note(self, content: str) -> int:
        with SQLiteHelper("session").open(write_mode=True) as db:
            cur = db.execute("INSERT INTO notes (content) VALUES (?)", (content,))
            note_id = cur.lastrowid
            db.commit()
        assert note_id is not None
        return note_id

    def list_notes(self) -> list[dict]:
        with SQLiteHelper("session").open(row_factory=True) as db:
            rows = db.fetchall("SELECT note_id, content, created_at FROM notes ORDER BY note_id")
        return [dict(r) for r in rows]

    def delete_note(self, note_id: int) -> bool:
        with SQLiteHelper("session").open(write_mode=True) as db:
            row = db.execute("SELECT note_id FROM notes WHERE note_id = ?", (note_id,)).fetchone()
            if row is None:
                return False
            db.execute("DELETE FROM notes WHERE note_id = ?", (note_id,))
            db.commit()
        return True

    def get_all_note_contents(self) -> list[str]:
        with SQLiteHelper("session").open(row_factory=True) as db:
            rows = db.fetchall("SELECT content FROM notes ORDER BY note_id")
        return [r["content"] for r in rows]
```
