# Implementation: Update `scripts/agent/orchestrator.py` to use `WorkflowExecutionPolicy`

## Goal

Replace 4 scattered `workflow_mode` string comparisons in `orchestrator.py` with calls to
`self._policy` (a `WorkflowExecutionPolicy` instance). Remove the redundant `self._workflow_mode`
field (or keep it only if referenced in `workflow_status()` display).

## Scope

- In-Scope:
  - Import `WorkflowExecutionPolicy` in `orchestrator.py`.
  - Add `self._policy = WorkflowExecutionPolicy(workflow_mode)` in `__init__()`.
  - Replace 4 string comparisons with policy method calls (see table below).
  - Update `workflow_status()` to use `self._policy.mode` instead of `self._workflow_mode`.
  - Remove `self._workflow_mode` field if no longer referenced.
- Out-of-Scope: No changes to `__init__()` public signature. No changes to workflow database
  schema or approval logic.

## Assumptions

1. `scripts/agent/workflow_execution_policy.py` exists (prerequisite: `workflow_execution_policy_py.md`).
2. The 4 string comparisons are at lines 131, 135, 147, 191 (approximate) as identified in the plan.
3. `Orchestrator.__init__` currently stores `self._workflow_mode = workflow_mode`.
4. `workflow_status()` currently returns `{"mode": self._workflow_mode, ...}`.

## Implementation

### Target file

`scripts/agent/orchestrator.py` (existing)

### Procedure

1. Read `orchestrator.py` `__init__()` and the lines containing `self._workflow_mode` comparisons.
2. Add import: `from agent.workflow_execution_policy import WorkflowExecutionPolicy`.
3. Add in `__init__()`: `self._policy = WorkflowExecutionPolicy(workflow_mode)`.
4. Replace string comparisons per the table below.
5. Update `workflow_status()` to use `self._policy.mode`.
6. Remove `self._workflow_mode` field if no longer referenced.
7. Run `uv run ruff check scripts/agent/orchestrator.py` — expect 0 errors.
8. Run `uv run mypy scripts/agent/orchestrator.py` — expect no new errors.

### Method

**Import:**
```python

```

**In `__init__()`:**
```python
self._policy = WorkflowExecutionPolicy(workflow_mode)
```

**String comparison replacements:**

| Original | Replacement |
|---|---|
| `if self._workflow_mode != "disabled":` | `if self._policy.is_workflow_enabled():` |
| `if self._workflow_mode == "required":` (startup check) | `if self._policy.requires_startup_definition():` |
| `if self._workflow_mode == "required":` (error handling) | `if self._policy.fail_closed_on_creation_error():` |
| `if self._workflow_mode == "disabled":` | `if not self._policy.is_workflow_enabled():` |

**`workflow_status()` update:**
```python
def workflow_status(self) -> dict[str, str]:
    return {
        "mode": self._policy.mode,
        "tracking": "enabled" if self._workflow_def is not None else "not_loaded",
    }
```
