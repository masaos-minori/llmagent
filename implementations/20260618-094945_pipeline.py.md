# Implementation: scripts/rag/pipeline.py

## Goal

Modify `RagPipeline.run()` to populate `ctx.stage_results` and expose via
`self.last_stage_results` so callers can inspect per-stage outcomes.

## Scope

- `scripts/rag/pipeline.py` — add `_get_stage_status()` private method,
  modify `run()` loop, add `last_stage_results` attribute, log summary.

## Assumptions

1. `StageResult` is imported from `rag.stage` (already imports `PipelineContext` from there).
2. `last_stage_results` parallels `last_timings` — set from `ctx.stage_results` at end of `run()`.
3. Fallback detection uses `self._cfg` flags (not isinstance checks on stage internals):
   - MqeStage: fallback when `not self._cfg.use_mqe`
   - SearchStage: fallback when `not ctx.search_results` after stage runs
   - FusionStage: fallback when `not self._cfg.use_rrf`
   - RerankStage: fallback when `not self._cfg.use_rerank`
   - AugmentStage / PluginHooks: always "success"
4. `isinstance` checks on stage objects determine which cfg flag to apply.
5. Exception during a stage: not currently caught in the loop (exceptions propagate).
   Status "failure" is reserved for future use when exception handling is added.
6. Summary log at INFO level after all stages complete.

**`_get_stage_status()` private method:**

```python
def _get_stage_status(
    self, stage: object, ctx: PipelineContext
) -> tuple[str, str | None]:
    """Return (status, fallback_reason) for the given stage based on cfg and ctx state."""
    if isinstance(stage, MqeStage):
        if not self._cfg.use_mqe:
            return "fallback", "use_mqe=False"
        return "success", None
    if isinstance(stage, SearchStage):
        if not ctx.search_results:
            return "fallback", "no search results"
        return "success", None
    if isinstance(stage, FusionStage):
        if not self._cfg.use_rrf:
            return "fallback", "use_rrf=False"
        return "success", None
    if isinstance(stage, RerankStage):
        if not self._cfg.use_rerank:
            return "fallback", "use_rerank=False"
        return "success", None
    return "success", None
```

**Modified `run()` loop** (replace existing for-loop):
```python
for stage in pre_augment_stages:
    t0 = time.perf_counter()
    await stage.run(ctx, db=db)
    elapsed = time.perf_counter() - t0
    self.last_timings[stage.__class__.__name__] = elapsed
    status, reason = self._get_stage_status(stage, ctx)
    ctx.stage_results.append(
        StageResult(
            stage_name=stage.__class__.__name__,
            status=status,
            elapsed_seconds=elapsed,
            fallback_reason=reason,
        )
    )
```
