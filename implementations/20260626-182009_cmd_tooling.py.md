# Implementation: cmd_tooling.py — filter obsolete tool_results rows

Source plan: `plans/20260626-180403_plan.md` — Phase 3

---

## Goal

Update `/tool list` to hide `obsolete=1` rows by default, and display `[UNDONE]` marker on obsolete rows when `--all` flag is used. Update `/tool show` to indicate when a result belongs to an undone turn.

---

## Scope

**In-Scope**
- Modify the DB query in `/tool list` (in `db/store.py` or `db/store_impl.py`) to add `WHERE obsolete = 0` by default
- Add `--all` flag to `/tool list` to include obsolete rows with `[UNDONE]` prefix in display
- `/tool show`: if the result row has `obsolete=1`, prepend `[UNDONE]` to the output header

**Out-of-Scope**
- Redesign of `/tool` command structure
- Hard-delete of obsolete rows

---

## Assumptions

1. The DB query for `/tool list` is in `db/store_impl.py` or `db/store.py` — need to verify with grep.
2. `ToolResultRow.obsolete: bool` is added via the schema_sql.py document.
3. `parse_command_args()` handles `--all` flags via the existing arg-parsing utility.
4. Existing `/tool list` does not currently have an `--all` flag.

---

## Implementation

### Target file
`scripts/agent/commands/cmd_tooling.py` (and the DB store it calls)

### Procedure
1. Grep for the DB query function called by `/tool list`.
2. Add `obsolete` filter to the list query: `WHERE obsolete = 0` (default) or no filter (with `--all`).
3. Update `cmd_tooling._cmd_tool_list()` to parse `--all` flag and pass it to the store query.
4. Update `cmd_tooling._cmd_tool_show()` to display `[UNDONE]` when `row.obsolete` is True.

### Method

**Locate the list_recent query:**
```bash
grep -n "list_recent\|tool_results\|SELECT.*tool_results" scripts/db/store*.py
```

**Updated list query (example):**
```sql
SELECT id, session_id, turn, tool_name, args_masked, summary, is_error, obsolete, created_at
FROM tool_results
WHERE session_id = ?
  AND (? OR obsolete = 0)  -- param: include_obsolete boolean
ORDER BY id DESC
LIMIT ?
```

**`cmd_tooling.py` changes:**

```python
def _cmd_tool_list(self, args: str) -> None:
    parsed = parse_command_args(args.strip().split())
    include_all = "--all" in parsed.flags  # add flags support
    rows = self._ctx.services.tool_store.list_recent(
        session_id=self._ctx.session.session_id,
        limit=20,
        include_obsolete=include_all,
    )
    if not rows:
        self._out.write_no_data("No tool results.")
        return
    table_rows = []
    for r in rows:
        prefix = "[UNDONE] " if r.obsolete else ""
        table_rows.append([str(r.id), f"{prefix}{r.tool_name}", r.summary or "", r.created_at])
    self._out.write_table(["ID", "Tool", "Summary", "Created"], table_rows)
```
