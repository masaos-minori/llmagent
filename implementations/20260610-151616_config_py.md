# Implementation: config.py

## Goal

Remove `Any` type usage from `config.py`, change `load_config()` return type from `dict[str, Any]` to a typed `RawConfig` TypedDict, and ensure `load_config()` raises `ConfigLoadError` only for anticipated failures — re-raising unexpected errors unwrapped.

## Scope

- Target: `scripts/agent/config.py`
- Replace `dict[str, Any]` return type on `load_config()` with a `RawConfig` TypedDict
- Remove `from typing import Any` (no other usages after this change)
- Keep `load_config()` wrapping `ConfigLoadError` only for `OSError` / `toml.DecodeError` / `KeyError` — let other exceptions propagate as-is
- `build_agent_config()`: replace `Any` in builder helpers (`_build_*`) with typed parameters
- `document_repo.py` uses `list[Any]` for params — out of scope (separate doc)

## Assumptions

1. `ConfigLoader.load_all()` returns a dict with known top-level keys (`llm`, `rag`, `mcp`, `tool`, `memory`, `observability`, `approval`).
2. `_build_*` helper functions receive `dict[str, ...]` sections, not raw `Any`.
3. No callers of `load_config()` depend on `dict[str, Any]` type; they access via `build_agent_config()`.

## Implementation

### Target file

`scripts/agent/config.py`

### Procedure

1. Define `RawConfig` as a `TypedDict` with the known top-level keys, each typed as `dict[str, object]`.
2. Change `load_config() -> RawConfig`.
3. In `load_config()`, replace the broad `except Exception` with:
   - `except (OSError, KeyError, ValueError) as e: raise ConfigLoadError(...) from e`
   - Let all other exceptions propagate without wrapping.
4. In `_build_*` helpers, replace `cfg.get(key, default)` patterns with typed accessors (e.g. `int(cfg.get("llm_max_retries", 3))`).
5. Remove `from typing import Any` once no remaining usages exist.

### Method

`TypedDict` with `total=False` for optional subsections. Builder helpers use `cast()` only where the value type cannot be inferred at static analysis time.

### Details

```python
from typing import TypedDict

class RawConfig(TypedDict, total=False):
    llm: dict[str, object]
    rag: dict[str, object]
    mcp: dict[str, object]
    tool: dict[str, object]
    memory: dict[str, object]
    observability: dict[str, object]
    approval: dict[str, object]


def load_config() -> RawConfig:
    try:
        raw = ConfigLoader().load_all()
    except (OSError, KeyError, ValueError) as e:
        raise ConfigLoadError(f"Config load failed: {e}") from e
    return raw  # type: ignore[return-value]  # ConfigLoader returns dict; shape matches RawConfig
```

The `type: ignore[return-value]` requires an inline justification comment.
