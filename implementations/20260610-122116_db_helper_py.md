# Implementation: db/helper.py — DI, fail-fast, conditional vec load, remove class vars

## Goal

Refactor `SQLiteHelper` to:
1. Remove the `_ensure_config()` lazy class-level loader; resolve config in `__init__` (fail-fast).
2. Delete class-level cache variables (`_RAG_PATH`, `_SESSION_PATH`, `SQLITE_VEC_SO`, etc.).
3. Store resolved config as instance variables set in `__init__`.
4. Add `load_vec` parameter to `open()` so vec extension is not loaded for session DB operations.
5. Replace `assert conn is not None` with explicit `RuntimeError` raises.

External API (`SQLiteHelper("rag")`, `SQLiteHelper("session")`, `.open()`, `.execute()`, etc.)
is unchanged.

## Scope

- `scripts/db/helper.py` — primary target
- `tests/test_sqlite_helper.py` — update tests that relied on class var behaviour or lazy loading

## Assumptions

1. The string targets `"rag"` and `"session"` remain the primary construction API.
   No callers pass a raw `db_path` in this step (that is future work).
2. `build_db_config()` may raise; the error propagates as `RuntimeError` wrapping the original.
3. `open()` for `"rag"` target: default `load_vec=True`.
   `open()` for `"session"` target: default `load_vec=False`.
   Callers that explicitly need vec on session can pass `load_vec=True`.
4. `assert conn is not None` in `health_check`, `checkpoint`, `vacuum`, `begin_immediate`,
   `begin_exclusive` are replaced with `if self.conn is None: raise RuntimeError(...)`.
5. `assert self.conn is not None` in `execute` and `executemany` (guarded by `_check_ready`)
   is already protected — remove the redundant assert.

## Implementation

### Target file

`scripts/db/helper.py`

### Procedure

1. Delete all class-level variables: `_RAG_PATH`, `_SESSION_PATH`, `SQLITE_VEC_SO`,
   `_SQLITE_TIMEOUT`, `_BUSY_TIMEOUT_MS`, `_config_loaded`.
2. Delete `_ensure_config()` classmethod.
3. In `__init__`, after target validation, call `build_db_config()` and store:
   - `self._db_path: str`
   - `self._vec_so: str`
   - `self._sqlite_timeout: int`
   - `self._busy_timeout_ms: int`
   Wrap in `try/except Exception as e: raise RuntimeError(f"DbConfig load failed: {e}") from e`.
4. Update `DB_PATH` property to return `self._db_path`.
5. Update `_connect()` to use `self._db_path` and `self._sqlite_timeout` (remove `_ensure_config` call).
6. Update `_load_vec_extension()` to use `self._vec_so`.
7. Update `_apply_connection_pragmas()` to use `self._busy_timeout_ms`.
8. Add `load_vec: bool` parameter to `open()`:
   - Default: `True` for `"rag"`, `False` for `"session"` (determined in `__init__`; store as `self._default_load_vec`).
   - If `load_vec=True` and `self._vec_so` is empty, skip (do not raise).
9. Replace every `assert self.conn is not None` with:
   `if self.conn is None: raise RuntimeError("DB not open — call open() first")`.
10. In `execute` / `executemany`: remove the redundant `assert self.conn is not None`
    (already guarded by `_check_ready`). Use `self.conn` directly after the guard.

### Method

Direct textual edit (file is small enough; no LibCST needed).

### Details

`__init__` structure:
```python
def __init__(self, target: str = "rag") -> None:
    if target not in ("rag", "session"):
        raise ValueError(f"target must be 'rag' or 'session', got: {target!r}")
    self._target = target
    self._default_load_vec = target == "rag"
    self.conn: sqlite3.Connection | None = None
    try:
        db_cfg = build_db_config()
    except Exception as e:
        raise RuntimeError(f"DbConfig load failed for target={target!r}: {e}") from e
    self._db_path = db_cfg.rag_db_path if target == "rag" else db_cfg.session_db_path
    self._vec_so = db_cfg.sqlite_vec_so
    self._sqlite_timeout = db_cfg.sqlite_timeout
    self._busy_timeout_ms = db_cfg.sqlite_busy_timeout_ms
```

`open()` signature:
```python
def open(
    self,
    *,
    write_mode: bool = False,
    row_factory: bool = False,
    load_vec: bool | None = None,   # None = use instance default
) -> "SQLiteHelper":
    use_vec = self._default_load_vec if load_vec is None else load_vec
    conn = self._connect()
    if use_vec and self._vec_so:
        self._load_vec_extension(conn)
    ...
```
