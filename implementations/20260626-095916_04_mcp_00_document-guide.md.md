   ```
   ## Migration Notes

   ### POST /v1/search (removed)

   The `POST /v1/search` endpoint on the `rag-pipeline-mcp` server has been removed.

   **Before**:
   ```http
   POST /v1/search
   {"query": "...", "top_k": 5}
   ```

   **After** (canonical MCP tool call):
   ```http
   POST /v1/call_tool
   {"name": "rag_run_pipeline", "args": {"query": "...", "top_k": 5}}
   ```

   Update any `rag_service_url` callers to use the MCP tool call format.
   This change is not backward-compatible — no compatibility shim is provided.
   ```

### Method
Documentation-only change.
