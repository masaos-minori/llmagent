# Goal

Narrow all four `except Exception` clauses in `helper.py` to specific exception
types, change `health_check()` to return `DbHealthMetrics`, change `checkpoint()`
to return `WalCheckpointCounts`, and update the two callers that use dict access
on those return values.

# Scope

- `scripts/db/helper.py`
- `scripts/agent/services/db_maintenance_service.py` (health_check caller)
- `scripts/db/maintenance.py` (checkpoint caller)

# Assumptions

1. `build_db_config()` raises only `ValueError` (confirmed by inspecting config.py).
   The `except Exception` in `__init__` narrows to `except ValueError`.
2. `close()` failure is always `OSError` (file descriptor close failure).
3. `begin_immediate()` and `begin_exclusive()` ROLLBACK can fail with
   `sqlite3.OperationalError` when the connection is already closed or read-only.
4. `DbHealthMetrics` and `WalCheckpointCounts` are in `db/models.py` (Step 1 prerequisite).
5. `db_maintenance_service.py:50-55` accesses `raw.get("integrity")` and
   `raw.get("db_size_bytes", 0)` — change to `raw.integrity` and `raw.db_size_bytes`.
6. `maintenance.py:74-78` accesses `raw["busy"]`, `raw["pages_in_wal"]`,
   `raw["pages_checkpointed"]` — change to `raw.busy`, `raw.log_size`,
   `raw.pages_checkpointed`.

# Implementation

## Target file

`scripts/db/helper.py`, `scripts/agent/services/db_maintenance_service.py`,
`scripts/db/maintenance.py`

## Procedure

### A. `helper.py` — except Exception changes

1. `__init__` line 35: `except Exception as e:` → `except ValueError as e:`

2. `close()` line 126: `except Exception as e:` → `except OSError as e:`

3. `begin_immediate()` inner except (line 143):
   ```python
   except Exception:
       pass
   ```
   → `except sqlite3.OperationalError: pass`

4. `begin_exclusive()` inner except (line 159): same change as above.

### B. `helper.py` — health_check() return type

```python
# Before
def health_check(self) -> dict[str, Any]:
    ...
    return {
        "journal_mode": journal_mode,
        "integrity": integrity,
        "page_count": page_count,
        "page_size": page_size,
        "freelist_count": freelist,
        "db_size_bytes": page_count * page_size,
    }

# After
def health_check(self) -> DbHealthMetrics:
    ...
    from db.models import DbHealthMetrics  # noqa: PLC0415 — local import to avoid circular at module level
    return DbHealthMetrics(
        journal_mode=str(journal_mode),
        integrity=str(integrity),
        page_count=int(page_count),
        page_size=int(page_size),
        freelist_count=int(freelist),
        db_size_bytes=int(page_count) * int(page_size),
    )
```

Add `from db.models import DbHealthMetrics` to the top-level imports (it's safe
since `db.models` only imports from stdlib).

### C. `helper.py` — checkpoint() return type

```python
# Before
def checkpoint(self, mode: str = "TRUNCATE") -> dict[str, int]:
    ...
    result = {"busy": row[0], "pages_in_wal": row[1], "pages_checkpointed": row[2]}
    logger.info(f"WAL checkpoint ({mode}): {result}")
    return result

# After
def checkpoint(self, mode: str = "TRUNCATE") -> WalCheckpointCounts:
    ...
    result = WalCheckpointCounts(
        busy=int(row[0]),
        log_size=int(row[1]),
        pages_checkpointed=int(row[2]),
    )
    logger.info(f"WAL checkpoint ({mode}): {result}")
    return result
```

Add `from db.models import WalCheckpointCounts` to imports.

### D. `db_maintenance_service.py` — health_check caller update

```python
# Before (lines 50-55)
raw = db.health_check()
return DbHealth(
    integrity_ok=raw.get("integrity") == "ok",
    wal_pages=0,
    size_bytes=raw.get("db_size_bytes", 0),
)

# After
raw = db.health_check()
return DbHealth(
    integrity_ok=raw.integrity == "ok",
    wal_pages=0,
    size_bytes=raw.db_size_bytes,
)
```

### E. `db/maintenance.py` — checkpoint caller update

```python
# Before (lines 74-78)
raw = db.checkpoint(mode)
return WalCheckpointCounts(
    busy=raw["busy"],
    log_size=raw["pages_in_wal"],
    pages_checkpointed=raw["pages_checkpointed"],

# After
raw = db.checkpoint(mode)
return WalCheckpointCounts(
    busy=raw.busy,
    log_size=raw.log_size,
    pages_checkpointed=raw.pages_checkpointed,
)
```

## Method

4× except narrowing + 2× return type changes + 2× caller attribute access updates.
